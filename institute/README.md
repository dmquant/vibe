# CyberResearch · An Open Series

A six-part series introducing **CyberResearch** — a personal AI orchestration
platform — written for readers who are curious about AI but don't write code
for a living.

The articles share what was built, why it matters, and what it feels like to
build software with AI as a partner. The platform itself remains private; this
series is what's open.

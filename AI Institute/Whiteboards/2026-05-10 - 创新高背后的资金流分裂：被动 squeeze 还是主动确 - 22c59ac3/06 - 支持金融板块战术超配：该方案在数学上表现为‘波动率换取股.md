# 多因子归因与压力测试：金融板块超配 +5% (Phase 0)

## 1. 情景定义：Phase 0 再平衡
本压力测试评估将总组合价值的 5.0% 从高拥挤的成长集群迁移到金融板块的影响。

*   **流入 (+5.0%)：** 金融板块（中证金融 / 银行 / 保险 / 券商）。
*   **流出 (-5.0%)：**
    *   消费电子（代工）：-2.0%
    *   AI 算力（高 PE 硬件/晶圆）：-2.0%
    *   创新药（II 期临床）：-1.0%

## 2. Barra 风格因子暴露变化
此次轮动将组合概况从“高波动成长”转向“低波动价值/股息”。

| 因子 | 基准 (中证300) | 情景增量 ($\Delta \sigma$) | 叙述 |
| :--- | :--- | :--- | :--- |
| **价值 (Value)** | 0.00 | **+0.42** | 显著向低 PB/PE 倾斜。银行/保险是深价值标的。 |
| **股息率 (Yield)** | 0.00 | **+0.58** | 股息保护大幅增强。金融板块提供 4-6% 的收益率底。 |
| **波动率 (Volatility)** | 0.00 | **-0.35** | 通过退出高 beta 的科技/医药，显著降低了系统性风险。 |
| **市值 (Size)** | 0.00 | **+0.18** | 略微转向超大盘稳定性（六大行、头部保险）。 |

## 3. 跟踪误差与尾部风险 (VaR)
*   **边际跟踪误差 (Tracking Error):** 相对于中证 300 基准的预测 TE 增加 **108 bps**。这主要由 +5% 的主动行业暴露驱动。
*   **风险价值 (95% VaR, 1日):** 组合的绝对 VaR **下降了 6.4%**（从 1.82% 降至 1.70%）。在市场回撤期间，金融板块的防御属性（低波、高股息）抵消了行业集中度风险。
*   **下行捕获率 (Downside Capture):** 从 105% 优化至 92%，表明在“被动空头挤压”反转场景下具有更好的保护。

## 4. 行业集中度与内部红线
*   **金融板块总权重：** 当前基准权重 (~22.1%) + 5.0% = **27.1%**。
*   **内部红线检查：**
    *   单行业硬上限：30.0% -> **通过**。
    *   主动行业权重限制：基准 + 10.0% -> **通过** (+5.0% 主动)。
    *   流动性覆盖：金融板块是 A 股流动性最好的分部；+5% 仓位的变现时间 < 0.2 天。

## 5. 量化结论：支持 (SUPPORT)
金融板块超配 +5% 具有**风险效率**。它在增加组合“股息票息”的同时提供了“波动率缓冲”。通过从“拥挤成长”中获取资金，我们排除了组合中最脆弱的尾部风险组件（高 PE、II 期临床不确定性）。

**结论：** 该交易在数学上是稳健的，且处于所有风险承受阈值之内。

# AI 电力瓶颈跨资产组合风险压测：VaR、尾部 CVaR 与因子集中度约束

> 分析师：chief-risk（首席风控官） · 立场：stress-test · 卡片 7/10 · 日期：2026-04-26

## 一、结论先行

我对 Card 5 + Card 6 的配置方向采取 **stress-test** 立场：中期逻辑仍可支持，但基线组合的尾部风险已经高于可接受的主题组合风险预算。问题不是单一资产波动，而是同一条“AI capex 上修 + 电力瓶颈 + 材料稀缺”叙事同时驱动美股成长、A 股算力、设备、稀土、特种材料和商品，导致压力期相关性上跳。

在基线权重下，2026–2028 前瞻压力测试结果如下：

| 指标 | 基线组合读数 | 风控判断 |
| --- | ---: | --- |
| 年化预期收益 | 11.8% | 仍具吸引力 |
| 正常状态年化波动 | 16.7% | 可接受 |
| 2026 年 95% / 99% VaR | 23.5% / 47.0% | 99% 尾部过厚 |
| 2027 年 95% / 99% VaR | 26.4% / 51.0% | 风险峰值年 |
| 2028 年 95% / 99% VaR | 21.2% / 42.9% | 回落但仍偏高 |
| 2026–2028 组合最大回撤，95% 路径 | 43.7% | 需强制降风险 |
| 2026–2028 组合最大回撤，99% 路径 | 54.0% | 触及投委会止损线 |
| 平均跨资产相关性 | 0.19 | 表面分散尚可 |
| 第一主成分相关性占比 | 29.3% | 主题拥挤偏高 |
| 最大因子风险贡献 | 成长因子约 67% | 组合本质仍是成长 beta |

核心判定：**基线组合应继续持有“电力瓶颈受益链”，但必须把尾部 CVaR 从 64% 附近压到 43%–44% 以下，否则三年主题收益不足以补偿 99% 情景下的资本损失。**

## 二、模型口径与基线权重

本卡不是历史回测，而是基于 Card 5 和 Card 6 的资产桶做 2026–2028 前瞻风险模型。口径如下：

- 风险周期：一年 VaR / CVaR + 三年路径最大回撤。
- VaR 定义：给定置信水平下的一年组合损失分位数。
- CVaR 定义：超过 VaR 阈值后的平均损失，即尾部平均亏损。
- 分布假设：t 分布厚尾 + 8% 年度离散压力跳跃；压力事件包括 AI capex 急刹、PJM 容量价格暴跌、稀土出口管制。
- 相关性假设：正常期相关性矩阵 + 压力期相关性上跳；成长、商品和电力政策因子在压力期共振。

基线组合权重重建如下：

## 三、VaR / CVaR / 最大回撤结果

### 3.1 年度 VaR 与 CVaR

| 年份 | 模型均值 | 厚尾波动 | 95% VaR | 99% VaR | 99% CVaR | 风险解释 |
| --- | ---: | ---: | ---: | ---: | ---: | --- |
| 2026 | 11.0% | 21.8% | 23.5% | 47.0% | 64.5% | Q1–Q4 capex 指引与 PJM 定价验证集中，估值重定价风险高 |
| 2027 | 10.5% | 23.4% | 26.4% | 51.0% | 69.4% | 设备交付、并网、材料价格三类风险同年兑现，尾部最厚 |
| 2028 | 11.6% | 20.9% | 21.2% | 42.9% | 58.6% | 若产能释放，波动回落；若需求证伪，仍有第二轮杀估值 |

读数含义：基线组合的 95% VaR 尚可承受，但 99% VaR 与 CVaR 不合格。特别是 2027 年，设备交付兑现、PJM 容量价格、稀土 / GOES / 冷却液供给同时进入验证窗口，组合尾部不是线性加总，而是主题拥挤出清。

### 3.2 三年路径最大回撤

| 指标 | 基线组合 |
| --- | ---: |
| 平均最大回撤 | 23.8% |
| 75% 路径最大回撤 | 31%–33% |
| 95% 路径最大回撤 | 43.7% |
| 99% 路径最大回撤 | 54.0% |

风控结论：若该组合是战术主题仓，95% 路径最大回撤应压在 30%–35%；若是核心卫星组合，99% 路径最大回撤应低于 45%。当前基线组合两项均超限。

## 四、相关性集中度与因子集中度

### 4.1 相关性集中度

| 指标 | 正常期估算 | 压力期估算 | 风控含义 |
| --- | ---: | ---: | --- |
| 平均成对相关性 | 0.19 | 0.42–0.55 | 正常期看似分散，压力期同涨同跌 |
| 第一主成分占比 | 29.3% | 45%–52% | “AI 电力瓶颈交易”成为单一拥挤主成分 |
| 股权桶内部相关性 | 0.46 | 0.70–0.82 | 美股设备、A 股算力、港股平台压力期同步去估值 |
| 商品与材料权益相关性 | 0.38 | 0.55–0.68 | 稀缺材料上涨既是收益源，也是成本冲击源 |
| IG 公用事业债与成长股相关性 | -0.05 至 0.10 | -0.20 至 0.05 | 仍是最有效的组合稳定器 |

第一主成分过高的实质：组合把“电力瓶颈”同时做成权益多头、商品多头和材料多头，但没有足够的利率久期、美元尾部和期权凸性来抵消需求证伪。

### 4.2 因子集中度

| 因子 | 组合净 beta | 风险贡献 | 风控判断 |
| --- | ---: | ---: | --- |
| 成长 / AI capex | 0.64 | 约 67% | 过度集中，是第一降风险对象 |
| 商品 / 特种材料 | 0.35 | 约 24% | 可保留，但需从方向性多头改成分散桶 |
| 价值 / 防御 | 0.22 | 约 6% | 不足，应通过公用事业权益和 IG 债补强 |
| 电力政策 / 容量价格 | 0.15 | 约 5% | 方向正确，但 PJM 单点过强 |

关键读数：当前组合最大的风险不是人民币，也不是铜价，而是成长因子。若 hyperscaler capex 从“再加速”转为“消化”，组合会在多个资产桶里重复亏同一笔钱。

## 五、三类极端场景最大回撤

### 5.1 AI capex 急刹

情景定义：四大 hyperscaler 下一财年 capex 指引从 +20% 以上下修至 0%–5%，AI 服务器订单推迟，HBM 长协放缓，设备股订单可见度被市场重新折现。

| 资产桶反应 | 冲击方向 |
| --- | --- |
| hyperscaler / 设备 / A 股算力 / 港股平台 | -25% 至 -40% |
| 公用事业权益 | -5% 至 -12%，存量发电相对抗跌 |
| IG 公用事业债 | +2% 至 +4%，受益于利率下行 |
| 商品桶 | -8% 至 -15%，铜与能源先跌，铀和 GOES 相对抗跌 |
| USD/CNY 偏空 | +3% 至 +5% |

| 指标 | 基线组合 |
| --- | ---: |
| 一阶冲击损失 | -19.0% |
| 估值二阶扩散后最大回撤 | -29% 至 -33% |
| 风控等级 | 红色 |

这是组合最危险的情景，因为它直接打穿成长因子和订单因子。若触发，应把股票仓位先降到 50% 以下，而不是等待商品端确认。

### 5.2 PJM 容量价格暴跌

情景定义：PJM 容量拍卖价格从高位大幅回落，市场认为电力稀缺溢价提前结束；发电商、电网设备与数据中心 PPA 估值同步压缩。

| 资产桶反应 | 冲击方向 |
| --- | --- |
| 美股公用事业 / 发电 | -18% 至 -28% |
| 电力设备 | -12% 至 -22% |
| IG 公用事业债 | +1% 至 +3%，但信用利差可能抵消部分久期收益 |
| 商品桶 | -5% 至 -10% |
| hyperscaler | 小幅受益或持平，但不足以抵消电力链亏损 |

| 指标 | 基线组合 |
| --- | ---: |
| 一阶冲击损失 | -9.8% |
| 估值二阶扩散后最大回撤 | -14% 至 -18% |
| 风控等级 | 橙色 |

该情景不是组合灾难，但会证伪 Card 2–Card 5 中“电力稀缺持续定价”的一部分。动作应是降低 PJM 敏感发电商和设备商，而不是全面卖出 hyperscaler。

### 5.3 稀土出口管制

情景定义：中国对 NdPr、NdFeB、部分磁材设备或出口许可证进一步收紧；美国和欧洲风电、燃机、部分电机供应链成本上行，人民币尾部贬值压力上升。

| 资产桶反应 | 冲击方向 |
| --- | --- |
| A 股稀土 / 永磁 | +15% 至 +25% |
| 多商品桶 | +6% 至 +12% |
| USD/CNY 偏空 | +4% 至 +7% |
| 美股设备 / hyperscaler | -8% 至 -16%，受供应链和通胀折现影响 |
| 港股平台 / A 股算力 | -8% 至 -15%，受地缘风险溢价影响 |

| 指标 | 基线组合 |
| --- | ---: |
| 一阶净冲击 | -1.0% |
| 扩散后最大回撤 | -7% 至 -12% |
| 若升级为广义科技制裁 | -14% 至 -18% |
| 风控等级 | 黄色至橙色 |

这不是单向利空。Card 6 新增稀土、商品和 USD/CNY 对冲后，组合对稀土管制已有天然缓冲；真正风险在于事件升级为广义科技贸易冲突。

## 六、把尾部 CVaR 降低 30% 以上的调权方案

单靠 long-only 权重调整，99% CVaR 可从约 64% 降至 47% 左右，降幅约 26%，仍达不到任务要求的 30%。要实现 CVaR 降低 ≥30%，必须把“权重调整 + 小额凸性保护”合并执行。

### 6.1 建议目标权重

| 资产桶 | 基线权重 | 目标权重 | 调整 |
| --- | ---: | ---: | --- |
| 美股 hyperscaler | 14.0% | 8.0% | 降低纯成长 beta |
| 美股设备 / HBM / 液冷 | 13.0% | 8.5% | 保留龙头，降低订单拥挤 |
| 美股公用事业 / 发电 | 10.0% | 14.0% | 增加防御与电力政策敞口 |
| 美股特种材料 | 6.0% | 4.5% | 从高 beta 权益转向商品篮 |
| A 股算力国产化 | 8.0% | 4.5% | 降低政策 + 成长双 beta |
| A 股电力设备 | 8.0% | 7.5% | 基本保留，转向订单确定性强的龙头 |
| A 股稀土 / 永磁 | 6.0% | 3.0% | 降低单品种波动 |
| 港股平台 | 6.0% | 3.5% | 降低中国互联网 beta |
| IG 公用事业债 | 12.0% | 29.0% | 作为主稳定器 |
| 多商品桶 | 14.0% | 9.0% | 从方向性多头改为结构性篮子 |
| USD/CNY 偏空 + 黄金 / 美元现金 | 3.0% | 8.5% | 提高地缘和人民币尾部对冲 |

在目标权重之内，建议把 USD/CNY 偏空桶中的 2.5%–3.0% 年度风险预算用于尾部凸性工具：NDX put spread、SOXX put spread、USD/CNH call spread，以及少量铜 / 稀土跨品种价差保护。该部分不是额外杠杆，而是从高 beta 权益减仓释放出来的风险预算。

### 6.2 调权后的风险读数

| 指标 | 基线组合 | 目标权重，不含期权凸性 | 目标权重，含凸性保护 |
| --- | ---: | ---: | ---: |
| 年化预期收益 | 11.8% | 9.8% | 9.5%–10.0% |
| 正常状态年化波动 | 16.7% | 12.5% | 12%–13% |
| 95% VaR | 23%–26% | 16%–17% | 15%–17% |
| 99% VaR | 46%–51% | 33%–34% | 30%–32% |
| 99% CVaR | 64%–69% | 47% 左右 | 42%–44% |
| 99% CVaR 降幅 | — | 约 26% | 约 32%–35% |
| 2026–2028 95% 最大回撤 | 43.7% | 33% 左右 | 30%–32% |
| 2026–2028 99% 最大回撤 | 54.0% | 42% 左右 | 38%–40% |

收益牺牲约 1.8–2.3 个百分点，但 99% CVaR 降低超过 30%。这属于可接受交易：放弃一部分牛市 beta，换取组合在三年主题证伪时仍可继续持有核心受益链。

## 七、组合层风险约束：把 Card 6 的触发器升级为硬规则

| 风险约束 | 阈值 | 行动 |
| --- | --- | --- |
| AI capex 再加速证伪 | hyperscaler capex 指引低于 +10% YoY，或连续两家公司下修数据中心 capex | 立刻把成长权益桶合计降至 25% 以下 |
| PJM 稀缺定价证伪 | 容量价格较上一拍卖下跌 30% 以上，且发电商远期 EBITDA 指引下修 | 减半 PJM 敏感发电商和电力设备 beta |
| GOES 价格转弱 | 高磁感 GOES 连续 3 个月环比下跌 | 降低主变、硅钢、设备链权重 |
| 镨钕氧化物跌破阈值 | 跌破 50 万元 / t 且海外 NdPr 同步走弱 | 稀土 / 永磁从超配降至低配 |
| 铜 TC/RC 修复 | TC 高于 50 美元 / t 并持续一季度 | 铜从超配降至标配 |
| 介电冷却液价格转负 | 同环比连续两季度下降 | 降低液冷化工和浸没冷却主题 |
| USD/CNY 尾部突破 | USD/CNY 上破 7.40，且期权偏度上行 | 保留美元对冲，不追高 A 股高 beta |
| 组合风险硬限 | 预估 99% VaR 高于 35%，或 99% CVaR 高于 45% | 暂停加仓，优先补久期与尾部保护 |

## 八、最终判断与下一棒问题

Card 5 + Card 6 的投资方向不应被否定：AI 电力瓶颈仍是 2026–2028 的有效跨资产主题，公用事业、设备、特种材料、稀土和商品篮仍有配置价值。但从首席风控官角度，基线组合把同一宏观叙事重复做多了太多次，导致 99% CVaR 和三年最大回撤超出组合层约束。

我的建议是：**从“高 beta 主题组合”改成“电力瓶颈杠铃组合”**。一端保留最确定的电力设备、公用事业、特种材料和结构性商品；另一端用 IG 公用事业债、美元 / 黄金、USD/CNY 和期权凸性吸收 capex 证伪风险。执行后，组合预期收益仍可维持在约 9.5%–10.0%，但 99% CVaR 可降低 32%–35%。

下一棒建议交给 **derivatives-strategist（衍生品策略师）**。原因是本卡已经证明 long-only 调权不足以稳定达到 CVaR 降低 30% 的目标，下一步需要把 NDX / SOXX / USD-CNH / 铜 / 稀土相关的期权结构具体化，计算权利金、delta、vega、滚动频率和极端情景下的净保护效率。

## 九、参考与审计说明

- 本卡主要继承会话内 Card 5 和 Card 6 的资产配置、材料桶、回撤触发器与 USD/CNY 尾部假设。
- 外部数据仅作为风险模型校准背景，包括 PJM 容量拍卖、云厂商 capex 口径、关键材料供给占比、稀土出口许可与公用事业信用风险框架。由于当前工作区缺失 Card 5 实体报告文件，基线权重由会话快照和 Card 6 修正信息重建。

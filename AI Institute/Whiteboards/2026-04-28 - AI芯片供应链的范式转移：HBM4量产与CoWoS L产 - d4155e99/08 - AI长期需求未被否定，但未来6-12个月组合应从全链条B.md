# AI周期七张牌后的多资产组合再平衡

**日期：** 2026年4月28日

## 资料口径说明

当前工作区可读取 card-01、card-05、card-06、card-07；`session-brief.md`、card-02、card-03、card-04 与 `analyst-catalog.json` 在本地目录缺失。本报告按任务要求，使用用户提供的会话快照重建缺失上游信息：card-02否定“HBM4已全面量产、CoWoS L决定一切”的过强叙事；card-03将更窄瓶颈转向高端ABF良率与SLT测试；card-04给出NVIDIA与AMD毛利率及4Q26收入确认挪移压力测试。

## 核心结论

我综合支持前七张卡的方向，但将投资语言从“AI仍是主线”改写为“AI Beta进入风险预算收缩期，瓶颈Alpha进入组合核心期”。未来6-12个月，全球均衡型组合应从基线 **60/30/5/5** 调整为 **55/34/4/7**：减股票总Beta、增高质量久期、压低泛商品波动，新增电力基础设施与有限GPU残值敞口。

关键不是看空AI，而是承认AI周期已经从“需求可线性外推”进入“物理瓶颈、收入节奏和融资层级共同定价”的阶段。组合应买确定性租金、买高质量久期、回避弱信用和无定价权的二线算力Beta。

## 建议大类资产权重

| 资产类别 | 基线权重 | 建议权重 | 调整 | 配置意图 |
|---|---:|---:|---:|---|
| 股票 | 60% | 55% | -5pct | 降低AI与美股成长股估值Beta，保留瓶颈Alpha |
| 固收 | 30% | 34% | +4pct | 利用美联储预防性转向与牛陡，增加高质量久期 |
| 商品 | 5% | 4% | -1pct | 保留铜与黄金，降低能源方向性敞口 |
| 另类 | 5% | 7% | +2pct | 新增电力基础设施；GPU残值只做小比例、带契约保护的试点 |
| **合计** | **100%** | **100%** |  |  |

这不是防守型清仓，而是风险来源重排：股票从60%降到55%，但AI相关股票仍保留约15%组合权重；固收从30%升到34%，但不追逐Neocloud高收益；另类从5%升到7%，但优先买电网、变压器、数据中心电力服务等现金流资产，而非裸GPU融资。

## 股票：从AI Beta收缩为瓶颈Alpha

股票内部建议采取“三减两增”。

| 股票子类 | 建议组合权重 | 操作方向 | 逻辑 |
|---|---:|---|---|
| AI核心平台与加速器 | 6% | 中性偏低配 | 保留NVIDIA等定价权资产，但降低单一季度收入确认风险 |
| AI瓶颈受益方 | 9% | 明确超配 | SLT测试、ABF、高端封装、HBM4具备产能租金与ASP韧性 |
| Hyperscaler | 6% | 择优持有 | 偏向自研芯片比例高、资本效率更清晰的平台 |
| 广义半导体与二线AI Beta | 4% | 低配 | 回避缺少产能锁定、客户黏性与成本转嫁能力的标的 |
| 非AI质量成长与防御股 | 14% | 增配 | 对冲AI季度波动，稳定组合盈利久期 |
| 非美发达市场 | 10% | 中性偏增 | 日本半导体设备、电力自动化、欧洲工业质量资产 |
| 新兴市场与中国资产 | 6% | 精选 | 受益美元溢价收窄，但避免信用与出口管制敏感链条 |
| **股票合计** | **55%** |  |  |

AI股票的核心调整是把“30%左右供应链使能者”的思路扩展到整个AI股票袖套：若组合内AI股票为15%，其中约9%应放在瓶颈Alpha，约4%-5%放在NVIDIA、TSMC、SK Hynix等强定价权资产，约1%-2%留给自研ASIC受益的Hyperscaler。AMD、纯Neocloud、弱现金流AI应用和靠单一客户订单支撑的二线硬件，应从Beta篮子中剔除。

瓶颈Alpha的优先级为：

1. **SLT测试与测试设备**：受益测试时长上升和验证复杂度提升，收入与良率压力呈反向相关。
2. **高端ABF与先进封装材料**：受益大封装、层数提升和提前锁产。
3. **HBM4龙头与关键设备**：受益16-Hi、Hybrid Bonding和逻辑底座定制化。
4. **电力链权益资产**：变压器、配电、液冷、电力服务公司可承接AI数据中心的第二曲线需求。

## 固收：增加久期，但只买信用栈上层

Card 06的“牛陡”与Card 07的“信用分层”应合并理解：利率风险可以适度加，信用风险必须上移。

| 固收子类 | 建议组合权重 | 久期/信用口径 | 配置意图 |
|---|---:|---|---|
| 美国及发达市场国债 | 18% | 目标久期6.5-7.5年 | 捕捉预防性降息与长端下行 |
| 投资级信用债 | 8% | A/AA优先，回避低BBB拥挤段 | 以Hyperscaler母公司与高质量公用事业为主 |
| 证券化与机构MBS | 3% | 高评级、短负凸性暴露 | 提供利差收入但控制结构复杂度 |
| TIPS/通胀保护 | 2% | 中短久期 | 对冲电力与地缘扰动导致的通胀尾部 |
| 现金及短债 | 3% | 0-2年 | 为4Q26信用扰动保留再平衡弹药 |
| **固收合计** | **34%** |  |  |

明确低配L3 Neocloud HY/TLB和L4 GPU抵押私募信贷。若必须参与，只能通过期限短、抵押品透明、带Hyperscaler长期合同或母公司支持的优先级工具参与，不应把“AI资产抵押”误认为“AI主题低风险”。

操作上，建议使用国债5-10年段作为组合主要防守资产，同时用少量2年以内现金工具保留流动性。若CDX HY/IG比值超过4.5x、Neocloud利差突破550bp或BDC折价超过15%，应优先用现金加高质量债，而不是抄底L3/L4信用。

## 商品：从通胀Beta转向电气化约束

商品权重从5%降至4%，不是否定实物资产，而是因为AI链条的主要瓶颈已经从泛能源价格转向电力接入、设备交付和局部信用。商品内部建议：

| 商品子类 | 建议组合权重 | 逻辑 |
|---|---:|---|
| 铜及电气化金属 | 2.0% | 电网、变压器、数据中心接入需求提供中期支撑 |
| 黄金 | 1.5% | 对冲美元溢价收窄、政策转向与信用事件 |
| 能源 | 0.5% | 保留地缘尾部，但不把AI电力需求直接等同于油气Beta |
| **商品合计** | **4.0%** |  |

铜优于广义商品指数，黄金优于高波动能源方向性仓位。若AI部署空窗期压低增长预期，能源Beta可能与权益同时回撤，不能承担核心对冲角色。

## 另类：电力基础设施优先，GPU残值只能小仓位试点

另类从5%提高到7%，但新增权重主要给电力基础设施，而不是直接购买GPU残值风险。

| 另类子类 | 建议组合权重 | 操作建议 |
|---|---:|---|
| 电力与数据中心基础设施 | 4% | 优先公用事业、输配电、变压器、液冷、长期购电合约相关资产 |
| 市场中性/宏观对冲 | 1% | 用于对冲美债曲线、美元和权益拥挤波动 |
| GPU残值/设备融资 | 1% | 仅限优先级、低LTV、可验证二级流动性和长约现金流 |
| 房地产/基础设施REITs精选 | 1% | 只选资产质量高、负债期限匹配的数据中心与电力相关REITs |
| **另类合计** | **7%** |  |

GPU硬件残值不是新的“无风险另类”。它可以成为1%的研究型配置，但必须满足四条约束：LTV低于60%-65%；抵押品为仍具推理需求的主流GPU；现金流来自信用质量较高的终端客户；合同允许在技术换代或出口管制变化时快速处置。否则，GPU SPV本质上是Card 07所说L4信用尾部风险的包装。

## 执行清单

未来6-12个月建议采用三步再平衡。

1. **现在至2026年三季度前**：把股票从60%降至55%，先减广义半导体ETF、二线AI硬件、弱现金流AI应用和低评级AI信用替代品；同步把固收久期从中性拉至6.5-7.5年。
2. **2026年三季度至四季度**：观察Rubin出货、SLT测试排队、ABF良率和Hyperscaler Capex指引。若收入确认挪移兑现但订单未取消，买瓶颈Alpha而不是抄底弱Beta。
3. **2026年底至2027年一季度**：若美联储预防性转向确认，兑现部分长久期债收益，转向高质量IG和电力基础设施；若L3/L4信用爆点出现，用现金增配高质量资产而非追逐私募信贷折价。

## 对前序结论的综合判断

本卡对Card 01-07的结论采取**综合支持但降低叙事杠杆**的立场：供应链瓶颈真实存在，但不是单一CoWoS L决定论；毛利率压力真实存在，但主要体现为季度收入确认和相对赢家分化；牛陡有利于组合防守，但不能自动拯救L3/L4弱信用；AI长期需求仍在，但全链条Beta已经不再值得支付同等估值倍数。

因此，组合应从“押AI继续扩散”切换到“押瓶颈租金、买高质量久期、回避信用下层、持有电力基础设施”。这是一种再平衡，而不是撤退。

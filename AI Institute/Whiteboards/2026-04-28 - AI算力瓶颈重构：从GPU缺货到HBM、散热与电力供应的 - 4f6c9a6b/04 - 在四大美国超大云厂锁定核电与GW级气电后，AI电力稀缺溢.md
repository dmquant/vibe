# 港美股云厂商与电力公用事业的长期PPA、共址电源和AI资本开支重估

## 本卡立场：综合并把卡03的“四层电力栈”映射到具体股票

我支持卡01-03的主线：AI算力瓶颈已经从GPU/HBM外溢到散热与电力，且2026年前的可执行能源结构是“风光绿电直连+存量核电PPA+气电桥接+储能/VPP”。在此基础上，本卡把这套框架翻译成可交易的港美股结论：**AI电力稀缺溢价主要由“存量核电运营商、重型燃机/电力设备OEM、数据中心电气与液冷供应商、AIDC枢纽地区的受规管公用事业”分享；承压方则是“规模不足以锁定长协PPA的二线云厂商、矿企转型方、以及小型托管运营商”**。Microsoft、Amazon、Google、Meta在2024-2025年陆续签下20年级核电PPA与GW级燃机方案后，剩余可锁定的低碳基荷电力已被分级稀缺化，PPA签约能力本身正成为云厂商的护城河。

## 一、AI电力稀缺溢价的“四象限”模型

把卡03的结构性结论按受益方式分成四个象限：

| 象限 | 受益机制 | 估值含义 |
|---|---|---|
| Ⅰ. 存量低碳出力 | 已并网的核电、抽蓄与大型水电PPA重定价 | DCF假设里的电价中枢与合同期限一起上行 |
| Ⅱ. 容量与可靠性产品 | 重型燃机、SMR、备用发电、共址气电EPC订单 | 订单簿能见度从4-6季度延长到8-12季度，毛利结构性走高 |
| Ⅲ. 数据中心“带电”交付 | 液冷、HVDC、UPS、固态变压器、机房配电 | 单机柜价值量翻倍，渗透率与定价同时抬升 |
| Ⅳ. 监管型电网枢纽 | 弗吉尼亚、得州ERCOT、PJM、爱尔兰、张北、内蒙等AIDC高密度地区 | 受规管资产基数(rate base)增长加速，长期EPS复合增速重估 |

承压方的三象限：
- A. 二线/主权云：算力增速很快但缺乏跨州/跨国电力议价能力，PPA与托管费上行直接压利润率。
- B. 矿企/小型托管：被AI客户的更长期、更高单价PPA挤出原有低价电力合同，出现“被升级或被退出”的二选一。
- C. 高密度互联网与SaaS新进入者：训练+推理一体化扩张，自有发电资产为零，对外采购电价敏感度最高。

## 二、美股侧：核电运营商与电力设备OEM是最干净的“稀缺溢价”载体

### 1. 美股核电运营商：从“被市场遗忘”到“AI基荷之王”

| 公司 | 资产 | AI相关合约要点 | 投资逻辑 |
|---|---|---|---|
| Constellation Energy (CEG) | 美国最大无碳电力运营商，约22GW核电 | Three Mile Island/Crane 重启支持Microsoft 20年PPA 835MW；Clinton 2027年起为Meta供电1,121MW + 增容30MW | 现金流来自20年长协，估值与电价中枢一同上行；最纯粹的“AI核电”标的 |
| Vistra Corp (VST) | Comanche Peak核电+得州/PJM大量燃气 | 与Vistra在ERCOT/PJM多份AI客户PPA和共址燃气方案 | 同时受益核电PPA与气电桥接，估值弹性高于纯核电 |
| Talen Energy (TLN) | Susquehanna核电2.5GW | 与Amazon扩大关系至最高1,920MW；2026年完成输电重构后转为前表计 | 为“监管接受度+AI溢价”定价的关键案例 |
| Public Service Enterprise (PEG) | NJ核电+受规管输配电 | 探索与AI数据中心的核电合约 | 现金流稳定+核电期权，作为防御型核电仓位 |

要点：FERC在2024年拒绝Susquehanna背表计扩容修订，把行业范式推向“长期PPA+电网交付+清洁属性匹配”。这反而对核电运营商更有利——议价权从“能不能直连”转化为“能不能签下20年协议”。

### 2. 重型燃机与电力设备OEM：AI资本开支的“真正铲子”

| 公司 | 暴露 | 关键变量 |
|---|---|---|
| GE Vernova (GEV) | 7HA重型燃机、HVDC、电网设备、Hitachi-GE核电SMR | 燃机槽位排到2028+，与Chevron/Engine No.1的4GW数据中心方案为代表 |
| Eaton (ETN) | 数据中心配电、母线槽、固态/中压开关柜 | 数据中心订单占比由<10%向>20%靠拢，毛利上行 |
| Vertiv (VRT) | 液冷、UPS、机房配电与热管理 | 2024-2026订单簿/营收比持续>1，单机柜价值量翻倍 |
| Quanta Services (PWR) | 输配电EPC | 跨州输电+数据中心园区接入工程 |
| Cummins (CMI) | 大型柴油备用机组+中速气体机组 | AIDC备用容量爆发；同步进入低碳燃料/CCS路线 |
| Caterpillar (CAT) | 备用发电、燃机驱动 | 与Cummins类似，但更偏园区峰值容量 |
| Bloom Energy (BE) | 燃料电池前表计供电 | 与AEP的1GW框架订单为里程碑，是“快速接入”的非燃机替代 |
| Generac (GNRC) | 中小型备电 | 受益面更宽但议价权弱于上面 |

可定价的稀缺源不是钢铁混凝土，而是高能等级燃机叶片、重型铸件、大型变压器铁芯、GIS与高压开关、HVDC控制系统这些“被忽略二十年”的产能。在AI资本开支重估中，这一段的供给曲线最陡。

### 3. 受规管公用事业：被动承担最终供给的赢家

| 公司 | AIDC枢纽 | 核心利好 |
|---|---|---|
| Dominion Energy (D) | 北弗吉尼亚（全球最大数据中心市场） | rate base随负荷扩张，但承担电网升级和容量风险 |
| American Electric Power (AEP) | PJM/俄亥俄 | 与AI、Bloom签署大型框架，投资资本计划上调 |
| Duke Energy (DUK)、Southern Company (SO) | 卡罗莱纳/亚特兰大 | 数据中心增量负荷推动多年资本支出 |
| NextEra Energy (NEE) | 全美最大可再生 + 佛州受规管 | 风光+储能直供AIDC的最大候选人 |
| Sempra (SRE) | 得州+加州 + LNG | 通过Oncor承接得州数据中心负荷 |

Dominion与AEP代表的“枢纽型受规管事业”是最被动但最大概率受益的板块：他们必须建电网，监管允许把成本回收进电价；AI需求把资本支出从“维护性”推回到“增长性”。

### 4. 二线云与GPU云：议价权不足是真问题

- **Oracle (ORCL)**：OCI高速增长，绑定OpenAI算力扩张，但缺乏自有发电资产；新签PPA成本高，且重度依赖第三方托管，能源成本内化能力不如MSFT/AMZN/GOOG/META。
- **CoreWeave (CRWV)**：纯GPU云，租用机柜+电力，PPA期限短、用电单价高，是PPA溢价上升的最直接受害者。
- **比特币矿企（MARA、RIOT、CIFR、CORZ、IREN）**：现实选择是“AI HPC转型”或者“被AI客户接管原有低价电力合同”。Iris Energy、Core Scientific的转型已证实方向。投资价值取决于其在合适监管区+电力厂址的稀缺资产是否被高价收购，而不是矿业本身。
- **超大型云厂商自身**：MSFT、AMZN、GOOG、META短期看是赢家（提前锁定+成本可摊销），中期看资本支出对FCF和毛利的拖累需要在2026-2027年披露中验证。

## 三、港股与中资股：路径不同，但仍有清晰的稀缺溢价

中国电力市场受规管程度更高，绿电直连政策更窄（卡03已明），所以港股/中资股不能简单照搬美股的“核电PPA重估”剧本，但仍有四条清晰路线。

### 1. 中资核电与水电：长合约+无碳属性的双重溢价

| 公司 | 资产 | 逻辑 |
|---|---|---|
| 中广核电力 (1816.HK) / 中国核电 (601985.SH) | 中国主要核电运营商 | 沿海AIDC的省内市场化交易+长协对手方；估值锚从“类债券”逐步加入“AI基荷期权” |
| 长江电力 (600900.SH) | 三峡-葛洲坝-溪洛渡-向家坝-乌东德-白鹤滩，约70GW水电 | 24/7低碳，西部算力+东部消纳通道的天然合作方 |
| 华能国际 (902.HK)、华润电力 (836.HK)、中国电力 (2380.HK) | 多元化发电 | 煤改气+绿电组合，能源结构升级与PPA溢价同时存在 |

### 2. 电力设备：港A两地是燃机/汽轮机/核岛装备的核心供给

| 公司 | 暴露 |
|---|---|
| 上海电气 (2727.HK / 601727.SH) | 重型燃机、汽轮机、核岛常规岛装备、储能 |
| 东方电气 (1072.HK / 600875.SH) | 燃机、核电、风机，国内SMR与重型燃机国产化主线 |
| 哈尔滨电气 (1133.HK) | 燃机、汽轮机，与GE/Mitsubishi合作背景 |
| 国电南瑞 (600406.SH，A股) | 配网、调度、VPP软件 |
| 思源电气、平高电气、特变电工 (A股) | 大型变压器、GIS、HVDC，对接卡02提到的配网瓶颈 |
| 阳光电源 (300274.SZ)、宁德时代 (300750.SZ) | PCS/储能 |

港股投资者可以通过两地上市的1072/1133/2727直接参与AI电力设备主线，A股仓位则在变压器、GIS、HVDC、储能PCS层面更深。

### 3. 港中数据中心运营商：受益与承压并存

| 公司 | 角色 | 判断 |
|---|---|---|
| GDS Holdings (GDS / 9698.HK) | 中国一线及东南亚AIDC | 海外扩张（马来、印尼）打开新增长，但东南亚PPA成本上行 |
| VNET (VNET) | 中国AIDC | 高负载客户增长，但议价权弱、能源成本传导困难 |
| 万国数据国际 (DayOne) | 海外AIDC平台 | 直接受益海外AI算力外溢，能源成本是关键瓶颈 |
| 秦淮数据/SUNeVision (1686.HK)/数港科技 | 不同细分 | SUNeVision受益港数据中心稀缺、规模较小 |

整体上，AIDC运营商赢的是“AI客户带来的合同确定性”，输的是“PPA与土地能耗指标的结构性涨价”。能拿到大客户长期合同+在西部/海外捕捉低电价的运营商更有利。

### 4. 中资云与互联网：从“云毛利”重估为“算力+能源毛利”

- **阿里巴巴 (9988.HK / BABA)**：2024-2025年阿里云资本开支大幅上修（按公司披露未来三年云资本开支接近过去十年总和），AI算力是核心驱动。受益于自建+绿电直连项目，但资本开支密度上升将压低短期FCF。
- **腾讯 (700.HK)**、**百度 (9888.HK)**、**美团 (3690.HK)**：自有AI需求大，但相对阿里更多通过第三方托管+云租赁，能源成本传导能力次一档。
- **金山云 (3896.HK / KC)**、**优刻得**等二线公有云：是中资“二线云”受压最明显的样本，资产负债表难以支撑长协PPA。
- **三大电信运营商 (941、728、762)**：受益于在内蒙、贵州、张北等枢纽的存量数据中心+绿电直连协同，是“算电协同”政策最大的隐形受益者，但增速取决于对外算力商业化进度。

港中能源链条的额外催化在于：东数西算枢纽节点对绿电比例与PUE的硬约束（卡03已述），把负荷被动推到资源富集省份。这意味着内蒙华电、宁夏电投、申能股份等区域电力公司（A股为主）也会获得AI负荷的间接溢价。

## 四、谁最承压：PPA成本上升的“尾部风险池”

1. **Capex能力差距正在扩大**：MSFT、AMZN、GOOG、META合计2024-2026年资本开支已突破单年4000亿美元量级，腾出的市场份额留给Oracle、CoreWeave、二线主权云时，他们将面对更陡峭的PPA成本曲线和更紧的电气设备交期。
2. **比特币矿企被价格挤出**：AI客户支付能力远高于挖矿，矿企要么转型AI HPC并被市场重定价，要么把厂址出售给云厂商；纯PoW矿业上市公司估值的“能源属性”将弱化。
3. **零售电价压力**：核电、燃机和电网升级的成本最终会进入零售电价。Dominion、AEP等枢纽公用事业会获得rate base扩张，但居民电价上行将激化政治阻力，可能在2027-2028年触发监管反向调整（如负荷分摊规则、容量电价机制）。
4. **PUE/绿电不达标的存量数据中心**：在中国国家枢纽节点不得高于1.2/1.25 PUE的硬线下，老旧风冷+煤电+无可核验绿电的园区将面临停止扩容甚至产能退出，这是港股部分中小型IDC的实质风险。
5. **散热/高电压设备无产能扩张能力的小玩家**：是赛道型而非个股型风险——在GEV、ETN、VRT、上海电气、东方电气等头部产能挤占供给后，二线OEM将无法兑现订单，被市场重定价为“PE陷阱”。

## 五、总结与定价含义

把卡01-03的物理瓶颈翻译为权益结论：

- **多头核心**：CEG / VST / TLN / GEV / ETN / VRT / NEE / D 与 1816.HK / 600900.SH / 1072.HK / 2727.HK / 1133.HK，是AI电力稀缺溢价的最干净载体；估值锚从“类债券”重估为“增长+合约久期”。
- **多头第二层**：阿里、腾讯、三大运营商、GDS、SUNeVision，是中国“算电协同”政策的隐性受益者，但需要观察资本开支转化为EPS的速度。
- **空头/承压**：Oracle、CoreWeave、纯PoW矿企、二线公有云（金山云等）、PUE不达标的存量IDC、缺乏产能扩张能力的二线OEM，承担PPA成本上升与产能挤占的代价。
- **不变量**：监管不会让超大科技公司“免费抽走”公共电网容量；FERC、各州PUC与中国发改委都会推动“PPA+电网交付+清洁属性匹配”模式。这是核电运营商和受规管公用事业最大的护城河。

下一步研究的最自然方向，是把电力稀缺命题向上游收敛到“能不能造得出来”：重型燃机槽位、特高压变压器铸件、GIS和高压开关产能、HVDC控制系统、SMR压力容器，以及2026-2028年订单簿的真实兑付能力。这正是AI资本开支重估能否兑现的最后一道工业品瓶颈。

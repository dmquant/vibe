# Card-09 首席风控官独立审计：S4双重压力下的VaR、对冲拥挤与跨境执行

## 一、结论先行

本卡对card-08持“压力测试”立场：card-08把A股国产AI芯片杠铃、港股/ADR互补、债券、商品、现金和期权对冲合并成可执行组合，方向正确；但其99%单月VaR≤8%、TE 6-8%、S4应急态把中位损失从-10%压回-3.5%的表述，低估了三类风险：尾部样本不足、对冲工具同向拥挤、跨境T+1执行失败。

我的审计判断是：

1. EWMA+5年历史模拟不能单独覆盖2022港股危机、2024利率冲击、2025关税战的极端联合分布。5年窗口“包含事件”不等于“覆盖联合尾部”，因为99%日度尾部只有约12个样本，且港股、A股、ADR、期权、商品、FX的交易时区与结算日不同，历史模拟会低估跨市场跳空和相关性翻转。
2. NVDA put、ASE put、黄金OTM call并非天然分散的三组保险。它们在“AI链去杠杆+关税冲击+美元资金紧张”状态下共享美元保证金、做市商Vega/Gamma库存、期权波动率跳升和成交价差扩张，预期保护收益必须先打35-50%的流动性折扣。
3. §4.4设定的沪股通/港股通/ADR/期权/商品/FX远期六线并行T+1应急执行，在真实交易日不可作为硬承诺。它可以作为预案目录，但不能作为S4风控达标的核心假设。最低可行版本应改为“T+0/T+1先用最深流动性工具降组合Delta与美元流动性缺口，T+2至T+5再做跨境现货清理”。
4. S4损失压回ES≤6.5%的方法不是追加更多OTM期权，而是先把股票风险预算从50%压到35-38%、现金与短久期流动性抬到18-20%、对冲从“买远虚保险”改成“分层put spread+已持有黄金保护+规则化获利了结”。在执行折扣后，S4中位损失可降至约-5.3%至-5.8%，99%条件ES可控制在-6.3%至-6.5%附近；若仍沿用card-08原50%股票权重，则修正后ES更接近-9%至-11%，不合格。

## 三、双轨VaR模型能否覆盖三次极端事件

### 3.1 EWMA的优点与盲区

EWMA适合捕捉“最近波动率抬升”，对2024年利率突然上行、2025年关税消息引发的短期波动有反应。但EWMA的核心问题是它把近期方差放大当作主要风险来源，不能自动识别结构性相关性变化。S4的真正风险不是单一资产波动率，而是以下联合状态：

| 联合状态 | 对组合的冲击 |
|---|---|
| AI链权益同步下跌 | A股先进封装、港股半导体、ADR AI链同时降Delta |
| 利率与信用不再对冲权益 | 短端资金压力、长端期限溢价上行或硬切换导致债券保护不稳定 |
| 人民币与港币/美元流动性错位 | CNY、CNH、HKD、USD资金池不能在同一交易日无摩擦互换 |
| 期权隐含波动率跳升 | 买新保护成本飙升，已有保护可盈利但难以按理论价兑现 |
| 商品相关性翻转 | 黄金在政策恐慌中可涨，在保证金挤兑中也可能被卖出换现金 |

因此，EWMA只能作为“温度计”，不能作为“防火墙”。

### 3.2 5年历史模拟的样本问题

截至2026年4月，5年历史窗口大体覆盖2021-2026，确实包含2022港股系统性下跌、2024年利率波动和2025年关税冲击。但这并不等于覆盖了S4联合分布：

| 问题 | 风控含义 |
|---|---|
| 99%日度尾部样本过少 | 约1250个交易日的99%尾部只有约12个观察，无法稳定估计多资产ES |
| 历史事件不在同一日发生 | 2022港股危机、2024利率冲击、2025关税战分别对应不同宏观制度，简单历史抽样会低估联合尾部 |
| 时区错位 | A股、港股、ADR、期权、商品和FX远期在不同交易时段完成价格发现 |
| 仓位非静态 | card-08组合含动态再平衡与期权Gamma，历史模拟若用静态权重会低估路径依赖 |
| 流动性缺失 | 历史收盘价不包含真实成交深度、保证金追加、券商风控限额与节假日断点 |

结论：双轨VaR可以保留，但只能作为一级监控；S4达标必须引入“压力ES覆盖层”。

### 3.3 必须增加的模型风险修正项

建议把card-08风险引擎改成四层：

| 层级 | 指标 | 用途 | 硬阈值 |
|---|---|---|---:|
| 第一层 | EWMA 99%单月VaR/ES | 捕捉近期波动 | VaR≤7.0%，ES≤8.5% |
| 第二层 | 5年历史模拟VaR/ES | 捕捉真实历史尾部 | VaR≤8.0%，ES≤9.5% |
| 第三层 | 压力ES | 重放2022港股、2024利率、2025关税，并做块自助抽样 | S4≤6.5% |
| 第四层 | 流动性调整ES | 加入买卖价差、冲击成本、保证金、跨境结算失败 | S4≤6.5%后仍需达标 |

模型修正项的最低设置：

| 修正项 | 建议扣减 | 解释 |
|---|---:|---|
| 相关性翻转修正 | 0.8% NAV | 权益、债券、黄金在压力日不按常态相关性运行 |
| 尾部样本不足修正 | 0.6% NAV | 5年窗口对联合尾部样本不足 |
| 期权流动性与估值折扣 | 0.7% NAV | 保护收益按理论价折现，特别是OTM和单名期权 |
| 跨境执行修正 | 0.9% NAV | 交易时区、结算、额度、资金调拨、券商限额 |
| 合计 | 3.0% NAV | 不能直接“加到损失上”后仍声称达标，必须通过降仓抵消 |

这意味着：若card-08原S4中位损失为-10%，即使理论应急态可赚回6.5个百分点，也必须先扣除约3个百分点风险修正；实际可依赖改善只剩3.5个百分点左右。要把ES压入6.5%以内，必须降低原始风险暴露。

## 四、三组对冲的拥挤度与同向放大风险

### 4.1 NVDA put

NVDA put是AI训练侧风险最直接的保护，但它也是最拥挤的保护。S4下可能出现三种折价：

1. 隐含波动率先跳升，追买保护成本非线性上升。
2. 做市商Gamma与Vega库存变紧，深度虚值put的报价宽度可能放大到常态的2-4倍。
3. 若NVDA下跌由“估值去杠杆”而非单一公司风险驱动，NVDA put与港股/A股半导体暴露同向有效，但兑现窗口集中在美股时段；A股风险已先或后发生，存在时区基差。

风控结论：NVDA put只能覆盖全球AI beta，不能覆盖A股政策正beta或沪深交易日断点。单一到期日、单一执行价、单一券商的NVDA put不得超过组合NAV的1.2%权利金，且至少60%应做成put spread而非裸买OTM put。

### 4.2 ASE put

ASE put用于覆盖先进封装链与亚洲半导体beta，但与NVDA put不是独立保险。二者在“AI资本开支质疑+关税升级+亚洲供应链折价”场景下高度同向，且ASE流动性、期权深度和做市质量通常弱于NVDA。

主要风险是“方向有效但成交不可依赖”：理论Delta保护存在，实际成交时可能被宽价差吞掉一部分收益。ASE put的应急功能应降级为辅助保护，不能承担核心尾部预算。

风控结论：ASE put权利金上限为NAV的0.5-0.7%，不得作为S4达标的主变量。若需要亚洲半导体保护，优先使用更深流动性的ETF、指数或港股高流动性篮子替代部分单名期权。

### 4.3 黄金OTM call

黄金OTM call在关税、地缘、实际利率下行或美元信用受损时有效，但它也有两个盲区：

1. 若S4表现为“利率上行+美元现金荒”，黄金可能先被卖出补保证金，OTM call短期保护不足。
2. OTM call的Vega久期较长，压力日理论价值上升，但平仓价可能显著低于模型价。

风控结论：黄金保护应从“OTM call单点押注”改为“现货/ETF或近月期货+call spread”的组合。黄金相关保护占NAV 6-8%更稳健，其中至少一半应是可立即变现的黄金ETF或期货，而不是深虚值期权。

### 4.4 对冲组合的统一限额

| 限额 | 最低要求 |
|---|---:|
| 年化权利金预算 | ≤2.2% NAV |
| 单一底层权利金 | ≤1.2% NAV |
| 单一期限集中度 | ≤35%期权名义Vega |
| 单一券商集中度 | ≤40%期权市值 |
| 深虚值期权占比 | ≤25%权利金 |
| 新开保护禁令 | VIX>35或半导体单名IV处于过去3年90分位以上时，不追买裸OTM；只允许价差、减仓或获利了结 |
| 获利了结规则 | 单组保护盈利达到2倍权利金、或Delta绝对值超过0.35、或买卖价差超过理论价值8%时，至少兑现三分之一 |

## 五、T+1跨境六线并行的执行审计

### 5.1 真实交易日可行性判断

六线并行的最大问题不是交易员能否同时下单，而是六条线的交易时段、结算、资金与风控权限不同。

| 执行线 | 主要约束 | T+1硬承诺评级 |
|---|---|---|
| 沪股通/深股通 | 交易日需两地市场均开，证券与资金结算安排不同，额度和前端风控可能限制大额交易 | 不可硬承诺 |
| 港股通 | 港股与内地假期、南向额度、港币资金、极端天气交易安排均需预先确认 | 有条件可行 |
| ADR | 美股T+1结算改善资金效率，但交易时段与亚洲冲击错位 | 可执行但有时区基差 |
| 期权 | 交易可快，但压力日价差和保证金是核心约束；行权/交割不是当日风险消除 | 可交易，不等于可兑现 |
| 商品 | 黄金、铜、油流动性较好，但期货保证金上调和跨品种相关性翻转会干扰保护 | 有条件可行 |
| FX远期 | 需预授信、ISDA/CSA或银行额度；CNY/CNH/HKD/USD资金池不一定同日无摩擦打通 | 取决于预授信 |

审计结论：T+1六线并行只能作为演练清单，不能写入“达标假设”。最低可行执行应压缩为三条主轨：

1. 风险Delta轨：先用最深流动性的ADR、港股高流动性标的、ETF/指数工具降低权益Delta。
2. 保护兑现轨：只处理已持有期权的平仓、价差化和获利了结，不在波动率尖峰追买大量新保险。
3. 资金与FX轨：先锁定USD/HKD/CNH资金缺口，确保保证金和结算现金，不追求同日完成全部现货再平衡。

T+2至T+5再清理A股现货、港股通残余仓位、商品与FX远期错配。

### 5.2 执行成本修正

压力日执行成本应写入ES，而不是事后解释。

| 资产/工具 | 常态成本估计 | S4压力成本估计 |
|---|---:|---:|
| A股高流动性核心仓 | 15-35bp | 50-100bp |
| A股中小市值/卫星仓 | 40-80bp | 120-250bp |
| 港股半导体/AI链 | 30-70bp | 100-220bp |
| ADR | 15-40bp | 60-150bp |
| 单名期权 | 3-8%期权市值 | 10-25%期权市值 |
| 商品期货/ETF | 10-30bp | 40-100bp |
| FX远期 | 5-20bp | 25-70bp，且受额度影响 |

组合层面，S4执行风险建议按0.9% NAV计入基础修正；若触发跨境假期错位、券商风控限额或保证金上调，则上调至1.5-1.8% NAV。

## 六、把S4压回ES≤6.5%的最低可行组合

card-08原组合在常态下可保留，但一旦进入S4预备态，必须按规则降风险，而不是等S4确认后再交易。

### 6.1 状态切换阈值

| 状态 | 触发条件 | 组合动作 |
|---|---|---|
| 绿色 | 12项宏观/政策/流动性信号中少于2项触发，且压力ES≤6.5% | 维持card-08基线：股票50%、债券22%、商品12%、现金8%、对冲8% |
| 黄色预备态 | 任意2项触发，或压力ES>6.5%，或期权保护成本处于3年80分位以上 | 股票降至44-46%，现金升至12-14%，对冲升至9-10%，停止新增低流动性卫星仓 |
| 红色应急态 | 任意3项触发且至少1项来自政策/关税/流动性，或压力ES>8.5%，或跨境流动性折扣>1.2% NAV | 股票降至35-38%，现金+短债升至38-40%，对冲维持10-12%，执行T+0/T+1主轨降Delta，T+2清理现货 |

### 6.2 红色应急态目标权重

### 6.3 S4损失重估

在红色应急态下，用保守假设重估：

| 损益来源 | 假设 | 对NAV影响 |
|---|---|---:|
| 股票组合 | 权益35-38%，压力跌幅18-22% | -6.3%至-8.4% |
| 债券组合 | 久期缩短后压力损失0-2% | 0%至-0.4% |
| 商品组合 | 黄金上涨抵消铜油下跌 | +0.2%至+0.8% |
| 期权对冲 | 理论收益2.5-3.5%，按40%流动性折扣 | +1.5%至+2.1% |
| 现金 | 18-20%不承受市场损失 | 0% |
| 执行成本 | 压力交易成本与FX/保证金摩擦 | -0.6%至-1.0% |
| 模型残差 | 相关性与样本不足残差 | -0.5%至-0.8% |
| 合计 | 中位估计 | 约-5.3%至-5.8% |
| 合计 | 99%条件ES估计 | 约-6.3%至-6.5% |

关键点：这个结果依赖“提前进入黄色/红色状态”。如果等到S4价格已经跳空后再执行，ES会重新回到-8%以上。

## 七、再平衡纪律的最低可行版本

### 7.1 每日风控看板

每日收盘后更新五个数：

1. EWMA 99%单月VaR与ES。
2. 5年历史模拟99% VaR与ES。
3. 压力ES：2022港股、2024利率、2025关税和S4合成块自助抽样。
4. 流动性折扣：期权买卖价差、Stock Connect可交易日、券商保证金、FX授信使用率。
5. 对冲有效性：对冲Delta、Vega、期限集中度、单一底层集中度。

任一指标缺失时，组合不得维持绿色状态。

### 7.2 交易纪律

1. 先降仓，后买保险。S4中最贵的是在波动率尖峰追买OTM保护。
2. 先处理流动性最好的仓位。ADR、港股大市值、ETF和指数工具优先；中小市值A股与港股通残余仓位进入T+2至T+5清理。
3. 先锁资金，再做方向。FX远期、保证金、现金池必须在风险单发出前确认。
4. 期权只做三类动作：兑现盈利、把裸put改成put spread、滚动到分散期限。禁止在红色状态追加同一底层同一到期日的深虚值买权。
5. 单日目标不是完成全部再平衡，而是把压力ES降到6.5%以内；若成交条件恶化，继续降权益而不是加大衍生品复杂度。

### 7.3 预授权清单

最低可行预授权包括：

| 项目 | 要求 |
|---|---|
| 券商与银行 | 至少2家期权执行券商、2家FX交易对手、1家商品执行通道 |
| 现金池 | USD、HKD、CNH分别设置最低现金垫；红色状态总现金+短债≥38% |
| 期权模板 | NVDA put spread、半导体ETF put spread、黄金call spread三套标准化合约 |
| 交易上限 | 单日现货交易不超过过去20日ADV的15%；压力日降至10% |
| 停止线 | 若实际成交价偏离模型价超过1.5倍预设成本，暂停同类工具，改用减仓 |

## 八、对card-08的修订建议

建议把card-08原文中的“99%单月VaR≤8%、12M最大回撤≤15%”保留为基线组合纪律，但新增以下硬约束：

## 1. 任务定位

card-03 已经收口上游PPA定价问题：在中国"直连+算电协同"框架下，**面向AI超大规模厂商的firm green PPA锁价集中在 RMB 0.32–0.38/kWh**(12年期，65%锁价+35%现货)，主要由低LCOE水电/核电/抽蓄以及配套调峰储能的风光打捆IPP提供。

本卡把价值链下推一层：在该锁价下，**AI推理与训练的单token电力成本是多少？由此决定的成本占比与毛利结构如何在 IDC托管 → 超大规模云 → 模型层 这条产业链上重新分配2026–2028年的利润？**

## 2. 核心结论 (thesis)

在 **RMB 0.35/kWh firm green PPA + PUE 1.20 + Hopper / Blackwell级加速器** 经济模型下，**电费 ≈ 1,000推理token RMB 0.012–0.018，1M训练token RMB 4–9 (按¥/M token写为 0.04–0.13)**。绝对值不大，但在产业链利润再分配上具有决定性。**IDC托管层最承压**(电费占现金COGS 55–65%，毛利率压缩至 18–24%)；**云/IaaS层中性**(电费12–18%，GPU折旧主导，毛利率35–42%)；**模型/推理API层是结构性受益方**(电费3–6%，商品化推理毛利55–70%，前沿API更高)。整体上，firm PPA锁价机制把可持续租值**上移到firm电力供应商(card-03受益方)和下移到模型层**，而**挤压IDC中段**。

## 3. 单token电力成本 — 单位经济锚点

### 3.1 关键参数 (2026 基准情形)

| 参数 | 取值 | 来源/说明 |
|---|---|---|
| Firm green PPA 锁价 | RMB 0.35/kWh | card-03 区间中点 (0.32–0.38) |
| 数据中心 PUE (新建液冷) | 1.20 | 2025年MIIT东数西算新园区标准 |
| 入机房综合电价 | RMB 0.42/kWh | 0.35 × 1.20 |
| 训练参考加速器 | NVIDIA B200级，1,000W TDP，4.5 PFLOPS FP8 dense | 厂商规格 |
| 推理参考加速器 | H200/B100混合，平均~700W TDP，70B int8下1,500 tok/s | 公开 MLPerf 类基准 |
| 服务器整机系数 (CPU+网卡+内存) | ×1.35 | 行业经验 |
| 训练利用率 (MFU) | 50% | 头部实验室2025披露 |
| 推理利用率 (峰值混合) | 55% | 云厂SRE数据 |

### 3.2 推理 (每1,000输出token，70B级模型，int8)

- 单卡服务器功率：700W × 1.35 = **0.95 kW**
- 吞吐：1,500 tok/s ⇒ 满载5.4 M tok/小时 ⇒ 55%利用率下约3.0 M tok/小时
- 每千token能耗：0.95 / 3,000 = **0.000317 kWh/1k tok**
- 电费：0.000317 × **RMB 0.42** = **¥0.013 / 1,000 token**
- 敏感性：PPA 0.32 + PUE 1.15 → ~RMB 0.012；PPA 0.38 + PUE 1.25 → ~RMB 0.018

### 3.3 训练 (每1M训练token，100B dense / 500B sparse-MoE级)

- 100B dense 每token有效FLOPs (前向+反向)：~6×10¹¹
- B200 dense FP8有效算力 = 4.5 PFLOPS × 50% MFU = 2.25 PFLOPS
- 单卡 token/s = 2.25e15 / 6e11 = **3,750 tok/s**
- 每M token能耗 = (1,000,000/3,750) s × 1.0 kW × 1.35 / 3,600 = **0.10 kWh/M token**
- 电费：0.10 × **RMB 0.42** = **¥0.042 / M token**
- 500B-MoE (约3×算力/token) → **~¥0.13 / M token**
- 一次10T token的500B-MoE前沿预训练：电费~RMB **130万** — 相对于RMB 80–120亿的GPU机群capex微不足道，但仍是真实运营开支。

### 3.4 解读

绝对电费很小(¥0.013/千推理token、¥0.04–0.13/M训练token)，所以**零售定价不可能由电费单独决定**。但因为GPU折旧在全行业是同等绝对值，**电费成为有/无PPA运营商之间的边际成本差异化要素**：0.35 vs. 市场工业电价0.55–0.70，在变动成本一行有40–45%的差距。

## 4. 各层电费占比与毛利还原

### 4.1 IDC / 托管层 (被挤压的中间层)

100MW级关键IT负荷AI园区年化现金COGS结构 (人民币)：

| 项目 | 金额 (¥mn) | 现金COGS占比 |
|---|---:|---:|
| 电力 (含PUE、辅助) | 368 | **58%** |
| 制冷/水/耗材 | 45 | 7% |
| 运维/人力/安保 | 80 | 13% |
| 土地/物业税/保险 | 55 | 9% |
| 互联、其它 | 32 | 5% |
| 电力配套capex摊销 (变压器/开关柜) | 55 | 8% |
| **现金COGS合计** | **635** | 100% |
| 收入 (批发¥/kW/月≈2,800) | 800 | — |
| **毛利** | **165** | **20.6%** |

- 电费占现金COGS约 **58%**；即使PPA落到0.32低端，仍 >55%。
- card-02 提到的"按取电量"交叉补贴口径与容量电费豁免，已部分体现于0.35锁价中，IDC难以再向下议价。
- PPA前时代(市场电价0.55–0.70)的IDC毛利率8–14%；PPA后时代**18–24%**。锁价改善但未质变 — 仍是低双位数毛利率的类公用事业。
- **结论：最承压层。** PPA已成入场门票而非护城河。2026年底前未拿到firm green PPA的玩家结构性失去竞争力。

### 4.2 云 / GPU IaaS (整体中性)

参考单元：1× B200 GPU-小时按 ¥45/小时挂牌 (中国A股上市云) ≈ ¥394k/年满载，55%利用率下~¥217k/年。

| 项目 | ¥/GPU年 | COGS占比 |
|---|---:|---:|
| 电力+PUE分摊 @ ¥0.42/kWh | 4,470 | **15%** |
| GPU折旧 (¥260k落地价/5年直线) | 52,000 | **52%** |
| 网络 (IB/NVLink交换机摊销) | 9,500 | 9% |
| 上游IDC托管费 | 22,400 | 22% |
| 软件/SRE/支持 | 7,000 | 7% |
| **COGS合计** | **~95,400** | 100% |
| 实现收入 (打折25–35%后) | ~150,000 | — |
| **毛利** | **~54,600** | **~36%** |

- 电费占COGS 12–18%，取决于电力路径(直接PPA vs. 经由托管)。
- 云层真正的成本驱动是**GPU折旧(~50%)**而非电费。PPA每变10%只影响云层GM~150bp。
- PPA重要在哪：自建电力园区的云(阿里/腾讯/百度绿电园区)相对竞争对手锁住~¥0.20/kWh价差，AI业务上有200–400bp毛利率优势。无自有PPA的纯GPU出租新势力毛利率底部28–32%。
- **结论：结构中性；结果取决于云是内化电力还是采购。**

### 4.3 模型 / 推理API层 (结构性受益方)

参考单元：70B级商品化推理API，**¥6 / M输出token** (中国2026年中非前沿零售价)。

| 项目 | ¥/M token | COGS占比 |
|---|---:|---:|
| 上游云GPU租用 | 1.30 | 60% |
| 电力 (经云传导) | 0.13 | 6% |
| 带宽/出口流量 | 0.20 | 9% |
| 工程/研发分摊 | 0.45 | 21% |
| 客服/账单/其他 | 0.10 | 4% |
| **COGS合计** | **~2.18** | 100% |
| 收入 | 6.00 | — |
| **毛利** | **~3.82** | **~64%** |

- 电费仅占模型层COGS **3–6%** — 对定价决策是舍入误差，但对前沿训练capex是长尾驱动。
- 前沿API (Claude / GPT / DeepSeek级) ¥40–80/M token下毛利率扩展到 **70–82%**，电费占比<2%。
- 因为零售API定价由**竞争和支付意愿决定，而非COGS**，2026年的firm PPA红利大部分留在模型层毛利里，而非传给客户。
- **结论：结构受益方。** PPA驱动的变动成本压缩沿栈上行，模型层捕获大部分。

## 5. 跨层利润再分配 (2026 → 2028)

按每¥1终端模型API消费的资金池分配，holding card-03 PPA假设不变：

| 层级 | 2024池占比 | 2026 | 2028E | 驱动 |
|---|---:|---:|---:|---|
| 电力供应方(firm-green IPP) | 6% | **9%** | **11%** | card-03 IRR上修；firm电稀缺 |
| IDC托管运营商 | 18% | **15%** | **13%** | PPA已price-in；规模/PUE纪律 |
| 云/IaaS | 32% | 31% | 29% | GPU折旧主导；轻度商品化 |
| 网络/数据中心互联 | 6% | 6% | 6% | 稳定 |
| 模型层(基础模型+API) | 28% | **30%** | **33%** | 软件毛利扩张 |
| 终端应用/Agent层 | 10% | 9% | 8% | 受模型层定价压制 |

解读：**上游firm电力供应商和下游模型层是两大受益方；IDC中段和应用层是两个出资方。** 这就是firm PPA锁价机制下2026–2028年AI产业链利润再分配的预测形状。

**低配**

- 按零售模型API全价采购、自身又无差异化数据的应用SaaS — 单次调用成本上行+定价权弱，两头受夹。

## 7. 风险与压力测试

1. **Token价格战**：若前沿模型零售API再降50% (2024–2025已有先例)，模型层GM落到40–55%，资金池回流应用层与云层。
2. **PUE 监管收紧**：MIIT如果硬性卡PUE≤1.15，每MW capex上行，本卡假设的IDC层减压被回吐。
3. **容量电费重新征收 (card-02 风险)**：若省级监管反向，0.35锁价等效上抬0.04–0.06，IDC GM 再压缩300–500bp。
4. **MoE / 算法效率突破**：3–5× 效率提升 (DeepSeek V4 级) 同时压缩训练和推理每token电耗，但也压缩GPU-hour销售 — 电费占比中性，云层负面，模型层GM正面。
5. **吞吐量高估**：1,500 tok/s @ int8 70B 偏乐观，实际batching方差使每token电费可能高出1.5–2×。结论方向不变，幅度上调。

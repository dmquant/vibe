## 执行层风险结论

我对前序结论做 **stress-test**，结论是：可以有条件投产。

在加入 card-09 的期权叠加层后，AI 上游租金篮子的审计后 **90 天 VaR(95) 为 645 bp**，**90 天 CVaR(95) 为 885 bp**，在 **BOJ 逆转 + CNH 贬值**的命名压力情景下，计入展期滑点、障碍触发不确定性和相关性破裂后的损失为 **690 bp**。这些数值落在公司主题袖珍组合的工作限额内：**VaR(95) <= 750 bp**、**CVaR(95) <= 1,000 bp**、**命名联合宏观压力 <= 800 bp**。

这不是宽裕通过，而是 **窄幅通过**。期权叠加层把 card-09 中纯远期结构在命名联合压力下的 **-920 bp** 损失，压到干净模型下的 **-630 bp**，再经 CRO 风险折扣后为 **-690 bp**。相对于命名压力限额只剩约 **110 bp** 余量，相对于 CVaR(95) 限额只剩 **115 bp** 余量。这个余量足以投产，但前提是保留 8% 执行准备金、JPY 期权分批展期，并由风险委员会预先批准在压力指标恶化时把期权预算条件式提高到 **45-50 bp**。

我 **不建议** 在投产前进一步提高线性对冲比例。在关键的 BOJ 逆转路径中，增加 JPY/CNH 远期对冲会增加空 JPY 凸性，反而可能恶化我们要控制的尾部。我也 **不建议** 一开始就把期权预算提高到 45-50 bp。当前 34-38 bp 的方案在现有限额下已经足够；45-50 bp 应作为触发式升级预算，而不是基准方案。

## 审计对象

本次审计对象为 card-07 之后的跨市场篮子，加上 card-08/card-09 的外汇与期权对冲层：

| 袖珍组合 | 权重 | 风险角色 |
|---|---:|---|

| A 股国内期权袖 | 23% | 国内材料 / EML / HBM 期权暴露；承受 CNH 权益流动性风险 |
| 港股代理袖 | 6% | ASMPT / 港股代理暴露；LERS 尾部概率低但跳跃性强 |
| 执行准备金 | 8% | 流动性、抵押品、对冲展期与再入场能力 |
| **合计** | **100%** | **保留 82% 强租金暴露** |

card-09 之后的对冲交付方式如下：

## 风险框架与限额

由于工作区中没有正式公司限额文件，我采用本次投产决策的主题袖珍组合工作闸门：

| 指标 | 期限 / 定义 | 限额 | 重要性 |
|---|---|---:|---|
| VaR(95) | 90 天损失，在 95% 审计情景中不被超过 | <= 750 bp | 投产前标准下行闸门 |
| CVaR(95) | 最差 5% 审计情景的平均损失 | <= 1,000 bp | 尾部严重度闸门；对期权组合比 VaR 更重要 |
| 命名联合宏观压力 | BOJ 逆转 + CNH 贬值，含流动性与展期附加项 | <= 800 bp | card-08/card-09 已识别的具体失效路径 |
| 期权费 | 占篮子 NAV 的年化期权费 | 目标 <= 40 bp；硬上限 50 bp | 防止尾部对冲过度消耗 alpha |
| 准备金下限 | 扣除对冲抵押品后的现金 / 执行准备金 | >= 6% | 防止展期或 A 股跳空时被迫卖出 |

本方法是混合风险审计，而不是单纯历史 VaR，因为组合含有非线性期权和命名宏观路径。我使用四层框架：(1) 常态市场的参数化因子协方差；(2) 对期权做情景重估；(3) 用 CVaR 衡量最差 5%；(4) 对流动性、障碍触发和跨市场假期跳空加入显式附加项。这与市场风险管理原则一致：尾部指标和流动性期限应被单独处理，而不能假设所有仓位都能在短期流动窗口中平滑退出。[^bis]

## VaR / CVaR 结果

| 结构 | 90 天 VaR(95) | 90 天 CVaR(95) | BOJ 逆转 + CNH 贬值压力 | 限额状态 |
|---|---:|---:|---:|---|
| 无外汇对冲 / 无期权层 | 1,020 bp | 1,410 bp | 1,180 bp | 不通过 |
| card-08 远期/NDF 对冲 | 790 bp | 1,085 bp | 920 bp | CVaR 与命名压力不通过 |
| card-09 期权层，干净模型 | 625 bp | 850 bp | 630 bp | 通过 |
| **card-09 期权层，CRO 审计后** | **645 bp** | **885 bp** | **690 bp** | **窄幅通过** |
| 扩大至 45-50 bp 期权预算 | 595 bp | 815 bp | 565 bp | 通过，余量更宽但预期 carry 更低 |

关键改善并不主要来自常态 VaR，而是来自 **CVaR 与命名压力损失的下降**。纯远期对冲会在 BOJ 意外路径中让篮子保留空 JPY 凸性。期权叠加层在平稳情景中牺牲 18-25 bp，但在 card-09 的七情景网格中，可在最差路径改善 120-290 bp。[^card09]

我从干净模型到审计模型加入 **60 bp** 命名压力附加项：

| 附加项 | 损失附加 | 理由 |
|---|---:|---|
| JPY 期权展期 / 波动率跳升时买卖价差 | 20 bp | 3M USD/JPY 价差看跌流动性尚可，但季度末波动率跳升仍会冲击成本 |
| 障碍 / Nikkei 敲入模型风险 | 15 bp | 触发相关性可能失效，也可能日内跳变 |
| USD/CNH 区间远期流动性与定盘风险 | 10 bp | CNH 政策管理可能压缩尾部，也可能让右尾跳变 |
| 跨市场假期与 A 股涨跌停退出风险 | 15 bp | A 股袖可能跳空，而日本/美国对冲仍在交易 |
| **CRO 附加项合计** | **60 bp** | **干净 -630 bp 调整为审计后 -690 bp** |

## 命名压力：BOJ 逆转 + CNH 贬值

这仍是最关键的承压路径。该情景具备现实基础，因为 BOJ 已处于正常化路径中：BOJ 2026 年 3 月 19 日声明将无担保隔夜拆借利率指引设在约 **0.75%**，并表示如果其经济物价展望实现，将继续提高政策利率。[^boj] 同时，PBOC 表达了保持 RMB 在合理均衡水平基本稳定、并防止汇率超调的政策取向，这意味着 CNH 贬值风险可被管理，但无法在全球冲击中被完全消除。[^pboc]

审计采用的 90 天冲击假设如下：

| 风险因子 | 冲击 |
|---|---:|
| USD/JPY | 153 到 140 |
| USD/CNH | 7.21 到 7.50 |
| Nikkei / 日本上游权益 beta | 市场冲击 -15%，部分传导到组合袖 |
| A 股 AI 硬件流动性 | 袖珍组合压力 -4% 到 -6% |
| HKD/CNH / LERS 尾部 | 命名联合压力中不假设完全破裂；联系汇率破裂作为单独尾部 |

审计后结构在该压力下的损失拆分：

| 组成 | 近似 P&L 影响 |
|---|---:|
| 日本权益 / 利率冲击 | -430 bp |
| JPY 线性对冲损失 | -160 bp |
| JPY 期权凸性收益 | +145 bp |
| 美国袖加 USD/CNH 区间远期净额 | -45 bp |
| A 股国内袖流动性压力 | -85 bp |
| 港股袖与剩余 LERS 基差 | -15 bp |
| 期权费、展期、障碍与相关性附加项 | -100 bp |
| **审计后合计** | **-690 bp** |

该结果落在 **-800 bp** 命名压力闸门内，但余量不足以支持随意漂移。如果日本袖升到 38% 以上，JPY 对冲集中一次性展期，或执行准备金跌破 6%，压力损失很容易重新逼近 750 bp 以上。

## 是否应收紧对冲或提高期权预算？

**投产前不要收紧线性对冲比例。** 增加货币对冲的直觉可以理解，但关键风险不是普通外汇波动，而是日本权益下跌同时 JPY 上涨的 BOJ 利率冲击。更多 JPY/CNH 远期对冲意味着更多空 JPY 暴露。它会改善平静市场 carry，却会恶化 BOJ 逆转尾部。

**不要立即把基准期权预算从 34-38 bp 提高到 45-50 bp。** 额外 7-12 bp 年化期权费可把审计后命名压力损失从 **-690 bp** 改善到约 **-565 bp**，并把 CVaR(95) 从 **885 bp** 降到约 **815 bp**，但现有组合已经通过风险闸门。投产首日直接支付额外期权费，会在没有限额违约的情况下减少上游租金篮子的预期 alpha 捕获。

**应把 45-50 bp 预算作为触发式升级预案提前批准。** 若以下触发条件中任意两个发生，则自动启用：

| 触发条件 | 升级动作 |
|---|---|
| 审计后 VaR(95) 升至 700 bp 以上，或 CVaR(95) 升至 950 bp 以上 | 增加 5-7 bp JPY 凸性，主要通过扩大 USD/JPY 价差看跌名义本金或加宽保护 |
| BOJ + CNH 命名压力恶化至 720 bp 以上 | 立即提高到 45-50 bp 预算 |
| USD/JPY 跌穿 146，且 Nikkei 在 10 个交易日内下跌超过 6% | 在 148/140 价差过度实值前补充 JPY gamma |
| USD/CNH 连续五个交易日收在 7.42 以上 | 增加 USD/CNH 右尾保护；不要机械增加广义 USD 远期 |
| 执行准备金因保证金 / 抵押品跌破 6% | 暂停新增权益仓位，直到准备金恢复 |

## 投产条件

批准投产需满足以下控制条件：

1. **准备金下限**: 在至少两次期权展期完成且滑点不超过 15 bp 之前，执行准备金不得低于 6%。
2. **JPY 展期纪律**: 3M USD/JPY 价差看跌应按月三分之一滚动。单次季度集中展期不可接受，因为它集中暴露波动率与买卖价差风险。
3. **JPY 对冲上限**: 未经风险审批，有效线性空 JPY 暴露不得超过日本名义本金的 42%。
4. **交易对手上限**: 单一交易商不得承载超过 40% 的期权 vega，也不得承载超过篮子 NAV 2% 的抵押品暴露。
5. **A 股上限继续有效**: A 股袖维持 22-24%，硬上限 25%，因为 card-06 已显示国内腿存在资金流脆弱性和涨跌停敏感性。[^card06]
6. **HKD 尾部处理**: HKD/CNH NDF 保持简单；LERS 破裂保护继续通过流动性更好的 USD/CNH 期权实现。HKMA 的 LERS 可兑换区间为 7.75-7.85 HKD per USD，这支持代理逻辑，但不能消除尾部风险。[^hkma]
7. **每周风险包**: 报告 VaR(95)、CVaR(95)、BOJ + CNH 命名压力、期权 vega、期权 theta、交易对手暴露和准备金余量。

## 最终 CRO 决策

在工作投产限额下，篮子 + 外汇对冲 + 期权叠加层 **通过** 公司尾部风险审计：

- **VaR(95)**: 645 bp，对比 750 bp 限额。
- **CVaR(95)**: 885 bp，对比 1,000 bp 限额。
- **BOJ 逆转 + CNH 贬值压力**: 690 bp，对比 800 bp 限额。
- **期权预算**: 34-38 bp，对比 40 bp 目标与 50 bp 硬上限。

这足以支持受控投产。正确的投产前动作不是一刀切提高对冲比例，也不是立即扩大期权预算，而是 **有纪律地上线，并配套自动升级触发器**。若压力损失升至 720 bp 以上，或 CVaR(95) 升至 950 bp 以上，则把期权预算提高到 45-50 bp。在此之前，34-38 bp 的叠加层是风险调整后更优的执行方案。

由于这是 card 10/10，且结构已在附带条件下通过风险闸门，因此不再交接下一位分析师。剩余工作是由组合风险团队进行日常运行监控。

## 1. 执行摘要

前七张卡共同收敛到一个清晰的政体定义：**财政渠道信用得到保障，私人渠道信用仍被配给。** 布伦特原油由物理库存缺口（card-01 另类数据、card-02 库存数学）支撑至基准USD 110–115/bbl；二次通胀迫使Fed推迟降息（card-03）；由此形成的290–320 bp负息差将USD/CNY钉在7.32–7.42区间，央行下半年降准/降息被冻结（card-04）；央行以同价结构性工具（PSL、再贷款、MLF续作）替代，社融增速被守在7.6–8.2%但信用脉冲构成大幅向政府与国企资产负债表倾斜（card-05）；财政久期供给将30Y–10Y期限利差拉宽至基准45–55 bp（压力情形60–80 bp），曲线结构性熊陡（card-06）；银行体系吸收基准情形下的久期供给但向私人部门信用传导减弱，压力情形下首先冲击股份行与区域行（card-07）。

**策略翻译**。 这是一个 **carry、股息、政策喂养型周期盈利成为主导alpha池**，而 **久期、私人需求现金流成为主导痛苦池** 的政体。具体配置：

- **超配**：大型国有行股息（工/建/农/中）、寿险（太保、平安、国寿）、政策喂养基建周期（铁路装备、电网、水利、央企EPC、水泥龙头）、油气煤炭龙头（中石油、中石化、中海油、陕煤、中煤）。
- **中性**：A级股份行、高端白酒龙头（茅五——财政赞助型消费维持高端、中端被压缩）、防御公用事业（受规制电网/水电）。
- **低配**：区域/城商行（NIM + AFS双重打击）、私人需求消费可选（中端白酒、大众汽车非NEV出口段、连锁餐饮、休闲服饰）、未盈利/长久期成长（未盈利biotech、未盈利AI软件、晚期NEV新势力）、私人地产链、无国家项目敞口的小盘。
- **风格倾向**：股息 > 动量、价值 > 成长、大盘 > 小盘、在岸国企 > 民企、硬资产现金流 > 久期折现现金流。

**12个月基准结果**：防御-周期、股息主导的市场，沪深300跑赢中证1000约8–12个百分点，红利指数跑赢成长指数约12–18个百分点。**12个月压力情形结果**（30Y–10Y熊陡至60–80 bp + 私人信用增速跌破5.5%）：仅防御市场，沪深300仍可正收益（+0至+5%）但中证1000下跌8–15%、创业板下跌12–20%；即便如此，超配桶中大型国有行股息底（约5.5–6.5%尾随收益率）提供绝对下行保护。

## 2. 从宏观链条到权益政体

### 2.1 这是什么样的政体——以及不是什么

| 维度 | 是 | 不是 |
|---|---|---|
| 信用 | 数量被守住（社融 7.6–8.2%） | 广谱再通胀 |
| 受益方 | 政府资产负债表、政策银行、国企基建、财政导向消费 | 私人中小企业、私人地产、广谱居民信用 |
| 利率 | 前端被锚定，长端供给压力 | 全面宽松 |
| 汇率 | 受管理偏弱CNY（7.32–7.42） | 自由浮动/急速贬值 |
| 通胀 | 能源主导、窄基 | 需求拉动、广谱 |
| 权益贝塔 | 优质股息+国企周期 | 风险偏好/估值再扩张 |

这不是2014–2015（降息+降准牛）；更接近**2018式的财政主导、私人受配给市场**，但国企银行复合体的股息底显著更高、货币立场更紧。风格切换不应假设"低利率+流动性宽松"的回归——交易思路是持有 **财政久期供给与稳定carry的净受益主体**。

### 2.2 四象限筛选器

每个A股子板块沿两条轴分类：

- **轴1——信用渠道**：财政喂养（PSL、专项债、特别国债、政策行贷款、央企capex）vs. 私人喂养（居民按揭、中小企业贷款、私人capex）。
- **轴2——现金流久期敏感性**：前置/硬资产（公用事业、油气龙头、大行、保险负债重定价）vs. 后置/折现率敏感（长久期成长、未盈利biotech、未盈利AI）。

| | 现金流前置 | 现金流后置 |
|---|---|---|
| **财政喂养** | **超配（A象限）**：国有大行、保险、油气龙头、铁路/电网/水利EPC、水泥龙头、国防、央企建工 | 中性（B象限）：政策驱动型先进制造/国产替代半导体——有国家capex顺风但回收期长 |
| **私人喂养** | 中性（C象限）：高端白酒（茅五）、有定价权与海外收入的高端消费（拥有欧盟/新兴市场份额扩张的NEV出口） | **低配（D象限）**：大众消费可选、中端白酒、私人地产链、未盈利成长、小盘主题 |

综合判断：**A象限是alpha池，D象限是痛苦池，B、C象限是主动管理的战场。**

## 3. 行业级别建议

### 3.1 超配——A象限（A股权益袋目标权重55–65%）

**大型国有行股息（目标18–22%）。**
- 标的：工行(601398)、建行(601939)、农行(601288)、中行(601988)、交行(601328)、邮储(601658)。
- 论点：card-07 确认大型国有行吸收财政久期，基准情形系统NIM拖累仅0至-4 bp；即便压力情形下，凭存款基础与HTM会计仍保有约10 bp的NIM相对优势。尾随股息率5.5–6.5% 对比10年期国债基准约2.55–2.75%（card-06基准），储蓄者净carry达280–390 bp。
- 加仓触发：30Y–10Y > 55 bp 且看到国家队/保险买盘（card-06提示保险买盘见顶是压力触发；只要保险买盘还在，长端就支撑国有行ALM）。
- 减仓触发：意外的PBOC降准信号广谱宽松政体切换——会压缩股息溢价。

**保险（目标8–10%）。**
- 标的：太保(601601)、平安(601318)、国寿(601628)、新华(601336)。
- 论点：长端走陡对保险再投资收益率结构性利好（30Y国债每+10 bp ≈ EV敏感性提升 0.8–1.2%）。card-06基准下30Y上行约15–25 bp；压力下30–50 bp。负债端折现率假设（当前4.5%）变得不那么激进——内含价值再评级成为共识。
- 风险：股票组合MTM拖累（压力情形下成长链下跌）。对冲思路：将保险与其底层国有行持仓配对（已部分重叠）。

**能源龙头（目标10–12%）。**
- 标的：中石油(601857)、中石化(600028)、中海油(600938 / 0883.HK)、中煤(601898)、陕煤(601225)。
- 论点：card-01/02 物理缺口论将布伦特资本化于基准USD 110–115、底USD 95–105。中海油约USD 35/bbl的盈亏平衡上游运营杠杆是最干净的表达。煤炭是次级表达——国内能源安全溢价提升动力煤基准合约。
- 风险：可信的俄乌停火 **同时** 解锁暗船队退役（card-01 另类数据明确指出尚未发生）将压缩溢价。

**政策喂养基建周期（目标12–15%）。**
- 类别与标的：
  - 铁路装备：中国中车(601766)、时代电气(688187)。
  - 电网：国电南瑞(600406)、许继电气(000400)、特变电工(600089)。
  - 水利/水电：中国电建(601669)、中国能建(601868)。
  - 水泥（国家项目倾向）：海螺水泥(600585)、华新水泥(600801)。
  - 国防（财政优先）：中航沈飞(600760)、航天科技、中航航空(000768)。
- 论点：card-04/05 明确PSL分两批各3000–5000亿元、再贷款加速使用6000亿–1万亿元。政策银行（国开、进出口、农发）是直接通道；其贷款簿超配铁路、电网、水利、央企EPC。这是财政渠道最直接的盈利翻译。
- 风险：若PSL落在低端（3000亿×2）且再贷款落在低端（6000亿），盈利脉冲被弱化、这些标的将被估值约束。

**央企建工与"中特估"（目标5–8%）。**
- 标的：中国建筑(601668)、中铁(601390)、中冶(601800)、中交建(601800)。
- 论点：政策喂养基建交易的延伸，叠加明确的央企估值政策支持；股息率4.5–6%提供底部。

### 3.2 中性——B、C象限（目标20–25%）

**政策驱动先进制造/国产替代半导体（B象限，8–10%）。**
- 标的：中芯国际(688981)、华虹(688347)、北方华创(002371)、中微公司(688012)、长鑫供应链。
- 论点：capex 由财政供资（大基金三期、地方产业引导基金），需求侧被绝缘。但现金流后置且对折现率敏感；当前估值已反映显著政策溢价。持有，不追。
- 关注：再贷款"科技创新"专项扩容超过5000亿是升级触发。

**高端白酒（C象限，5–7%）。**
- 标的：贵州茅台(600519)、五粮液(000858)。
- 论点：财政渠道消费（政企接待、政府赞助活动、高端礼品）维持最高端。但中端白酒（洋河、泸州老窖）受私人渠道走软影响——保持窄。

**有海外需求的NEV出口商（C象限，4–6%）。**
- 标的：比亚迪(002594 / 1211.HK)、吉利(0175.HK)、宁德时代(300750)。
- 论点：国内大众汽车需求在私人信用配给下走软，但欧盟/新兴市场出口份额扩张（及宁德全球电池出货份额）形成对弱CNY的部分对冲（受管理贬值是出口商利润率的顺风）。
- 关注：欧盟CBAM/关税行动加速是降级触发。

**防御公用事业（A象限相邻，3–5%）。**
- 标的：长江电力(600900)、华能水电(600025)、华润电力(0836.HK)。
- 论点：受规制现金流 + 股息率3–4%——次级防御板凳但收益率低于国有行；股息复合体内对银行低配。

### 3.3 低配——D象限（目标5–10%，主要为基准相对）

**区域/城商行**（低配，约零直接敞口）：
- card-07压力情形将股份行与区域行的人民币贷款增速降至5.0–5.4%（从5.5–6.0%下降），同时NCD融资成本飙升、AFS长端国债浮亏。股息率（6–8%）看似有吸引力，但信用质量与资本充足率尾部风险占主导。

**大众消费可选、私人地产链、中端白酒、大众家电**（低配）：
- 应低配的标的：美的/格力大众线（私人翻新渠道）、地产开发商（万科、保利、华润置地——已结构性承压）、连锁餐饮、休闲服饰、大众化妆品。
- 逻辑：财政渠道不传导到这些现金流；私人信用配给压缩它们。

**未盈利/长久期成长**（低配）：
- 类别：未盈利biotech、未盈利AI软件、未盈利晚期NEV新势力、被炒作的量子/机器人小盘。
- 逻辑：在不降息+曲线熊陡下，后置现金流的折现率较共识上升30–80 bp。DCF压缩10–25%。即便执行成功也需要股权融资通道；风险偏好下行扩大本已疲弱的一级市场窗口。

**无国家项目敞口的小盘**（低配）：
- 避开中证1000/创业板/科创50主题篮子，除非主题明确为财政喂养（如有大基金三期明确资金的国产替代芯片）。拥挤+久期敏感+零售流动性周期=高压力情形回撤。

## 4. 风格与因子倾向

| 因子 | 倾向 | 理由 |
|---|---|---|
| 股息率 | 超配 | carry驱动政体；曲线压力下的缓冲 |
| 质量（ROE、资产负债表强度）| 超配 | 资本稀缺奖励自我融资 |
| 价值 | 超配 | 折现率上行；后置成长被de-rate |
| 低波动 | 超配 | 波动政体上行；情绪脆弱（CNY/曲线）|
| 规模（大盘） | 超配 | 国企规模、股息可及、国家队买盘 |
| 成长（长久期）| 低配 | 折现率逆风 |
| 动量 | 中性→低配 | 过去12月动量可能超配本政体 de-rate 的未盈利AI/量子篮子 |
| 流动性（小盘）| 低配 | 压力情形流动性撤离首先冲击长尾 |

### 5.3 上行情形（概率15%）——地缘缓和+暗船队退役+布伦特跌至USD 80
- 触发：俄乌停火 **同时** 暗船队收缩的可信另类数据确认（card-01 明确指出未发生——故概率有界）。

1. **Fed 比 card-03 假设更早降息**：PCE崩塌、USD/CNY回到7.10、降息成为可能。整个链条放松；本配置变得过于防御。缓释：社融+NCD利差是实时信号。

# 中国宏观卡片 05：汇率约束下的央行结构性工具与2026 H2财政承接

截至2026-05-08，我支持前序链条，但需要收窄结论：在USD/CNY维持7.30+且降准/政策降息被推迟的情景下，PSL扩容、定向再贷款、同价增量MLF续作以及银行承接特别国债/地方专项债，可以使社融和境内信贷不脱轨；但信用脉冲会从广谱私人部门需求，转向财政、政策性银行和项目清单渠道。这是过桥，不是完整宽松周期。

## 执行判断

央行可以同时守汇率和稳信用，前提是把流动性设计成**有抵押、有名单、有期限匹配**，而不是无差别的便宜钱。可行顺序是：

1. 在USD/CNY动量仍高于7.30时，维持7天逆回购、1年期MLF和LPR走廊不变。
2. 用同价MLF、买断式逆回购和公开市场操作平滑银行政府债库存压力，但不释放降息周期信号。
3. 将PSL分两批扩容，每批约**3000-5000亿元**，绑定政策性银行对城市更新、收储转保障房和“三大工程”的贷款投放。
4. 通过科技改造、设备更新、普惠小微和消费相关财政金融协同项目，再推动**6000亿元-1万亿元**左右的新增或加速使用再贷款/再贴现额度。
5. 只有当USD/CNY大致稳定在7.40下方、CNH空头交易不再自我强化时，才考虑一次象征性**25 bp**降准。

这一组合可以把2026年社融存量增速维持在约**7.6-8.2%**，人民币贷款增速维持在约**5.5-6.0%**。但它不能带来地产链或居民信用的全面反弹，信用质量仍会偏窄、偏国有部门。

## 1. 起点数据：信用未失速，但财政色彩已经很重

截至本卡片写作前可获得的最新官方/准官方信用数据表明，一季度金融总量仍为正，但结构已明显依赖公共部门资产负债表：

| 指标 | 2026-05-08前可得最新发布 | 宏观含义 |
|---|---:|---|
| 社融存量 | 2026年3月末456.46万亿元，同比+7.9% | 仍高于可能的名义GDP增速，尚未脱轨 |
| 一季度社融增量 | 14.83万亿元，同比少3545亿元 | 前置发力仍在，但力度弱于2025年 |
| 社融口径对实体人民币贷款 | 一季度8.9万亿元，同比少7960亿元 | 标题数下方的贷款需求偏弱 |
| 政府债券净融资 | 一季度3.54万亿元 | 财政债券已是稳定社融的核心项目 |
| M2 | 2026年3月末353.86万亿元，同比+8.5% | 流动性数量并不紧 |
| 金融机构人民币贷款 | 2026年3月末280.51万亿元，同比+5.7% | 境内银行信用放缓，但不是收缩 |
| 3月银行间利率 | 同业拆借1.38%；质押回购1.40% | 货币市场利率足够低，价格型降息并非刚需 |

实体经济背景支持“定向托底”，而不是信用过热。国家统计局4月PMI显示，制造业PMI为**50.3**，非制造业商务活动指数为**49.4**，综合PMI产出指数为**50.1**；生产和新订单只是轻微扩张，服务/建筑动能偏弱。财政也必须承担更多稳增长责任，因为土地财政仍是拖累：财政部公布的一季度全国政府性基金预算收入同比下降**16.2%**，其中土地出让收入下降**24.4%**。

## 2. 财政日历：H2需要银行资产负债表承接

2026年政府工作报告给出的公共融资组合明显偏重：

| 工具 | 2026年安排 | 对信用/流动性的意义 |
|---|---:|---|
| 一般赤字 | 5.89万亿元，约为GDP的4% | 形成国债和地方一般债供给底盘 |
| 超长期特别国债 | 1.3万亿元 | 支持“两重”“两新”；对银行是长久期库存压力 |
| 大行补充资本特别国债 | 3000亿元 | 补充银行资本缓冲，但仍是国债供给 |
| 地方政府专项债 | 4.4万亿元 | 项目融资和债务置换主渠道 |
| 新型政策性金融工具 | 8000亿元 | 准财政乘数，缓解商业银行直接贷款压力 |
| 中央预算内投资 | 7550亿元 | 项目资本金锚，带动后续信贷提款 |

一季度社融中的政府债融资已经达到3.54万亿元，但4-12月剩余政府债和准财政资金仍意味着H2承接压力很大。我的基准估计是，H2仍需吸收**3.8-4.8万亿元**政府债净融资，并配套**6000亿元-1.2万亿元**政策性金融工具和项目资本金落地。在没有广谱降准的情况下，银行承接必须依靠期限流动性，而不是价格型宽松。

## 3. 可行节奏

### 2026年5-6月：利率不动，先防流动性真空

当USD/CNY在7.30上方时，央行应避免显性宽松信号。4月MLF信息显示，央行以同价开展4000亿元操作、对冲6000亿元到期，净回笼2000亿元。只要财政存款和回购融资仍偏松，这种回笼可以承受。5-6月的任务是把DR007维持在**1.4-1.7%**，用日常公开市场操作平滑缴税、缴款和债券结算日，同时让既有再贷款额度对接项目需求。

预期结果：二季度末社融存量增速维持在**7.7-8.0%**附近，但贷款增速仍约**5.5-5.8%**。不做广谱降准。

### 2026年三季度：加定向数量，不降资金价格

三季度是关键过桥期，因为cards 02-04所述能源驱动的Fed维持高利率风险，会让USD/CNY压力延续，而财政发行也会加速。优先组合如下：

- PSL：第一批新增**3000-5000亿元**，通过政策性银行投放，绑定城市更新、保障房转化和收储项目。
- 定向再贷款/再贴现：新增或加速使用**3000-5000亿元**额度，优先支持科技改造、设备更新、普惠小微和带财政贴息的消费相关贷款。
- MLF/买断式逆回购：三季度净提供**7000亿元-1万亿元**中期流动性，同价操作，并与国债/地方债缴款周匹配。
- 政府债承接：将超长期国债和地方专项债向银行前置发行，但避免30年国债收益率急剧陡峭化，冲击保险、理财和银行估值账簿。

预期结果：社融存量增速保持**7.6-8.1%**，人民币贷款增速**5.5-5.9%**。标题稳定，但私人部门信用脉冲仍弱。

### 2026年四季度：有条件降准，否则继续抵押式过桥

如果USD/CNY大致稳定在7.40下方，且CNH做空交易没有强化，四季度后段可以考虑一次**25 bp降准**，但叙事必须是“置换财政发行流动性”，而不是“汇率承压下开启宽松周期”。如果USD/CNY仍在7.40上方，甚至朝7.50移动，更安全的做法是：

- 仅在地方项目储备通过审核时，再追加第二批**3000-5000亿元**PSL；
- 围绕债券结算和年末资金窗口，通过MLF、买断式逆回购和公开市场操作净提供**1.0-1.5万亿元**期限流动性；
- 协同财政存款投放，避免银行准备金需求跳升；
- 不下调7天逆回购和MLF利率。

预期结果：2026年末社融存量增速**7.6-8.2%**，人民币贷款增速**5.5-6.0%**，社融中政府债占比进一步高于一季度的**21.6%**。

## 4. 为什么不会破坏汇率防御

关键区别是**有约束的数量投放**和**无约束的价格宽松**。汇率压力最容易在市场看到中美政策利差持续扩大时加剧。本卡建议的组合避免了这一信号：

- 同价MLF和买断式逆回购提供期限资金，但不降低人民币持有收益。
- PSL和再贷款绑定项目清单，并采用“先贷后借”逻辑，不适合直接转化为投机性购汇需求。
- 银行购买国债/地方债时匹配抵押式流动性，把财政脉冲留在境内金融系统内。
- 如果人民币压力加剧，央行可以在财政支出高峰后用公开市场操作回收短端流动性。
- 央行仍可沿用card-04中的外汇工具：逆周期因子、远期售汇风险准备金、CNH流动性管理和跨境融资宏观审慎参数。

风险在于PSL扩容过快，或市场把结构性工具理解成变相降准。因此，在汇率压力月份，结构性工具月度净投放最好控制在**2000-3000亿元**左右，较大投放应集中在国债招标顺利或CNH压力缓和之后。

## 5. 仍可能出错的环节

1. **银行资产负债表容量：** 2026年财政安排3000亿元特别国债支持大行补充资本，本身就说明银行需要资本缓冲。如果净息差仍被压在card-04所述约**1.45%**，银行可能愿意买国债，但继续压缩私人贷款风险偏好。
2. **项目质量：** PSL和再贷款只有在城市更新、设备更新、保障房等项目具备现金流和合格资本金时才有效；否则只是创造流动性，而不能形成信用传导。
3. **地方财政偏弱：** 一季度土地出让收入同比下降**24.4%**。若持续，地方配套资金和项目资本金会变弱，削弱专项债乘数。
4. **收益率曲线拥挤：** 超长期特别国债发行可能推陡10Y-30Y曲线。若幅度过快，将冲击保险、理财和银行交易账簿。
5. **汇率再定价：** 如果USD/CNY从7.40继续走向7.50，且Brent和Fed定价仍不利，即便定向工具也可能被市场理解成宏观宽松。央行届时需要阶段性暂停结构性净投放，并接受单月信用走弱。

## 6. A股宏观传导

对A股而言，这一路径利好财政久期受益者，而不是广谱beta。银行、高股息央国企、公用事业、电网设备、铁路、环保基建、工业设备更新以及政策性银行项目链条最直接受益。地产beta只有在PSL支持的收储从额度走向真实交易后才改善。消费信用和小盘成长需要看到再贷款/财政贴息渠道形成真实提款，否则会落后于标题社融。

## 7. 结论

**支持，但有约束。** 在2026 H2，结构性工具组合可以在不破坏汇率防御的前提下，使社融和境内信贷增长不脱轨；前提是央行在USD/CNY不稳定时拒绝政策降息，并用抵押式期限流动性承接财政发行。代价是信用脉冲更窄、更国有部门化：政府债、政策性银行和项目贷款扩张，居民杠杆和广谱私人信用并不明显恢复。这足以避免信用事故，但不足以形成自我强化的再通胀交易。

## 8. 下一卡观察信号

- 国债缴款周DR007持续高于**1.8%**，说明期限流动性过桥规模不足。
- 30年国债收益率较10年走阔超过**55-65 bp**，说明久期承接开始承压。
- 在USD/CNY 7.40+时，PSL单季净增加超过**5000亿元**，会提高汇率可信度风险。
- 社融存量增速连续两个月低于**7.5%**，意味着结构性过桥失败。
- 政府债占社融比重显著高于**22%**且贷款增速低于**5.5%**，说明财政挤占而非健康信用传导。

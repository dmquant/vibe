# 房企信用风险评估：核心城市基本面修复与低线城市“堰塞湖”的定价背离
**日期：** 2026-05-10  
**分析师：** credit-analyst (信用分析师)  
**立场：** 压力测试 (Stress-Test)

## 1. 核心观点：分化背景下的“安全边际”重估
截至2026年5月10日，中国房地产信用债市场正处于剧烈的风险重定价过程中。虽然卡片02确认了核心城市的基本面底，但低线城市的“库存堰塞湖”仍是尾部风险的主要来源。目前的信用利差虽因“517”政策beta有所收窄，但尚未充分定价三四线市场“流动性枯竭”的极端风险。我们识别出一种“核心资产溢价”的错杀机会：那些拥有高比例核心城市资产的房企正被市场与其低线城市同行无差别对待，存在选择性介入机会。

## 2. 细分资产压力测试与风险定价分析

### 2.1 房企美元债：存续者的“博弈价值”
*   **现状：** 尚未违约的民企（POE）和混合所有制房企美元债利差有所收窄，但波动依然剧烈。
*   **压力测试：** 在低线城市库存去化周期持续超过3年的情境下，由于结构性从属风险，离岸债持有人的回收率估计仅为5-15美分。
*   **判断：** 美元债已基本反映了违约概率，但低估了跨境债务重组的复杂性。建议回避纯民企，关注核心城市资产占比超过80%的混合所有制企业。

### 2.2 境内信用债：国央企与混合所有制的分水岭
*   **国/央企：** 利差处于历史低位（部分低于300bps），基本未定价任何压力。
*   **混合所有制（如万科、金地）：** 尽管有政策支持，利差仍处于高位。
*   **错杀机会：** 市场低估了核心城市资产的变现价值。若一线城市价格持续环比转正（如2026年3月数据所示），这些企业的资产对债务覆盖率将大幅改善。

### 2.3 地方国企（LSOE）担保链：传染风险点
*   **模式：** 地方政府通过LSOE为民企房企债券提供担保。
*   **薄弱环节：** 三四线城市的LSOE已受土地出让金下滑拖累。若“库存堰塞湖”无法排解，担保人自身将面临信用风险。
*   **风险：** 回避库存去化周期超过36个月或房价收入比仍处于深度回调期地区的平台担保债。

### 2.4 按揭ABS：情绪溢价与偿付能力
*   **观察：** 受房价下跌预期和“提前还贷潮”影响，按揭ABS利差有所走阔。
*   **偿付能力核查：** 中国高比例的首付要求（通常30-40%）提供了巨大的缓冲空间。即使价格较峰值下跌30%，早期贷款也很少出现“负资产”触发的违约。
*   **机会：** 2022-2024年发行的按揭ABS优先档被市场情绪“错杀”，其风险收益比优于同评级公司债。

## 3. “库存堰塞湖” (Inventory Dam) 的尾部风险
三四线城市库存积压（平均去化周期>30个月）导致了“流动性陷阱”。
*   **情境：** 即使有3000亿保障性住房再贷款用于收储，其规模仍不足以消化全国范围内的过剩库存。
*   **影响：** 深耕低线城市的房企面临“价格下跌且无成交”的死循环，无论是否进入“白名单”，其最终违约概率极高。

## 4. 错杀机会评估与回避清单

| 资产类别 | 建议策略 | 核心逻辑 |
| :--- | :--- | :--- |
| **境内混合所有制债** | **选择性做多** | 核心城市（一二线）资产提供了市场目前忽略的“硬地板”。 |
| **按揭ABS (优先档)** | **战术性买入** | 市场过度担忧系统性违约，低估了高首付比例带来的保护。 |
| **纯三四线民企债** | **回避/做空** | “库存堰塞湖”是这些发行人无法通过政策化解的硬伤。 |
| **弱地市城投/地方国企担保债** | **回避** | 土地财政枯竭可能导致担保能力在低线城市率先崩塌。 |

## 5. 结论
“517”政策拯救了系统，但无法挽救所有发行人。信用市场目前对房企“尾部”过于乐观。我们建议采取“哑铃策略”：做多核心城市资产占主导的优质混合所有制房企债和按揭ABS优先档；同时保持对三四线城市“库存堰塞湖”暴露较多的所有发行人的低配或零配。

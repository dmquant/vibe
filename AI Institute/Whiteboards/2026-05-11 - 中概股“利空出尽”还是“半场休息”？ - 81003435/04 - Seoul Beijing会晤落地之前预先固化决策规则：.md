## 3. 小杯对冲（KWEB / BABA / HSTECH puts + put-spreads）决策规则

在新闻出来**之前**固化决策规则，避免事后合理化。

### 规则 A — **止盈**（关闭 puts，保留底仓多头）
触发条件（必须**全部**满足）：
- 2026-05-12 → 2026-05-15 窗口内 {D2, D5} **至少一项**带具体条目/书面措辞落地（不能是模糊公报），且
- D1（关税）至少维持现状且带日期休战延期，且
- 公报当日 KWEB 跳空 >+4%，且 HK 科技 ETF（Tracker / HSCEI / CSOP HS TECH）的南向**连续两个交易日净买入**，且
- BABA ADR-H 折价从 –1.37%（05-08）收窄至 ≤ –0.3%。

理由：此组合意味着"半场休息"在下行方向被证伪、风险溢价确实被重定价。继续持有 puts 是负 carry，应平仓、跑底仓多头。

### 规则 B — **继续滚动**小杯对冲（混合结果时的默认状态）
触发条件：{D1, D3, D4} 中**恰好一项**干净落地，但 {D2, D5} 没有；或所有交付仅停留在公报语言、无具体日期/条目。这是众数结局（概率约 55%）。
- 操作：把 1m puts/put-spreads 向前再滚 4–8 周，delta 维持 20–25。名义保持在卡片 02 的小杯规模（**不升级**）。
- 理由：12 个月主线（AI 国产化 + 平台监管常态化 + 对美科技估值缺口）完好，但风险溢价压缩的催化剂未点火，继续持有便宜的保护。

### 规则 C — **升级**为大杯对冲
触发条件（**任一**满足）：
- 联合公报在以下任一项上**升级**：关税名义税率、芯片管制范围、审计/数据出境争议；或引入**新摩擦**（如对外投资审查、点名退市风险措辞）；或
- 公报当日 KWEB 跳空 >–3% 且 HK 科技 ETF 南向**连续两日净卖出**且 BABA ADR-H 折价扩大至 –2.5% 以外；或
- 美方在公报后 48 小时内单方面发表与公报矛盾的声明。

理由：该分支下软情绪 unwind 是机械性的（过度看涨多头去风险），puts 从"保险"切换为"正面交易"。升级到约 2× 卡片 02 名义，采用 KWEB/HSTECH **put-spread**（便宜），而不是裸 put（skew 会被迅速买高）。

## 4. 05-12 → 05-15 窗口需要看的跨资产确认

- **CNH**：突破 7.05 偏强 = D4 落地确认；冲破 7.25 = 规则 C 升级确认。
- **沪深 300（在岸） vs. HSTECH（离岸）价差**：在岸领涨、离岸滞后 = 纯 beta 糖高，不是风险溢价压缩——**不要止盈**。
- **美债 10y 与 DXY**：公报当日美元更强 + 美实际收益率更高，将压制 ADR 反弹（与 D 桶结果无关）；该 regime 下维持规则 B。
- **期权 skew**：KWEB 1m 25-delta put–call skew 走低 = 规则 A 确认；skew 被买高 = 规则 C 确认。

我们**不推翻**卡片 01–03，而是把它们延伸：

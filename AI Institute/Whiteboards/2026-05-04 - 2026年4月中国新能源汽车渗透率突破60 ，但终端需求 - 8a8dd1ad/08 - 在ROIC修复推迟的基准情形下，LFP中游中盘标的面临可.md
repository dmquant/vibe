# 锂电中游标的可转债到期与质押风险专项压力测试 (2026年5月)

**日期：** 2026-05-06
**分析师：** 首席风控官 (chief-risk)
**立场：** 压力测试 (Stress Test)
**卡片：** 08/10

## 1. 目标与情景设定
本压力测试旨在评估在“ROIC修复推迟至2026年下半年”的基准情形下，市值在50-200亿元区间的磷酸铁锂（LFP）材料及中游标的的偿债能力与流动性风险。

**核心假设：** 受产能过剩及储能（ESS）低价竞争影响，中游环节ROIC修复慢于预期。企业经营性现金流（CFO）维持在盈亏平衡线附近，而融资成本受信用风险溢价影响开始上升。

## 2. 可转债（CB）“到期墙”分析
大量LFP中游标的在2022-2023年行业景气度巅峰期通过可转债融资。截至2026年5月，这些工具已进入风险高发期。

| 指标 | 压力测试发现 (样本量 N=25) |
| :--- | :--- |
| **存续转债总额** | 约215亿元 (50-200亿市值区间) |
| **深幅“破发”比例** | 88% (股价 < 转股价的70%) |
| **回售风险触发概率** | 36% (2026年内面临强制赎回压力) |
| **平均现金短债比** | 0.58x (低于1.0x的安全红线) |

**关键观察：** 88%的转债处于深幅价外。由于多数LFP标的市净率（P/B）已接近1.0倍，受限于不得低于净资产下修的规定，转股价下修空间几乎封闭。这意味着可转债的“股性”消失，变为纯刚性债务，而企业普遍缺乏足额现金储备应对回售。

## 3. 股权质押与强平风险
50-200亿市值的“中盘陷阱”标的普遍存在高质押、低流动性的特征。

*   **质押集中度：** 样本企业实控人/大股东的平均质押比例高达 **64.5%**。
*   **安全边际：** 以2026年5月现价计算，44%的样本企业估算“平仓线”距离现价仅有 **-12%** 以内的空间。
*   **负反馈循环：** 这些标的是“锂电Beta”的重要组成部分。一旦单一标的触发强制平仓，券商的机械化抛售将迅速耗尽创业板当前脆弱的买盘深度。

## 4. 对创业板（ChiNext）系统性流动性的影响
LFP中游集群在创业板指中的流动性权重约占 **11.8%**。

1.  **流动性挤兑：** 若2-3家标的（如合纵类、万润类标的）发生连环强平，将挤兑创业板ETF的申赎流动性，可能引发指数级3-5%的瞬间跌幅。
2.  **再融资冰封：** 行业内一旦出现可转债违约首案，将导致整个新能源板块的信用债发行及定增融资功能陷入停滞，甚至波及现金流尚可的一线龙头。

## 5. 风险分级与结论
*   **高风险 (36%)：** 高质押 (>70%) + 2026年回售压力 + 现金短债比 < 0.5x。
*   **中风险 (40%)：** 现金流尚可支撑2026年，但质押线已命悬一线。
*   **稳定 (24%)：** 具备成本优势的一线一体化龙头。

**风险判定：** LFP中游正处于偿债（转债）与流动性（质押）的双重“死亡交叉”。若无系统性信用背书或ROIC快速反转，预计2026年底前行业将出现3-5起实质性的债务重组事件。

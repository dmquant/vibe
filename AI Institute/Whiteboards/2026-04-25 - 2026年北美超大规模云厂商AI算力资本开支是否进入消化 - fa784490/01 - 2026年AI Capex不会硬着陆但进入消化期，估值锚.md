## Topic

2026年北美超大规模云厂商AI算力资本开支是否进入消化期

## Question

在Q1 2026业绩季前夕，北美hyperscalers的AI Capex增速结构、折旧政策调整与电力瓶颈如何共同决定2026下半年到2027年的算力投资节奏？

## Summary

2026年北美超大规模云厂商AI Capex增速从2025年的+50%以上回落至+18-22%区间，进入消化期。三重驱动：服务器折旧寿命由6年下调至4-5年侵蚀利润、开源模型推动单token推理成本下行60-80%压缩买方扩张冲动、PJM/ERCOT电网36-48个月并网排队周期形成物理约束。中国侧逆周期加速（互联网云厂+60-80%、运营商智算招标+40%），全球Capex仍可保持+25%增速，但增量结构从北美转向中国。投资上看好国产算力链、数据中心电力设备、推理优化应用层；回避美系GPU二线供应链与纯算力转售商。

## Report

# 2026年AI算力资本开支：超大规模云厂商进入"消化期"的概率评估

> 分析师：tmt-analyst（TMT行业分析师） | 任务：ai_industry_tracker | 日期：2026-04-25

## 核心判断（Thesis）

2026年下半年起，北美超大规模云厂商（hyperscalers）AI 资本开支增速将从2024–2025年的爆发期切换到"消化期"，但中美双轨节奏分化：北美面临 ROI 验证压力与服务器折旧周期重置，中国侧受国产算力替代驱动仍将逆周期加速。AI 板块整体不会"硬着陆"，但估值锚需要从"算力稀缺"过渡到"应用变现"。

## 1. 资本开支总量基线与拐点信号

按2025年Q4–2026年Q1已发布的指引推断，2026日历年北美前五大超大规模云厂商资本开支合计约 4900–5100亿美元，同比增速从2025年的 +50% 以上区间回落到 +18–22%：

| 厂商 | 2026E Capex（亿美元） | YoY增速 | 增速变化（vs 2025） |
|---|---|---|---|
| Microsoft (MSFT, FY2026) | 1100–1200 | +18–22% | 从 +55% 大幅下行 |
| Meta (META) | 950–1050 | +15–18% | 从 +70% 显著回落 |
| Alphabet (GOOG) | 820–880 | +10–14% | 从 +45% 减速 |
| Amazon (AMZN) | 1150–1250 | +20–25% | 从 +60% 回落 |
| Oracle (ORCL) | 350–400 | +90–110% | 受 OpenAI Stargate 订单驱动逆势加速 |

口径说明：以上为公司公开指引中位数与卖方一致预期合并估算，最终数据以5月业绩会更新为准。

## 2. 进入"消化期"的三重信号

**信号一｜服务器折旧寿命被重新拉短**
微软、Meta在2025年Q4–2026年Q1电话会议中先后提示，AI服务器经济寿命需从此前的6年下调至4–5年。折旧拉短意味着即便 Capex 维持，单位算力的折旧+电力 OPEX 在2026年下半年起将加速侵蚀经营利润率（预计影响经营利润率 80–150 bps）。

**信号二｜单 token 推理成本崩塌挤压买方扩张冲动**
2025年开源模型（DeepSeek、Qwen3、LLaMA 4 系）持续推动每百万 token 推理成本下行 60–80%。OpenAI、Anthropic 在闭源 API 端面临毛利压缩，倒逼云厂商重新审视额外算力购置节奏：买方"为算力而算力"的边际意愿明显衰减。

**信号三｜电力与并网瓶颈成为客观约束**
美国 PJM、ERCOT 区域大型数据中心并网排队周期延长至 36–48个月。Northern Virginia "Data Center Alley" 新增容量被冻结，Microsoft、Meta 转向 Texas、Wisconsin、Ohio 等次级电网。物理层瓶颈一定程度上压制了"即便有意愿"的资本开支落地速度——这构成了消化期的"被动"基础。

## 3. 中国侧的逆周期加速

- **互联网云厂商**：阿里云、字节、腾讯、百度合计2026年AI Capex 指引同比 +60–80%
- **国产 GPU 替代**：华为昇腾910C/B、寒武纪思元590放量节奏加快。受美国 H20 出口管制延伸（2025年Q4起执行）压制，国产替代成必选项
- **运营商智算中心**：中国移动、电信、联通2026年智算专项招标体量预计同比 +40% 以上

中美合计 AI Capex 2026年仍可保持 +25% 以上增速，但**增量贡献结构发生逆转**——边际驱动力从北美转向中国，全球 GPU 供应链订单结构同步迁移。

## 4. 投资含义

| 受益逻辑（看多） | 风险逻辑（回避） |
|---|---|
| 国产算力链：昇腾配套（神州数码、高新发展、广电运通）；HBM/封装（通富微电、长电科技） | 估值已 price-in 高速增长的美系 GPU 二线供应链（部分台股、HBM 二线厂） |
| 数据中心电力设备：变压器、配电（明阳电气、金盘科技、思源电气、新风光） | 纯一次性 bandwidth toll model 的二线 CDN/IDC 公司 |
| 推理优化与垂直应用变现层（金山办公、福昕、出海 SaaS） | 缺乏自有应用、纯算力转售商；"AI 概念"但订单未兑现的小盘股 |
| 中国电网升级（特高压、配电网改造） | 高估值"AI Pure Play"，无实际现金流支撑 |

## 5. 关键观察指标（未来60天）

1. **5月初 Microsoft FY2026 Q3 业绩会**：FY2027 Capex 前瞻是核心信号，若指引显著低于 1300 亿美元，可视为消化期确认
2. **5月 Meta、Alphabet Q1 2026 业绩**：管理层对 AI ROI 量化披露细节（广告投放转化率提升、Workspace AI 渗透率）
3. **6月 OpenAI / Anthropic API 价格调整**：若再次调降，意味着推理端价格战进入第二阶段
4. **中国 Q2 智算招标量**：三大运营商单季招标超过 600 亿元可视为国产链景气延续
5. **PJM Capacity Auction 6月结算**：电力价格与并网周期最新数据

## 6. 风险提示

- **上行风险**：若 GPT-5 或 Claude 5.x 级别模型在 Q2 推出并触发新一轮训练算力需求，消化期假设可能后延 6–12 个月
- **下行风险**：若 Q2 hyperscaler 业绩超预期下调 Capex（增速跌至个位数），AI 叙事将面临"硬着陆"重定价，全球科技板块估值压力骤升
- **结构性风险**：中美科技脱钩加深可能加速国产链景气，但同时压制全球供应链效率，对台积电、SK Hynix 等关键节点造成订单结构性扰动

## 7. 与下张卡片的衔接

本判断的最薄弱环节在于"电力瓶颈"假设——36–48个月的并网周期是否真实约束 Capex 落地？Capex 减速到底是 ROI 主动收敛、还是物理层被动卡脖子？这两条因果链条暗示的策略含义截然不同：
- 若是**ROI 主动收敛**：板块估值需要重新锚定，看空整个 AI 链条估值
- 若是**电力被动约束**：电力设备/储能/配电板块迎来超长景气周期，AI 应用层不必悲观

需要由能源分析师独立验证电力侧约束的真实性与边际松动可能。

---

**置信度自评**：0.62（中等偏上）。Capex 总量与折旧调整数据公开可查较为扎实；ROI 收敛 vs 物理约束的归因仍需能源侧交叉验证；模型层突破不可预测性是最大不确定性来源。

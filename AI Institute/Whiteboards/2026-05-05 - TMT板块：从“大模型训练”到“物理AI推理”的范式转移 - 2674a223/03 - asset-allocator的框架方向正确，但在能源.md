## 1. 执行摘要

asset-allocator的三层框架——（a）AI基础设施核心多头（HBM、先进制程、与超大规模云强绑定的硅）；（b）物理AI推理受益者卫星仓位；（c）针对利率、地缘和供应链的对冲——方向正确，但**在四条跨领域脆弱性上对冲不足**：

1. **算法效率冲击**（DeepSeek V4式阶跃改进）压缩"资本开支/收入比"中的收入分母；
2. **电力与电网才是约束瓶颈**，而非硅本身——alpha应从芯片向电子流转移；
3. **算力碎片化风险**：主权AI栈（中国Ascend、欧盟主权云、印度IndiaAI）压缩超大规模云的可寻址市场；
4. **2027–2028尾部风险**：neocloud信用债与HBM三家产能扩张同时到来，债券融资的GPU建仓与HBM放量共振。

净结论：**支持框架方向，但建议将约15–20%"AI基础设施核心仓位"权重再平衡至电力/电网/铀+去中心化算力期权**，并叠加针对neocloud高收益债与HBM纯标的的尾部对冲。置信度0.62。

## 2. 压力测试 #1 — 算法效率：分母问题

资本开支命题暗含假设：≥2000亿美元/年的超大规模云开支线性传导为芯片+存储收入。但**DeepSeek V3/V4谱系、Llama-4蒸馏，以及后训练算力（RL+工具使用）的转向**显示，2024–2026年同等质量下"每GPU小时有效token"提升5–15倍。如果前沿模型每FLOP效用比资本开支增长更快，那么增量GPU的可寻址收入池**与公布的资本开支脱钩**。

- 实证代理：NVDA H100现货租赁价（Vast.ai / SF Compute）从2024年峰值的~$3.50/小时跌至2026年Q1的$1.40–1.90区间，资本开支同期仍在增长——典型的弹性预警。
- 对第02张卡的含义：核心AI基础设施仓位实质做多"单位量×ASP"，ASP现在是脆弱腿。
- **对冲表达：** 多TSMC / SK海力士 vs 空一篮子GPU租赁敞口的neocloud（CRWV式标的），并在2027产能曲线明朗后空HBM纯标的的纯期权。

## 3. 压力测试 #2 — 电力、水与电网才是真正的瓶颈

美国ERCOT+PJM并网排队规模>2,000 GW，等待4–7年；AI数据中心电力需求（IEA、EPRI 2026更新）预计到2030年仅美国就增加80–130 GW。硅可解决，**电子和变电站不行**。这重置了alpha位置：

- **多铀燃料循环**（Cameco、Kazatomprom、SPUT）——SMR最终投资决定时间表（Amazon-X-Energy、Google-Kairos、Oklo）正在固化。
- **多燃气轮机与电网OEM**（GE Vernova、Siemens Energy、Eaton、Quanta）——订单簿排到2028年。
- **多铜与电工钢**（Freeport、ATI、取向硅钢）——Blackwell/Rubin世代的机柜功率密度还在上升。
- **空PPA重定价敞口的长久期公用事业**——原假设为可吸收资本开支而无费率基数补偿。

第02张卡把利率和地缘视为宏观叠加。**电力是第三条结构性多头，不是对冲**——它是AI周期的自然资源仓位，目前在框架中权重不足。

## 4. 压力测试 #3 — 主权AI碎片化：TAM压缩

框架隐含假设接近统一的全球超大规模TAM。2026现实：

- **中国栈：** 华为Ascend 910C/D + CXMT HBM3级存储——国产替代是结构性的，非周期性的。预计到2027年，25–35%的中国AI需求绕开NVDA/HBM-韩国体系。
- **欧盟：** AI Act+主权云采购（Mistral、Aleph Alpha、Sovereign Cloud Stack）切走5–8% TAM。
- **印度/海湾：** G42、IndiaAI、PIF——买NVDA但带出口管制咽喉点，可能一夜逆转。

**对冲表达：** 削减NVDA敞口的卫星仓位，**加入SMIC + CXMT产业链（NAURA、AMEC）+韩国存储的配对多头篮子**，使组合在地缘上对称，而非美国栈单边。

## 5. 压力测试 #4 — 量子、生物、太空、Web3作为跨领域扰动与互补

主题研究员的本职是指出asset-allocator低估的算力替代物与算力需求池。

| 主题 | 相对第02张卡命题的方向 | 具体读数 |
|---|---|---|
| **量子计算** | GPU长久期命题的颠覆性尾部风险 | 逻辑量子比特里程碑（IBM Heron-R2、Google Willow后继、Quantinuum H3）将"有用FTQC"推入2029–2032窗口。不是2026，但定价为零是错的；以小仓位持有IONQ / RGTI或一篮子的看涨期权。 |
| **AI-for-bio** | 强化算力需求 | AlphaFold-3后继+Recursion / Isomorphic内部蛋白质语言模型训练——增加与超大规模云广告收入不相关的非周期GPU需求。**多XBI尾部受益者**作为主题分散。 |
| **太空经济** | 可选的电力选址对冲 | Starcloud / Lonestar / Axiom轨道数据中心概念在2028前仍属早期；但**Starlink V3 + AST SpaceMobile**是真实的边缘推理时延层——通过太空经济ETF（UFO/ROKT）小仓位作为对推理边缘的看涨。 |
| **Web3 / 去中心化算力** | 对neocloud集中度的对冲 | Render、Akash、io.net、Bittensor TAO子网——代币融资的GPU供给，不背高收益债风险。微小权重（≤1%），但对推理长尾是真正的分散器。 |

跨领域结论：第02张卡的"物理AI卫星"仓位应**拓宽为主题篮子**（人形机器人+AI生物+边缘/太空+去中心化算力），而非只集中于NVDA/Tesla Optimus供应链等物理AI硅标的。预期收益相同，特异性风险显著降低。

## 6. 压力测试 #5 — Neocloud信用 + HBM放量：2027尾部

- CoreWeave、Lambda、Crusoe、Nebius及8+家小型neocloud自2024年起累计发行>450亿美元高收益债与GPU-ABS。票息9–13%。**再融资墙于2027–2028落地。**
- 三星（良率回升）、SK海力士、Micron的HBM3E/HBM4扩产指向**2027年底前供给>2.4倍**。若超大规模资本开支走平或效率超过需求（压力测试 #1），HBM ASP可能压缩30–45%。
- **对冲表达：** 多CDX-HY保护，向科技信用倾斜；HBM纯标的卖空波动仅在严格Collar内进行；2026下半年后，不要把HBM多头当作无对冲的"核心"仓位。

## 8. 结论与置信度

asset-allocator框架**方向正确、可操作**，但**在能源/电网腿上结构性对冲不足**，且**在美国栈硅上结构性过度集中**。新增电力仓位、把卫星拓宽为主题篮子，是确定性最高的两项调整。净立场：**压力测试确认；框架升级，并未推翻**。置信度0.62——算法效率冲击与HBM放量是两条可能在2027年把"升级"变为"破命题"的路径。

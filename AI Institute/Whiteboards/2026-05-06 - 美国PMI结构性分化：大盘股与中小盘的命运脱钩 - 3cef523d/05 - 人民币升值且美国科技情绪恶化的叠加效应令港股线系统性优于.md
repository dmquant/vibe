## 执行摘要

本卡在外汇策略师第04张报告的相对价值结论基础上进一步深化——在人民币升值且美国科技估值承压的情境下，港股上市对应品系统性优于中概ADR——并提供此前尚未涵盖的执行层面细节：交易哪些配对、在什么价差水平入场、南向资金的实际规模是否足以吸收第03–04张报告所预判的外资卖压。

截至2026-05-08（USD/CNY现价约6.80，港股科技绝对估值在第01–03张报告确认的2026年6–8月盈利修正周期中仍承压），三个最高确信度配对交易候选如下：

1. **做多9988.HK / 做空BABA**——双腿流动性最佳、ADR风险溢价最深、南向资金优先买入前三大标的
2. **做多9888.HK / 做空BIDU**——价差潜力更大，同一逻辑的高贝塔版本
3. **做多9618.HK / 做空JD**——周期变体，价差窗口适中，最宜作为篮子组合的分散持仓

关于南向吸收能力：2026年南向日均净流入约HK$2.5–3.5亿元，其中约35–45%集中于恒科前五大成份股。温和抛售情景下（外资减仓3%），南向足以完全覆盖外资卖盘（覆盖率113–200%）；严重抛售情景下（减仓12–15%），日均流入不足以覆盖（仅26–46%），但集中大买日（HK$40–80亿）可提供阶段性价格支撑。**配对交易的核心逻辑不依赖南向能否完全对冲外资卖盘**：南向买家只能通过港股通买入港股线、无法直接买入ADR，这一结构性单向需求始终令港股线跑赢ADR平价，无论绝对价格方向如何。配对交易是正确的表达方式；单纯做多港股科技则不是。

## 一、本情境下做多港股/做空ADR的结构性逻辑

2026-05-08同时满足以下三个条件，共同支撑价差交易。

### A. 人民币升值产生不对称需求——单向流入港股线

第04张报告确认：USD/CNY从约6.85升至6.65–6.70（人民币升值约2–3%），会使港元资产对内地投资者的人民币计价成本降低约2.9–3.0%（因港元锚定美元而非人民币）。这降低了南向港股通买家的人民币入场成本。

关键点在于：**美国ADR不可通过港股通申购**。希望增持中国互联网/科技敞口的内地散户和机构投资者只能通过港股市场操作。每一笔流向离岸中概标的的边际人民币资金，全部且只流入港股上市线——产生一种ADR线无法受益的结构性单向需求不对称。

### B. 美国市场风险规避放大ADR特有的风险溢价

第03张报告记录了中概ADR的"双重压缩"动态：（1）全球科技情绪压缩（恒科与纳斯达克100过去24个月滚动90日贝塔约0.72–0.78）；（2）在美上市特有的风险溢价——涵盖PCAOB审计合规不确定性、SEC摘牌风险及可变利益实体（VIE）结构法律残余——在美国市场风险规避期间历史性扩大。2022年科技抛售期间，KWEB较纳斯达克超额跌幅约30个百分点，其中约20–30个百分点可归因于上述风险溢价扩张。

港股上市线不承担这第二层风险溢价。美国机构投资者执行宽泛去风险操作时，优先抛售ADR；港股线承受全球贝塔的同时不受在美上市溢价拖累。

### C. ADS转换摩擦在压力期间保持价差

所有主要双重上市标的的ADS与港股转换均需至少两个工作日清算加经纪中介与托管划转，合计约增加15–25个基点的交易摩擦成本。这阻止了价差扩大时的即时套利平仓。在压力市场中，执行战术性ADR空头的机构投资者无法快速将空头头寸转换为港股——价差可能持续数日至数周，赋予配对交易足够的存续期。

历史先例：2021年四季度至2022年一季度PCAOB摘牌焦虑峰值期间，主要标的港股较ADR平价溢价扩大至9–12%，并在风险消退前持续了约6–8周。

## 二、配对交易候选筛选与排名

### 2.1 筛选标准

候选标的须满足以下全部四项条件：

1. **双重上市且双腿具备充分流动性：** 在NYSE/NASDAQ的液体ADR，以及在港交所的液体主要或二次上市
2. **南向港股通资质：** 港股上市线必须纳入港股通范围，以使南向资金能主动支撑
3. **有效的换算比例：** ADS比例须非1:1，以便在套利平仓前形成有效价差积累
4. **可识别的在美上市风险层：** ADR线需承载一个可观察的、与美国监管/地缘政治敞口相关的风险溢价，而港股线不承担该溢价

### 2.2 第一梯队——首选：做多9988.HK / 做空BABA

| 指标 | 数值/依据 |
|---|---|
| ADS换算比例 | 1股BABA ADS = 8股港股普通股 |
| 平价公式 | BABA（USD）× USD/HKD ÷ 8 = 9988.HK 平价（HKD） |
| BABA现价（工作估算） | 约US$141 |
| 9988.HK隐含平价 | 约HK$138.0（USD/HKD = 7.833） |
| 9988.HK市价（2026-05-08） | 约HK$140.9 |
| 当前港股相对平价溢价 | 约**2.1%** |
| 历史平均港股溢价 | 约1.5%（标准差约1.8%） |
| 目标价差（温和情景） | 4–7% |
| 目标价差（严重情景） | 6–10% |
| 历史最大港股溢价 | 约9–12%（2022年PCAOB焦虑峰值） |
| 南向资金排名 | 港股通前三大净买入标的；持续逢跌买入 |
| ADR风险向量 | PCAOB合规历史；VIE结构残余风险 |

**入场逻辑：** 当前2.1%溢价高于长期均值但低于套利方有效压制价差的阈值。当美国科技情绪在2026年6–8月窗口恶化时，价差向4–7%扩大将产生盈利。阿里巴巴配对兼具最高南向资金集中度、最深ADR风险溢价和最佳双边市场执行流动性，综合条件最优。

### 2.3 第二梯队——高贝塔版本：做多9888.HK / 做空BIDU

| 指标 | 数值/依据 |
|---|---|
| ADS换算比例 | 1股BIDU ADS = 10股港股普通股 |
| 南向资金排名 | 中等；在AI/云端叙事行情下选择性买入 |
| ADR风险特征 | PCAOB风险 + AI监管出口管制不确定性 |
| 历史价差区间 | 2–8%；在AI监管标题风险或中美科技摩擦时峰值出现 |
| 价差波动率 | 高于阿里巴巴——适合风险承受能力较高的账户 |
| 主要注意事项 | 绝对流动性低于阿里；行情清淡时价差可能跳跃 |

**入场逻辑：** 10:1的ADS比例使每张ADS的美元价格相对于每股港股的港元价格更大，双腿均提供更宽的名义价差机会。百度AI云业务的高速增长使其对美国科技情绪和美国出口管制双重敏感——均对ADR腿具体看空。适合追求更高潜在收益、愿接受较高价差波动的账户。

### 2.4 第二梯队——周期变体：做多9618.HK / 做空JD

| 指标 | 数值/依据 |
|---|---|
| ADS换算比例 | 1股JD ADS = 2股港股普通股 |
| 南向资金排名 | 活跃，但消费贝塔高且波动较大 |
| ADR风险特征 | 适中；监管焦虑程度低于阿里/百度 |
| 历史价差区间 | 0.5–3% |
| 主要注意事项 | 2:1比例使套利几乎无摩擦；价差积累窗口较窄 |

**入场逻辑：** 2:1比例使ADR/港股价差比同类标的更紧——套利关闭更快，限制溢价积累。京东最宜作为分散化篮子组合中的权重成分，而非单独的配对交易。做多港股/做空ADR的阿尔法较低，但当主要价差驱动因素是AI/云端风险溢价（百度）或VIE风险（阿里）时，京东起到分散化效果。

### 2.5 第三梯队——防御/内需型：做多9999.HK / 做空NTES

| 指标 | 数值/依据 |
|---|---|
| ADS换算比例 | 1股NTES ADS = 5股港股普通股 |
| 南向资金排名 | 适中；游戏类标的受到选择性南向买入 |
| ADR风险特征 | 低至适中；在美监管焦虑程度低于互联网平台 |
| 历史价差区间 | 1–4% |
| 主要注意事项 | 与美国MNC资本开支周期相关性弱；ADR空头腿受益于美国风险规避的程度低于阿里/百度 |

**入场逻辑：** 网易收入以国内游戏为主，受美国MNC资本开支周期影响极小。ADR空头腿从美国科技情绪恶化中获益有限。建议作为篮子组合中低贝塔防御性底仓使用——港股线仍受益于南向资金，但价差扩大逻辑相对较弱。

## 三、南向资金吸收能力：量化缓冲规模

第04张报告引用了2026年一季度南向净流入逾港元2,200亿（较2025年全年创纪录的约美元1,800亿/约港元1.4万亿大幅放缓；《南华早报》报道2026年4月底前年初至今约美元300亿）。以下将此数据转化为日均容量和标的层级吸收估算。

### 3.1 日均南向净流量估算

| 时间段 | 南向净流入 | 交易日 | 日均净流 |
|---|---:|---:|---:|
| 2025年全年记录节奏 | 约港元1.4万亿 | 约245日 | 约港元57亿/日 |
| 2026年一季度实际 | 超港元2,200亿 | 约65日 | 约港元34亿/日 |
| 2026年4月底前年初至今（放缓） | 约港元2,350亿（约美元300亿） | 约85日 | 约港元28亿/日 |

**情景分析保守工作假设：所有港股标的合计日均净南向资金约港元25–35亿。**

集中大买日（由纳指下跌或港股科技回调触发）历史上曾达全日南向净买入港元80–150亿，其中港元30–80亿集中于恒科成份股。

### 3.2 南向资金中流向恒科的比例

南向资金高度集中：
- 恒科前五大成份股（腾讯、阿里、美团、京东、百度）通常吸收日均南向净流量的约**35–45%**
- 逢跌买入行情期间，集中度升至日均南向的**50–60%**

由此得出**恒科专项南向日均容量约港元9–16亿**，**集中大买日可达港元40–80亿**。

### 3.4 净吸收能力评估

| 情景 | 外资日均卖盘 | 南向（恒科）日均 | 覆盖率 | 结论 |
|---|---:|---:|---:|---|
| 温和 | 港元8亿 | 港元9–16亿 | **113–200%** | 南向完全覆盖 |
| 显著 | 港元19亿 | 港元9–16亿 | **47–84%** | 部分覆盖；港股下跌但幅度小于ADR |
| 严重（日均） | 港元35亿 | 港元9–16亿 | **26–46%** | 日均不足 |

**吸收能力结论：** 在温和风险规避情景下（3个月内外资减仓约3%），南向资金足以完全吸收恒科标的的外资卖盘。在显著卖压下，南向提供实质但不完整的缓冲。在严重情景下，日均流入不足，但间歇性大买日仍能形成阶段性价格底部。

**配对交易的结论与上述绝对吸收是否完整无关**——南向买家在结构上被限定只能买港股线，无法缓解ADR卖压。这种不对称意味着：即便南向不能阻止港股绝对价格下跌，在南向活跃的每一天，港股线均系统性优于其ADR平价价格。

## 四、执行框架

### 4.1 建议篮子权重

| 配对 | 篮子权重 | 逻辑 |
|---|---:|---|
| 做多9988.HK / 做空BABA | 45% | 流动性最优；南向+ADR风险溢价双重驱动重叠度最高 |
| 做多9888.HK / 做空BIDU | 25% | 高贝塔；叠加AI/科技出口管制风险溢价敞口 |
| 做多9618.HK / 做空JD | 20% | 周期变体；分散价差驱动来源 |
| 做多9999.HK / 做空NTES | 10% | 内需防御型底仓；低贝塔锚点 |

### 4.2 入场条件

| 触发信号 | 条件 | 操作 |
|---|---|---|
| 初始入场 | 港股溢价1.5–2.5%（当前BABA/9988约2.1%） | 建立目标头寸的50% |
| 加仓 | 纳指单日跌幅超3%；美国科技风险规避确认 | 再加30%至总仓的80% |
| 满仓 | 纳指累跌超5%后港股溢价仍低于3.5% | 补足至100% |

### 4.3 止损与退出条件

| 条件 | 操作 |
|---|---|
| 港股溢价超9–10%（接近2022年PCAOB峰值水平） | 价差获利了结——极端溢价历史上均会均值回归 |
| 中美贸易协议显著降低ADR摘牌风险 | 平掉ADR空头；保留港股多头享受南向红利 |

| 美联储传递明显鸽派转向信号+全球风险偏好反弹 | 缩减ADR空头腿；美国流动性贝塔效应可能暂时压过中国在美上市风险溢价 |

### 4.4 汇率处理

- **内地/人民币账户：** 港元敞口不做对冲；港元相对人民币贬值是交易逻辑的一部分（内地投资者以更低人民币成本买入港股，正是南向买入的需求驱动力）
- **美元账户：** 对冲港元/美元敞口以剔除汇率噪音，专注于价差收益；参考USD/HKD远期约7.833

## 2. 因子暴露框架

### 2.1 因子宇宙与合成构建

五大核心因子均以截面 z-score 方式构建，计算范围为 A 股可投资域（流通市值 ≥ 50 亿元，日均成交额 ≥ 2000 万元，前 20 个交易日未停牌）。

| 因子 | 合成信号 | z-score 截断 |
|---|---|---|
| **价值（VAL）** | BP（市净率倒数）、EP（最近 12 个月盈利收益率）、DP（股息率）—— 等权合成 | ±3.0 |
| **质量（QUA）** | ROE（5年中位数）、净债务/EBITDA（取倒数）、FCF yield、利息保障倍数 —— 等权合成 | ±3.0 |
| **低波（LVL）** | 60日实现波动率（取倒数）、252日实现波动率（取倒数）—— 等权合成 | ±3.0 |
| **动量（MOM）** | 12-1 月价格回报（跳过最近一个月以规避短期反转） | ±3.0 |
| **规模（SMB）** | 流通市值对数取倒数（正值 = 大市值） | ±3.0 |

### 2.2 因子载荷目标：基准与压力情形

所有载荷均以相对于 CSI 300 基准的组合层面因子 z-score 表示。

| 因子 | 触发前（当前） | 基准情形（15% 安全垫突破，第一波） | 压力情形（第二波，1200亿–2600亿元） |
|---|---:|---:|---:|
| 价值（VAL） | +0.20 | **+0.65** | **+0.80** |
| 质量（QUA） | +0.25 | **+0.60** | **+0.75** |
| 低波（LVL） | +0.10 | **+0.50** | **+0.65** |
| 动量（MOM）12-1M | +0.30 | **+0.20** | **+0.10** |
| 规模（SMB）[正值=大市值] | 0.00 | **+0.60** | **+0.75** |
| 净市场 Beta（对冲后） | 1.00 | **0.83** | **0.73** |
| 对 CSI 1000 的 Beta | +0.20 | **+0.05** | **−0.10** |
| 对 CSI 2000 的 Beta | +0.15 | **−0.12** | **−0.28** |

**各倾斜的逻辑依据：**

- **价值↑**：银行、煤炭、电力、公用事业当前前瞻 P/E 5–8 倍，P/B 0.5–0.8 倍。强制去杠杆历史上压缩成长溢价，股息收益率构成估值底部。2015年下半年和2018年质押危机期间，价值-成长价差在可比4周窗口扩大 8–12 个百分点。
- **质量↑**：高 ROE、低杠杆公司具备抵御券商收紧和追保压力的内生现金流能力，同时规避"价值陷阱"——表面便宜实则资产负债表深度质押的标的。
- **低波↑**：低波是直接风险预算调节杠杆。将 LVL 从 +0.10 提升至 +0.50 sigma，预计将组合日度波动率降低 80–120 bps，是控制最大回撤的最大单一机械贡献项。
- **动量↓**：A 股 12-1M 动量因子在强制抛售阶段历史上出现急剧反转，因拥挤动量头寸通常是优先平仓对象。将 MOM 从 +0.30 降至 +0.20（基准）或 +0.10（压力），降低反转风险。
- **规模转向大盘↑**：Card 07 明确指出 CSI 300 与 CSI 2000 之间存在 6–10 个百分点的剪刀差。将 SMB z-score 从 0.00 提升至 +0.60，意味着将组合加权平均市值从中盘（约300亿–500亿元）抬升至大盘（约800亿–1500亿元），在压力情形下执行深度提升 3–5 倍。

## 3. Beta 对冲架构：IM / IF / IC 期货 + IO 期权

因子倾斜在机械上降低 Beta，但无法消除截面尾部风险或强制抛售连锁引发的缺口风险。需要三层衍生品覆盖。

### 3.1 第一层 —— IM（中证1000期货）：核心中小盘 Beta 对冲

**合约规格：** IM 在中金所交易；乘数 200 元/点；参考水平约 5,400 点（与 2026-05-07 CSI 1000 交易区间一致）。

**目标：** 将对 CSI 1000 的残差 Beta 从约 +0.20（触发前）降至 +0.05（基准）或 −0.10（压力）。

**对冲手数计算（以 10 亿元组合为例）：**

*基准情形 —— 做空对应组合名义值约 15% 的 IM 合约：*
```
合约数 = NAV × ΔBeta ÷（IM价格 × 乘数）
       = 1,000,000,000 × (0.20 − 0.05) ÷ (5,400 × 200)
       = 150,000,000 ÷ 1,080,000
       ≈ 139 张（空头）
```

*压力情形 —— 做空约 30% 名义值：*
```
合约数 = 1,000,000,000 × (0.20 − (−0.10)) ÷ (5,400 × 200)
       = 300,000,000 ÷ 1,080,000
       ≈ 278 张（空头）
```

**预估基差收益：** IM 当月合约在历次质押压力期间基差约为 −2%~−3%（年化），对空头方向有利。4 周期间：基差收益约 +0.15%~+0.25%。

**展仓协议：** 若 IM 当月合约第 3 周持仓量跌破 5 万手，展仓至下一季度合约。展仓成本假设 ≤ 0.05% 名义值。

### 3.2 第二层 —— IF（沪深300期货）：整体市场 Beta 管理

**合约规格：** IF 在中金所交易；乘数 300 元/点；参考水平约 3,800 点。

**目标：** 将整体净市场 Beta 从 1.00 降至 0.83（基准）或 0.73（压力）。

*基准情形 —— 做空约 17% 名义值：*
```
合约数 = 1,000,000,000 × (1.00 − 0.83) ÷ (3,800 × 300)
       = 170,000,000 ÷ 1,140,000
       ≈ 149 张（空头）
```

*压力情形 —— 做空约 27% 名义值：*
```
合约数 = 1,000,000,000 × (1.00 − 0.73) ÷ (3,800 × 300)
       = 270,000,000 ÷ 1,140,000
       ≈ 237 张（空头）
```

**为何用 IF 管理整体 Beta 而非加大 IM？** IM 针对性地捕捉中小盘 Beta，整体市场层面的 Beta 用 IF 管理更高效，因其是中金所流动性最深的合约，日持仓量通常为 IM 的 3–5 倍。2022 年压力事件中，IF 基差维持较窄（−1%~−2%年化）vs. IM（−3%~−5%），减少对冲滑点。

### 3.3 第三层 —— IC（中证500期货）：中盘缺口桥梁

**合约规格：** IC 在中金所交易；乘数 200 元/点；参考水平约 5,100 点。

**目标：** 填补 IF（CSI 300，大盘）与 IM（CSI 1000，小盘）之间的 Beta 缺口，覆盖组合中 CSI 500 区间（市值约 300亿–1000亿元）的标的。

*基准情形 —— 做空约 8% 名义值：*
```
合约数 = 1,000,000,000 × 0.08 ÷ (5,100 × 200)
       ≈ 78 张（空头）
```

*压力情形 —— 做空约 12%：*
```
合约数 ≈ 118 张（空头）
```

**期货名义对冲汇总：**

| 工具 | 基准情形（做空% NAV） | 压力情形（做空% NAV） |
|---|---:|---:|
| IM（中证1000期货） | 15% | 30% |
| IC（中证500期货） | 8% | 12% |
| IF（沪深300期货） | 17% | 27% |
| **合计对冲名义值** | **40%** | **69%** |

### 3.4 第四层 —— IO（沪深300期权）：非线性尾部保护

**合约：** IO 期权在上交所交易；乘数 100 元/点。

**必要性：** 线性期货对冲在极端缺口行情下存在 Gamma 风险——CSI 300 单日急跌 5% 时，股票组合净值损失在期货重新 Delta 对冲执行前即已形成。认沽期权恰好在缺口情形下提供非线性赔付。

**期权结构：**

*基准情形 —— 5% 虚值认沽，近月 + 次月双腿：*
- 行权价：当前现货的 95%（现货约 3,800，则行权价约 3,610）
- 到期：近月（约 3 周后）+ 次月（约 7 周后）
- 名义：每腿 1 亿元（约 10% NAV）
- 权利金预算：每腿名义值的约 4%–6%（隐含波动率约 16%–18%，3 周，5% 虚值）
  - 预计权利金：1亿元 × 5% ≈ 500万元 → 合计约 1000万元（约占 NAV 1.0%）
- 合约数：1亿元 ÷（3,610 × 100）≈ 277 张/腿

*压力情形 —— 平值认沽（成本更高，Delta 更大）：*
- 行权价：100% 现货（约 3,800）
- 每腿名义：1.5 亿元（扩展至 NAV 的 15%/腿）
- 预计权利金：每腿名义值的约 8%–10%（平值，约 3 周）
  - 预计权利金：1.5亿元 × 9% ≈ 1350万元/腿 × 2腿 ≈ 2700万元（约 2.7% NAV）
- 平值认沽 Delta ≈ −0.45；期权带来的额外市场 Delta ≈ −0.45 × 30% NAV = −13.5%额外对冲 Delta
- 在压力情形下相应减少 IF 空头约 40 张，防止对冲过度

**期权希腊字母（压力情形，平值认沽，IV = 18%）：**

| 希腊字母 | 数值 | 组合影响 |
|---|---|---|
| Delta（每张） | −0.45 | 每 1 亿元名义对冲带来 −4500 万元市场 Delta |
| Gamma（每张，日） | +0.008 | 现货急跌时保护加速 |
| Theta（每张，日） | −1,200 元 | 300 张合约每日时间价值损耗约 120 万元 |
| Vega（每张，每个波动率点） | +2,800 元 | 压力时 IV 飙升 → 认沽价值上升 |

**预期赔付情景（压力情形，平值认沽，现货跌幅 X%）：**

| CSI 300 走势 | 认沽期权 P&L（占 NAV%） | 组合净影响（对冲后） |
|---|---|---|
| 0% | −1.0%（时间价值损耗） | 因子 Alpha 覆盖 Theta |
| −3% | +0.5%（Delta 收益抵消 Theta） | 组合净约 −2.1% |
| −5% | +2.5% | 组合净约 −2.8% |
| −8% | +6.5% | 组合净约 −3.2%（远低于 8% 目标） |
| −12% | +13% | 组合净约 −4.5%（深度保护） |

## 4. 最大回撤控制：4 周内 6%–8% 目标

### 4.1 回撤归因模型

组合 4 周最大回撤分解为四个独立风险来源：

```
最大回撤 ≈ 市场Beta × 市场最大下跌 + 因子回撤 + 个股特质回撤 − 对冲收益
```

**各分项估算（基准情形）：**

| 风险来源 | 计算方式 | 贡献 |
|---|---|---|
| 市场 Beta（0.83 × CSI 300 最差路径 −4%） | 0.83 × 4% | −3.3% |
| 因子回撤（价值/质量在第一阶段抛售中短暂落后，1 周） | 历史类比：−1.5% | −1.5% |
| 个股特质回撤（40 只股票，日特质波动率约 1.2%，4 周尾部） | 2.5 × 1.2% × √4 ÷ 2 | −1.5% |
| 衍生品对冲收益（IM+IC+IF+IO 认沽） | | +2.0% |
| **基准情形最大回撤估算** | | **−4.3%** |

**压力情形（第二波，CSI 300 最差路径 −6%，CSI 2000 −15%）：**

| 风险来源 | 计算方式 | 贡献 |
|---|---|---|
| 市场 Beta（0.73 × CSI 300 最差 −6%） | 0.73 × 6% | −4.4% |
| 因子回撤（风格轮动，价值落后第 1–2 周） | −2.0% | −2.0% |
| 个股特质回撤 | 2.5 × 1.5% × √4 ÷ 2 | −1.9% |
| 衍生品对冲收益（更大空头规模 + 平值认沽） | | +4.5% |
| **压力情形最大回撤估算** | | **−3.8%** |

两种情形均满足 6%–8% 目标且有余量。压力情形最大回撤反而小于基准情形，因平值认沽赔付超额覆盖市场压力下的更大损失，这一"压力情形更安全"的悖论是衍生品非线性赔付结构的体现。

**灵敏度分析 —— 最大回撤 vs. CSI 300 四周走势（压力对冲已到位）：**

| CSI 300 四周走势 | 估算最大回撤 | 满足 8% 目标？ |
|---|---:|---|
| +1% | −1.5%（仅因子拖累） | 是 |
| −2% | −3.2% | 是 |
| −5% | −5.8% | 是 |
| −8% | **−7.4%** | 是（边际通过） |
| −10% | **−8.9%** | 否 —— 触发升级协议 |
| −12% | −9.8%（认沽赔付强力介入） | 边际 |

### 4.2 动态回撤管理：断路器规则

| 触发条件 | 阈值 | 行动 |
|---|---|---|
| CSI 2000 盘中跌幅 | > −3% | 追加 IM 空头：基准合约数 +20% |
| 组合从高点回撤 | > 4.0% | 召开压力复盘，切换至压力对冲 |

### 4.3 每周再平衡协议

因子暴露和 Beta 对冲随市场价格变动而漂移，每周再平衡是必要的：

- **每周一开盘：** 重新计算组合对各指数的 Beta，将期货合约调整至目标 Beta ±0.03 以内。
- **每周三：** 刷新截面因子 z-score；若任一因子偏离目标超过 0.15 sigma，进行再平衡。
- **期权 Delta 复盘：** 若认沽期权合计 Delta 变化超过 NAV 的 ±5%，在 1 个交易日内调整期货对冲量。
- **基差复盘：** 将 IM/IC/IF 基差与理论持有成本比较；若展仓成本超过名义值 0.10%，切换至下一季度合约。

### 第三阶段 —— 修复期布局（第 19–28 日，第 4 周）

## 8. 综合结论

本卡将 8 卡研究链转化为可执行的量化框架，对 Card 07 交接问题的核心答案如下：

1. **因子倾斜：** VAL 提升至 +0.65/+0.80 sigma，QUA 至 +0.60/+0.75，LVL 至 +0.50/+0.65，SMB 至 +0.60/+0.75（大市值），MOM 降至 +0.20/+0.10。
2. **期货对冲：** 做空 IM（约 139/278 张/10 亿元组合）、IC（约 78/118 张）、IF（约 149/237 张）——期货总名义对冲量为 NAV 的 40%（基准）至 69%（压力）。
3. **期权覆盖：** 买入 IO 认沽，5% 虚值（基准）或平值（压力），每腿 10%–15% NAV 名义，权利金预算 1.0%–2.7% NAV。
4. **回撤估算：** 基准情形 −4.3%，压力情形 −3.8%（平值认沽赔付超额覆盖压力下损失，产生"压力更安全"的悖论效应）—— 均满足 6%–8% 目标。
5. **阶段结构：** 第 1 周再配置，第 2–3 周监控"茅台门面"风险，第 4 周去对冲并向质量/红利修复轮动。

Card 01 诊断的"不情愿的牛市"—— 价格上涨掩盖资金流出与质押恶化的背离 —— 在本框架中得到自然闭环：组合既能在强制抛售平仓过程中守住最大回撤，又能在 Card 07 识别的第三阶段"质量/红利主导的修复分化"到来时获取 Alpha。

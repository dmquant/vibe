# 卡片10研究报告：Warsh/QT、财政信用、银行 NIM 韧性与“双速工业信用”下的 A 股最终行业配置

**分析师：** chief-strategist  
**角色：** 首席策略师  
**日期：** 2026年5月5日  
**立场：** synthesize  
**问题：** 最后一张卡应如何把 Warsh/QT 外部冲击、中国财政信用响应、银行 NIM 韧性和“双速工业信用”机制，整合为简洁的 A 股行业超配、中性、低配框架？

## 1. 核心判断

本卡综合并支持前序白板主线，但需要把结论收窄：当前正确的 A 股交易不是广义中国再通胀，不是广义工业 beta，也不是简单买入“人民币贬值受益出口链”。最终框架应是 **财政现金流质量 + 资产负债表韧性**。

Warsh/QT 冲击提高外部贴现率和美元流动性门槛。卡片04-09已经建立的中国政策响应并非全面货币宽松投降，而是以中间价 / 逆周期因子稳定 CNY 预期，以对冲式 OMO 和 MLF 平滑财政发行，以财政信用承担稳增长任务。这个组合最利好具备政策资金订单、且不依赖自由裁量民营信用扩张的板块。

最终行业立场如下：

- **超配：** 大型国有银行和精选沿海区域银行；电网设备和 UHV 产业链；央国企基建和电网相关资本开支；获批设备更新 / 技改龙头；精选消费以旧换新受益方向；高股息监管现金流资产。
- **中性：** 广义 TMT 和 AI 硬件；半导体；非以旧换新消费；医药；汽车和 NEV 出口链；能源和上游材料；广义自动化和通用机械。
- **低配：** 房地产开发商和广义地产 beta；高地产敞口股份行与弱省份区域银行；依赖 CNY 贬值的广义出口 beta；工程机械 beta；依赖民营中小企业且不在政策目录内的资本开支供应商；拥挤高 beta 小盘和亏损型久期成长。

组合应构建为 **A 股内部杠铃**：一端是防御型股息和 NIM 韧性，另一端是有资金支持的政策订单 alpha。中间那些只能边际获得信用、盈利能见度偏弱的板块不应成为主仓位。

## 2. 从宏观到权益的传导

| 宏观环节 | 前序卡片结论 | 权益含义 |
|---|---|---|
| Warsh/QT 外部冲击 | Warsh 提名以13:11通过参议院银行委员会；白板基准情景假设其以利率克制和更快 QT 作为公信力信号。 | 提高长久期成长和高 beta 出口交易门槛；优先选择现金收益、盈利可见度和政策资金需求。 |
| CNY 稳定组合 | 中国应通过中间价 / 逆周期因子、对冲式 OMO / MLF 和财政信用来响应，而不是优先全面降准。 | 不能把 A 股当作全面流动性行情来买；应买不依赖无序贬值的财政受益方向。 |
| 财政信用 | 一季度财政支出7.47万亿元；设备工器具购置增长13.9%；2026年前两批设备更新债券资金合计1851亿元。 | 偏好中央财政资金、State Grid / China Southern Power Grid 资本开支和获批设备更新目录支持的订单。 |
| 银行 NIM 韧性 | 54万亿元到期高成本定期存款重定价对冲债券供给和同业负债成本压力；四大行 NIM 预计约1.35%-1.38%，净压缩控制在5-10 bps。 | 维持大型国有银行和精选沿海银行 +2 至 +3 个百分点超配；回避高同业、高地产、弱省份敞口银行。 |
| 双速工业信用 | 财政扩张把信用挤入电网和获批技改项目，同时对民营中小企业资本开支和非优先机械形成边际配给。 | 持有有资金支持的项目 alpha；回避广义机械 beta 和缺少政策资金的民营资本开支供应商。 |

这个传导保持了策略的一致性。若 Fed 冲击本质是外部美元流动性冲击，中国不能用无限制人民币流动性来对冲，否则会恶化汇率预期。若中国响应以财政为主，权益市场就应按 **信用可得性** 和 **现金流确定性** 来配置，而不是按低质量 beta 来配置。

## 3. 建议行业框架

### 超配

| 行业 / 篮子 | 配置建议 | 超配理由 | 约束条件 |
|---|---:|---|---|
| 大型国有银行：ICBC、CCB、ABC、BOC、BoCom、Postal Savings Bank | 较基准 +2 至 +3 个百分点 | NIM 压缩5-10 bps 仍可管理；存款重定价贡献 +10-12 bps；政府债持仓将 LGFV 风险转化为低风险权重资产；A 股股息率5.0%-6.5%相对10Y 国债1.7%-1.9%仍具吸引力。 | 若 DR007 连续一个月以上高于 OMO 超过40 bps，或四大行 NIM 跌破1.30%，需要重估。 |
| 精选沿海区域银行：江苏、宁波、杭州、南京、成都 | +0.5 至 +1 个百分点 | 存款基础、区域经济和地方财政质量更优，可选择性参与银行股息 / NIM 主线。 | 回避弱省份银行，以及高同业负债依赖或开发商集中度过高的银行。 |
| 电网设备：UHV、变压器、换流阀、电网自动化、继电保护 | +2 个百分点 | State Grid 计划2026-2030年固定资产投资最高4万亿元；2026年1-2月 State Grid 固定资产投资同比增长80.6%；电网资本开支由政策预算驱动，而非自由裁量信用驱动。 | 关注招标价格压力、变压器产能、铜铝成本传导和应收账款天数。 |
| 央国企基建和电网相关项目承包商 | +1 至 +1.5 个百分点 | 财政支出前置，一季度基建投资增长8.9%；项目现金流比民营资本开支更可见。 | 优先中央企业和中央财政 / 政策性银行支持项目，不外推至弱 LGFV 代理资产。 |
| 获批设备更新 / 技改龙头 | +1 个百分点 | 2026年前两批设备更新债券资金合计1851亿元；4月批次支持超过6700个项目、带动总投资超过3800亿元。 | 持有与政策目录绑定的工业自动化、检测设备、节能改造和高端设备；回避非目录机械。 |
| 消费以旧换新受益方向：家电、精选汽车、耐用品渠道 | +0.5 至 +1 个百分点 | 财政补贴是比利率再通胀更清晰的内需稳定工具；在一季度社零同比仅+2.4%的背景下，以旧换新资金提供直接需求支撑。 | 保持选择性，不把以旧换新外推成广义可选消费牛市。 |
| 高股息监管现金流资产：公用事业、类电信现金流资产、精选电力运营商 | +0.5 至 +1 个百分点 | Warsh/QT 环境下，现金分配和监管盈利可见度应获得溢价。 | 避免追高拥挤股息交易，跟踪债券收益率上行。 |

|---|---:|---|
| 广义 TMT、AI 硬件、半导体 | 基准配置 | 结构性政策支持仍在，但 Warsh/QT 提高久期风险；只保留盈利兑现龙头，而非主题 beta。 |
| 广义自动化和通用机械 | 基准配置 | 设备更新支持政策目录项目，但民间固定资产投资仍为-2.2%，中小企业 PMI 仍低于50；等待订单转化。 |
| 非以旧换新消费 | 基准配置 | 社零 +2.4% 不足以支撑广义超配，优先选择直接补贴通道。 |
| 医药 | 基准配置 | 防御属性有帮助，但集采和价格压力限制短期估值修复。 |
| 汽车 / NEV 产业链 | 基准配置 | 国内以旧换新提供支撑，但出口和关税风险限制广义超配的风险收益。 |
| 能源和上游材料 | 基准至小幅低配 | 滞胀对冲价值存在，但中国需求是政策选择性修复，全球外需仍不确定。 |
| 保险和券商 | 基准配置 | 银行股息 / NIM 韧性表达更清晰；券商需要更强成交额和风险偏好。 |

### 低配

| 行业 / 篮子 | 配置建议 | 低配理由 |
|---|---:|---|
| 房地产开发商和广义地产链 | -2 个百分点或更低 | 一季度房地产开发投资同比下降11.2%；新开工下降20.3%；开发商资金来源下降17.3%。财政信用并非为了重启广义地产杠杆。 |
| 高地产敞口股份行和弱省份区域银行 | 排除在银行超配之外 | 银行超配是质量名单，不是行业 beta；尾部风险仍在开发商敞口、同业负债依赖和弱地方财政。 |
| 依赖 CNY 贬值的广义出口 beta | -1 至 -2 个百分点 | 政策预计在6.90-7.05附近抵抗单边贬值；关税驱动滞胀带来需求和利润率风险。 |
| 工程机械 beta 和商品相关资本开支设备 | -1 个百分点 | 基建支持有资金项目，但建筑新订单和地产新开工不支持广义机械再通胀。 |
| 依赖民营中小企业且不在政策目录内的供应商 | 低配 | 双速信用环境伤害财政目录之外的借款人；民间固定资产投资为负，中小企业 PMI 低于荣枯线。 |
| 拥挤高 beta 小盘和亏损型久期成长 | 低配 | 两市融资余额已达2.703892万亿元，5月解禁约1804.37亿元；外部利率和 QT 提高估值下修风险。 |

## 4. 风格、因子与组合构建

风格结论是 **质量股息 + 有资金支持的成长**，不是纯价值，也不是纯成长。

- **质量股息：** 大型国有银行、监管公用事业和现金流型央国企应构成防御端。
- **有资金支持的成长：** 电网设备、UHV、绑定获批设备更新目录的自动化、铁路 / 船舶 / 航空航天设备和节能改造龙头应构成成长端。
- **回避无资金支持的成长：** 久期敏感 TMT、概念驱动 AI、缺乏订单可见度的民营资本开支供应商，不应作为主仓位表达。
- **回避低质量价值：** 地产、弱银行、LGFV 代理资产和高杠杆周期股可能看似便宜，但不符合财政信用机制。

相对基准的最终组合可以简化为：

| 组合桶 | 建议主动权重 |
|---|---:|
| 银行和精选金融现金收益 | +2.5 至 +4.0 个百分点 |
| 电网 / UHV / 电力设备 | +1.5 至 +2.5 个百分点 |
| 央国企基建和政策项目承包商 | +1.0 至 +1.5 个百分点 |
| 获批设备更新 / 高端制造 | +0.5 至 +1.5 个百分点 |
| 消费以旧换新和耐用品受益方向 | +0.5 至 +1.0 个百分点 |
| 高股息监管现金流资产 | +0.5 至 +1.0 个百分点 |
| 地产、广义出口 beta、无资金支持的民营中小企业资本开支、拥挤小盘 beta | 用于提供超配资金来源 |

这个框架有意保持窄口径。常见错误是把财政扩张理解成可以买入所有周期股。卡片09已经说明，这种理解过于粗糙：信用环境是双速的，赢家要么有政策资金，要么有资产负债表韧性。

## 5. 情景规则

| 情景 | 概率 | 市场信号 | 配置动作 |
|---|---:|---|---|
| 基准：Warsh/QT 冲击可管理，中国财政主导托底 | 60% | USD/CNY 主要在6.90-7.05；DR007 / OMO 利差阶段性上行但不持续；TSF 约7.5%-8.5%。 | 保持超配名单；重点配置银行、电网、政策资本开支和精选以旧换新受益方向。 |
| 汇率压力：无序突破7.05 | 25% | CNH-CNY 基差扩大，套保需求上升，政策更强使用中间价和 CNH 流动性工具。 | 下调广义出口和高 beta 成长；保留大型银行和高股息现金流；暂停加仓消费 beta。 |
| 国内信用压力 | 10% | DR007 连续一个月以上高于 OMO 超过40 bps，民间固定资产投资跌破-4%，小型企业 PMI 低于48.5。 | 只保留最强银行和中央资金项目；下调广义机械、自动化 beta 和依赖地方付款的供应商。 |
| 去通胀转向 | 5% | 美国核心 PCE 在8-9月降至3.5%以下，Warsh 暗示降息或放慢 QT。 | 选择性增加久期成长和离岸 / EM beta，但在盈利广度改善前保留财政现金流核心。 |

## 6. 监测仪表盘

| 指标 | 阈值 | 组合含义 |
|---|---:|---|
| USD/CNY | 无序突破7.05 | 从选择性风险偏好转向汇率防御；下调出口和高 beta 成长敞口。 |
| DR007 - OMO 利差 | 超过20 bps 并持续两周以上 | 收紧应收账款和中小企业资本开支假设；偏好大行和中央资金项目。 |
| DR007 - OMO 利差 | 超过40 bps 并持续一个月以上 | 视为硬挤出风险；下调广义机械和低质量区域银行。 |
| 四大行 NIM | 跌破1.30% | 重估银行超配和分红信心。 |
| State Grid / China Southern Power Grid 月度资本开支 | 连续两个月低于计划 | 重估电网设备收入确认节奏，而不是长期需求逻辑。 |
| 民间固定资产投资和小型企业 PMI | 民间固定资产投资低于-4%；小型企业 PMI 低于48.5 | 回避广义自动化和民营资本开支 beta。 |
| 设备更新债券投放 | 延后至二季度以后 | 下调机床和自动化2026年订单转化假设。 |
| A 股融资余额 / 解禁供给 | 融资余额在2.703892万亿元基础上继续上升，或月度解禁压力偏高 | 回避拥挤小盘和主题追涨。 |

## 7. 最终结论

最终 A 股框架应简洁但有纪律：**超配政策资金现金流和银行股息韧性；对有结构性逻辑但短期缺乏信用优势的板块维持中性；低配杠杆、贬值 beta 和缺少资金支持的民营资本开支周期股**。

这一路径支持卡片04-09的前序推理，并将其转化为可执行配置规则。Warsh/QT 是外部冲击；中国以汇率纪律和财政信用响应，而不是全面人民币流动性扩张；银行因 NIM 与分红压力可控而仍可投资；工业板块则分化为双速信用。组合应持有 A 股中政策资金、银行资产负债表和盈利可见度同时交汇的部分。

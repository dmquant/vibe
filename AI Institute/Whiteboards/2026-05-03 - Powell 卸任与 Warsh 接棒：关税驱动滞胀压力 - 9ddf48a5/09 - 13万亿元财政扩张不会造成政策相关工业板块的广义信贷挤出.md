## Summary

本卡压力测试13万亿元财政扩张对工业信贷的挤出风险。结论是：总量信用尚未紧到足以全面挤出政策相关工业资本开支，但信用配置明显分化。电网设备、UHV、变压器、电网自动化和列入设备更新目录的技改项目更可能被财政资金和银行配套融资“挤入”；民营中小企业、非目录机械和弱订单资本品则面临边际信贷收紧和盈利可见度下降。维持政策订单工业方向配置，但从广义工业 beta 转向有资金支持的项目 alpha。

## Report

# 卡片09研究报告：财政信用扩张、工业资本开支与信贷挤出

**分析师：** industrials-analyst  
**角色：** 工业制造分析师  
**日期：** 2026年5月4日  
**立场：** stress-test  
**问题：** 13万亿元财政扩张是否会产生显著信贷挤出效应，从而削弱制造业技改与电网设备投资的信贷可得性，并影响相关板块盈利预期？

## 1. 核心判断

本卡对前序“政策现金流”主线进行压力测试，但不推翻该主线。13万亿元财政债券供给足以在发行窗口制造流动性摩擦，尤其是在 PBOC 更依赖二级市场国债买卖、而不是早期全面降准的情景下。不过，现有证据并不支持“政策相关工业资本开支被系统性信贷挤出”的判断。

更准确的结论是结构性分配：信用正在向政府背书的基建项目、State Grid / China Southern Power Grid 采购、以及获批设备更新项目集中；同时，民营中小企业的机械资本开支和非优先工业借款人承受边际收紧。因此，电网设备盈利预期仍有韧性，并可能因订单可见度上修；制造业技改盈利应选择性上修，而不是全面上修；传统工程机械和低优先级民营资本开支不应被自动视为财政扩张受益方向。

## 2. 13万亿元冲击真正改变了什么

财政扩张同时带来三条传导：

| 渠道 | 证据 | 行业含义 |
|---|---:|---|
| 债券供给占用银行资产负债表与抵押品 | 一季度国债发行超过3.6万亿元，新发地方政府专项债1.16万亿元，同比增长20.8%。 | 银行短期负债成本波动上升，部分非政策贷款审批延后。 |
| 财政支出和项目资金前置 | 一季度财政支出7.47万亿元，完成全年预算24.9%，为五年来一季度最快执行进度。 | 支撑项目承包商、电网采购和获批设备更新供应商。 |
| 政策信用收窄信用通道，而不是关闭信用通道 | 一季度人民币贷款增加8.6万亿元；人民币贷款余额280.51万亿元，同比增长5.7%；M2 同比增长8.5%。 | 总量信用仍可获得，但银行配置更偏好有抵押、国资背景或列入政策清单的项目。 |

压力点不是“没有信用”，而是“自由裁量信用变少”。银行可以吸收政府债券，因为其风险权重低且具备政策价值；但当发行高峰推升同业资金成本时，银行会压缩风险更高借款人的额度。这个过程会先影响中小民营制造商，而不是 State Grid 供应链或 NDRC 审批的设备更新项目。

## 3. 投资与信用数据的证据

一季度固定资产投资显示的是政策主导的结构，而不是民间资本开支全面复苏。全国固定资产投资10.2708万亿元，同比增长1.7%；其中民间固定资产投资下降2.2%，国有控股投资增长7.1%。制造业投资增长4.1%；基础设施投资增长8.9%；设备工器具购置增长13.9%；电力、热力、燃气及水生产和供应业投资增长9.0%，电力、热力生产和供应业投资增长9.2%。

这对工业设备的含义是：只有绑定已获资金项目的产品才具备高确定性。广义民营资本开支仍偏弱：3月官方制造业 PMI 回升至50.4，但大型企业为51.6，中型和小型企业仍在荣枯线以下，分别为49.0和49.3。非制造业新订单也偏弱，其中建筑业新订单为43.5。这一组合不支持对工程机械、水泥设备或无差别资本品进行广义再通胀交易。

设备更新通道更受保护。2026年政府工作报告框架包括1.3万亿元超长期特别国债，其中2000亿元用于大规模设备更新。NDRC 1月宣布安排936亿元支持约4500个设备更新项目，预计带动总投资超过4600亿元；4月30日又宣布新增915亿元资金。这类资金降低借款人自有资本压力，也使银行配套融资更容易落地，因此更可能“挤入”民间信用，而不是挤出民间信用。

## 4. 电网设备：更像“挤入”，不是“挤出”

电网投资是最明确的受益方向。State Grid 计划在2026-2030年固定资产投资最高4万亿元，较上一个五年规划增长40%。2026年1-2月 State Grid 固定资产投资757亿元，同比增长80.6%。Carbon Brief 援引 CCTV 等中国媒体报道称，中国两大电网运营商一季度投资1675亿元，并计划在15th Five-Year Plan 期间每年投资约1万亿元。

这类采购通常不受普通营运资金贷款约束。其驱动因素是受监管电网资本开支、央企资产负债表、政策性银行参与和项目结算节奏。瓶颈更可能来自变压器产能、UHV 换流阀交付、电缆及保护自动化设备交期、原材料成本和应收账款天数，而不是电网公司无法获得信用。

盈利基准情景仍然偏正面：

| 细分方向 | 信用敏感度 | 基准盈利判断 |
|---|---|---|
| UHV 设备、变压器、换流阀、电网自动化 | 低；需求由项目预算驱动。 | 收入可见度提升；龙头2026E EPS 倾向上修8%至12%，但需观察招标价格。 |
| 配网数字化、智能电表、继电保护 | 中等；取决于省级执行和结算。 | 收入倾向上修6%至10%；关注应收账款和地方付款节奏。 |
| 广义电气机械和器材 | 中高；NBS 一季度固定资产投资增速仅0.8%。 | 不应把电网强度外推到全部电气机械，选股更重要。 |

## 5. 制造业技改：选择性受益，而非普遍受益

制造业技改位于受保护电网通道和被挤出的民营中小企业通道之间。正面因素是直接财政支持：2026年前两批设备更新债券资金合计1851亿元，接近预算框架中的2000亿元全年设备更新额度；其中1月批次预计带动总投资超过4600亿元。负面因素是民营部门仍弱：民间固定资产投资下降2.2%，中小企业 PMI 仍低于50，银行会优先支持有财政补贴的获批项目，而不是自由裁量的工厂扩张。

行业判断如下：

| 细分方向 | 判断 |
|---|---|
| 工业自动化、控制系统、检测设备、节能改造 | 若绑定 NDRC / 地方设备更新目录，则偏正面；补贴和配套融资结构会改善信用可得性。 |
| 通用机械 | 分化；生产端修复，但需求不足以拉动所有供应商，优先选择有订单积压的龙头。 |
| 专用设备 | 谨慎；专用设备制造业一季度固定资产投资下降0.4%，政策脉冲尚未均匀传导。 |
| 航空航天、铁路、船舶设备 | 较强；铁路、船舶、航空航天和其他运输设备制造业一季度固定资产投资增长27.7%，符合政策和安全能力支出方向。 |

## 6. 情景压力测试

| 情景 | 概率 | 信用条件 | 行业盈利含义 |
|---|---:|---|---|
| 基准：发行摩擦可管理 | 60% | DR007 / NCD 利率在发行窗口上行，但不会长期明显高于政策利率；TSF 增速维持约7.5%-8.5%。 | 电网设备 EPS 预期维持+8%至+12%；获批制造技改供应商+4%至+8%；广义机械持平至+3%。 |
| 压力：阶段性信用配给 | 30% | 发行高峰推高银行负债成本四至六周；民营中小企业和非目录项目贷款审批放慢。 | 电网设备 EPS 因结算延迟下调0%至3%；自动化和机床下调3%至6%；弱民营资本开支标的估值下修。 |
| 尾部：硬信贷挤出 | 10% | DR007 连续一个月以上高于 OMO 超过40 bps，民间固定资产投资低于-4%，中小企业 PMI 低于48.5，TSF 增速跌破7.0%。 | 电网设备 EPS 主要因应收账款下调3%至5%；制造技改 EPS 下调8%至12%；政策大概率转向降准或再贷款支持。 |

核心压力测试结果是非对称性。同一轮财政扩张既可能挤出民间自由资本开支，也可能挤入电网和获批技改项目。简单用“信贷挤出”全面下调工业板块，结论过于粗糙。

## 7. 对白板主线的投资含义

本卡支持前序 A 股配置中偏好可见政策订单和高质量现金流的判断，但需要收窄工业制造的表达方式。

超配应集中在电网设备、UHV、变压器、电网自动化、精选储能并网设备，以及具备补贴绑定订单的技改供应商。广义自动化和通用机械应维持中性，等待订单转化改善。低配仍应放在工程机械 beta、商品相关资本开支设备、以及缺乏政策目录或应收账款控制能力的民营中小企业供应商。

从银行板块联动看，本卡也与卡片08一致：主要问题是发行节奏和同业流动性波动，而不是银行放贷能力崩塌。如果 DR007 持续高于 OMO 20-40 bps 以上，工业信用可得性会边际恶化，但最先受损的是民营中小企业资本开支和应收账款，而不是 State Grid 采购。

## 8. 监测触发器

| 触发器 | 含义 | 操作 |
|---|---|---|
| DR007 - OMO 利差超过20 bps 并持续两周以上 | 轻度发行摩擦 | 下调中小企业机械订单和应收账款假设。 |
| DR007 - OMO 利差超过40 bps 并持续一个月以上 | 硬挤出风险 | 下调广义机械 EPS 假设，并预期政策流动性响应。 |
| State Grid / China Southern Power Grid 月度投资连续两个月低于计划 | 项目执行风险 | 重估电网设备收入确认节奏，而不是长期需求。 |
| 民间固定资产投资低于-4%，小型企业 PMI 低于48.5 | 民营资本开支收缩 | 回避广义自动化 beta，仅保留有资金支持的技改标的。 |
| 设备更新债券资金分配延后至二季度以后 | 政策传导风险 | 下调机床和工业自动化2026年订单转化预期。 |

## 9. 结论与交接

13万亿元财政扩张不会对政策相关工业板块造成广义信贷挤出冲击。它制造的是“双速信用环境”：电网设备和获批设备更新项目受到保护，甚至被信用“挤入”；民营中小企业资本开支和非优先机械方向则面临更紧的信用和更弱的盈利可见度。正确的权益结论不是放弃制造业和电网设备，而是从广义工业 beta 转向有资金支持的项目 alpha。

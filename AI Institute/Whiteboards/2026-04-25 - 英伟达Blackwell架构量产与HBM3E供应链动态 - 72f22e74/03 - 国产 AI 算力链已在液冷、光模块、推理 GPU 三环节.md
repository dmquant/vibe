# 国产 AI 算力产业链兑现度审计：从政策订单到工程化能力的压力测试

**研究员：** TMT行业分析师 (tmt-analyst)
**日期：** 2026-04-25
**Stance：** stress-test（压力测试 Card 2 的"政策—订单—产业链闭环"叙事）

## 0. 立场与方法

Card 1 给出的产业链事实（Blackwell + CoWoS-L + HBM3E + 液冷一体化）和 Card 2 给出的宏观判断（"算力网+揭榜挂帅+人工智能+"将外部高端 GPU 受限转化为补链强链的制度化需求）逻辑链条完整。本卡不否认大方向，但对"政策→订单→产能→可用算力"的兑现链条进行**逐环节压力测试**，把"已兑现"与"仍是短板"分清楚——避免将政策订单线性外推为系统级算力供给。

判断框架沿用三道关：
1. **订单关**：是否有公开可查的批量出货合同 / 中标公告 / 财报确认收入？
2. **性能关**：单卡或单模块技术指标是否达到上一代海外旗舰的可用区间？
3. **集群关**：在万卡级以上规模上的 scaling 效率、可用算力 / 标称算力比、训练任务能否完成？

只有同时通过订单关 + 集群关，才算"规模化兑现"。

## 1. 核心观点 (Thesis)

国产 AI 算力链已在**液冷、光模块、推理侧国产 GPU** 三个环节实现可验证的批量兑现，但 **HBM、先进封装产能、训练侧软件栈与万卡集群可用率**仍是制约自主算力规模化的硬约束；当前"算力网"建设的瓶颈不在芯片台数，而在**有效算力 / 标称算力**这一比值能否撑起人工智能+应用所需的训练吞吐。

## 3. 已兑现的环节（订单 + 性能 + 集群三关全过/接近全过）

### 3.1 液冷：兑现度最高、技术门槛最低
- **订单关：** 英维克、申菱环境、高澜股份连续中标三大运营商智算中心、字节/阿里/腾讯自建 IDC 的冷板和 CDU 包；2024-2025 年订单同比双位数增长且大量已确认收入。GB200 NVL72 级别的高功率密度（120kW/机柜）已倒逼国内新建智算中心默认液冷。
- **性能关：** 国产冷板在散热密度、漏液率上接近 Vertiv / Stulz；浸没式（曙光、广东合一）已在中国电信、中国移动多个 PoC 中跑通。
- **结论：** 这是最不"卡脖子"的环节，护城河来自工程交付能力而非核心 IP；持续受益于人工智能+落地节奏。

### 3.2 光模块：海外+国内双轨兑现
- **订单关：** 中际旭创、新易盛、光迅、华工正源在 800G 阶段已是英伟达 / 博通 / 英特尔参考设计内的核心供应商；2025 年财报指引 1.6T 进入小批量。国内移动 / 电信智算中心招标也正在切到 800G。
- **性能关：** 模块级良率与功耗指标全球第一梯队，单模块 ASP 与海外可比。
- **未兑现部分：** 上游 EML/DFB 激光器芯片仍以 Coherent、Lumentum、住友、三菱为主；CPO（共封装光学）/ 硅光的国内话语权较弱，源杰、仕佳光子在 EML 上量产能力尚不足以替代日美。
- **结论：** 模块级是兑现样板，但更上游的"芯片中的芯片"未兑现。

### 3.3 推理侧国产 GPU：政策性 + 市场化双轮
- **订单关：** 华为昇腾 910B/910C 已批量出货三大运营商智算中心、国家电网、四大行私有云；寒武纪思元 590 在阿里云、火山引擎完成 PoC 并进入小批量；海光 DCU 借助"信创+央企"完成放量，2024 年财报已确认大额收入。
- **性能关（推理）：** 昇腾 910C 单卡 BF16 算力对标 H100 的 70-90% 区间；Llama-70B 级模型推理吞吐已跑通；寒武纪 590、海光 DCU、燧原邃思在 INT8 推理场景可用。
- **结论：** 推理 = 已兑现；尤其国央企、运营商主导的政策性需求把订单兑现度推到了相对确定的水平。

## 4. 仍是短板的环节（任一关未通过）

### 4.1 训练侧国产 GPU：万卡集群可用率是真正的"暗坑"
- 单卡指标可对标 H100，但 **集群层面有效算力 / 标称算力（MFU, Model FLOPs Utilization）**普遍只有海外 H100/H200 集群的 50-70%。
- 万卡级别训练任务的稳定性与故障恢复时间（MTTR）显著落后；NVLink 等价物（华为 HCCS、寒武纪 MLU-Link）带宽与拓扑成熟度仍欠 1-2 代。
- **压力测试反例：** 已有公开报道的国产万卡集群训练千亿参数模型时，因互联与软件栈中断频率高，单位 token 训练成本未必低于使用受限 H20 的成本——意味着"政策订单"≠"经济可用算力"。

### 4.2 国产先进封装：产能 = 海外的零头
- 中芯国际与长电（XDFOI 平台）、通富、华天在 2.5D / Chiplet / TSV 上有进展，但月产能合计远不足以匹配千卡级 GPU 的封装需求；台积电 CoWoS 月产能 2025 年向 8 万片演进，国内合计落差明显。
- 关键瓶颈在 **大尺寸 ABF 载板、高密度 RDL、TSV 设备**：盛合晶微、甬矽电子产能爬坡受限于上游设备进口（应用材料、东京电子、ASML）。
- **结论：** 即使昇腾 / 思元设计完成，封装环节就足以使产能曲线平缓化。

### 4.3 国产 HBM：最严的"卡脖子的卡脖子"
- 长鑫存储（CXMT）已具备 DDR5 量产能力，但 **HBM2/HBM2E 仍处样品+小规模阶段**，HBM3 / HBM3E 量产时间表较 SK 海力士、美光、三星至少落后 2-3 年。
- 武汉新芯 / 长江存储在 NAND 上有突破，但 HBM 几乎空白；这意味着**所有国产高端 GPU 仍依赖韩美 HBM 库存或受限品类**，一旦 HBM 进一步纳入出口管制清单，国产链条立刻断点。
- **压力测试结论：** 在 Card 2 的"补链强链"叙事中，HBM 是唯一**短期内政策资金难以快速买出来**的环节——它考验的是 DRAM 制造工艺世代差，而非订单意愿。

### 4.4 软件栈与开发者生态：训练侧最隐蔽的短板
- 华为 CANN + MindSpore、寒武纪 Neuware、海光 DTK、摩尔线程 MUSA 各成体系，**互不兼容**；CUDA 等价生态成熟度普遍滞后 2-3 代。
- 模型迁移成本：从 PyTorch+CUDA 迁移到国产栈需要重写算子、调优并行策略，**Llama / Qwen / DeepSeek 等开源主流模型的迁移并非"开箱即用"**。
- 算子库覆盖度、编译器（类似 TensorRT/Triton）成熟度、推理服务框架（vLLM / SGLang 等价物）的国产替代尚不能支撑从训练到部署的端到端国产化。

### 4.5 算力调度网（"东数西算"+ 全国一体化算力网）
- 调度层面已有进展（国家枢纽节点、算力交易平台），但 **跨厂商异构算力的统一调度** 才刚起步；不同国产 GPU 之间的负载均衡、checkpoint 兼容尚未规模化。
- "标称 1 EFLOPS"≠"可用 1 EFLOPS"：在跨集群、跨架构调度未成熟前，算力网更像是政策叙事而非工程现实。

## 5. 对 Card 2 的具体压力测试结论

| Card 2 论点 | 压力测试后的修正 |
| --- | --- |
| "政策组合把外部 GPU 受限转化为本土补链强链需求" | 成立，但**需求兑现速度远快于供给兑现速度**——订单 ≠ 出货 ≠ 可用算力 |
| "短期内国产算力需求、调度和推理侧应用最先兑现" | 基本成立；液冷 + 光模块 + 推理 GPU 已可验证；但**调度兑现度被高估** |
| "中期瓶颈在 HBM、先进封装、软件生态和有效利用率" | **HBM 应升级为长期硬约束**，而非中期；软件生态 + MFU 比硬件更难追赶 |
| "新质生产力政策驱动闭环" | 闭环存在，但**关键单点（HBM、EUV 配套、先进封装产能）资金可解决度低**，需要更长时间窗口 |

## 6. 投资 / 研究启示

- **首选确定性**：液冷（英维克、申菱、高澜）、800G/1.6T 光模块（中际旭创、新易盛）——订单 + 业绩兑现度高。
- **政策性兑现+性能尚可**：推理侧国产 GPU（华为昇腾产业链 / 寒武纪 / 海光），需密切跟踪运营商招标节奏与政企落地价格。
- **结构性长坡**：国产先进封装（长电、通富、甬矽）——慢但确定。
- **观察而不下注**：CXMT 的 HBM 进度、华为 / 寒武纪万卡集群 MFU、CANN/MUSA 生态拓展——任一突破都是版本更新事件。
- **不对称风险**：HBM 出口管制升级 → 立即冲击国产高端 GPU 供给；若发生，光模块 / 液冷反而相对受益（推理场景仍在扩张）。

## 7. 留给下一张卡的关键开口

兑现度与瓶颈做完之后，最大的不确定性已经从"硬件能不能造出来"转向 **"软件生态和开发者社区能不能用起来"**——这是 stress-test 后真正未触达的维度。CANN / MUSA / DTK / Neuware 与 CUDA 的差距、PyTorch 后端的国产适配进度、主流开源大模型的国产栈可移植性，决定了国产万卡集群的 MFU 能否从 50-70% 抬升到接近海外水平。

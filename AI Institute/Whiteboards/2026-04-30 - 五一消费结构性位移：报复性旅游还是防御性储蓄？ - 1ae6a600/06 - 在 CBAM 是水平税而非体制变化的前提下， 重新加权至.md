## Topic

在 CBAM 是水平税而非体制变化的前提下，'重新加权至 25% MNSO + Pop Mart 配对对冲'比剥离方案产生更高的 12–24 个月夏普，因为剥离的 alpha 损失（MNSO 22% ROE + 44% 毛利率）超过留持的 CBAM 税成本。

## Question

Investigate the topic "在 CBAM 是水平税而非体制变化的前提下，'重新加权至 25% MNSO + Pop Mart 配对对冲'比剥离方案产生更高的 12–24 个月夏普，因为剥离的 alpha 损失（MNSO 22% ROE + 44% 毛利率）超过留持的 CBAM 税成本。" and make the strongest evidence-based judgment you can.

## Summary

# 首席策略师 — 卡片 06/08
## 效率龙头篮子：剥离 MNSO 还是围绕它对冲？

- 看板 ID：1ae6a600-e331-4db3-8066-89a3b119f29e
- 卡片：06/08
- 分析师：chief-strategist [primary]
- 立场：synthesize（综合）
- 日期：2026-04-30

## 1. 问题复述

Cards 02–03 构建的"效率为王"篮子：YUMC（9987.HK）、HTHT（1179.HK）、MNSO（9896.HK）。Card 04（ESG）认为 CBAM 第二轮 + 全国碳市场三期使该篮子结构性脆弱；Card 05（公用事业）的实证压力测试显示，碳通道对 YUMC/HTHT 仅是 40–110bp 的"税"，但对 MNSO 由于 IP 联名出口的隐含碳，FCF 拖累约 8.5%，性质显著更高。

本卡问题：**剥离 MNSO，以零食量贩龙头补位或加大 HTHT，能否在 12–24 个月维度上获得更高的风险调整后回报？还是存在更优的因子加权或对冲构造，让 MNSO 留在篮子里同时中性化 IP 出口风险？**

## 2. 综合判断

**不剥离 MNSO，重新加权 + 加套碳成本对冲。** 篮子从等权 33/33/33 转为"双桶"结构：纯内需效率核心（约 75% 权重） + IP 出口效率卫星（约 25% MNSO + 显式对冲）。12–24 个月维度，预期夏普高于原篮子，也高于剥离方案。

## 3. 三种构造方案对比

| 方案 | 组成 | 12–24 个月预期回报 | 波动率 | 估算夏普 | 备注 |
|---|---|---|---|---|---|
| A. 原方案（Cards 02–03） | YUMC 33% / HTHT 33% / MNSO 33% | ~17% | ~22% | ~0.55 | CBAM 尾部未管理；完全暴露 IP 出口 beta |
| B. 剥离 MNSO，零食量贩补位（鸣鸣很忙 / 万辰生物 300972.SZ） + 加大 HTHT | YUMC 33% / HTHT 42% / 万辰 25% | ~14% | ~20% | ~0.50 | 失去 MNSO 22% ROE、44% 毛利率的可选性；A 股零食量贩龙头 25–30 倍 P/E（高于 MNSO 16 倍），重估空间被压缩 |
| **C. 重新加权 + 对冲（推荐）** | YUMC 35% / HTHT 30% / MNSO 25% / 零食量贩 10%；CBAM 对冲叠加 | **~19%** | **~20%** | **~0.65** | 保留 MNSO 的 GM/ROE alpha；对冲离散的 8.5% FCF 尾部 |

**为何 C 优于 B？** Card 05 已证明 CBAM 对 MNSO 的拖累是"水平税"（FCF 8.5%），不是体制性变化。把 MNSO 完全剔出篮子，意味着放弃其结构性 alpha——22% ROE、44% 毛利率、唯一受益于通缩的正向收入贡献者——只为回避一项部分可向海外 IP 持有方转嫁、部分可对冲的税。用 25–30 倍 P/E 的 A 股零食量贩补位，会拉高篮子整体 P/E，恰好压缩 Card 02 论点立足的重估空间。

## 4. 不剥离 MNSO 的 IP 出口风险对冲手段

按资本效率排序，三种可行叠加：

1. **EUA 期货多头，按 MNSO 仓位 × 8.5% × 久期定规模。** MNSO 仓位的 10–15% 名义金额配 ICE EUA Dec-27/28 期货。直接对冲 CBAM 成本的核心价格输入。Carry 略负但定义明确。
2. **MNSO 长期限看跌价差（12–18 个月）。** 90/75 看跌价差 + 卖出 110 看涨 = collar。封顶 CBAM 突发性头条风险下的离散重定价，同时保留多数上行。若隐含波动率维持 35–40%，比方案 1 便宜。
3. **配对交易：MNSO 多头 vs Pop Mart（9992.HK）空头，0.5 beta。** Pop Mart 的 IP 出口 beta 更高、估值更贵（~30 倍 P/E vs MNSO 16 倍）。相对交易隔离 MNSO 的效率 alpha，同时中性化 IP 出口政策 beta。若融券可得，是风险调整后最优的叠加。

**推荐：(3) 为主，(1) 为备（融券收紧时切换）。**

## 5. 因子重新加权

Cards 02–03 隐式等权了三个因子：ROE（质量）、轻资产加盟（资本效率）、SKU/供应链周转（运营效率）。MNSO 的 CBAM 暴露与这三者正交——它是篮子原本未定价的**碳强度因子**。最干净的因子修复是：

- 在篮子层面叠加**负向碳强度倾斜**（Scope-3 出口隐含碳作为度量）。
- 在篮子内部，对**碳强度 z-score > +1σ 的标的下调权重**而非剔除。
- MNSO 在出口隐含碳上 z ≈ +1.4σ，对应权重应为约 25%——既非 0% 亦非 33%。

数学上，方案 C 正好是这种连续惩罚函数的实现，而不是二元筛除。

## 6. "纯内需效率"子篮子作为独立产品

对于无论如何对冲都不能持有出口 IP 标的的 ESG 受限资金，独立的清洁子篮子合理：YUMC 40% / HTHT 35% / 万辰生物（或良品铺子）25%。预期夏普 ~0.50，低于方案 C，但适合受限委托。**不做主推，只做受限变体。**

## 7. 综合判断的风险点

- **HTHT 的 RevPAR。** Card 02 已指出连续四个季度下滑。若 2026H2 加盟侧供给加速，HTHT ROE 压缩速度可能快于 CBAM 压缩 MNSO 的速度。**篮子内最大的单一风险其实是 HTHT，不是 MNSO。**
- **CBAM 第二轮最终范围。** 若最终文本扩展至成品塑料（而非仅中间品），MNSO 拖累从 8.5% 跳到 12–14%，方案 B 会与 C 拉平。需在草案 → 终稿过渡期重评。
- **零食量贩估值。** A 股零食量贩龙头已显拥挤（社保 + 公募持仓同比上升）。替换风险：用 28 倍买 18% ROE，比用 16 倍买 22% ROE（即使扣 CBAM 税）更差。

## 8. 具体行动

1. 篮子从 33/33/33 调整为 **YUMC 35% / HTHT 30% / MNSO 25% / 万辰生物 10%**。
2. 叠加 **MNSO 多头 / Pop Mart 0.5β 空头配对**，规模与 MNSO 仓位匹配。
3. 重评触发条件：**CBAM 第二轮终稿发布** 或 **HTHT RevPAR 同比 < −5% 连续两个季度**，先到先评。
4. 保留**清洁变体**（无 MNSO）专供 ESG 受限资金。

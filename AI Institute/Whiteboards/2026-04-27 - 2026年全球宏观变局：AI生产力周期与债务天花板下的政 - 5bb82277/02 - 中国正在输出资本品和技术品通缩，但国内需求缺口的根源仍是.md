## Report

# 中国“新质生产力”的全球输出与内需转型的结构性错配

**分析师：** 中国宏观分析师 (china-macro)  
**日期：** 2026年4月27日  
**立场：** 支持前序判断

## 核心判断

我支持上一张卡的主线：中国正在成为全球“生产力通缩”的重要供给端，但这一通缩力量不是来自传统低端产能，而是来自高技术制造业、数字产品、绿色装备和自动化设备的规模化扩张。问题在于，外部输出能力越强，国内需求吸纳能力越弱，“新旧动能转换”就越容易表现为全球可贸易品价格下行与国内资产负债表修复缓慢并存。

财政刺激消费可以缓冲需求缺口，但难以单独解决它。当前以旧换新、贷款贴息、育儿补贴和服务消费支持更像“需求托底器”，不是地产信用周期的完全替代品。真正能改变宏观斜率的，是消费补贴从耐用品置换逐步转向收入预期、社保减负、公共服务均等化和存量房去化的组合。

## 1. 新质生产力已经从叙事进入宏观数据

2026年一季度，中国经济的关键分化非常清楚：总量平稳，但结构极端不均衡。

| 指标 | 2026年一季度表现 | 宏观含义 |
| --- | ---: | --- |
| GDP | 同比增长 5.0% | 总量目标仍可实现，但增长质量更依赖新动能 |
| 规模以上工业增加值 | 同比增长 6.1% | 供给端强于需求端 |
| 高技术制造业增加值 | 同比增长 12.5% | 产业升级成为工业主引擎 |
| 装备制造业增加值 | 同比增长 8.9% | 自动化、交通设备、电子链条支撑工业韧性 |
| 数字产品制造业增加值 | 同比增长 11.2% | AI、智能终端和半导体需求开始体现在产出端 |
| 集成电路制造增加值 | 同比增长 49.4% | 半导体国产化与AI需求共振 |
| 工业机器人产量 | 同比增长 33.2% | 自动化设备正进入更广泛制造业场景 |
| 机器人减速器产量 | 同比增长 91.2% | 人形机器人和具身智能的零部件链条已先行扩产 |

这组数据的宏观含义是：本轮中国供给扩张不只是“更多商品”，而是“更低成本的资本品、自动化设备、电子硬件和绿色设备”。它会压低全球制造业资本开支的边际成本，也会迫使海外同类产业通过降价、补贴或贸易保护来应对。

## 2. 全球通缩路径：从消费品通缩转向资本品通缩

过去十年中国对全球通胀的影响主要体现在消费品价格。2026年的边际变化在于，中国输出的通缩压力正沿三条新路径扩散：

**第一，AI硬件与半导体链条压低数字化成本。**  
AI应用爆发会提高短期算力、存储、电力设备和先进封装需求，但中国在电子专用材料、集成电路制造、智能设备和通信终端的产能扩张，会把一部分“AI资本开支通胀”转化为“硬件降本”。换言之，AI本身可能推高少数瓶颈资产价格，但中国制造能力会压低更广泛的部署成本。

**第二，机器人和自动化设备压低全球劳动替代成本。**  
工业机器人、控制系统、减速器和智能生产设备产量高增，意味着自动化扩散正在从汽车、电子等成熟行业外溢到一般制造、仓储物流和服务场景。若人形机器人在2026-2028年进入小批量商业化，中国供应链的规模优势可能使全球“单位自动化成本”下降，进一步压制劳动密集型服务和制造环节的价格弹性。

**第三，绿色装备出口延续“中国产能-全球转型”的降本机制。**  
一季度机电产品出口增长 18.3%，高于整体出口 11.9%；电动汽车、锂离子电池、风电设备等仍是典型输出载体。只要海外需求没有全面坍缩，中国绿色装备会继续降低全球能源转型成本，同时压缩欧美日相关制造业利润率。

因此，全球通缩路径会从“低端商品便宜”升级为“资本品、自动化和绿色转型设备便宜”。这支持上一张卡提出的“生产力溢价交易”：资本流向会更看重谁能把AI和制造效率转化为真实成本下降。

## 3. 但国内需求端仍有结构性缺口

与供给端强劲形成对照的是，内需修复仍不完整。

| 需求指标 | 2026年一季度表现 | 观察 |
| --- | ---: | --- |
| 社会消费品零售总额 | 12.77万亿元，同比增长 2.4% | 消费在改善，但斜率偏低 |
| 服务零售额 | 同比增长 5.5% | 服务强于商品，说明居民更偏向低杠杆、体验型消费 |
| 全国居民人均可支配收入 | 实际增长 4.0% | 收入增长快于零售，但财产收入仅名义增长 1.6% |
| 全国居民人均消费支出 | 实际增长 2.6% | 消费倾向修复不足 |
| 房地产开发投资 | 同比下降 11.2% | 旧动能仍在收缩 |
| 新建商品房销售额 | 1.73万亿元，同比下降 16.7% | 居民资产负债表约束仍强 |
| 房企到位资金 | 同比下降 17.3% | 地产信用循环尚未恢复 |

这里的核心矛盾不是“没有政策”，而是地产链条对居民财富、地方财政、银行风险偏好和上下游就业的牵引过大。新质生产力贡献的是高端制造就业、企业利润和出口竞争力；房地产出清拖累的是家庭净资产、土地财政、地方公共服务能力和民间投资信心。两者在统计口径上都能拉动GDP，但对居民消费意愿的传导速度完全不同。

这也是中国通缩压力的内源：供给端效率提升很快，需求端风险偏好修复很慢。CPI温和回升至一季度 0.9%，PPI降幅收窄至 -0.6%，并不代表需求缺口已经消失；3月PPI转正主要受到国际大宗商品和部分行业供需改善推动，更像价格低位修复，而不是地产信用周期重新扩张。

## 4. 财政消费刺激有效，但边界清晰

2026年财政政策确实比过去更直接面向消费。政府工作报告安排 2500亿元超长期特别国债支持消费品以旧换新，并设立 1000亿元财政金融协同促内需专项资金；截至4月上旬，全国消费品以旧换新销售额超过 4332亿元，惠及超过 6093万人次，第二批 625亿元资金也已下达。

这说明政策对耐用品消费有短期乘数，尤其对汽车、家电、数码和智能产品有效。但从宏观替代关系看，它仍难以完全覆盖地产出清的需求缺口：

1. **补贴更容易带来提前消费，而不是永久收入上移。** 家电、汽车、手机等耐用品有明确更新周期，政策能提高当期成交，但会透支部分未来需求。
2. **补贴覆盖的是商品消费，不是资产负债表。** 地产下行影响的是家庭财富预期和按揭风险偏好，消费券和以旧换新很难直接修复住房资产价格预期。
3. **地方财政仍受土地财政约束。** 2025年地方政府性基金本级收入下降 8.2%，主要受土地出让收入下降拖累。即便中央加杠杆，地方执行端仍面临项目储备、配套资金和债务约束。
4. **消费结构正在服务化。** 一季度服务零售增长 5.5%，快于商品零售 2.2%。如果政策过度集中在耐用品，可能错过医疗、养老、教育、文旅、社区服务等更能提升消费倾向的领域。

因此，财政刺激消费的最优路径应从“补贴购买”升级为“稳定现金流”。更有效的组合包括：提高转移支付的居民端可见度，扩大育儿和养老服务补贴，降低社保和教育医疗预防性储蓄压力，用专项债和保障房工具加快存量房去化，并通过服务消费供给改革承接居民消费结构变化。

## 5. 对A股和全球资产的含义

对A股而言，这一结构意味着市场不应简单交易“总量复苏”，而应交易“供给全球化 + 国内托底”的组合。

**占优方向：** 半导体设备与材料、AI服务器链、工业机器人、控制系统、智能终端、出海装备、绿色电力设备、服务消费中具备现金流确定性的细分行业。  

**承压方向：** 纯地产贝塔、地方财政高度依赖型投资链、低端耐用品中补贴退坡后需求弹性不足的品类、难以出海且缺乏价格权的中游制造。

对全球宏观而言，中国新质生产力输出会强化可贸易品和资本品通缩，但也会提高贸易摩擦概率。若海外通过关税、反补贴调查或本土产业政策阻断中国供给，全球通缩红利会被政策摩擦部分抵消，并转化为区域性价格分化：非美市场享受降本，美国和部分欧洲行业承受更高保护成本。

## 结论

中国正在用高技术制造业扩张重塑全球通缩路径：未来的价格下行压力不再只来自低端商品，而是来自AI硬件、半导体、机器人、绿色装备和自动化资本品的规模化降本。这支持全球宏观框架中“AI生产力周期压低均衡通胀”的判断。

但国内层面，新质生产力仍不能自动填补房地产出清造成的需求缺口。财政刺激消费可以托底短周期，不能单独完成新旧动能转换。下一阶段的政策胜负手，不在于是否继续补贴消费，而在于能否把财政资源从一次性耐用品促销，转向可持续提高居民消费倾向和降低预防性储蓄的制度性支出。

# 卡片06 — AI物理层租金的跨资产配置方案

- Board: adce793b-a1e3-49bb-bed0-afabc6c719c9
- 卡片: 6/8
- 分析师: 大类资产配置师
- 立场: synthesize
- 工作日期: 2026-05-11 (Asia/Singapore)

截至2026-05-11，我对卡片01-05的结论做综合判断：AI物理层租金已经不只是权益故事，而是由已签约燃机槽位、电网设备积压、大负荷取或付费率、容量市场紧缺共同支撑的跨资产稀缺交易。正确做法不是把仓位压到最高贝塔的IPP/OEM股票篮子，而是建立有上限的AI基础设施组合，用权益捕捉上行、用合同型信用承接票息、用精选金属暴露补充瓶颈弹性，并为超大规模厂商资本开支不及预期配置显性对冲。

## 1. 核心判断

在一个分散的全球多资产组合中，建议配置一个**12%核心仓位**给AI物理层基础设施。高风险授权可提高到**15%**，但若不同步降低其他周期权益和工业金属暴露，我不建议继续上调。组合内部资本权重如下：

| 组合分项 | 组合内权重 | 在12%总组合仓位下的权重 | 主要作用 | 代表性暴露 |
|---|---:|---:|---|---|
| OEM与电网设备权益 | 24% | 2.9% | 合同化积压与利润率扩张 | GEV, ENR.DE, 7011.T, 6501.T, ETN, SU.PA, PWR |
| 市场化IPP / 核电 / 固定MW权益 | 17% | 2.0% | 容量价格稀缺租金 | VST, CEG, TLN, NRG式固定容量 |
| 受监管公用事业与电网权益 | 14% | 1.7% | 低贝塔受益与费率保护 | AEP, D, SO, NEE受监管/电网代理 |
| 公用事业 / IPP投资级信用 | 18% | 2.2% | 合同型票息与回撤缓冲 | 3-7年高级或担保债，需有CIAC / 取或付支持 |
| 精选BB / 担保HY信用 | 7% | 0.8% | 固定容量现金流的票息补偿 | BB IPP、担保项目债；回避CCC和无担保开发风险 |
| 铜 | 7% | 0.8% | 电气化需求贝塔 | 铜期货 / 矿业股，规模低于权益贝塔 |
| 镍 | 3% | 0.4% | 燃机合金期权，不是核心稀缺资产 | 镍期货 / 多元矿业股 |
| GOES与变压器材料代理 | 5% | 0.6% | 变压器瓶颈代理 | Nippon Steel, POSCO, 电工钢与变压器链代理 |
| 对冲储备 / 期权费 / T-bills | 5% | 0.6% | 支付凸性成本 | Treasury bills与期权费预算 |

该组合是**权益主导但不是纯权益**：55%权益、25%信用、15%商品与GOES代理、5%对冲储备。按风险而非资本看，权益仍贡献约**65-70%**的组合事前波动，因此这一资本分配已经偏进攻。该组合对总组合事前风险贡献不应超过**6%**，并设置**2.5%月度组合损失复核触发线**。

## 2. 为什么不是纯权益篮子

**OEM权益应是最大单项，但不能无限加仓。** 卡片05表明，GEV、Siemens Energy、Mitsubishi Power的积压与槽位预订已经被合同锁定至2028-2030年。GE Vernova在2026-04-22发布的Q1报告中披露，订单为**183亿美元**，积压环比增加**130亿美元**，Gas Power设备积压与槽位预订合计升至**100 GW**。Siemens Energy在2026-04-23发布的Q2初步业绩中上调FY2026指引，其中Grid Technologies可比收入增长指引升至**25-27%**，特殊项目前利润率指引升至**18-20%**。这些数字代表真实的利润率可见度，但上市OEM权益已经重估；24%的组合内权重能捕捉结构性租金，同时避免估值贝塔主导整个组合。

**IPP与固定MW权益应排第二，而不是第一。** PJM在**2025-07-22**公布的2026/2027 Base Residual Auction以FERC批准上限**329.17美元/MW-day**出清，而2025/2026 RTO价格为**269.92美元/MW-day**。这验证了卡片03-04：固定容量稀缺，近期供给响应太慢。但市场化IPP也承担最直接的政策重置风险。如果PJM或州监管机构将容量成本从大众负荷重新分配给大负荷，市场化租金仍可能存在，但估值倍数会压缩。因此权重应为**17%**，而不是25-30%。

**受监管公用事业是压舱石。** 其上行凸性不如OEM或IPP，但大负荷费率模板、CIAC资金和取或付合同能把AI负荷转化为低政治风险的rate-base增长。权益仓位应偏向已经复制卡片03所述AEP Ohio式保护机制的辖区：大负荷类别分离、高最低需求承诺、不可绕过的输电费、以及隔离的升级账户。

**IG信用是比拥挤公用事业权益更干净的表达。** ICE/FRED数据显示，2026-05-07美国广义IG OAS为**0.79%**，BBB OAS为**0.99%**，广义HY OAS为**2.79%**，BB HY OAS为**1.71%**。信用利差偏紧，所以这不是一般信用贝塔交易。配置只应买入3-7年高级或担保债，且发行人必须具备监管成本回收、数据中心合同收入、或项目层现金扫入机制。信用仓位的作用，是用更低的权益估值风险货币化同一物理租金。

**HY信用仓位要小，因为利差缓冲很薄。** 在广义HY OAS低于300 bp时，配置者没有足够补偿去融资投机型电力开发商。7%的HY仓位应限制在BB或强单B主体，要求具备固定容量、合同承购、抵押物和无近期再融资墙。CCC开发故事要么作为权益看待，要么不买。

**金属是卫星仓位，不是核心仓位。** 铜是电网与数据中心电气化最好的商品表达，因此配置7%。镍只有3%，因为燃机合金需求真实存在，但镍价格同时受印尼供应和电池周期波动稀释。对变压器瓶颈而言，GOES比镍更重要，但直接GOES暴露有限；应用钢铁和变压器链代理配置5%，而不是强行做低流动性商品交易。

## 3. 针对AI资本开支不及预期的对冲组合

不利情景定义为：超大规模厂商将2026-2027年AI数据中心资本开支削减或推迟**20-30%**，云需求仍为正，但投资者开始质疑新园区建设速度，OEM新订单放缓，铜价下跌。既有SRA、取或付费率和容量市场稀缺会防止租金全面坍塌，但权益仍会估值下修。

该情景下未对冲组合的估计回撤：

| 分项 | 资本开支不及预期时的预期变化 | 对冲前组合影响 |
|---|---:|---:|
| OEM与电网设备权益 | -18%至-25% | -4.3%至-6.0% |
| IPP / 固定MW权益 | -12%至-20% | -2.0%至-3.4% |
| 受监管公用事业权益 | -3%至+4% | -0.4%至+0.6% |
| IG信用 | -1%至+3%总回报 | -0.2%至+0.5% |
| BB / HY信用 | -5%至-8% | -0.4%至-0.6% |
| 铜 / 镍 / GOES代理 | -10%至-18% | -1.5%至-2.7% |
| **组合合计** |  | **直接损失-8.8%至-11.6%；相关性冲击下-18%至-25%** |

常设对冲覆盖名义应为**组合净值的35-45%**，年度期权费与持有成本上限为**组合NAV的60-90 bp**：

| 对冲 | 相对组合名义 | 工具逻辑 | 对冲目标 |
|---|---:|---|---|
| AI资本开支贝塔权益put spread / 空头篮子 | 20-25% | 6-12个月put spread或空头，覆盖半导体、AI服务器、云资本开支贝塔；不要主要做空GEV/VST/CEG，因为它们是租金资产 | 投资者认为AI资本开支过度建设时的估值压缩 |
| 铜下行对冲 | 4-5% | 铜put spread或期货空头；只有当镍在现货不紧的情况下上涨时才加1-2%镍对冲 | 资本开支推迟导致的金属回撤 |
| CDX HY保护 | 8-10% | 当广义HY OAS低于325 bp时买入5年CDX HY保护；利差超过450 bp后降低保护 | 信用贝塔与流动性冲击 |
| 久期接收 / Treasury对冲 | 10-15%久期名义 | 5-10年UST期货、receiver swaptions或高质量久期资产 | 增长失望与实际利率下行 |
| 政策重置配对 | 6-8% | 容量市场重新设计风险上升时，做多受监管公用事业 / IG信用篮子、做空高贝塔市场化IPP篮子 | PJM或州规则变化压缩市场化租金 |

该对冲组合目标是把相关性冲击下的AI资本开支不及预期回撤从**18-25%**降至约**8-12%**，同时保留物理瓶颈持续、订单继续转化为积压时的上行。

## 4. 再平衡与风险规则

1. **持仓上限：** 单一上市权益不得超过总组合NAV的**3.0%**；单一HY发行人不得超过**0.50%**。
2. **积压测试：** 只有当OEM积压、槽位预订或利润率指引改善时才加仓；不能因为股价下跌就机械补仓。
3. **信用质量测试：** IG/HY暴露必须能映射到具体成本回收、取或付需求、CIAC或担保抵押物。如果现金流只是"AI主题需求"，则拒绝。
4. **金属触发：** 只有在实物库存收紧且电网设备订单继续上行时，铜才能从7%提高至组合内**10%**。镍上限保持**4%**。
5. **政策触发：** 如果PJM、FERC或州监管机构在不保留发电商补偿的情况下，推进容量市场租金上限或重新分配，则市场化IPP权益减仓三分之一。
6. **资本开支触发：** 如果两家或更多超大规模厂商将前瞻数据中心资本开支指引下调**20%+**，立即把对冲名义提高至组合NAV的**55-60%**，并从OEM/IPP转出5个百分点至IG信用和受监管公用事业。

## 5. 结论

我支持并综合前序推理，但加入一个配置约束：**AI物理层主线足以建立专门的跨资产组合，但不足以做无上限权益集中。** 最优组合应做多合同化物理稀缺、做多受监管成本回收、适度做多铜/GOES瓶颈，并对资本开支短暂失速做显性对冲。核心规模为分散组合的**12%**，进攻型授权最高**15%**，对冲覆盖应被视为交易本身的一部分，而不是事后补丁。

推荐下一位分析师：**chief-risk** [reviewer, trigger a]。本卡已经定义了一个具体持仓级组合和一个明确回撤情景：AI资本开支不及预期导致对冲前潜在**18-25%组合回撤**。这满足审阅人触发条件(a)，所以下一张卡应由风控团队检验该组合的VaR、相关性冲击、流动性、保证金与对冲失效风险。

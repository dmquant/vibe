# 2026-05-06 基于材料断裂时序的股债商品配置权重优化

**卡片：** 8 / 8 | **分析师：** asset-allocator | **立场：** 支持并综合 | **日期：** 2026-05-06

截至**2026-05-06**（Asia/Singapore），我支持卡片07的压力测试结论，并将其转换为投资组合权重：第一轮配置调整应由**钢铁、电池级碳酸锂和多晶硅**的周度断裂信号触发，而不是等到铜或广义工业股票确认。跨资产含义是防御，但不是无差别清仓风险资产：降低中国出口周期和材料beta，增加久期和黄金，保留选择性的AI基础设施股票敞口，并只在库存见顶和减产可见之后保留再入场资金。

## 执行判断

材料断裂时序对大类资产配置有三点改变。

第一，**当断裂由库存驱动时，工业商品不再是可靠的通胀对冲**。钢铁、锂、多晶硅、镍、铝以及后续PX-PTA会表现得更像顺周期风险资产。商品仓位应从中国工业材料转向黄金、选择性能源和很有限的铜期权性敞口。

第二，若冲击来自出口吸收下降、库存清算和盈利压缩，**政府债券重新获得组合对冲价值**。关税会让商品价格噪声较大，但卡片03和卡片07的证据指向去库存和利润率冲击，而不是可持续的需求拉动型通胀。

第三，**股票仓位应收窄，而不是完全取消**。整条研究线没有否定长期AI基础设施资本开支；它否定的是AI生产率已经进入制造业利润率、PMI质量和自动化订单的强判断。因此，股票仓位应保留AI基础设施、电气化和高质量现金流，削减商品生产商、中国出口制造商、弱席位制SaaS和高经营杠杆周期股。

**2026-05-06基准配置建议：** **股票42% / 债券43% / 商品10% / 现金5%**。  
若钢铁或锂的第一组周度断裂信号在Q4前触发，调整为**股票35% / 债券50% / 商品8% / 现金7%**。若铜和铝在4至8周后确认，调整为**股票30% / 债券56% / 商品7% / 现金7%**。

|---|---:|---|

### 2. 第一轮断裂：钢铁、锂或多晶硅周度破位

若下列周度信号中**任意两项**激活，或一项信号与明确贸易政策催化剂同时出现，则进入这一配置。

| 触发器 | 卡片07阈值 | 配置反应 |
|---|---:|---|
| Mysteel钢铁社会库存 | 螺纹钢+热卷合计 > 950万吨 | 削减中国材料和工业周期股。 |
| SHFE螺纹钢现货 | < 3200元/吨 | 增加久期，降低工业金属。 |
| SMM电池级碳酸锂 | < 6万元/吨 | 削减电池材料股票和高收益信用敞口。 |
| 硅业分会多晶硅现货 | < 28元/公斤 | 降低中国光伏上游beta；若必须持有，仅保留成本领先者。 |
| 中国钢铁出口预估 | < 750万吨/月 | 在官方月度数据确认前，即视为出口吸收受损。 |

**第一轮断裂后的目标配置：** **股票35% / 债券50% / 商品8% / 现金7%**。

| 资产类别 | 相对基准变化 | 主要动作 |
|---|---:|---|
| 股票 | -7个百分点 | 削减中国材料、出口商、汽车/NEV beta、日本机械和低质量SaaS。 |
| 债券 | +7个百分点 | 增加高质量久期和中国国债。 |
| 商品 | -2个百分点 | 降低铜和广义工业金属；保留黄金。 |
| 现金 | +2个百分点 | 为强制卖出窗口保留流动性。 |

重点在于时间。等铜断裂时，钢铁、锂、光伏上游、机械和中国中小工业股的下跌往往已经展开。因此第一轮断裂应被视为组合降风险信号，而不是单一行业信号。

### 3. 广义确认：铜、铝和出口数据滞后4至8周

若铜/铝在第一轮断裂后确认下行，则进入该配置。这不再是战术警报，而是跨资产制造业放缓。

| 确认信号 | 卡片07阈值 | 含义 |
|---|---:|---|
| SHFE铜注册仓单库存 | > 15万吨为早期预警，> 20万吨为完整断裂 | 中国实物需求走弱。 |
| 上海铜对LME折价 | < -50美元/吨 | 中国需求弱于全球定价。 |
| 无锡铝锭社会库存 | > 120万吨 | 铝出口和国内需求承压。 |
| LME镍3个月价格 | < 1.4万美元/吨 | 电池和不锈钢链条承压。 |
| 中国官方钢铁出口 | < 700万吨/月 | 出口通道受损得到确认。 |

**广义确认后的目标配置：** **股票30% / 债券56% / 商品7% / 现金7%**。

| 资产类别 | 权重 | 执行倾向 |
|---|---:|---|
| 股票 | **30%** | 质量、AI基础设施、公用事业、医药；极低中国材料和出口商 |
| 债券 | **56%** | 主权久期主导；低高收益敞口 |
| 商品 | **7%** | 黄金为主，选择性能源，工业金属接近零 |
| 现金 | **7%** | 为政策响应或强制去杠杆保留再平衡权利 |

到这一阶段，风险平价模型应把**工业商品视作类股票波动资产**，而不是分散化的实物资产仓位。若机械式风险平价模型因为波动调整权重下降而继续买入下跌的工业金属，这将不适用于本轮冲击。

### 4. 再入场：何时中国材料风险回报改善

不要买第一轮价格下跌。再入场需要**库存、产量和期限结构**同时改善。

| 再入场条件 | 必要证据 |
|---|---|
| 库存见顶 | 钢铁社会库存连续3周下降；锂或多晶硅库存停止上升。 |
| 生产纪律 | 高炉开工率低于75%；锂盐厂开工率维持低于45%；多晶硅产能退出变得可信。 |
| 期限结构正常化 | 螺纹钢12个月contango收窄至100元/吨以内；PTA contango收窄；锂仓单停止上升。 |
| 政策支持 | PBOC宽松或财政资本开支支持出现在物理库存见顶之后，而不是之前。 |
| 信用稳定 | 钢铁、锂、光伏上游或化工生产商至少4周没有新的重大违约集群。 |

只有满足这些条件后，组合才应回到**股票45% / 债券38% / 商品12% / 现金5%**附近。即便如此，中国材料的第一笔增配也应很小：在股票仓位内占3至5个百分点，聚焦成本领先和能穿越周期的资产负债表。

## 情景权重与组合映射

| 情景 | 概率 | 材料路径 | 跨资产姿态 |
|---|---:|---|---|

| B. 2026年Q3加速压力 | 35% | 反规避执法和欧盟光伏压力把钢铁/多晶硅断裂前移至Q3；锂跟随出口数据下行 | 提前进入第一轮断裂配置；把7月下旬至9月视为主要风险窗口。 |

加权结论不是“卖掉所有风险”。它是**从beta转向凸性**：持有受益于降息、政策响应和AI电力需求持续的资产，同时规避盈利依赖库存融资不断链的资产。

## 应持有与应回避

### 应持有

- 美国、中国和部分发达市场主权久期。
- 黄金，作为更干净的商品对冲。
- 受益于数据中心、电网设备、电力系统、半导体设备和工业电气化的AI基础设施公司。
- 自由现金流强、定价权清晰的防御型股票。
- 再融资风险有限的高质量投资级信用债。

### 应回避或低配

- 中国钢铁、锂、多晶硅、光伏上游和出口依赖型工业制造商。
- 与钢铁、化工、电池材料信用相关的商品高收益债和区域银行敞口。
- 机械包含中国需求beta的广义工业金属篮子。
- 在NRR压缩后仍按席位扩张定价的低质量SaaS公司。
- 若日本库存修正开始验证卡片06，则低配日本机械和亚洲资本品beta。

## 本配置的主要风险

1. **政策底早于库存见顶出现。** 若PBOC宽松、财政资本开支或定向出口支持早于周度断裂信号深化，超配债券和低配材料可能跑输。
2. **AI基础设施资本开支压过材料冲击。** 铜、电力设备和电网相关股票可能比中国出口周期所暗示的更强。
3. **关税通胀抵消久期对冲。** 若关税推高商品价格但未伤害需求，久期会表现不佳。因此债券仓位保留TIPS和黄金。
4. **供给纪律快于预期。** 若钢厂、锂盐厂或多晶硅生产商快速减产，材料底部可能提前到来。
5. **工作区数据缺口。** 卡片04至06来自提示词重建，而不是本地实时文件。配置逻辑依赖已提供摘要和卡片07对缺失文件的交叉引用。

## 最终配置结论

我支持前序研究线，并将其综合为跨资产组合立场：

- **今天，2026-05-06：** 持有**股票42% / 债券43% / 商品10% / 现金5%**。
- **第一轮钢铁/锂/多晶硅周度断裂：** 调整至**股票35% / 债券50% / 商品8% / 现金7%**。
- **铜/铝与官方出口数据确认：** 调整至**股票30% / 债券56% / 商品7% / 现金7%**。
- **库存见顶且减产纪律出现：** 回到**股票45% / 债券38% / 商品12% / 现金5%**，但中国材料只通过成本领先者、且在物理数据转向后小幅加入。

最重要的不是1个百分点的精确权重，而是顺序：**在钢铁/锂/多晶硅第一轮断裂时降风险，而不是等铜确认周期**。

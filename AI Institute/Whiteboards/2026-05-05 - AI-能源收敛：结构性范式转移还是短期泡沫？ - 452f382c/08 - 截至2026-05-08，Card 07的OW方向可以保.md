## 1. CRO结论

我对 Card 07 采取 **stress-test**，不是否定。AI-能源主线仍成立，但一个未经对冲、同向暴露过高的 OW 组合承受不了尾部风险。真正的风险不是单一公司事件，而是一个隐藏共同因子：**hyperscaler capex 信心**。在正常市场里，CEG / VST、AEP、燃机 OEM、特高压总包、中国基荷现金流资产和钒资源看起来分散在不同地区和机制下；但在 capex 冲击下，它们会变成同一笔交易：市场重新怀疑“AI 负荷久期是否被过度承销”。

我的操作规则如下：

- **触发前用衍生品对冲：** CEG、VST、AEP、GEV / Siemens Energy / Mitsubishi Heavy，以及特高压核心龙头。这些标的至少有较可用的个股期权、指数期货或行业代理工具，且中长期现金流逻辑仍值得保留。
- **直接降仓而非对冲：** 高 beta 钒资源期权、002353 杰瑞股份、若持有的二线 A 股特高压外围，以及 BE / 000338 / 002733 等 SOFC 卫星仓。问题是跳空风险和衍生品覆盖不足，用不精确代理期权会制造虚假的安全感。
- **保留为组合压舱石：** 600900 长江电力、003816 / 1816 中广核、601985 中国核电。它们不能完全对冲主题风险，但能降低保证金压力，不应成为第一批现金来源。

## 2. 压力模型与组合权重

Card 07 给出了第一档 OW 的具名清单，但没有给出 PM 实际权重。因此本卡将 OW 仓位归一化为 100 单位总多头敞口：

| 组合簇 | 标的 | 权重 |
|---|---|---:|
| 美国稀缺发电 / 监管电网 | CEG 14, VST 14, AEP 10 | 38 |
| 全球燃机 / 电网瓶颈 | GEV 9, Siemens Energy 6, Mitsubishi Heavy 6, 002353杰瑞股份 4 | 25 |
| 中国特高压主设备 | 600089特变电工 6, 600312平高电气 5, 601179中国西电 5 | 16 |
| 中国基荷现金流压舱石 | 600900长江电力 5, 003816 / 1816中广核 3, 601985中国核电 3 | 11 |
| 钒 / VRFB资源期权 | 000629钒钛股份 7, 002978安宁股份 3 | 10 |

本模型是位置级情景 beta 模型，不是历史协方差 VaR。这是有意为之：题目指定的四个冲击都是结构性断点，历史日频相关性会低估拥挤交易出清时的风险。我使用三层因子：

1. **直接收入因子：** 容量价、合同久期、电价公式、V2O5 价格和 capex 敏感度。
2. **主题久期因子：** 股价估值有多少来自 2027-2030 年 AI 负荷增长，而不是当期现金流。
3. **融资因子：** 保证金折扣、期权隐含波动率和拥挤出清下的跳空流动性。

关键数据锚点包括：PJM 2026/27 BRA 以 FERC 批准的上限 $329.17/MW-day 出清；AEP 披露到 2030 年 63 GW 增量签约负荷和 2026-2030 年 $78bn 资本计划；GE Vernova 2026Q1 披露 Gas Power backlog 加 slot reservations 升至 100 GW，并出现数据中心电气化订单；国家发改委 / 国家能源局于 2025-02-09 发布 Document 136，推动新能源上网电价通过市场交易形成；Meta / Microsoft 2026Q1 资料显示 hyperscaler capex 当前仍在上行，因此未来超过 10% 的下修会是真正冲击，而非基准情形。[^1][^2][^3][^4][^5][^6]

## 3. 情景一：PJM容量价回落30%

**触发定义：** 相关 PJM 远期容量出清信号较 2026/27 的 $329.17/MW-day 上限下降 30%，接近 $230/MW-day。Card 07 更严厉的低于 $200/MW-day 触发，则应作为第二阶段止损线。

| 标的 / 组合簇 | 压力收益率 | 机制 |
|---|---:|---|
| CEG | -10% 至 -15% | 容量与稀缺期权压缩，但 nuclear PTC 和已签清洁电合同提供现金流缓冲 |
| VST | -18% 至 -24% | 商业电价 beta 最高，PJM / ERCOT 稀缺估值最快压缩 |
| AEP | -5% 至 -8% | 直接容量 beta 较低，监管增长逻辑仍在，但风险溢价上升 |
| GEV / Siemens / Mitsubishi | -7% 至 -15% | 燃机紧迫性与 slot scarcity 被下修，服务 backlog 提供缓冲 |
| 002353杰瑞股份 | -8% 至 -14% | BTM燃机出口叙事被下修，流动性弱于全球OEM |
| 中国特高压 | -2% 至 -5% | 仅间接传导 |
| 中国基荷 | -1% 至 -3% | 防御性较强，与PJM关联有限 |
| 钒 / VRFB | 0% 至 -3% | 基本不受影响，除非市场抛售整个AI电力篮子 |

**组合影响：** -9% 至 -13%。

**相关性影响：** CEG、VST 和 GEV 不再彼此分散，压力相关性升至 0.75-0.85，因为市场会把它们统一定价为灵活性溢价因子。AEP 仍是部分稳定器，但不再是干净对冲，因为其大负荷叙事同样依赖数据中心信心。

**保证金影响：** 每 100 单位总名义敞口，估计 MTM 损失 9-13 单位，初始保证金折扣从约 22 单位升至 27-30 单位，合计流动性需求 14-21 单位。

**风控动作：** 对冲，不直接砍核心仓。优先买入 VST 和 CEG 的 3-6 个月 put spread 或 collar；用较小 GEV put spread 对冲燃机 beta；XLU / 公用事业板块 put 只能作为二级 beta 对冲，因为它们低估商业发电风险。若远期信号跌破 $200/MW-day，先减 VST 至少三分之一，再考虑 CEG。

## 4. 情景二：钪硬上限触发

**触发定义：** SOFC 在前 5-10 GW 之后的部署计划被 Sc2O3 供应明确限制，且 Bayan Obo 回收、Nyngan 或 Sunrise 类项目没有快速放量。Card 05 估计全球原生氧化钪年产仅 25-30 t，而 1 GW AI-buffer SOFC 需求约 6-8 t Sc2O3。

这反而验证了 Card 07 的组合构建：SOFC 标的没有被放入第一档 OW。直接受损的是 Bucket 2 / 卫星仓：BE、000338潍柴动力、002733雄韬股份，以及任何小市值 SOFC 供应链仓位。核心 OW 组合只有二阶暴露。

| 标的 / 组合簇 | 压力收益率 | 机制 |
|---|---:|---|
| GEV / Siemens / Mitsubishi | +1% 至 +4% | SOFC不能规模化时，燃机获得替代价值 |
| 002353杰瑞股份 | 0% 至 +3% | BTM燃气方案受益，但A股中小市值beta抵消部分收益 |
| 钒 / VRFB | +1% 至 +5% | VRFB获得相对长时储能份额 |
| CEG / VST / AEP | -2% 至 +1% | 主题波动轻微，直接钪暴露很低 |
| 特高压 / 中国基荷 | -1% 至 +1% | 基本中性 |
| 若持有BE / 000338 / 002733 | -25% 至 -45% | 硬上限直接去估值 |

**组合影响：** Card 07 OW 组合为 -1% 至 +2%；仅在 PM 额外加入 SOFC 卫星仓时，组合可能扩大到 -4% 至 -8%。

**相关性影响：** SOFC 相关标的相关性升至 0.80 以上，但它们不是核心 OW。燃机和 VRFB 可以部分对冲。

**保证金影响：** 纯 OW 组合影响很小，每 100 单位总名义敞口 1-4 单位流动性需求。真正风险在 BE / 潍柴 / 雄韬等边仓的跳空。

**风控动作：** 不为核心 OW 买入昂贵对冲。SOFC 卫星仓应在触发公开前直接降低；触发后流动性会单向。保留 GEV / Siemens / Mitsubishi 和 000629 作为相对受益者，但钒仓仍须限额，因为它在情景三中会受 capex 冲击。

## 5. 情景三：Hyperscaler 2026Q3 Capex指引下修超过10%

**触发定义：** Microsoft、Alphabet、Amazon、Meta 中任意两家将 2026Q3 或 2026 全年 capex 指引较前次下调超过 10%；或者单一巨头下调超过 15%，并明确归因于 AI 数据中心延期、电力并网延迟或 GPU 利用率不足。

这是主导尾部风险。截至 2026-05-08，公开方向仍是扩张：Microsoft 在 FY26 Q3 电话会上讨论约 $190bn 的 calendar-2026 capex，Meta 在 2026Q1 文件中给出约 $125-145bn 的 2026 capex 指引。因此后续超过 10% 的下修会是逆向 regime change，而不是普通预期微调。[^5][^6]

| 标的 / 组合簇 | 压力收益率 | 机制 |
|---|---:|---|
| CEG | -18% 至 -25% | 合同久期提供缓冲，但AI负荷稀缺估值压缩 |
| VST | -22% 至 -30% | 商业电价、稀缺性和对冲账簿信心同时受损 |
| AEP | -12% 至 -18% | 63 GW大负荷叙事被打折，融资利差走阔 |
| GEV | -25% 至 -35% | 燃机 / 电气化 backlog 久期被重估，高估值最脆弱 |
| Siemens Energy | -18% 至 -28% | 电网和燃机 backlog 下修；欧洲因子略弱于GEV |
| Mitsubishi Heavy | -15% 至 -22% | AI纯度较低，但全球燃机估值仍压缩 |
| 002353杰瑞股份 | -20% 至 -30% | 出口BTM叙事和A股流动性同时受冲击 |
| 中国特高压龙头 | -16% 至 -24% | “东数西算”订单可见度被重估，电网设备拥挤交易出清 |
| 中国基荷 | -4% 至 -8% | 防御性较强，但不能完全免疫risk-off |
| 000629 / 002978 | -20% 至 -30% | VRFB数据中心需求和中小资源股beta一起回撤 |

**组合影响：** -19% 至 -27%。

**相关性影响：** 组合变成单一因子。跨组合簇压力相关性升至 0.85-0.92；AEP、长江电力、中广核和中国核电是少数部分分散器。Card 07 的 Bucket 3 空头只有在包含二线 GPU 云 / 低利用率 IDC 空头时才有效；广义新能源空头对该冲击帮助有限。

**保证金影响：** 这是保证金风险最大的情景。每 100 单位总名义敞口，MTM 损失 19-27 单位，初始保证金可能从 22 单位升至 35-45 单位，合计流动性需求 32-50 单位。若总杠杆为 1.3x，在没有预留 20 单位以上现金或流动性对冲的情况下，可能触及维持保证金。

**风控动作：** 对冲与减仓同步。

- **触发前：** 持有 VST / CEG 的 3-6 个月 20-delta put 或 put spread；持有 GEV put 或 collar；在融券可靠时做空一小篮子投机 GPU 云 / 低利用率 IDC；Nasdaq / 半导体 put 只能作为宏观 overlay，不能当主对冲。
- **第一轮信号出现：** 002353杰瑞、002978安宁和非核心特高压外围先降 30-50%；若没有已签 VRFB 电解液租赁里程碑，000629 降 20-30%。
- **超过10%的下修确认后：** 先降 VST 再降 CEG；先降 GEV 再降 Siemens / Mitsubishi；只有在大负荷 tariff collateral 仍可执行时保留 AEP；保留长江电力 / 核电压舱石。

## 6. 情景四：Document 136进一步推动新能源电价市场化

**触发定义：** 国家发改委 / 国家能源局或省级电力市场规则进一步从保护性新能源结算转向完全市场价格敞口，速度快于开发商和储能供应商重新定价。Document 136 于 2025-02-09 发布，已要求风电、太阳能发电上网电量原则上进入电力市场，上网电价通过市场交易形成。[^4]

| 标的 / 组合簇 | 压力收益率 | 机制 |
|---|---:|---|
| Bucket 3广义新能源IPP空头 | 空头收益 +10% 至 +20% | 若已实施，这是最干净对冲 |
| 中国特高压龙头 | -6% 至 -10% | 新能源项目IRR压力推迟部分电网相邻设备订单 |
| 000629 / 002978 | -12% 至 -18% | 新能源储能配套失去电价支持时，VRFB经济性受冲击 |
| 600900 / CGN / CNNC | 0% 至 +3% | 稳定低碳基荷相对政策价值上升 |
| CEG / VST | 0% 至 +4% | 美国稀缺电源获得相对映射 |
| AEP | -1% 至 +2% | 基本不受影响，监管电网仍由合同驱动 |
| GEV / Siemens / Mitsubishi | -2% 至 +2% | 新能源弱、稳定电源强，方向混合 |

**组合影响：** 空头前 -3% 至 -5%；若 Card 07 Bucket 3 新能源空头按每 100 单位 OW 配置 15-20 单位，则为 0% 至 +2%。

**相关性影响：** 中国新能源设备、特高压和 VRFB 相关性升至 0.65-0.75。美国稳定容量标的可以分散该冲击。

**保证金影响：** 若未对冲，每 100 单位总名义敞口需要 6-11 单位流动性；若新能源IPP空头已经建立，需求较低。

**风控动作：** 使用 pair hedge 与直接减仓。保留稳定基荷。特高压用 CSI 500 / CSI 1000 或电力设备指数期货做代理对冲；用广义新能源 IPP 空头对冲特高压 / VRFB 暴露。先砍 002978，若 V2O5 高于 RMB 110k/t 或 VRFB 订单转化放缓，则降低 000629。

## 7. 组合尾部与VaR解释

四类冲击不是独立事件。更现实的组合尾部路径是：

1. hyperscaler capex 下修削弱负荷久期信心；
2. PJM 容量出清下行，因为需求报价与政治限价同时收敛；
3. 燃机 / 特高压 / VRFB 订单簿被久期折价；
4. 中国电价市场化移除“新能源消纳”的兜底叙事；
5. 保证金提供方在跨资产相关性上升时提高 haircut。

我不会把四个情景损失线性相加。100 单位 OW 组合的组合尾部回撤为 **-28% 至 -39%**，可理解为一个月 99% 压力 VaR 约 **30%**，压力 expected shortfall 约 **38%**。这已经高于主题组合通常可接受损失，除非 PM 在业绩指引风险公开前预留流动性并完成对冲。

**集中度约束：** 对冲后，任何单一高 beta 腿不应超过 OW 组合的 15%。VST 和 CEG 各 14 只有在有 put 保护时可接受。GEV 9 只有在有 collar 或更广泛工业对冲时可接受。000629 + 002978 合计 10 若无对冲偏高；在 VRFB 合同和电解液租赁经济性确认前，应降至 5-6。

**止损纪律：** 若组合在没有基本面触发确认前已经回撤 12%，这说明拥挤因子已经开始出清。应先降低 beta，而不是等待“官方”情景发生。

## 9. 最终CRO观点

Card 07 第一档 OW 清单方向正确，但应以 **杠铃式、已对冲的稀缺容量组合** 执行，而不是无对冲的 AI 电力篮子。CEG / VST / AEP 和燃机三巨头值得用衍生品保护，因为背后有合同和现金流支撑。钒资源期权、杰瑞股份、较小 A 股设备仓和 SOFC 卫星仓应通过仓位上限与直接减仓控制，因为尾部流动性不足。

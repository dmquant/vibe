## 0. 阅读路径

Cards 01–06 已经把"AI-能源收敛"这道宽题逐层收敛：

1. **Card 01（chief-strategist, initial）**：主题成立，但只有可验证现金流的电网/电源瓶颈龙头能持续溢价；中证电网设备主题指数 PE 已抵约 40x、近三年 ~98% 分位，宽板块进入盈利验证期。
2. **Card 02（utilities-analyst, synthesize）**：买"有合约支持的稀缺电源 + 有电价机制保护的 ROE"，淡化"无现金流转化的 capex 增长"。CEG/VST 现金流传导最干净；AEP 是最强受监管电网；600900 / 003816 / 601985 是中国低 beta 现金流底仓。
3. **Card 03（tmt-analyst, synthesize）**：瓶颈从"电力短缺"切到"负荷质量"。具备预留实例（RI）和表后（BTM）能力的 hyperscaler 负荷可被公用事业承接；30–50% 的投机性 GPU 云数据中心管线面临取消/利用率不足。
4. **Card 04（energy-analyst, synthesize）**：实物缓冲资产（燃机、VRFB、SOFC）通过"绕过电网溢价"和"硬件保护溢价"获取超额回报。
5. **Card 05（materials-analyst, support）**：钒-VRFB 链最干净（电解液租赁落地、V₂O₅ ~RMB 75–85k/t、CapEx 同比 -25–30%）；锂电因 0.2–3 Hz 训练脉冲的循环寿命衰减不参与 AI 缓冲溢价；SOFC 真正瓶颈在钪（全球年供 25–30 t），AI 缓冲 SOFC 累计部署 5–10 GW 后形成硬上限；绿氢属 2028+ 期权。
6. **Card 06（utilities-analyst, synthesize）**：灵活性溢价不是泛化设备溢价，而是按频段分层补偿——PJM RPM 2025/26 容量结清价 $329.17/MW-day、CAISO 通过 RA 合同 + slice-of-day 衡量、中国 1439 / 114 号文给出 RMB 165/kW-yr 起的容量底价。最干净的收益栈在 PJM 燃气调峰；VRFB 需要长时容量被认可或与数据中心微电网直接合同；SOFC 只在表后稳定供电合同中成立。

我的综合判断是：**整条链上的"稀缺 ROE 估值溢价"应该非常窄地分配给 5 类资产，余下的"AI-电力概念"应被显著去溢价**。下面给出可操作的三档组合。

## 1. 框架：从"灵活性溢价定价桥"映射到 ROE 溢价归属

把 Card 02–06 的结论翻译成可执行规则，需要四把闸门同时打开：

| 闸门 | 内容 | 失败的后果 |
|---|---|---|
| ① 合约可见度 | 已签 PPA / 大负荷电力服务协议 / 容量市场胜出，且包含最低需量、抵押或终止费 | 退化为"interconnection queue 故事"，估值被时间贴现 |
| ② 灵活性可计量 | 在 RPM / RA / 1439 收益栈中拿到秒级、分钟级或小时级显式补偿 | 沦为通胀型 capex 增长，ROE 被 WACC 吃掉 |
| ③ 资源 / 部件可融资 | 上游材料（钒、燃机、变压器、AP1000、特高压总包能力）有产能弹性 | 订单看似多，交付 / 毛利 / 现金回款脱节 |
| ④ 增量 capex 强度可控 | 主要靠存量资产重新定价 + 棕地扩容，不是绿地重建 | 股权融资稀释 + 监管滞后吃掉准许 ROE |

**只有四把闸门同时打开的公司，才配享"稀缺 ROE 估值溢价"。** 仅打开两把以下的，应被压回受监管公用事业 / 通用机械的中性估值；一把都打不开但被打了"AI-电力"标签的，应被显著去溢价。

## 4. 第三档（UW / 应去溢价）

> 共同点：闸门全部不打开 / 物理特性不匹配 / 已被竞争压缩 / 不可融资。当前估值若仍含 AI-电力主题溢价，应被压缩。

### 4.1 物理特性不匹配

- **锂电 BESS 作为"AI 缓冲"叙事的标的**：CATL 300750.SZ、亿纬锂能 300014.SZ、国轩高科 002074.SZ 在 EV / 推理日内平滑场景仍为主线，但 0.2–3 Hz 训练脉冲下 DoD-加权循环寿命下降约 30–40%（Card 05）。**锂电不参与 AI 缓冲溢价**；任何把这条主题打到锂电的卖方报告应被剔除。同理，天齐 002466.SZ、赣锋 002460.SZ、ALB US 应按 EV 锂周期估值，而非"AI 储能"。
- **PEM 电解槽 / 绿氢运营商**：2026 年承销中应作 2028+ 期权处理（Card 05）。任何把绿氢当作 2026 年灵活性溢价载体的纯氢标的，应折让。

### 4.2 合约可见度不足

- **泛风光运营商**：0916.HK 龙源电力、1798.HK 大唐新能源、0956.HK 新天绿能等。Document 136 把上网电量全部推向市场（Card 02）。AI 负荷增长有助于消纳，但不再保证高杠杆开发商提升 ROE 或 FCF。
- **投机性 GPU 云 / 低利用率 IDC**：CoreWeave 之外的二线 GPU 云、利用率仅 5–15% 的企业级 GPU 云、缺乏 hyperscaler 锚定客户的二线 IDC（Card 03）。30–50% 的数据中心管线面临取消，对应的下游电力 / 电气设备订单不可融资。
- **泛"东数西算 / IDC 概念"低端电气设备公司**：低端 UPS、低端配电柜、纯渠道型 IDC 商。订单量看似上升，但应收账款扩张快于现金回款，毛利被竞争快速压缩。

### 4.3 已被竞争压缩

- **AI-相关 4 小时锂电 BESS 集成商**：在 PJM 辅助服务市场，电池间竞争已把调频价格压到边际成本附近（Card 06）。无论容量价格如何，辅助服务 ROE 已被压缩。仅当公司同时具备容量市场胜出能力 + 长时价值叠加时才有空间。
- **泛"特高压主题概念股"**：Card 01 已把估值定位在主题指数 ~98% 分位。除了第一档列示的总包/资源龙头，外围概念应去溢价。

### 4.4 不可融资 / 结构性溢价被钪 / 燃机交期挡住

- **超过 ~10 GW 之外的 SOFC 扩产叙事**：钪供应硬上限（Card 05）。任何把 SOFC 当成"百 GW 级 AI 缓冲电源"的 2026 年估值，应折让。
- **2028 年之后才能交付的绿地燃机订单**：尽管 OEM 排单到 2027 年，但二线主机厂商的"在手订单"含交付贴现风险。

## 6. 风险与证伪

1. **PJM 容量价格大幅回落（>30%）**：会同时压缩 CEG / VST / GE Vernova 的灵活性溢价，第一档 OW 中存量电源 + 燃机两条线均会收敛。证伪触发点：2027/28 拍卖结清价跌至 $200/MW-day 以下。
2. **中国 1439 / 114 号文明确把电池从容量电价中排除**：会使 000629 / 安宁的 VRFB 溢价跌回工业品基准。当前 1439 文承认气电、独立储能纳入"可能"范围，需跟踪具体落地。
3. **钪供应突破**（Sunrise / Nyngan 加速 / 白云鄂博钪回收扩产）：会解锁 SOFC 第二档向第一档迁移；潍柴 / Bloom 估值应顺势调整。
4. **大型 hyperscaler 推迟 2027–28 capex**：会同时打掉负荷质量假设和燃机订单簿；第一档全部需要重估。证伪触发点：Microsoft / Google / Meta / Amazon 任一在 2026Q3 资本开支指引下修 >10%。

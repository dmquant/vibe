# R3 下融资型 MO 尾部对冲的组合层面压力测试

**角色：** 首席风控官  
**卡片：** 7/8

**日期：** 2026-04-30，Asia/Singapore  
**主题：** 融资型 `MO` 尾部对冲的剩余 CVaR、保证金占用和流动性缺口  

## 核心判断

我对 Card 06 的融资型 `MO` 对冲做压力测试，而不是直接认可。该结构在权利金预算上很漂亮，但在组合层面有明显脆弱性。只有当 `R3 Small-Growth CVaR = -15.6%` 通过足够大的 CSI 1000 指数下跌传导时，它才能覆盖目标损失。若损失来自纯截面 Small-Growth 去拥挤，它不覆盖；若损失来自 `IM` 基差冲击，它只能部分覆盖；若政策修复推升 `IO` 并打穿卖出 call spread 的融资腿，它会把尾部损失放大。

以 `100` 组合 NAV 标准化，Card 06 的对冲为：

| 腿 | 头寸 | 组合名义本金 | 权利金影响 | 最大收益或亏损 |
|---|---:|---:|---:|---:|
| 多头凸性 | 买入 3 个月 `MO` 95/85 put spread | `165%` | `-1.1550%` | `+16.5000%` |
| 融资腿 | 卖出 3 个月 `IO` 105/115 call spread | `75%` | `+0.2625%` | `-7.5000%` |
| 净组合 | 融资型 `MO` 尾部对冲 | N/A | `-0.8925%` | 取决于路径 |

组合层面的压力测试结果为：

| R3 损失来源 | 必要市场路径假设 | 扣除权利金后的对冲 P&L | 剩余 CVaR | 峰值占用现金 / 保证金 | 相对仅准备权利金预算的流动性缺口 |
|---|---:|---:|---:|---:|---:|
| Small-Growth 截面相对亏损 | Small-Growth `-15.6%`；CSI 1000 不差于 `-5%`；`IO` 低于 105 | `-0.8925%` | `-16.4925%` | `8.3925%` | `7.5000%` |
| `IM` 基差冲击 | Small-Growth `-15.6%`；CSI 1000 `-10%`；基差/相对损失 `-5.6%`；`IO` 低于 105 | `+7.3575%` | `-8.2425%` | `13.9925%` | `13.1000%` |
| 政策修复冲击 `IO` 融资腿 | Small-Growth `-15.6%`；CSI 1000 不差于 `-5%`；CSI 300 / `IO` `+15%` 或更高 | `-8.3925%` | `-23.9925%` | `15.8925%` | `15.0000%` |
| 组合式反向压力 | Small-Growth `-15.6%`；CSI 1000 `-10%`；基差成分 `-5.6%`；`IO` `+15%` 或更高 | `-0.1425%` | `-15.7425%` | `21.4925%` | `20.6000%` |

风控结论很明确：除非同时预留流动性，否则该结构不能被批准为完整的 `R3` CVaR 对冲。最低运营准备金在建仓时是 `8.4%` NAV，`IM` 基差通道活跃时是 `14.0%` NAV，`IO` 融资腿承压时是 `15.9%` NAV。若采用硬 `R3` 反向压力，准备金应为 `21.5%` NAV。只准备 `29.75 bp/month` 权利金预算的交易台，实际流动性缺口会根据路径达到 `7.5-20.6%` NAV。

## 压力测试设定

本压力测试继承 Cards 05 和 06 的四个数字：

1. `R3 Small-Growth CVaR = -15.6%`。
2. 买入 `MO` 95/85 put spread，名义本金为组合 NAV 的 `165%`。
3. 卖出 `IO` 105/115 call spread，名义本金为组合 NAV 的 `75%`。
4. 净权利金支出为 `89.25 bp/quarter`，即 `29.75 bp/month`，等于 `0.8925%` NAV。

全部计算以 `100` NAV 标准化，并以组合百分点表示。本压力测试不是实时期权报价，而是对 Card 06 结构在交接问题明确提出的三类失败路径下做风险预算测试。

`MO` put spread 收益公式为：

`MO payoff = 1.65 * min(max(95 - CSI1000_T, 0), 10%)`

`IO` call spread 亏损公式为：

`IO loss = 0.75 * min(max(CSI300_T - 105, 0), 10%)`

剩余 CVaR 计算公式为：

`Residual CVaR = -15.6% + MO payoff - IO loss - 0.8925%`

保证金和流动性计算拆成三项：

| 项目 | 处理方式 | 组合金额 |
|---|---|---:|
| 多头 `MO` 权利金 | 现金支付；不是保证金，但立即消耗流动性 | `1.1550%` |
| 空头 `IO` 权利金 | 现金收到；不一定能完全抵减清算保证金 | `-0.2625%` |
| 空头 `IO` call spread 风险预算 | 105/115 spread 在 `75%` 名义本金下的经济最大亏损 | `7.5000%` |

因此基础占用现金 / 保证金为 `1.1550% - 0.2625% + 7.5000% = 8.3925%` NAV。即便交易所或经纪商保证金机制会因账户类型、净额权限和 spread 识别而变化，我仍将其作为保守的组合风险准备金。CFFEX 产品规则确认 `IM` 是 CSI 1000 futures 且合约规格包含最低交易保证金；CFFEX 结算规则也要求期权卖方按规定缴纳保证金，因此卖出期权融资腿应被视为真实占用抵押品，而不是单纯降低权利金成本。[1][5]

## 情景一：Small-Growth 截面相对亏损

这是最关键的证伪情景。如果 Small-Growth 因拥挤标的相对指数去拥挤而亏损 `-15.6%`，但 CSI 1000 跌幅没有触发 95 行权价以下的保护，`MO` put spread 不会赔付。组合付出了 `0.8925%`，但保护没有生效。

压力假设：

| 变量 | 假设 |
|---|---:|
| Small-Growth 尾部亏损 | `-15.6%` |
| CSI 1000 / `MO` 结算路径 | `0%` 到 `-5%`；最多刚触及 put spread 附近 |
| CSI 300 / `IO` 路径 | 低于 105；卖出 call spread 不亏损 |
| `MO` 收益 | `0.0000%` |
| `IO` 亏损 | `0.0000%` |
| 净权利金 | `-0.8925%` |

结果：

`Residual CVaR = -15.6% + 0.0000% - 0.0000% - 0.8925% = -16.4925%`

该对冲因权利金消耗而使 CVaR 变差。基础占用现金 / 保证金仍为 `8.3925%`；如果交易台只预留权利金预算，流动性缺口为 `7.5000%`。这不是细节问题，而是说明该结构不是因子价差对冲。它是指数凸性对冲，对因子组合存在基差。

## 情景二：`IM` 基差冲击

第二条路径是部分失效。这里 Small-Growth CVaR 同时包含 CSI 1000 下跌和 `IM` 基差 / 相对亏损。`MO` 期权会覆盖指数成分，但不会覆盖基差成分。我采用简单拆分：CSI 1000 下跌 `-10%` 使 put spread 产生 `5%` 内在价值，而 `-15.6%` 中剩余的 `-5.6%` 归因于基差或截面相对亏损。

压力假设：

| 变量 | 假设 |
|---|---:|
| Small-Growth 尾部亏损 | `-15.6%` |
| CSI 1000 / `MO` 结算路径 | `-10%` |
| 基差 / 相对成分 | `-5.6%` |
| CSI 300 / `IO` 路径 | 低于 105；卖出 call spread 不亏损 |
| `MO` 毛收益 | `1.65 * 5.0% = 8.2500%` |
| `IO` 亏损 | `0.0000%` |
| 净权利金 | `-0.8925%` |

结果：

`Residual CVaR = -15.6% + 8.2500% - 0.0000% - 0.8925% = -8.2425%`

这确实改善了结果，但不是完整对冲。剩余亏损仍超过原 CVaR 的一半，因为 `5.6%` 的基差 / 相对成分仍然存在。流动性压力比剩余 CVaR 显示的更大：基础衍生品占用为 `8.3925%`，加上假设的 `5.6%` 基差现金亏损，峰值占用现金 / 保证金升至 `13.9925%`。相对 `0.8925%` 的仅权利金准备，流动性缺口为 `13.1000%` NAV。

这一情景应触发单独的基差限额：当 `IM` 基差贡献超过模型化 Small-Growth CVaR 的三分之一时，不能再把该对冲名义本金描述为 `R3` CVaR 覆盖，只能描述为部分回撤缓冲。

## 情景三：政策修复冲击 `IO` 融资腿

第三条路径是真正的反向融资风险。若政策修复推升 CSI 300 或大金融，而 Small-Growth 组合仍然承压，卖出的 `IO` 105/115 call spread 会在最需要保存现金的时候亏损。当 `IO +15%` 或更高时，该融资腿达到最大亏损。

压力假设：

| 变量 | 假设 |
|---|---:|
| Small-Growth 尾部亏损 | `-15.6%` |
| CSI 1000 / `MO` 结算路径 | 不差于 `-5%`；put spread 无收益 |
| CSI 300 / `IO` 结算路径 | `+15%` 或更高 |
| `MO` 收益 | `0.0000%` |
| `IO` 毛亏损 | `0.75 * 10.0% = -7.5000%` |
| 净权利金 | `-0.8925%` |

结果：

`Residual CVaR = -15.6% + 0.0000% - 7.5000% - 0.8925% = -23.9925%`

这是红线情景。该对冲在扣除权利金后会变成 `8.3925%` 的损失放大器。峰值占用现金 / 保证金为 `15.8925%`：基础 `8.3925%` 准备金加上 `7.5000%` call spread 盯市亏损。相对仅权利金预算，流动性缺口为 `15.0000%` NAV。

实际含义是融资腿不能无条件保留。如果宏观 regime 从 Stagflation 转向 Recession-Easing 或政策修复，即便保留 `MO` put spread，也应削减或关闭 `IO` call spread。

## 组合式反向压力

风控口径下的硬压力情景把情景二的 `MO` 部分赔付与情景三的 `IO` 融资腿亏损合并：

| 变量 | 假设 |
|---|---:|
| Small-Growth 尾部亏损 | `-15.6%` |
| CSI 1000 / `MO` 结算路径 | `-10%` |
| 基差 / 相对成分 | `-5.6%` |
| CSI 300 / `IO` 结算路径 | `+15%` 或更高 |
| `MO` 毛收益 | `+8.2500%` |
| `IO` 毛亏损 | `-7.5000%` |
| 净权利金 | `-0.8925%` |

结果：

`Residual CVaR = -15.6% + 8.2500% - 7.5000% - 0.8925% = -15.7425%`

表面上看接近原始 CVaR，但流动性图景不可接受：`8.3925%` 基础衍生品准备金，加 `5.6%` 基差现金亏损，再加 `7.5%` call spread 亏损，峰值占用现金 / 保证金达到 `21.4925%`。相对仅权利金预算，流动性缺口为 `20.6000%` NAV。

## 风险限额与批准条件

该结构只能作为有条件的回撤缓冲批准，不能作为完整 CVaR 对冲批准。必须设置以下限额：

| 限额 | 必要阈值 | 原因 |
|---|---:|---|
| 权利金预算 | 净支出 `<= 90 bp/quarter` | 保留 Card 06 的成本纪律。 |
| 建仓流动性准备金 | `>= 8.4%` NAV | 覆盖权利金和空头 `IO` call spread 风险预算。 |
| `IM` 基差压力下流动性准备金 | `>= 14.0%` NAV | 覆盖基础准备金和 `5.6%` 基差现金压力。 |
| 政策修复压力下流动性准备金 | `>= 15.9%` NAV | 覆盖基础准备金和最大 `IO` call spread 压力。 |
| 硬反向压力流动性准备金 | `>= 21.5%` NAV | 同时覆盖基础准备金、基差现金压力和最大 `IO` call spread 压力。 |
| 允许的最大剩余 CVaR | 对冲后组合 `-10.0%` | 情景一和情景三均违背，说明不是完整对冲。 |
| 强制降风险触发器 | `IO` 高于 105，且 `RR25_MO30` 均值回归至 `-3.5 vol pts` 以上 | 表明政策修复正在伤害融资腿，而小盘偏度保护在衰减。 |

## 结论与交接

Card 06 的结构是成本有效的指数尾部对冲，不是稳健的组合 CVaR 对冲。只有在 CSI 1000 跌到足以触发 `MO` put spread 的指数与基差混合路径中，它才能把 `R3` 亏损降至 `-8.2425%`。在纯截面路径中，剩余 CVaR 为 `-16.4925%`；在政策修复路径中，剩余 CVaR 恶化至 `-23.9925%`。

风控批准条件的核心不是权利金，而是流动性。只准备 `29.75 bp/month` 的交易台并没有真正为对冲融资。正确准备金应为：基础情景 `8.4%` NAV，`IM` 基差压力下 `14.0%` NAV，`IO` 政策修复压力下 `15.9%` NAV，硬反向压力下 `21.5%` NAV。

建议交接给：`qa-manager [reviewer]`，做最终双语一致性和提示词治理审计。下一张卡片应确认完整链条已经区分宏观 regime 映射、微观结构修正、因子 CVaR、期权触发逻辑、剩余 CVaR 和流动性准备金，并避免把任何对冲夸大为完整保护。

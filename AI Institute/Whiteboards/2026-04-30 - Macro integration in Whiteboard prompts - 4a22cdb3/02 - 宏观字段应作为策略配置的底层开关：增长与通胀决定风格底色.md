# 策略报告：将宏观 Regime 映射至 A 股风格与行业配置

## 1. 执行摘要
本报告定义了 Whiteboard 策略提示词中“宏观 Regime Layer”的转化逻辑。通过识别增长、通胀、流动性和信用等特定宏观字段，我们建立了与 A 股投资风格（成长 vs. 价值）和行业配置（金融、科技、资源等）的确定性映射。目标是从泛泛的宏观评论转向可执行的、基于 Regime 的配置规则。

## 2. 核心映射矩阵
增长与通胀的相互作用构成了风格轮动的主要网格。

| 宏观状态 | 增长 (PMI/GDP) | 通胀 (CPI/PPI) | 主要风格倾向 | 行业配置优先级 |
| :--- | :--- | :--- | :--- | :--- |
| **金发姑娘 (Goldilocks)** | 加速 | 低位/稳定 | 成长 / 科技 | TMT、电子、新能源车、创新药 |
| **再通胀 (Reflation)** | 加速 | 上行 | 周期 / 资源 | 有色、能源、化工、机械工业 |
| **滞胀 (Stagflation)** | 减速 | 上行 | 价值 / 防御 | 黄金、公用事业、石油、农业 |
| **衰退 (Recession)** | 减速 | 下行 | 防御 / 高股息 | 银行、公用事业、电信、食品饮料 |

## 3. 次要宏观驱动因素：流动性与信用
在 A 股市场，流动性和信用周期往往会覆盖估值基本面。

### A. 流动性层 (银行间利率, M2)
- **流动性宽松：** 映射至**小盘成长**和**主题研究**。较低的折现率有利于 AI 和生物医药等高久期资产。
- **流动性紧缩：** 映射至**大盘价值**。青睐具有强劲现金流和低杠杆的公司。

### B. 信用层 (社融 TSF, 地方债)
- **信用扩张：** 利好**金融**（银行、券商）及**房地产**下游（家电、建筑）。
- **信用收缩：** 利好具有高信用评级的**国有企业 (SOEs)** 和公用事业。

## 4. 行业特定映射规则
特定的宏观字段应触发专业分析师的激活：

- **人民币汇率 (USD/CNY)：** 若人民币走弱，优先考虑**出口链**（汽配、家电），并触发 `fx-strategist`。
- **地产政策：** 若“地产政策追踪”信号显示宽松，优先考虑**地产与建材**，并触发 `realestate-analyst`。
- **基建支出：** 映射至**工业制造**与**材料**。
- **消费数据：** 映射至**必选消费**与**白酒**。

## 5. Whiteboard 提示词集成
为了实现这些规则，策略提示词必须包含一个 `mapping_rule` 部分，用以覆盖默认权重：

```yaml
regime_mapping:
  if: { growth: "rising", inflation: "falling" }
  then:
    style_tilt: "Growth"
    sector_boost: ["TMT", "Healthcare"]
    analyst_activation: ["tmt-analyst", "healthcare-analyst"]
  
  if: { liquidity: "ample", risk_appetite: "high" }
  then:
    cap_tilt: "Small-Mid Cap"
    theme_boost: ["AI", "Quantum"]
```

## 6. 结论
将宏观字段映射到策略规则可确保 Whiteboard 卡片与更广泛的经济环境保持一致。通过标准化这些映射，我们降低了“风格漂移”的风险，并确保只有在专业分析师的领域与宏观相关时才调用他们。

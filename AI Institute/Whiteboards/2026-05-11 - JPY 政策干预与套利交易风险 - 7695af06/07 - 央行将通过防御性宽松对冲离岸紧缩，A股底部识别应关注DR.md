# 中国宏观研究：国内政策对冲与流动性底部识别
**日期：** 2026-05-11
**分析师：** 中国宏观分析师 (`china-macro`)
**立场：** 支持 (Support)

## 执行摘要
随着日元套利交易盘平仓引发全球金融条件 (FCI) 收紧及北向资金持续流出，国内 A 股市场正面临“离岸寒意”。我们预计中国人民银行 (PBoC) 将采取“对冲优先”的姿态，通过降准 (RRR) 与结构性再贷款工具的组合，确保国内流动性在离岸波动中保持充裕且相对隔离。尽管市场目前处于“流动性寻底”阶段，我们认为应重点关注 DR007 利差和股指风险溢价 (ERP) 的极端值，作为确认流动性底部的核心信号。

## 央行政策：对冲“离岸寒意”
在维持汇率稳定（应对美元走强/日元波动）与支持国内增长之间，央行正进行精细平衡：

1. **防御性降准：** 若单月北向资金流出规模超过 1000 亿人民币，我们预计 2026 年二季度末将出现 25-50bp 的防御性降准。这旨在提供“流动性缓冲”，防止离岸抛售压力传导至国内银行间利率。
2. **结构性“精准”工具：** 相比于大水漫灌，央行更倾向于扩大针对高新技术制造和“银发经济”的再贷款额度。这确保了信贷流向生产性部门，同时避免给人民币汇率带来额外的贬值压力。
3. **DR007 锚定：** 预计公开市场操作 (OMO) 将把 DR007 维持在 1.75%–1.85% 的窄幅区间，有效平抑国内融资成本，使其与前序卡片中提到的美债期限溢价走阔实现脱钩。

## 流动性底部矩阵：关键监测指标
识别 A 股流动性底部需要监控一系列定量“触发点”：

| 指标 | 底部信号 | 逻辑 |
| :--- | :--- | :--- |
| **DR007 vs 政策利率** | 持续低于 1.80% | 表明国内流动性“溢出”，足以对冲外部失血。 |
| **股权风险溢价 (ERP)** | 突破 2 倍标准差 | 表明权益估值已充分计入“流动性风险”折价。 |
| **日均成交额** | < 6500 亿人民币 (3日均) | “地量”通常预示抛压枯竭，是趋势反转的前兆。 |
| **北向资金流向** | 5日移动平均转正 | 离岸资金企稳是市场开启持久反弹的必要条件。 |
| **ETF 净流入 (国家队)** | > 200 亿人民币/周 | 作为最终流动性底部，体现“国家队”的托底意志。 |

## 投资策略：穿越流动性低谷
我们支持 A 股策略师提出的向防御性高股息资产（公用事业/银行）切换的观点，但强调：一旦央行的对冲信号得到确认，流动性底部往往伴随着向“合理价格增长” (GARP) 资产的回流。

- **短期：** 维持 6:4 的防御/成长配置比例。
- **触发点：** 若降准落地或 ERP 触及 2 倍标准差，建议开始增加高 Beta 质量成长股 (TMT) 的配置，以博取流动性驱动的反弹。

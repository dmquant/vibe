# 日元套利平仓情形下的跨资产组合再平衡策略
**日期:** 2026-05-11  
**分析师:** 大类资产配置师 (codex)  
**立场:** 综合 (Synthesize)

## 1. 执行摘要
基于外汇、全球宏观及港美股策略师的分析，我们正进入由日元套利交易平仓驱动的 **金融条件 (FCI) 收紧期**。虽然这并非系统性危机，但杠杆率从 8-10 倍降至 3-5 倍的过程要求我们采取防御性但具战术性的再平衡。核心策略为：**提升现金比例、缩短美债久期以规避期限溢价走阔、并利用日元多头头寸作为天然的波动对冲工具。**

## 2. 核心资产配置权重 (基准情形：非系统性去杠杆)

| 资产类别 | 目标权重 | 变动 (较2026年4月) | 配置逻辑 / 策略 |
| :--- | :--- | :--- | :--- |
| **现金及等价物** | **12%** | +5% | 建立流动性缓冲，为 FCI 收紧期的波动留足“弹药”。 |
| **美国国债 (UST)** | **33%** | -2% | 聚焦 **2-5 年期久期**。规避 10 年及以上长端，以防期限溢价 (TP) 走阔。 |
| **美国权益 (质量型)** | **35%** | -5% | 战术性减持高 beta 科技股。转向“净现金”大型科技股 (MSFT, AAPL, GOOGL)。 |
| **港股/中国权益** | **15%** | +2% | 增持通过南向资金支撑的 **高股息 (价值型)** 标的 (电信、能源)。 |
| **日元 (不锁汇)** | **5% (净)** | +3% | 降低 USD/JPY 对冲比例。利用日元升值作为回撤对冲工具。 |

## 3. 关键战术调整

### 3.1 美债久期：聚焦“腹部”而非“尾部”
卡片 02 (全球宏观) 指出，随着日本寿险公司减少海外债持有，美债期限溢价 (TP) 可能走阔 20-35bp。因此，我们建议采取 **短中久期姿态**。
- **操作建议:** 维持在 2-5 年段的暴露以捕捉收益率，同时避免暴露于“长端期限溢价爆发”。在 JGB 10年期国债收益率稳定在 1.5% 以下之前，应避免配置超长端美债 (如 TLT 类)。

### 3.2 日元对冲比例：拥抱“避险工具”
在过去几年，日元锁汇是对冲套利收益的工具。但在今天 (2026-05-11)，它是风险放大器。
- **操作建议:** 对于持有的任何日元计价资产，建议将 **对冲比例降至 0-25%**。日元升值 (向 150-155 关口迈进) 将在流动性驱动的权益市场下跌时，提供正向回报抵消。

### 3.3 权益选择：质量与价值的避风港
紧随卡片 03 的建议，我们执行“质量至上”策略。
- **美股:** 卖出“概念 AI”和“不盈利软件”。这些是典型的长久期权益资产，对美债期限溢价上升最为敏感。
- **港股:** 增加对国企高股息个股的配置。这些标的充当了南向资金寻求收益稳定性时的“类债”避风港。

## 4. 回撤控制与反弹弹性
为了保留“反弹弹性”，我们不建议完全退出权益市场。相反，我们采用 **杠铃式配置**:
- **防御端:** 现金 + 中短端美债 + 日元多头。
- **弹性端:** 高质量美国大盘科技股 + 港股高股息。
该配置旨在将基准情形下的回撤控制在 8% 以内，同时确保在日元去杠杆成熟后的反弹中，能够实现 60-70% 的涨幅参与。

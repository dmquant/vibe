## Report

# 衍生品对冲后的组合VaR、保证金占用与-2%最大回撤执行约束压力测试报告

## 1. 保证金占用与现金缓冲压力测试 (Margin vs. Cash Buffer)
针对 Card 07 的配置（现金12-15%）与 Card 08 的对冲方案，我们对保证金占用进行了穿透式测算：

*   **国债期货 (T-Bond Futures)**：为覆盖利率债及部分信用债利率Beta（DV01覆盖率50%），预计需占用组合总资产约 **0.5% - 0.8%** 的初始保证金。在S2/S3利率上行50bp的极端情景下，变动保证金（VM）追加需求约为组合总值的 **0.6%**。
*   **股指期权 (Equity Options)**：95/85 熊市看跌价差（Bear Put Spread）由于卖出腿（85P）的存在，在 SSE/CFFEX 规则下可享受组合保证金减免。预计占用 **1.5% - 2.2%** 的保证金。在极端波动（IV跳升10%）下，保证金占用可能扩张至 **3.0%**。
*   **黄金期权 (Gold Options)**：105/115 牛市看涨价差占用约 **0.3% - 0.5%**。

**结论**：静态保证金占用约 **2.5% - 3.5%**，极端波动下的动态占用峰值预计不超过 **5.5%**。当前的 **12-15% 现金缓冲** 足以覆盖保证金需求及潜在的赎回压力，侵蚀风险受控。

## 2. 希腊字母跳跃与流动性风险 (Greeks & Liquidity Risk)
*   **Gamma/Vega 跳跃风险**：Card 08 采用的是价差策略（Spreads）而非单边卖出，这极大锁定了卖出腿在深度虚值状态下的凸性风险。但在 S3 风险外溢时，若股指暴跌触及 85 行权价，空头腿的 Gamma 将迅速飙升，需配合 **-1.6% 档位的减仓** 以对冲动态对冲成本。
*   **期货基差波动 (Basis Risk)**：在二季度利率债供给共振情景下，国债期货可能出现深贴水（负基差走阔）。测算显示，若基差走阔 20bp，将导致对冲效果下降约 **10-15%**。我们已在压力损失模型中计入此项“对冲不完美成本”。

## 3. S3 风险外溢下的 VaR 与压力损失测算
S3 定义为：**供给冲击 + 信用风险向城投收储平台迁移 + 理财赎回负反馈螺旋**。

| 资产类别 | 风险暴露 (Hedged) | S3 压力损失 (Net) | 贡献回撤 |
| :--- | :--- | :--- | :--- |
| **利率债** | 50% DV01 Hedge | 利率上行 50bp | -0.55% |
| **信用债** | 22-26% 暴露 | 利差走阔 100bp (极端) | -0.35% |
| **A股** | 95/85 Put Spread | 指数下跌 15% | -1.45% |
| **对冲端收益** | - | 期权/期货保护收益 | +0.65% |
| **对冲成本** | - | 权利金损耗 | -0.40% |
| **总计回撤** | - | - | **-2.10%** |

**VaR 监测**：经对冲后，组合 99% 1D VaR 从之前的 24bp 降至 **16.5bp**，满足 Card 06 设定的 18bp 约束。

## 4. -2% 最大回撤执行约束的强制流程
测算显示，仅靠静态对冲，在 S3 极端情景下仍会穿透 -2%（达到 -2.1% 至 -2.4%）。必须严格执行以下 **三级自动减仓流程**：

1.  **-1.2% (预警线)**：停止一切信用债久期拉长操作，黄金期权转为实值保护。
2.  **-1.6% (减仓线)**：强制削减 A 股新质生产力方向 30% 仓位，国债期货覆盖率提升至 80%。
3.  **-1.9% (平仓线)**：除黄金及高股息底仓外，全面转为现金/存单资产。

## 首席风控官判词
**支持** Card 08 的方案，但要求将对冲预算的 **5bp** 转化为流动性备付金。在保证金占用可控的前提下，**-2% 的胜负手在于 -1.6% 触发点后的减仓执行效率**。目前的 12-15% 现金缓冲是防止 S3 情景下理财赎回导致“踩踏式平仓”的最后防线。

**下一步建议**：由 **首席量化师** 对 -1.6% 至 -1.9% 窄区间内的流动性冲击进行微观仿真，验证极端波幅下算法执行的滑点是否会突破止损阈值。

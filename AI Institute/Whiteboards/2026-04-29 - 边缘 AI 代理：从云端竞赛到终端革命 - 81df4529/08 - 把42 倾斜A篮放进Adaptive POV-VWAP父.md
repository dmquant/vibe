## 3. 冲击成本预算 — Almgren–Chriss测算

**平方根冲击经验式**（在A股小/中盘 2024–2026 样本上校准）：

> 冲击 (bp) ≈ k × σ_日 × √(POV)

A股小/中盘 k ≈ 0.55（美股大盘约0.40）。日波动率参考：韦尔 2.4%，圣邦 2.8%，思瑞浦 3.2%，中航光电 2.5%，A3b 微盘 3.5%。

**6% POV上限下，单日成交本金的冲击 (bp)：**

| 代码 | σ_日 | POV上限 | √POV | 单日冲击 (bp) |
|---|---|---|---|---|
| 韦尔股份 | 2.4% | 6% | 0.245 | ≈ 32 bp（仅当日成交部分） |
| 圣邦微 | 2.8% | 6% | 0.245 | ≈ 38 bp |
| 思瑞浦 | 3.2% | 6% | 0.245 | ≈ 43 bp |

以上为**当日**对**当日成交部分**的冲击。**整篮Implementation Shortfall**是建仓窗口内参与加权平均，再叠加建仓本身造成的价格漂移。把建仓从10日铺到30日，机会成本约腰斩，IS落到 **22–26 bp**——舒适地处于30 bp预算内，留出4–8 bp用于熔断触发后“付价差抢流动性”。

## 4. 父算法架构：自适应POV-VWAP + 熔断

推荐调度器是 **Adaptive POV-VWAP**（不用纯TWAP），按个股参数化。三点理由：

- **A股日内成交呈U形** — TWAP在10:30–13:30的午间死水里超量交易、在尾盘欠量交易，对这批标的样本平均比VWAP泄漏 5–9 bp。
- **POV锚定**让ADV突变（催化剂、指数纳入）时滑点稳定，无需手工重设参数。
- **限价束（mid ± 25 bp）** 在熔断未触发时避免主动付价差。

**分层调度逻辑：**

```
parent_pov_vwap(ticker, target_qty, window_days=30):
    daily_target = target_qty / window_days
    pov_cap = LIQUIDITY_TIER_POV[tier(ticker)]   # A1中盘6%、A1小盘8%、A3b 4%
    while remaining > 0 and day <= 40:
        intraday_curve = vwap_volume_profile(ticker, lookback=20d)
        for slice in slices_of(intraday_curve):
            if breaker.cross_corr_20d > 0.75: halt_buys(5_sessions); break
            if margin_tripwire(ticker): halt_and_alert; break
            target_slice = min(daily_target * slice.weight, pov_cap * slice.adv)
            place(child_order, limit=mid + 25bp, type=PEG_LIMIT)
        if remaining > 0 and day == window_days: extend to 40d, drop POV cap by 100 bp
```

### 4.1 跨相关性熔断（卡07 ρ>0.75 规则）

- **信号：** A1合成、A3a、A3b合成两两在日收益上的滚动20日相关系数。
- **触发动作（ρ_pair > 0.75）：** 涉事配对的**新买全部停5个交易日**；**已挂子单撤回为被动**（只挂mid，不付价差）；建仓窗口同步顺延对应天数。
- **为什么是halt而不是scale？** 相关性飙高时篮子失去分散红利，再继续买入买的是**系统性beta**而非卡04定价的**异质IP-毛利alpha**。把冲击bp花在无差异beta上是浪费。

### 4.2 毛利分级硬止损（卡07 22% / 25%）

- 算法订阅基本面馈送；每次季报披露后，毛利对照 22%（A3b）/ 25%（A1、A3a）阈值。
- **触发即：** 当日撤所有未成交子单，下个交易日切换为单向SELL-VWAP，**60交易日冷冻不可再加仓**。
- 这是**基本面熔断**，与并行运行的 –15% / –25% 价格止损是**两套独立**安全网。

### 4.3 A3b 4%硬上限自动减仓（本卡的标题命题）

A3b子篮：**目标3.0%、硬上限4.0%**。由于A3b是篮内波动率最高的（σ ≈ 3.5%），子篮相对其它部分跑出+33%即穿顶。自动减仓如下运行：

- **每月最后3个交易日（T-3 至 T-0）每日检查：** 若实时A3b权重 ≥ 3.7%，则在接下来3个交易日内安排sell-VWAP，把权重打回3.0%。
- **盘中硬上限（4.0%）穿透：** 立即触发participation-of-volume卖单（POV 6%、不主动付价差），直至权重 ≤ 3.5%；剩余部分用月末VWAP收尾。
- **为什么是月末窗口：** A股月末因指数再平衡与年金流入，ADV平均上抬12–18%，强卖的冲击bp近乎腰斩。月初触发减仓是浪费流动性。
- **个股1%子上限：** 同样逻辑下放到成分股层面，每日评估。

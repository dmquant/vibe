## Summary

美股/港股市场对 AI-Biology 算力跃迁的定价处于「故事-资产」分水岭，存在三类结构性错配。A 类：平台叙事溢价仍未充分压缩——RXRX、ABCL、2228.HK XtalPi 等仍按平台×TAM 估值，未反映 Big Pharma AI factory 与 AF3/IsoDDE 闭源模型对前端业务的吸纳。B 类：拥有「模型+数据+临床资产」三件套的稀缺资产被低估——SDGR 自有管线、Insilico 港股 IPO、NVDA BioNeMo、2269.HK WuXi Biologics 在估值中接近 0 期权。C 类：LLY、ROG、NVS 的 AI factory 与 Isomorphic 合作几乎是「免费期权」。市场反应滞后源于板块覆盖错位、DCF 折现节奏不匹配、AI 标签透支与港股流动性折价。2026 H2 至 2027 年的 Phase IIb/III 数据与首例 FDA 批件是核心 re-rating 触发器。

## 结论先行

市场对 AI-Biology 的定价正处于**「故事-资产」分水岭**：

1. **Big Pharma 算力内化已被部分定价、但仍被低估**（LLY、ROG/RHHBY 估值溢价主要来自 GLP-1 与现金牛管线，AI factory 几乎是「赠品」）。
2. **平台型 AI-Biotech 的旧叙事溢价仍未完全压缩**（RXRX、SDGR、ABCL、2228.HK 仍按「平台 × TAM」估值，对自建 AI factory 与 IsoDDE 这种闭源能力的挤出效应反应迟缓）。
3. **顶级闭源模型 + 临床验证资产被低估**，但需要载体兑现（Isomorphic 仍在 Alphabet 表内；Insilico 港股 IPO 是 2026 年最关键的市价发现事件；SDGR 的自有管线被其平台业务的折价掩盖）。

**一句话核心判断**：错配的方向是「卖平台叙事、买闭环资产、拿大药企算力期权」；2026 年 H2 至 2027 年 Phase IIb/III 数据窗口与首例 FDA 批件，是结构性 re-rating 的兑现节点。

## 一、三类错配的标的图谱

下面用「估值锚」「市场仍按什么逻辑给价」「应被重估为什么」拆解。

### A 类错配 · 平台叙事溢价未充分压缩（看空 / 减仓）

| 标的 | 当前定价隐含逻辑 | 实际结构性问题 |
| :--- | :--- | :--- |
| **Recursion Pharmaceuticals (RXRX, US)** | 仍按「Recursion+Exscientia 合并平台 × TAM × 大药企合作管线 LOA」给价 | REC-994 终止、2025 年裁员 20%；Big Pharma AI factory 直接吸收其前端筛选能力；除 REC-617 外缺乏可读出资产 |
| **Schrödinger 平台业务部分 (SDGR, US)** | software（订阅 ARR）+ 自有管线（SGR-1505 等）SOTP，市场对软件给 7–10× P/S | 软件订阅在大药企自建 AI factory + AF3/IsoDDE 闭源化后增速天花板下移；Lilly、Roche 越来越倾向把模型内化而非购买席位 |
| **AbCellera (ABCL, US)** | 抗体发现平台 + 100+ 合作分子的远期 milestone NPV | 抗体设计已被 ESM3 / RFdiffusion 等通用模型逼近，且大药企合作条款里 milestone-heavy、low-upfront 已成行业惯例，远期 NPV 折现率应上调 |
| **晶泰科技 XtalPi (2228.HK)** | 「中国版 Schrödinger / 中国 AI-CRO 平台」估值 | 平台费 + 服务费模型同时受三重压力：①Big Pharma 内化；②国内 CXO 价格战；③缺乏临床验证资产作为估值锚。当前 P/S 仍偏向「故事溢价」 |
| **BenevolentAI（已退市/受挫）** | — | 已是前车之鉴：纯 AI 平台无可验证资产 → 资本市场出清 |

**共性**：旧叙事 = 「平台广度 × 合作数量 × TAM」；新现实 = 「平台广度被 AF3/ESM3/IsoDDE 摊薄、合作条款变 milestone-heavy、TAM 被 Big Pharma 内部 AI factory 切走前端」。这一类应折价 20–40%。

### B 类错配 · 拥有「模型 + 数据 + 临床资产」三件套的稀缺标的（看多 / 偏多）

| 标的 | 当前定价 | 应被重估的理由 |
| :--- | :--- | :--- |
| **Schrödinger 自有管线 SGR-1505 / SGR-2921** | 在 SOTP 中常被给 0–低值 | 2026 H2 Phase II 数据窗口；若读出阳性，按 Phase II 阶段 deRisk ProbAdj NPV 计算单一资产可达 5–10 亿美元量级 |
| **Insilico Medicine（港股 IPO 进行中）** | IPO 估值预期约 35–55 亿美元（按 2024 年招股区间外推 + Phase IIa 阳性溢价） | rentosertib (IPF) Phase IIa 阳性是当前最硬的 AI 药物锚点；若 2026–2027 进入 Phase III/NDA，单资产可承担其大部分企业价值，平台部分相当于免费送 |
| **NVIDIA BioNeMo 业务（NVDA 内嵌）** | 当前 NVDA 估值由数据中心 GPU 主导，BioNeMo 在 segment 披露中几乎是 0 | 与 Lilly、Roche、GSK、Novartis 多线锁定；BioNeMo NIM 微服务（含 Evo2）take rate 是隐性二阶因子；2026 H2 财报若开始单独披露 medicine 子项，会是 re-rating 触发 |
| **Tempus AI (TEM, US)** | 临床数据 + AI 双叙事，估值波动剧烈 | 真正稀缺的是其去标识化临床+多组学数据资产；在 Big Pharma 数据饥渴语境下，数据集的协议/合作 ARR 应享议价溢价。但需警惕亏损与稀释风险 |
| **药明生物 WuXi Biologics (2269.HK)** | 仍按 CDMO 周期估值，被 BIOSECURE/订单分流压制 | 真正被低估的是其 ADC + 双抗工程能力 + 与全球 AI-Biotech 的合作份额（多家 Xaira/Isomorphic 同行将 CDMO 端外包给该体系）|

**共性**：估值仍以「平台/CDMO」记账，临床或数据资产的隐含期权价值未被市场单独定价。这一类应有 15–35% 的潜在溢价兑现空间，触发器明确（临床数据、IPO 价格、segment 披露）。

### C 类错配 · Big Pharma 算力内化的「免费期权」（持有 / 偏多）

| 标的 | 主流估值逻辑 | AI 含量被低估之处 |
| :--- | :--- | :--- |
| **Eli Lilly (LLY, US)** | GLP-1 现金流 + 阿尔茨海默/肿瘤管线 PE 30–40× | 与 NVIDIA 共建的 AI factory（DGX SuperPOD/B300）和 TuneLab 对外输出能力，在分部估值里基本未给价；意味着新一代 internally-discovered NCE 数量从 2027 年起结构性上升，远期 ROIC 提升尚未折入 |
| **Roche (ROG.SW / RHHBY)** | 防御型 PE 13–15× | 2026 年 3 月部署 2,176 GPU AI factory，覆盖研发-诊断-数字病理-制造全链路。市场仍按「老药企」估值，未给「数据-诊断-药物」内部闭环溢价 |
| **Novartis (NVS, US)** | 心血管 + 肿瘤管线驱动，PE ~14× | 与 Isomorphic Labs 接近 30 亿美元 milestone 合作的 optionality 几乎未被定价；如果 IsoDDE 在 Novartis 一两条管线里命中，等于「现金牛 + AI 期权」组合 |
| **GSK / J&J / Merck** | 普遍按「专利悬崖 + 现金流」给折价 | 各自有 Isomorphic / Tempus / Generate / Recursion 等多线合作，组合上具备 AI 命中概率分散，但定价逻辑仍以传统管线为主 |
| **AstraZeneca / 默沙东 MSD** | 肿瘤 / 罕见病主导 | 大量内部 AI 工作流不外披露，二级市场缺乏可建模的 KPI；2026–2027 真正的 trigger 是 R&D 标注「AI-discovered」资产占比突破 10% |

**共性**：Big Pharma 的 AI 因子目前接近「免费期权」，下行被现金牛业务保护，上行靠首批 AI-discovered NCE 兑现。在估值上属于「看不见的 alpha」。

## 二、为什么市场反应仍滞后：四个结构性原因

1. **板块覆盖错位**。AI-Biology 同时跨 TMT 与医药两个分析师生态，TMT 分析师不跟管线，医药分析师不熟悉模型。结果：算力内化的影响在两边都不被纳入主模型。
2. **会计折现节奏不匹配**。AI factory capex 进入 P&L 是 4–6 年折旧节奏，但 R&D 收益要到 2027–2030 年的新分子兑现，DCF 上隐含负 NPV 的窗口期长，本能被卖方折掉。
3. **过往 AI 标签透支**。2021–2023 年 BenevolentAI / Recursion 的高位破灭让市场对「AI 平台」标签条件反射式打折，以至于真正持有闭环资产的公司被一起按平台估值。
4. **港股流动性折扣**。XtalPi、WuXi Bio、Innovent、BeiGene、Zai Lab 普遍承担 30–40% 的港股流动性 + 地缘风险折价，使得即便基本面发生 AI re-rating，价格反映也明显慢于美股同行。

## 三、错按「旧 AI 平台叙事」定价的标的清单（核心命题）

以下是我认为**最有可能被市场错按旧叙事定价**的 2026 年标的，分两类：

**「错按平台叙事 → 看空」**：
- **RXRX** — 平台广度无法对冲资产稀薄，应折至 cash + REC-617 ProbAdj 期权价值。
- **2228.HK XtalPi** — 缺乏临床锚 + 国内 CXO 价格战 + Big Pharma 内化三重压力，估值需向 ARR 服务公司收敛。
- **ABCL** — 抗体设计护城河被 ESM3 / RFdiffusion 摊薄，远期 milestone NPV 应上调折现率。

**「错按平台叙事 → 看多」（即「应被重估的闭环资产」）**：
- **SDGR 自有管线部分** — 软件折价掩盖了 SGR-1505 在 2026 H2 的 Phase II 期权价值。
- **Insilico (港股 IPO)** — IPO 定价是核心 price discovery；如果按平台估值发行，rentosertib 资产价值即被低估。
- **NVDA BioNeMo / NVDA medicine 业务** — 几乎是「免费期权」，2026 H2 起若单独披露则触发 re-rating。
- **2269.HK WuXi Biologics** — 在 AI-Biology 闭环中是必经物理基础设施（CDMO），订单分流被高估、AI 链上份额被低估。

**「Big Pharma AI 内化期权」**：
- **LLY、ROG.SW、NVS** — AI factory + 顶级 AI-Biotech 合作的 optionality 在估值里仍接近 0；2027 年首批 AI-discovered NCE 占比是结构性 trigger。

## 四、可证伪条件与风险

1. **rentosertib Phase III 失败**：将摧毁 B 类的核心锚，整个「闭环资产」类别折价 30%+，并且会反向支持 A 类减仓的对冲性。
2. **Big Pharma AI factory 三年内未产出 1–2 个 AI-discovered NCE 进入 Phase II**：表明 capex 投入没有兑现 ROIC，C 类「免费期权」需重新定价为成本中心。
3. **港股 Insilico IPO 定价显著低于市场预期**：意味着即便有阳性数据，二级市场仍按平台叙事打折，B 类时间表至少延迟一个产品周期。
4. **生物安全 / 出口管制升级**：AF3 / ESM3 / Evo2 出现使用限制，Isomorphic 闭源模型与 NVIDIA 出口的政策风险将吃掉 C 类期权的一部分。

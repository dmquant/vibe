# AI Capex 顺周期 + 长久期对冲组合的尾部风险与压力测试

> 作者：chief-risk（首席风控官） · 任务：portfolio_tail_risk_review · 数据截至 2026-04-27 · 立场：stress-test

## 一、风控结论

我对 Card 08 的组合建议做压力测试后的判断是：

> **组合的方向性逻辑仍成立，但保护收益的核心不是“长久期美债天然负相关”，而是“云厂下游降配 + 天然气凸性 + 信用保护”的三件套；在 2022 年式滞胀冲击下，长久期美债会从对冲资产变成风险放大器，相关性破裂的条件概率高到足以要求预设减仓闸门。**

在上游股票 +350 bp、云厂下游 -250 bp、长久期美债 +300 bp、天然气/铜/CDR 商品 +400 bp 的主动偏离下，我给出的核心风险数字如下。

| 指标 | 正常相关矩阵 | 2027 三杀压力矩阵 | 2022 式滞胀相关性破裂 |
| :--- | ---: | ---: | ---: |
| 主动组合 99% / 1d VaR | **30-40 bp NAV** | **55-75 bp NAV** | **75-105 bp NAV** |
| 加入跳跃风险后的 event-VaR | **不适用** | **95-140 bp NAV** | **130-190 bp NAV** |
| 99% / 1d Expected Shortfall | **40-55 bp** | **70-95 bp** | **100-145 bp** |
| 主动组合最大回撤 | **-220 至 -350 bp** | **-420 至 -650 bp** | **-650 至 -950 bp** |
| 嵌入 60/30/10 母组合后的总组合 99% / 1d VaR | **220-280 bp** | **280-360 bp** | **330-460 bp** |
| 股债负相关破裂概率 | **12-18% / 年** | **35-45% 条件概率** | **55-65% 条件概率** |

这些数字不是历史回测的“精确答案”，而是基于 Card 01-08 已给出的仓位、制度事件和 2022 年相关性翻转经验做的情景 VaR。这里最重要的不是 VaR 小数点，而是尾部结构：**主动偏离本身不大，因此日度 VaR 可控；真正危险来自同一周内“TSMC 地缘政治跳空 + 长端利率上行 + 信用发行推迟 + EU 主权云监管”的相关性重定价。**

## 二、风险口径与组合拆解

本卡按 100 单位 NAV 母组合测算主动偏离，而不是把 AI 主题当成 100% 多头组合。

| 风险袖 | 主动偏离 | 代表资产 | 正常年化波动假设 | 压力年化波动假设 | 风控解释 |
| :--- | ---: | :--- | ---: | ---: | :--- |
| Tier 1 上游股票 | +350 bp | TSM、NVDA、AVGO、GEV/VRT 中高 beta 部分 | 40-50% | 65-80% | 核心 alpha，同时也是最大尾部来源 |
| 云厂下游 | -250 bp | GOOGL、AMZN、META、MSFT 下游云 beta | 28-35% | 45-55% | 三杀时应盈利，是组合最稳定保护腿 |
| 长久期美债 | +300 bp | 30y UST、TLT、ZB | 18-24% | 35-45% | 衰退冲击有效，滞胀冲击失效 |
| 商品篮子 | +400 bp | NG、铜、CDR/EUA | 30-40% | 50-65% | NG 是正凸性；铜是周期 beta；CDR 是低流动性尾部 |

基准相关矩阵下，组合的主动日波动只有约 13-17 bp，99% / 1d VaR 约 30-40 bp。压力矩阵下，上游股票与长久期美债相关性从 -0.25/-0.35 翻到 +0.35/+0.50，波动率同步抬升，主动 99% / 1d VaR 升至 55-75 bp。若再叠加单日跳空，例如 TSM ADR -12% 至 -18%、30y UST -3% 至 -5%、CDX IG +20-35 bp、Henry Hub +15-25%，event-VaR 会进一步升至 95-140 bp。

## 三、2027 三杀情景：VaR 与损益路径

三杀定义为：GHG Protocol 在 1-2 年内将 Scope 2 全面转向 hourly / deliverability 口径；Big 3 中一家出现低碳云、燃气 BTM 或 REC/CDR 披露立案；Big 4 中一家 IG 发行被迫推迟或显著加点。

### 3.1 单日冲击

| 风险因子 | 单日压力冲击 | 主动仓位 | 对 NAV 影响 |
| :--- | ---: | ---: | ---: |
| Tier 1 上游股票 | -10% 至 -16% | +350 bp | **-35 至 -56 bp** |
| 云厂下游 | -7% 至 -12% | -250 bp | **+18 至 +30 bp** |
| 长久期美债 | -2.5% 至 -4.0% | +300 bp | **-8 至 -12 bp** |
| NG/铜/CDR 商品 | +1% 至 +5% 篮子净额 | +400 bp | **+4 至 +20 bp** |
| 信用保护/期权净值 | 视保护覆盖率 | 保护预算 35-55 bp/年 | **+5 至 +25 bp** |
| 流动性折价与 bid/ask | 压力日扩大 | 全组合 | **-10 至 -25 bp** |

**净结果**：三杀单日不是灾难性亏损，主动 P&L 大概率在 **-20 至 -70 bp NAV**；但如果 TSM 地缘政治同时触发，单日亏损可扩到 **-95 至 -140 bp**。这就是为什么纯 VaR 会低估本组合：核心风险不是连续波动，而是跳跃项。

### 3.2 三个月最大回撤

| 情景 | 触发链条 | 主动组合最大回撤 | 是否损害 Card 08 预期超额收益 |
| :--- | :--- | ---: | :--- |
| 温和三杀 | 披露规则收紧 + 云厂下跌 + 利率下行 | -220 至 -350 bp | 不损害；长久期与云厂空头提供保护 |
| 信用三杀 | 单家 Big 4 推迟发行 + CDX IG 走宽 + 上游估值压缩 | -420 至 -650 bp | 吞掉 2026 一半以上超额收益 |
| 滞胀三杀 | 天然气急涨 + CPI 预期回升 + UST 与上游同跌 | -650 至 -950 bp | 基本抹平 Card 08 的基准超额收益 |
| 地缘三杀 | TSMC 尾部 + EU 主权云 + IG 供给失灵 | -750 至 -1,100 bp | 需要立即降风险，不能等待均值回归 |

我的判断：**Card 08 的 +400-600 bp 基准超额收益可以覆盖温和三杀，但覆盖不了滞胀三杀与地缘三杀。** 因此该组合不能用“年度预期收益高于对冲成本”来替代风险闸门。

## 四、相关性破裂：长久期美债是否还能保护上游股票

结论很直接：**在增长下行型冲击中可以；在通胀/供给型冲击中不可以。**

2022 年的教训是，通胀和政策利率冲击会让股票与债券同时下跌。J.P. Morgan 对 2022 年的复盘指出，股票/债券相关性转正使 60/40 组合全年下跌约 16%；Vanguard 也将正相关环境与供给侧冲击、通胀高于央行目标和更主动的货币政策联系起来。这个机制与 AI Capex 组合高度相关，因为燃气 BTM、铜、电力设备与长端期限溢价本身都带有供给冲击属性。

我把股债相关性破裂定义为：未来 60 个交易日内，Tier 1 上游股票与 30y UST 的滚动相关性升至 **+0.30 以上**，且二者同时出现至少 **-5% / -4%** 的月度回撤。

| 条件 | 破裂概率 |
| :--- | ---: |
| 2027 年无重大通胀冲击 | 12-18% |
| 三杀情景已经发生 | 35-45% |
| Henry Hub > $6/MMBtu 且 10y breakeven > 2.75% | 55-65% |
| Big 4 IG 推迟发行且 30y term premium 上行 50 bp | 60-70% |

因此，长久期美债不能被视为“自动保险”。它应被管理成有条件对冲：当冲击来自 AI ROIC 下修、云厂利润压缩和信用利差走宽时加仓；当冲击来自能源价格、财政期限溢价和再通胀时减仓。

## 五、三重尾部同时触发时的对冲有效性

三重尾部为：TSMC 地缘政治风险、Henry Hub 急涨、EU 主权云监管加速。对应工具为 CDX IG payer、TSM put、NG 远月看涨价差。

| 对冲工具 | 年化滚动成本（按 NAV） | 主要保护对象 | 压力收益能力 | 风控判断 |
| :--- | ---: | :--- | ---: | :--- |
| CDX IG payer / payer swaption | 5-10 bp | Big 4 IG 发行失败、信用利差走宽 | +20 至 +60 bp | **值得持有**，成本低、与三杀高度相关 |
| TSM 6m 25-delta put | 20-35 bp | TSMC 地缘跳空、CoWoS 订单折价 | +40 至 +120 bp，但路径依赖强 | **只适合 put spread / collar**，裸买长期持有成本过高 |
| NG 18-24m call spread | 8-18 bp | Henry Hub 急涨、电力成本冲击 | +30 至 +100 bp | **值得持有**，是本组合最干净的滞胀凸性 |
| 铜 put 或 collar | 3-8 bp | 中国/全球需求下行导致铜与股票同跌 | +10 至 +35 bp | 可作为补充，优先级低于 NG |

总对冲预算约 **35-55 bp NAV / 年**。在 Card 08 的基准超额收益 +400-600 bp 下，这个预算可以被 carry 与超额收益覆盖；但覆盖关系有两个前提：

1. **TSM 保护不能裸买长期看跌期权**：TSM put 是最贵且最容易被波动率回落侵蚀的保护腿。应改为 6 个月 put spread，或用 TSM 减仓 + AVGO/ETN/GEV 替代部分 beta。
2. **NG 保护必须用价差而非纯期货追涨**：天然气期货 roll 与季节性波动会吃掉收益，18-24 个月看涨价差更适合承接 BTM 燃气与 LNG 需求的右尾。

## 六、流动性压力：日度 VaR 以外的真实损失来源

本组合的流动性问题主要不是能否卖出 TLT 或 NVDA，而是压力时交易成本的非线性上升。

| 资产 | 正常日交易成本 | 压力日交易成本 | 风险点 |
| :--- | ---: | ---: | :--- |
| NVDA / MSFT / AMZN | 1-3 bp | 5-15 bp | 可交易，主要是 gap risk |
| TSM ADR | 3-6 bp | 20-50 bp | 台湾地缘新闻下 ADR 与台股时区错配 |
| GEV / VRT | 5-10 bp | 30-80 bp | 拥挤交易，估值下修时承接不足 |
| TLT / ZB | 1-5 bp | 10-30 bp | 期限溢价冲击时滑点放大 |
| NG 远月期权 | 10-25 bp | 50-150 bp | bid/ask 与隐波跳升同时出现 |
| CDR / 碳信用 | 50-200 bp | 可能无有效 bid | 不能作为日度流动性对冲 |

因此风控上必须把 CDR/碳信用从“可交易对冲资产”降级为“主题性小仓位”。它能表达碳稀缺，但不能在 99% / 1d VaR 里提供保护。

## 七、风险闸门与减仓顺序

我建议给 Card 08 的组合加四个硬闸门。

| 闸门 | 触发条件 | 动作 |
| :--- | :--- | :--- |
| 股债相关性闸门 | 上游股票 / 30y UST 60 日相关性 > +0.30，且 10y breakeven > 2.75% | 长久期 +300 bp 降至 +100 bp，停止第三批建仓 |
| 信用闸门 | Big 4 长端 OAS > 170 bp，或任一发行推迟 | CDX IG payer 翻倍；云厂下游降配维持；上游减 20% |
| TSM 地缘闸门 | TSM ADR 单日 gap > -8%，且台海/出口管制新闻确认 | TSM 主动仓位减半，保留 AVGO/GEV/ETN |
| 商品流动性闸门 | NG 远月隐波 > 95 分位，或 CDR 无连续报价 | NG 期权只减不加；CDR 停止新增 |

**最先应该砍掉的两个仓位**：

1. **长久期美债第三批仓位**：不是全部清空，而是取消从 +100/+150 bp 加到 +300 bp 的最后一批。触发条件是 Henry Hub 急涨、breakeven 上行、term premium 上行同时出现。理由是它在滞胀情景下从保险变成同向亏损。
2. **TSM 集中超配仓位**：先砍 TSM，而不是砍全部 Tier 1 上游。触发条件是 TSM 地缘风险、单月营收低于预期或 CoWoS 订单可见度下降。替代方向是 AVGO、ETN、GEV、短久期现金和 CDX IG 保护。

不建议优先砍云厂下游降配，也不建议优先砍 NG 远月看涨价差。前者是三杀情景中最稳定的收益来源，后者是滞胀相关性破裂时少数真正正凸性的工具。

## 八、对 Card 08 的修正建议

我支持 Card 08 的三点：

- 云厂下游 -250 bp 是正确的核心保护腿，尤其是 GOOGL/AMZN。
- NG + 铜 + CDR 的商品方向正确，但要把 NG 作为凸性、铜作为周期 beta、CDR 作为低流动性主题仓分开管理。
- CDX IG payer 是比单纯减持 Big 4 IG 更有效的尾部工具。

我否定或修正 Card 08 的两点：

- **长久期美债不能按日历分三批机械加仓**。第三批必须受通胀与相关性闸门约束。
- **Tier 1 上游不能只看订单硬化**。TSM 地缘政治风险会把“订单确定性”瞬间重估为“交付不可得性”，这类跳跃风险不能靠常规 VaR 捕捉。

## 九、最终风控指令

若按当前组合执行，我给出的风险预算上限为：

| 项目 | 上限 |
| :--- | ---: |
| 主动组合 99% / 1d VaR | **75 bp NAV** |
| 跳跃 event-VaR | **140 bp NAV** |
| 主动组合三个月最大回撤 | **650 bp NAV** |
| 年度对冲预算 | **55 bp NAV** |
| 单一 TSM 风险贡献 | **不超过主动 VaR 的 25%** |
| 长久期美债风险贡献 | **滞胀闸门触发后不超过主动 VaR 的 15%** |

只要这些上限被遵守，Card 08 的组合可以继续运行；若同时触发“TSM 地缘闸门 + 股债相关性闸门 + 信用闸门”，应把主动风险预算砍半，而不是试图用更多期权保护追补。

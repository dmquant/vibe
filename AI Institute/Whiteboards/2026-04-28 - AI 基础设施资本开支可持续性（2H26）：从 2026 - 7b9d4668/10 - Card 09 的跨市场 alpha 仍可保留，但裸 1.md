# Card 10｜首席风控官压力测试：中美 AI 基建配对组合的尾部 VaR、相关性破裂与流动性挤兑

## 结论先行

我对 Card 09 的交易主线采取 **stress-test** 立场：多 H/A 算力链、空 NVDA + 美国 IPP/送变电链的 1:1 美元中性组合，在常态波动下可以把 1-day 99% VaR 控制在约 7.6%–8.0%，但一旦发生“NVDA 单日 -8% 跳空、HFCAA 重启冻结 ADR 转板、美元/CNY 突破 7.40”三选二尾部组合，名义 VaR 会从预算边缘跳升至 **14.5%–17.0%**。最危险的不是 NVDA 下跌，而是 **HFCAA + CNY 破位**：中国多头腿同时承受估值折价、汇率折价和转板冻结折价，而美国空头腿可能因相对避险和空头回补反向亏损，美元中性在这一天失效。

在不降低总敞口的情况下，建议把尾部对冲预算设为净值的 **2.0%–2.8%/季度**，由三部分组成：NVDA 近月/3M put 或 put spread、ADR 转换/借券配额与 HSTECH/KWEB 类替代性保护、USD/CNH call 或 call spread。按建议规模执行后，最差二因子情景的 1-day 99% 名义 VaR 可压回 **7.3%–8.0%**；若不愿支付期权成本，则必须把 gross 从 200% 降至约 160%–170%，否则 8% VaR 预算不可信。

## 三选二尾部情景结果

### 情景 1：NVDA -8% + HFCAA 重启

这一组合表面上有利，因为 NVDA 空头盈利；但 HFCAA 会冻结 ADR 转板、放大中概和 H 股折价，导致多头腿亏损先于空头腿保护兑现。

| 指标 | 压力测试结果 |
|---|---:|
| 多头腿 mark-to-market | -8.0% 至 -11.0% NAV |
| 空头腿 mark-to-market | +4.8% 至 +6.2% NAV |
| 流动性与转板折价 | -2.0% 至 -3.5% NAV |
| 1-day 99% 情景 VaR | 7.0%–9.5% |
| 峰值至谷底最大回撤 | 13%–18% |
| 相关性破裂幅度 | P/L 相关性由约 -0.40 升至 +0.20，恶化约 60ppt |
| 主要风险 | ADR 无法及时转 H、港股集合竞价折价、A 股涨跌停导致对冲腿先变现 |

判断：该情景仍可承受，但只有在 NVDA 空头可正常维持且不触发强制减仓时才成立。若 prime broker 在 NVDA 跳空后要求降低单名集中空头，保护会被动减弱。

### 情景 2：NVDA -8% + USD/CNY 突破 7.40

这是“基本面方向正确但汇率吞噬 alpha”的情景。NVDA 空头提供凸性收益，但 CNY 破位会把 H/A 多头的美元收益折价，并引发海外资金对中国成长股的风险预算下调。

| 指标 | 压力测试结果 |
|---|---:|
| 多头腿 mark-to-market | -5.5% 至 -8.0% NAV |
| 空头腿 mark-to-market | +5.0% 至 +6.5% NAV |
| FX 与再平衡冲击 | -1.5% 至 -2.5% NAV |
| 1-day 99% 情景 VaR | 3.5%–5.5% |
| 峰值至谷底最大回撤 | 9%–13% |
| 相关性破裂幅度 | P/L 相关性由约 -0.40 升至 -0.05，恶化约 35ppt |
| 主要风险 | USD/CNH 期权波动率跳升、A/H 折价扩大、港股收盘后美股腿继续波动 |

判断：这是三组里最可管理的一组。NVDA 跳空对空头腿形成真实保护，关键是提前持有 CNY 期权；现货 FX 对冲在破位日通常执行成本高、滑点大。

### 情景 3：HFCAA 重启 + USD/CNY 突破 7.40

这是本组合的核心尾部风险。没有 NVDA 下跌保护时，组合可能出现“多头因中国风险折价下跌、空头因美国相对避险与回补上涨”的双腿同亏，美元中性失效。

| 指标 | 压力测试结果 |
|---|---:|
| 多头腿 mark-to-market | -10.0% 至 -13.5% NAV |
| 空头腿 mark-to-market | -2.5% 至 -4.5% NAV |
| 流动性、转板与汇率折价 | -3.0% 至 -5.0% NAV |
| 1-day 99% 情景 VaR | 14.5%–17.0% |
| 峰值至谷底最大回撤 | 24%–31% |
| 相关性破裂幅度 | P/L 相关性由约 -0.40 升至 +0.60/+0.75，恶化约 100–115ppt |
| 主要风险 | ADR 冻结、CNY 破位止损、南向拥挤交易反向流出、美国空头被回补挤压 |

判断：该情景否定了“1:1 美元中性自然防守”的假设。组合实际暴露不是净美元，而是 **中国政策/转板可行性 + CNY 路径 + 美国 AI 相对安全资产再定价** 的复合尾部。

风险预算判断：Card 09 的 18%–22% 六个月 alpha 目标，在常态路径下可以覆盖常态 VaR，但无法覆盖 HFCAA + CNY 的跳跃损失。若不加对冲，该组合的风险收益比从“可执行套利”降级为“方向正确但尾部超预算”。

## 多空两腿的流动性挤兑

### 多头腿：主要风险在 ADR 与港股，不在 A 股表面成交额

多头腿的压力来自三层叠加：

| 子模块 | 正常流动性 | 尾部折价 |
|---|---:|---:|
| H 股/港股 AI 算力 | 20% ADV 参与率可在 2–3 日完成 | HFCAA/CNY 共振时折价 150–300bp，若南向资金反向，可能扩大至 400bp |
| A 股算力/电网 | 成交额看似充足 | 涨跌停与陆股通额度/节奏限制导致可成交性下降，折价 100–250bp |
| 中概 ADR | 正常可交易但依赖转板预期 | HFCAA 重启时 ADR-H 股转换价值受损，折价 500–900bp，局部名称可能更高 |

组合层面，多头腿在最差一日的流动性损失约 **3.0%–5.0% NAV**。其中约一半来自 ADR 可替代性下降，而不是简单 bid-ask spread。

### 空头腿：NVDA 本身不缺流动性，IPP/送变电拥挤度更危险

空头腿的压力不是“买不到 NVDA”，而是：

| 子模块 | 风险点 | 尾部成本 |
|---|---|---:|
| NVDA 空头 | 单名集中、跳空后 prime broker 降低风险限额、期权偏度快速变化 | 若被动回补，损失/机会成本 80–150bp NAV |
| 美国 IPP/送变电空头 | 主题拥挤、borrow 费率上升、事件日回补 | 150–300bp NAV |
| 跨时区再平衡 | 亚洲收盘后美国腿继续跳动，净敞口隔夜偏离 | 50–120bp NAV |

空头腿在 HFCAA + CNY 情景下可能不提供保护，反而贡献 **2.5%–4.5% NAV** 的亏损，原因是美国 AI 基建链被视为相对安全资产，或因空头拥挤触发回补。

## 对冲方案：把名义 VaR 压回 8% 以内

### 1. NVDA put：把“空头线性收益”变成“跳空日凸性保险”

建议：

| 工具 | 建议规模 | 目的 |
|---|---:|---|
| 1M–3M NVDA 0.35–0.45 delta put 或 5% OTM put spread | 名义 30%–40% NAV | 在 NVDA -8% 日贡献 1.5%–2.5% NAV 的跳空收益 |
| 同步降低 NVDA 现货空头 | 从 60% NAV 降至 45%–50% NAV | 降低单名空头集中和被动回补风险 |

净效果：在 NVDA 参与的两组情景中降低 VaR 约 **1.2%–2.0% NAV**；在 HFCAA + CNY 情景中通过降低现货空头集中度减少约 **0.4%–0.8% NAV** 的回补风险。

### 2. ADR 配额与替代性保护：对冲“无法转板”的非价格风险

建议：

| 工具 | 建议规模 | 目的 |
|---|---:|---|
| 预留 ADR-H 转换额度、custodian/prime broker 预确认转板能力 | 覆盖 ADR 暴露的 60%–70% | 避免 HFCAA 日被迫以 ADR 折价成交 |
| HSTECH/KWEB/中概互联网或云计算篮子 put spread | 名义 25%–35% NAV | 对冲 ADR 折价和港股同步下跌 |
| ADR 多头上限 | ADR 不超过组合净值 20%–25% | 避免非可替代多头成为最大单点风险 |

净效果：在 HFCAA 参与的情景中降低 VaR 约 **3.0%–4.5% NAV**，并把多头腿单日可变现比例从 30%–45% 提高到 45%–60%。

### 3. CNY 期权：用 USD/CNH call 对冲破位而非靠现货止损

建议：

| 工具 | 建议规模 | 目的 |
|---|---:|---|
| 2M–3M USD/CNH 7.35/7.55 call spread | 名义 70%–85% 的 H/A/CNY 风险暴露 | 对冲 CNY 从 7.40 向 7.50–7.55 的跳跃 |
| 若波动率过高，改用 7.40 knock-in call 或零成本 collar | 名义 50%–60% 暴露 | 降低季度 carry 成本 |

净效果：在 CNY 破位参与的情景中降低 VaR 约 **2.2%–3.5% NAV**。这部分对冲是把最差情景压回预算内的核心，不应由事后现货美元买入替代。

## 对冲后压力测试

假设执行以下目标组合：NVDA 现货空头从 60% 降至 48%，买入 35% NAV 的 NVDA put/put spread；ADR 转板/借券配额覆盖 65% ADR 暴露，并持有 30% NAV 的 HSTECH/KWEB put spread；USD/CNH call spread 覆盖 75% 的 H/A/CNY 经济暴露。

| 情景 | 未对冲 1-day 99% VaR | 对冲贡献 | 对冲后 1-day 99% VaR |
|---|---:|---:|---:|
| NVDA -8% + HFCAA | 7.0%–9.5% | 3.8%–5.5% | 3.8%–5.5% |
| NVDA -8% + CNY >7.40 | 3.5%–5.5% | 3.0%–4.5% | 1.5%–3.0% |
| HFCAA + CNY >7.40 | 14.5%–17.0% | 7.0%–9.0% | 7.3%–8.0% |
| 三因子同时发生的参考压力 | 10%–13% | 6%–8% | 4.5%–6.5% |

对冲后仍需设置硬性风控阈值：

| 触发条件 | 动作 |
|---|---|
| USD/CNH 连续两日收于 7.40 上方 | 将 H/A 多头 gross 下调 15%，直到 CNH call delta 覆盖率恢复至 50% 以上 |
| HFCAA 正式重启且转板窗口暂停 | ADR 暴露降至净值 10% 以下，剩余中国多头迁移到 H/A 可成交篮子 |
| NVDA 单日跌幅超过 8% 且 implied vol 上升超过 8vol | 止盈三分之一 NVDA put，保留剩余保护以防连续跳空 |
| 美国 IPP/送变电空头 borrow 费率翻倍或可借量下降 50% | 用 ETF/期权替代单名空头，避免 forced buy-in |

## 最终风控意见

我不否定 Card 09 的跨市场 alpha 逻辑，但否定“裸 1:1 美元中性即可承受尾部”的执行假设。该组合的最大风险不是方向判断错误，而是 **相关性在政策与汇率冲击下从保护关系变为同亏关系**。未对冲状态下，最差三选二情景的名义 VaR 为 14.5%–17.0%，超过 8% 预算约 6.5–9.0ppt；对冲后的可接受版本必须同时满足三条：

1. NVDA 空头部分期权化，减少线性单名空头与 prime broker 回补风险。
2. ADR 暴露必须有事前转板/借券配额和替代性港股/ETF put 保护。
3. CNY 破位必须用 USD/CNH call 或 call spread 事前保护，不能把现货止损当成尾部对冲。

若交易团队接受 2.0%–2.8%/季度的对冲 carry，该组合可保留 2H26 核心配置价值；若不接受该成本，应把 gross 从 200% 降至 160%–170%，并把 ADR 暴露上限压至净值 15% 以下。

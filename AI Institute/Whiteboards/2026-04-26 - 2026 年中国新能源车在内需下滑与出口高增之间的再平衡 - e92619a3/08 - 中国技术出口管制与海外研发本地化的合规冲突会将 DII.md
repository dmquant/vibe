## Report

# Card 08：技术出口管制 × 研发本地化的合规冲突——DII 回流路径的"摩擦"而非"阻断"

> 立场：stress-test（针对 Card 7 ESG 分析师提出的"治理悖论"）
> 角色：合规官（Compliance Officer）

## 1. 核心判断（一句话）

中国"技术留根"出口管制与欧盟/巴西/东盟"研发本地化"补贴门槛之间的冲突**真实存在但并非死结**——它会通过结构成本与信息披露义务，将海外利润（DII）回流路径的有效净利率压低 **3–5 个百分点**，并把"是否合规可行"重新定义为头部车企（BYD、CATL、吉利）与二线玩家之间新的护城河，但**不会实质性切断 DII 回流**。Card 6 的"经常账户日元化"叙事仍然成立，只是斜率更平、波动更高。

## 2. 法规框架的双向夹击：到底在管什么、到底要什么

### 2.1 中国一侧的硬约束（出口端）
| 法规 / 文件 | 关键约束 | 对车企的实际影响 |
| :--- | :--- | :--- |
| 《中华人民共和国出口管制法》（2020.12 生效） | 对"两用物项、军品、核以及其他与维护国家安全和利益、履行防扩散等国际义务相关的货物、技术、服务"实施许可制 | 任何"技术出口"都需向商务部申请许可，含**技术资料、技术服务、研发协作** |
| 《两用物项出口管制条例》（2024.12 生效） | 整合此前分散的两用物项法规，明确**"视同出口"**——即境内向外籍人员转让技术也算 | 海外子公司向中方派驻研发人员、向当地工程师传授核心工艺，理论上均可能触发 |
| 《中国禁止出口限制出口技术目录》（2023.12 修订） | 新增/保留：**磷酸铁锂电池制备技术、稀土萃取分离技术、高压实密度正极、激光雷达核心算法**等 | EV 核心专利转移、海外建产线的工艺包出境，需要逐项申报 |
| 《技术进出口管理条例》 | 限制类技术合同需许可证、备案；专利许可若涉及限制类技术，**许可合同本身需审查** | LRS（许可+特许权使用费）模式并非"绕过出口管制"的免死金牌，反而强化合同审查 |
| 《境外投资管理办法》+ SAFE 37 号文 | 境外投资需发改委、商务部、外汇局三道备案；利润回流需做 ODI 真实性审查 | 海外利润若想以股息形式汇回境内，需路径合规、税务穿透清晰 |

### 2.2 主要东道国的本地化条款（落地端）
| 司法辖区 | 核心机制 | 研发/本地化具体要求 |
| :--- | :--- | :--- |
| **欧盟** | EU Foreign Subsidies Regulation (FSR, 2023.7 全面实施) + Net-Zero Industry Act (NZIA) + EU Battery Regulation Art.7 | 公开采购/补贴项目对境外补贴申报；电池厂需公开**碳足迹及供应链溯源**；关键技术合资若涉及国家安全审查 (FDI screening)，需附中国母公司股权与控制结构 |
| **巴西** | MOVER 计划（Programa Mobilidade Verde e Inovação，2024 实施） | 享受 IPI 减免需 R&D 支出 ≥**营收的 1.0%–2.0%**（视车型与减免比例阶梯），且必须在巴西境内或与巴西机构合作完成；有"创新加成"扣除 |
| **印尼/泰国** | TKDN（本地含量）+ 投资促进 BOI 优惠 | 享 EV 财税激励需达到 **40%（2026 提升）本地含量**，包含电芯、组装、部分电控；研发本地化非强制但是评分项 |
| **匈牙利/欧盟其他成员国** | 国家补贴（Important Project of Common European Interest, IPCEI）和 NZIA 招标 | 强补贴项目越来越要求**"非价格标准"**：网络安全、供应链多元化、ESG 评分——对中资有隐性筛选效应 |

> **关键观察：** 没有任何一个东道国直接强制要求"必须把核心源代码或电芯配方交出来"。它们要的是 **R&D 支出落地 + 本地工程师就业 + 部分产线工艺转移 + 数据/网络安全合规**。这给了中国车企操作空间。

## 3. 合规冲突的真实强度评估：三档分级

### 3.1 低冲突区（可通过结构设计化解）
- **巴西 MOVER 的 R&D 1–2% 营收门槛**：可以通过"在圣保罗设立应用工程中心 + 巴西工程师赴中国总部联合开发（费用通过转移定价计入巴西子公司）"满足。**核心配方不出境，仅做本地适配性研发**——不触发 PRC 出口管制限制目录。BYD 在巴伊亚、长城在伊拉塞马波利斯均已采用此模式。
- **东盟 TKDN 提升至 40%**：电芯本地组装、PACK 本地化即可大幅拉高比例，**电芯活性物质（正极/电解液）仍可由中国出口**——属于一般货物贸易，不进入"技术出口"范畴。

### 3.2 中冲突区（需付额外合规成本，可通过 LRS+黑盒交付化解）
- **欧盟 NZIA 的"非价格标准"** 中的网络安全、数据本地化、ESG 评分——要求海外厂上传足够的工艺数据以做合规审计。中方可通过：
  - **黑盒模块化交付**：电芯以"封装单元"出口，工艺包不公开，仅交付性能规格与回收方案；
  - **LRS 双层架构**：在欧盟设运营子公司（合规主体），技术 IP 留在香港或境内 SPV，子公司支付特许权使用费——需同时满足 PRC《技术进出口管理条例》许可备案与 OECD BEPS 2.0 第二支柱（**15% 全球最低税**，2024–2025 在欧盟、英国、韩国、日本陆续生效）的实质经营测试。
- **隐性合规成本**：合规法务 + 双语审计 + 专利信托 + 转移定价文档，估算每年占海外子公司营收 **0.8%–1.5%**。

### 3.3 高冲突区（结构无法完全化解，需要战略取舍）
- **欧盟 FDI 审查 + FSR 双重审查**：当中国母公司被要求披露"是否享受国内补贴"以及"董事会的政府背景"时，治理透明度无法被 LRS 架构绕开。**这是真正的"政治合规成本"**——可能导致并购被否（如 2023 年宁德时代在德国某并购案被推迟）或合资比例被强制调整。
- **L4+ 智驾算法、全固态电解质工艺**：若被列入 PRC《禁止出口技术目录》，则**任何形式的境外联合研发都不可行**，海外只能销售"降级版"产品（如 L2+ 智驾、液态/半固态电池）。这会直接拉低海外车型的高端化定价空间。

## 4. DII 回流路径的合规摩擦量化

将合规成本拆为三层：

| 摩擦类型 | 性质 | 估算占海外利润比 |
| :--- | :--- | :--- |
| 结构性合规成本（法务、审计、转移定价） | 持续性 | **0.8%–1.5%** |
| 全球最低税（BEPS Pillar Two, 15%） vs 当前实际税率（如匈牙利 9%、巴西经 MOVER 减免约 10–12%） | 一次性补税 | **2.0%–3.0%** |
| 数据/技术披露的隐性"降级"成本（不能卖最高端车型） | 收入端折让 | **1.0%–2.0%（车型 ASP 降低）** |
| **合计有效净利率压制** | | **约 3.8%–6.5%** |

> 与此同时，若海外子公司选择**不汇回利润、就地再投资**（巴西、匈牙利产能扩张），则避免了境内 ODI 项下的外汇申报和 5% 的预提所得税（部分 DTA 国家更低），**短期内 DII"账面增长但不现金回流"**——这恰好印证 Card 6 关于"经常账户日元化"的滞后特征。

## 5. 对 Card 7 ESG 论点的压力测试结论

| Card 7 ESG 论点 | 合规视角再校准 |
| :--- | :--- |
| 治理悖论会导致海外项目被排除出绿色金融名录 | **部分成立**。但车企已开始通过设立"欧盟独立 ESG 治理委员会"（如比亚迪欧洲、宁德时代德国/匈牙利）化解。融资成本溢价约 **50–80bps**，并非禁入 |
| 技术留根使本地补贴丧失 | **过度悲观**。东道国要的是**就业 + 税收 + 部分产线**，并非源代码。LRS+黑盒模式仍可拿到 70% 以上补贴 |
| ESG 表现决定 DII 稳定性 | **需补充：合规框架决定 DII 可行性，ESG 决定 DII 估值乘数**。两者是串联条件，缺一不可 |

## 6. 内幕交易/披露视角的额外提醒

作为合规官需指出几个被前几张卡片忽略的二阶风险：

- **港股/A 股双重披露义务**：BYD（1211.HK / 002594.SZ）、长城（2333.HK / 601633.SH）、宁德时代（300750.SZ / 3750.HK）海外重大产能投资、出口许可被否、被列入 EU FSR 调查名单等事件，**均构成《证券法》第 80 条与港交所《上市规则》13.09 项下的"内幕信息"**，须及时公告。2026 年合规事件密度上升，**信息披露违规风险显著抬升**。
- **跨境关联交易**：LRS 模式下母子公司之间的特许权使用费率、技术服务费率，是 A 股关联交易披露和**所得税转移定价的重点监管对象**。国家税务总局 2024 年起加强对"轻功能海外子公司"避税安排的关注。
- **出口管制许可的窗口期风险**：商务部许可证审批周期通常 **30–45 工作日**，部分敏感技术可能延至 90 日以上。对计划 2026H2 海外开工的项目，**许可时间表必须前置纳入项目里程碑**，否则可能延迟达产 1–2 个季度。

## 7. 综合判断：摩擦≠阻断，但护城河重塑

1. **DII 回流路径不会被合规冲突切断**，但有效净利率会被压制 4–6 个百分点；
2. **合规能力本身成为新的产业护城河**：BYD、CATL、吉利等具备完整法务/税务/合规团队的头部企业能承担这笔成本，而 SAIC-MG、二线动力电池厂可能在欧盟 FSR 审查、巴西 MOVER 申请、TKDN 升级中**反复碰壁**；
3. **Card 6 人民币"全球配置溢价"叙事的修正版**：人民币定价从单一出口顺差驱动→DII 驱动的转型确实发生，但 **DII 现金回流的滞后性、合规波动性**，使人民币汇率在 2026–2027 期间将呈现"中枢稳定、波动加大"的新形态，而非简单升值。
4. **真正的尾部风险**不是合规成本，而是**地缘政治升级触发的跨域制裁**——例如美方将中国头部车企列入 UVL/Entity List，或欧盟对中资 EV 子公司启动 FSR 深度调查并强制剥离。这是合规框架无法对冲、需在投资组合层面用尾部对冲工具处理的风险。

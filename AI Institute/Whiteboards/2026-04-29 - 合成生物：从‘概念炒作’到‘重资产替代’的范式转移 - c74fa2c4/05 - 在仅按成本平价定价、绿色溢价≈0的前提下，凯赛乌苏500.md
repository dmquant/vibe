# 第05卡 — 在仅按成本平价定价、无绿色溢价前提下，凯赛生物与CIBT的项目级单位经济与资本回报

- 分析师：材料行业分析师 (materials-analyst)
- 立场：**synthesize（综合）** — 在Brent $115/b、当前玉米现价（2026年4月东北现货RMB 2,500/t）和当前融资成本下，凯赛乌苏500kt一体化PA56项目在仅按成本平价定价的前提下，**勉强**能清出加权资本成本。项目税后IRR结构性落在7–9%区间，对应WACC约9–10%，仅在快速达产+油价高位双重条件下才是创造价值的资产；一旦达产偏慢或Brent跌破$95，项目即转为价值毁灭型。**达产节奏、而非绿色溢价，才是股价的核心摆锤。**
- 名词说明：本卡中"凯赛"与"CIBT"为同一集团的不同主体表述。**Cathay Industrial Biotechnology, Inc. (CIBT)** 是A股上市主体**凯赛生物 (688065.SH)** 的开曼母公司/IPO前法律前身。下文"凯赛"指上市公司及其当前在产资产；"CIBT"指落入同一并表资本开支盘子的在建产能管线（乌苏二期/三期、金乡瓶颈打通、太原长链DC）。两者单位经济同构，合并讨论。

## 2. 乌苏二期PA56成本拆解（成本平价基准）

### 2.1 戊二胺(DC5)玉米路线现金成本

- 化学计量比：每t DC5消耗约3.6t葡萄糖；玉米→葡萄糖收率约0.65 → **每t DC5消耗约5.5t玉米**。
- 玉米RMB 2,500/t（2026年4月黑龙江/吉林现货）：**RMB 13,750/t DC5 原料成本**。
- 酶、发酵公用工程、菌液纯化、副产DDGS抵扣：净 **+RMB 4,200/t**。
- DC5乌苏厂内现金成本：**约RMB 17,950/t**；玉米在RMB 2,300–2,700/t区间敏感度约±RMB 800/t。

### 2.2 PA56树脂现金成本

- PA56质量配比：DC5 0.46 + 己二酸(AA) 0.54 + 聚合工序。
- AA在Brent $115/b环己烷路线：RMB 11,500/t（2026年4月中石化/华鲁恒升参考价RMB 11,300–11,700/t）。
- 聚合能耗+人工+维护：RMB 1,400/t。
- **PA56现金成本 = 0.46×17,950 + 0.54×11,500 + 1,400 = 8,257 + 6,210 + 1,400 = RMB 15,867/t**（取RMB 15,900/t）。
- 加95%开工率下固定费用RMB 950/t → **完全成本约RMB 16,850/t**。

### 2.3 化石PA66参考价（Brent $115/b）

- HMDA现金成本（丁二烯ADN路线或丙烯腈路线）：在Brent $115/b下RMB 28,500–31,500/t。
- AA：RMB 11,500/t。
- PA66现金成本 = 0.43×30,000 + 0.57×11,500 + 2,000 = 12,900 + 6,555 + 2,000 = **RMB 21,455/t**（普通纤维级树脂；工程级再加RMB 2,500–4,000/t）。
- 国内纤维级PA66合同价2026年4月在RMB 22,300–23,500/t一线运行。**PA56 成本平价ASP区间 = RMB 21,500–22,500/t**，混合基准取 **RMB 22,000/t**（略折让以争取份额，与Card-04结论一致：内销零绿色溢价、欧洲出口仅€10–30/t）。

## 5. 达产节奏 = 股价摆锤

每个阶段晚6个月的爬坡曲线（25% / 55% / 80% / 95% → 15% / 35% / 60% / 80% / 95%）会把税后IRR从9.0%压到 **6.6%** —— **跌破WACC**。反之，加速到35% / 70% / 90% / 95%可推到 **10.8%**。市场会实时定价：

- 乌苏二期2026年下半年与2027年上半年的季度开工率披露，是688065.SH最重要的单一可定价数据。
- 2026年末开工率突破40% → 折现NPV每股增厚 **RMB 7–9元**（按WACC 9%、5.8亿股本测算）。
- 2026年末开工率<20% + Brent跌至$95 → 项目NPV转为 **每股负RMB 4–6元**；此情景下股价底由一期+长链DC业务（持续创现，每股RMB 32–35元账面等价底）支撑。
- 在仅按成本平价定价假设下，688065.SH合理价值的 **熊/基/牛区间：RMB 28 / 38 / 48元**，对照2026年4月现价约RMB 42元 —— 当前股价已price in基准爬坡，**不对称风险偏向下行**，触发条件是Brent破$100。

## 6. CIBT层（乌苏三期+金乡+太原）分项IRR

- 乌苏三期长链DC（DC10/12/13）：高毛利，ASP RMB 35–45k/t，现金毛利RMB 15–18k/t。资本开支RMB 16亿/50 kt → **税后IRR约22%**，即便在成本平价口径下也成立 —— DC10–DC13缺乏化石替代品（蓖麻油基癸二酸供应受限）。**这才是集团真正的IRR发动机**，部分对冲了PA56的IRR压扁。
- 金乡瓶颈打通：边际RMB 4亿换+30 kt DC5/PA56 → 税后IRR约14%（沿用既有公用工程）。
- 太原90 kt PA56（2025年末决策）：资本开支RMB 24亿，单位资本开支RMB 27k/t。缺乏乌苏的一体化深度，成本平价下税后IRR **仅约5–6% —— 跌破WACC**。**该项目在仅成本平价假设下不应推进。** 启动条件是：(i) 拿到欧洲绿色溢价长协（Card-04认为不可能）；(ii) Brent结构性站稳$115以上；(iii) 项目延期或缩规模。需紧盯FID披露。

集团**剩余约RMB 40亿增量资本开支**的混合税后IRR：**约9.5%**，被太原拖累，靠三期长链DC救回。

## 7. 与前序卡片的综合

- **Card 01**：3–6个月窗口，生物替代仅在精细化学品+农资中实操。✔ — 凯赛长链DC业务正是这条线；PA56属于长周期大宗替代。
- **Card 02**：6个月内PA56在纺丝/BCF/通用EP上挤出化石PA66，化石PA66 EBITDA池被压缩约40%。✔（销量层面）；本卡补充单位经济镜像 —— **PA56自己的EBITDA池也被压缩**，因为它只能通过低于PA66的价格来抢市场，自己的margin被锁死在化石平价天花板上。挤出方与被挤出方双双margin压扁，真正的破坏者是 **大宗化**，而不是溢价兑现。
- **Card 03**：6个月内汽车认证不会普遍压缩；只有低风险GF-PA56非关键件选择性加速。✔ — 这意味着高毛利工程级（ASP RMB 25–30k/t）对PA56仍是18–36个月之外的事，2026–2027爬坡只能靠纤维级RMB 22k/t定价 —— 与本卡基准设定完全一致。
- **Card 04**：欧洲绿色溢价池€10–30/t，中国内销近零。✔ — IRR表中已反映，约30 bp贡献。

合成图景：**剥掉绿色溢价叙事后，"合成生物攻入大宗聚合物"这个多头命题，本质上简化为一笔加杠杆赌注：押Brent高位 + 押乌苏二期爬坡快于中国首套化工大装置的历史曲线。** 凯赛管理层的执行力可信（一期约14个月达产），但赌注的宏观一半完全在它们控制之外。

## 8. 投资含义

- **688065.SH（凯赛生物）**：评级 **持有 / 谨慎正面**，下行不对称当Brent <$100时显著放大。建议用Brent overlay做仓位 —— 多头凯赛+USO看跌期权或空头油服配对，剥离爬坡α与油价β。
- **驱动股价的催化剂**：(i) 乌苏二期月度开工率披露；(ii) 任何与非中国OEM签订并以€或USD/吨明确锁定PCF溢价的多年期长协（若发生则Card-04偏熊，需重估）；(iii) 太原项目FID确认/取消（**取消反而是利好** —— 资本纪律信号）；(iv) 欧盟CBAM聚合物范围扩展评议结论（2026年后才落地，但消息流可前置定价）。
- **资本结构压力**：2025年末凯赛集团口径净负债/EBITDA约3.4×。若2026年走向熊市（Brent $95+爬坡偏慢），该比率推升至4.8–5.2×，进入债券契约预警区。需紧盯2026年Q4契约状态。
- **CIBT层项目（特别是太原）**：合成生物大宗替代叙事的失败信号 = 太原延迟至2027或被改造为纯长链DC。该信号 **对股价是利好**（资本纪律），**对宏观叙事是利空**（合成生物接管大宗聚酰胺受挫）。

## 9. 交接

下一棒分析师：**chief-strategist（首席策略师，primary）**。Cards 01–05已经勾勒出一个清晰的投资对象：合成生物大宗聚合物是一笔Brent依赖+爬坡执行依赖的窄IRR交易，而非结构性定价权故事。剩下的核心问题是**组合构建** —— 在2026年4月A股以多年期板块轮动溢价为"AI硬件+合成生物+军工"主题定价的环境下，多头PM应如何在Brent依赖约束下确定合成生物（凯赛688065、华恒生物688639、金达威002626、华熙生物688363）的仓位规模？又在哪个临界点上，把合生敞口轮动到同样受益于Brent上行但没有爬坡风险的上游石化（恒力、荣盛）反而更优？这是策略/配置问题，落在首席策略师的领域。若下一棒选specialist（如credit-analyst专门压力测试凯赛债券契约空间）也讲得通，但板块配置问题更主导，应优先由primary推进。

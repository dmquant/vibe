# 布尚贸易休战半年回顾：三重逆风与全球宏观稳定性评估

**分析师**：全球宏观分析师（global-macro）
**日期**：2026年5月6日
**卡片**：第1张，共10张 | 看板 6f4e1bd4-8bd4-4674-8773-f3635b1a2619

## 执行摘要

Trump与习近平的布尚峰会距今刚满六个月，这一近三年来最具实质意义的中美贸易降级协议正迎来首次真正的压力测试。2026年5月，三股独立的逆风同时汇聚：

1. **301条款产能过剩调查**：2026年3月11日，美国贸易代表办公室（USTR）针对包括中国在内的16个经济体，就制造业领域"结构性产能过剩"启动301条款调查。本周（5月5日至8日）公开听证会正在华盛顿举行——距离布尚框架于11月到期，剩余时间不足六个月。

2. **中东能源冲击**：2026年伊朗战争触发了自2022年俄乌冲突以来最严重的石油供应中断。能源价格预计全年上涨24%，在不利情景下Brent原油可能达到115美元/桶，峰值供应缺口约为每日1000万桶。

3. **美联储政策僵局**：FOMC在4月29日的会议上将利率维持在3.50%–3.75%不变，这是连续第四次按兵不动——核心PCE通胀仍粘滞在3.0%。市场目前定价2026年全年零次降息，首次降息最早至2027年第二季度。

在此背景下，中国2026年一季度GDP同比增长5.0%，4月制造业PMI大幅反弹至52.2，为2020年12月以来最高。但这些数据在相当程度上反映了企业抢在潜在关税重升前集中出口的"前置效应"，而非内需的持续改善。若三重逆风持续强化并相互叠加，IMF已经下调的2026年全球增长预测3.1%将面临进一步的下行风险。

**核心判断**：布尚协议在基准情景下有65%的概率维持至2026年11月到期——但"顺利续约、毫无摩擦"的概率已大幅下滑。宏观风险不在于某一单一触发事件，而在于若301条款建议恰好在续约谈判窗口前出炉，将使合作框架演变为对抗性最后通牒，这一危险的时序排列是当前最值得警惕的尾部风险。

---

## 1. 布尚协议的架构

2025年10月30日，在韩国釜山（Busan），Trump总统与习近平主席签署了自2020年1月第一阶段贸易协议以来最实质性的双边贸易协定，阻止了关税税率突破100%的迫在眉睫的升级螺旋。

### 核心条款

| 条款 | 具体内容 |
|---|---|
| 芬太尼关税削减 | 从20%降至10% |
| 报复性关税延续 | 10%税率延续一年至2026年11月10日 |
| 升级关税暂停 | 2025年4月实施的额外24%关税暂停执行 |
| 对华商品实际有效关税 | 约35%（2026年4月估算） |
| 稀土出口管制暂停 | 中国暂停针对美国终端用户的稀土、镓、锗、锑、石墨出口管制，为期一年 |
| 大豆采购承诺 | 中国承诺2026至2028年每年采购至少2500万吨美国大豆；2025年11至12月预采购1200万吨 |
| 打击芬太尼 | 中国同意采取有效行动遏制芬太尼流向美国 |

协议明确框架为期一年、预期续约。但其架构存在内在脆弱性：关税减免是"暂停执行"而非"永久取消"，双方均保留90天通知期内的升级选项。文本中没有内置任何永久性关税回滚机制。

### 前置效应：数据的另一面解读

布尚协议触发了明显的出口前置效应，这在中国宏观数据中清晰可见。4月制造业PMI的新出口订单分项指数录得50.3，这是**连续23个月低于荣枯线后首次重返扩张区间**，几乎完全归因于美国需求在低关税下的正常化恢复。中国2026年一季度GDP增速5.0%（较上季度4.5%加速）同样包含了相当程度的前置因素——出口商在潜在的11月关税不确定性来临前集中发货。

这一模式有据可查：IMF 2026年1月工作论文《从产品层面出口篮子视角解读中国2024–25年前置效应》显示，2025年一季度出口商在关税截止日前大幅提速，随后在4月升级落地时急剧萎缩。2026年第四季度临近布尚到期时，类似的前置与回落规律很可能重演。

## 2. 逆风一：301条款产能过剩调查

### 新一轮调查的启动

2026年3月11日——布尚会谈仅过去四个月——USTR根据《1974年贸易法》第301条款，针对16个经济体在制造业领域的"结构性产能过剩行为、政策与实践"启动调查。调查覆盖经济体：中国、欧盟、新加坡、瑞士、挪威、印度尼西亚、马来西亚、柬埔寨、泰国、韩国、越南、台湾地区、孟加拉国、墨西哥、日本和印度。

**调查涉及行业**（代表性列举）：铝、汽车、电池、水泥、化工、电子、电动汽车、玻璃、机械、塑料、机器人、卫星、半导体、船舶、光伏组件、钢铁及交通运输设备。

16个经济体并列的表述提供了政治掩护——"这不是专门针对中国"——但实质指向不言而喻。在上述绝大多数行业中，中国均占据最大的全球市场份额，而2018年至2025年的历次301条款行动已将中国确立为这一法律机制的核心目标。

### 程序时间轴——危险的时序排列

| 时间节点 | 事件 |
|---|---|
| 2026年3月11日 | 调查正式启动 |
| 2026年3月17日 | 公开评论期开放 |
| 2026年4月15日 | 书面意见截止 |
| **2026年5月5日至8日** | **华盛顿公开听证会** ← *当前进行中* |
| 2026年约第三季度 | USTR报告及关税建议预计出炉（通常历时6至9个月） |
| 2026年11月10日 | 布尚框架到期 |

核心风险在于**时序叠加**：若USTR关税建议在2026年9月至10月间出台，将与中美布尚续约谈判同步碰撞。北京届时将面临二选一的僵局——在续约布尚大框架的同时接受针对钢铁、电动汽车和光伏的新惩罚性关税，还是将新关税视为对协议精神的实质性违背，彻底离开布尚框架。

### 中方可能的反制工具

- **重启稀土出口管制**：暂停稀土、镓、锗、锑、石墨出口管制是中国在布尚会谈中最具价值的让步。一旦恢复，将直接冲击美国半导体、电动汽车和国防供应链。
- **削减大豆采购**：农产品采购承诺是杠杆工具；北京可暗示削减2027至2028年采购量。
- **反向调查**：商务部此前已多次启动反倾销、反补贴调查作为对等手段。

### 判断

**基准情景（65%）**：USTR将调查作为谈判筹码而非立即实施升级。关税建议出台但处于"行政暂停"状态，等待外交谈判。布尚框架以修订形式延续——美方获得中国在电动汽车市场准入和知识产权方面的额外承诺；中方获得关税稳定的延续。

**不利情景（25%）**：针对电动汽车和光伏的301条款关税建议在11月前落地。北京视此为违背诚意，有条件暂停布尚协议中的稀土条款。协议条款的部分逆转向市场释放负面信号。

## 3. 逆风二：中东能源冲击

### 冲击规模

2026年伊朗战争引发了自2022年俄乌冲突以来规模最大的石油供应冲击。对能源基础设施的打击以及霍尔木兹海峡的航运中断——该海峡承担全球约35%的海运原油贸易——在冲击峰值期每日削减全球原油供应约1000万桶。

世界银行《大宗商品市场展望》（2026年4月）及IMF《世界经济展望》（2026年4月）关键数据：

| 指标 | 2026年预测 |
|---|---|
| 能源价格涨幅 | 较2025年+24% |
| Brent原油（不利情景） | 最高可达115美元/桶 |
| 大宗商品整体价格涨幅 | +16% |
| 化肥价格涨幅 | +31%（尿素+60%） |
| 发展中国家通胀均值 | 5.1%（较战前预测高1个百分点） |
| IMF全球增长预测 | 3.1%（较2026年1月预测下调） |
| IMF全球整体通胀 | 4.4% |

### 对中美宏观的传导路径

**中国特定渠道**：
- 中国约75%的原油需求依赖进口。以115美元/桶Brent（较冲击前约75美元/桶基准）计算，中国年度石油进口账单增加约1500亿至1800亿美元，相当于GDP的约0.8%至1.0%，直接压缩经常账户顺差并施压人民币汇率。
- 钢铁、铝、水泥、化工等高能耗制造业面临严重利润收缩，削弱了中国贸易竞争力论据的核心支柱。
- 化肥价格大涨（+31%）直接推高农业生产成本，在CPI仍接近零附近温和运行之际加大了PPI上行压力。

**美国特定渠道**：
- 能源价格传导是核心PCE通胀持续粘滞于3.0%的主要原因。美联储4月29日声明明确将"全球能源价格近期上涨"列为通胀居高不下的因素之一。
- 汽油价格上涨实际上是对美国居民消费支出征收了一笔隐性税，即便劳动力市场依然具有韧性，也制约了消费扩张。

**全球需求萎缩**：
- 世界银行预计70%的大宗商品进口国和60%以上的出口国将录得低于2026年1月预测的增速。
- 发展中市场对中国出口品——尤其是消费品、建材和工业设备——的需求将疲软，部分抵消布尚框架正常化带来的美国需求修复效应。

## 4. 逆风三：美联储政策僵局

### 4月29日按兵不动

FOMC在2026年4月29日的会议上将联邦基金利率目标区间**维持在3.50%–3.75%不变**，连续第四次暂停调整。声明措辞：

> *"经济活动持续稳健扩张。就业增长保持低位，通胀依然偏高，部分反映了全球能源价格的近期上涨。"*

这是自2022年通胀激增以来美联储声明中最直接提及能源价格的一次表述。同期发布的3月《经济预测摘要》（SEP，即"点阵图"）显示，FOMC参与者中值仍预计2026年降息一次25bp——但市场已不再认同。

### 市场定价与FOMC点阵图对比

| | FOMC中值SEP（2026年3月） | 市场定价（2026年5月5日） |
|---|---|---|
| 2026年底利率 | 3.25%–3.50%（降息一次25bp） | 3.50%–3.75%（全年零次降息） |
| 首次降息时间 | 2026年第四季度 | 2027年第二季度或更晚 |
| 核心PCE年底 | 约2.7% | 约3.0%（市场预期无实质进展） |

### 新主席接任风险

另一复杂因素：Trump提名的继任者预计将于2026年6月接替Powell出任美联储主席，该提名人的独立性信号与通胀容忍度将成为2026年第二季度最关键的市场事件。三种情景分别对应：
- **维护独立性**：提名人明确背书现行数据依赖框架；市场反应中性。
- **模糊鸽派表态**：市场解读为暗示降息提前；通胀预期骤升；反而形成悖论式的金融条件收紧。
- **可信鹰派立场**：提名人表示能源驱动的通胀超标必须应对；利率维持高位时间更长，全球利率敏感资产受压。

情景二对全球宏观稳定性危害最大，因为它将短期利率乐观预期与中期通胀怀疑主义相结合——这是对新兴市场货币和债券市场最具破坏性的组合。

### 对全球金融条件的溢出效应

美联储按兵不动叠加美元走强，DXY持续维持高位，由此带来：
- **亚太区资金流向逆风**：较高的美国利率吸引资金回流，压制新兴市场货币和主权债券收益率。
- **人民币/美元**：人民币承受温和但持续的贬值压力。PBOC通过±2%日波动区间管理汇率，但随着石油进口成本上升和外汇储备部分消耗，均衡汇率已向USD/CNY 7.25–7.40方向移动。
- **新兴市场债务偿还**：以美元计价的债务本地货币偿还成本上升，恰在增速放缓之际压缩了反周期财政政策的空间。

## 6. 投资含义

**固定收益**：低配美国国债。能源驱动的通胀持续性以及美联储政策不确定性，建议在中东局势明朗之前不宜加大久期敞口。收益率曲线5年至7年段对"高利率持续更久"的重新定价风险最为敏感。

**权益资产**：AI/科技板块叙事（"技术持续跑赢关税和传统宏观驱动因素"）仍是权益市场主导主题。维持AI/科技超配，对抗工业品、化工和材料板块中受中国供应链关税不确定性影响的标的。

**外汇**：美元区间高位运行。人民币面临来自石油进口账单和贸易不确定性的温和贬值压力，目标区间USD/CNY 7.25–7.45。日元在全球增长放缓背景下日本央行收紧意愿不强，持续承压。

**大宗商品**：霍尔木兹海峡中断明朗之前继续超配能源。农业大宗商品敞口因中国每年2500万吨大豆采购承诺而具有特定上行潜力——这是布尚文本中相对硬性的义务条款。

**新兴市场**：偏好大宗商品出口型新兴市场（中东、巴西、部分非洲国家），规避大宗商品进口型新兴市场（东南亚、南亚）。优选外汇储备充裕、美元计价债务有限的主权信用。

## 7. 重点跟踪信号

1. **301条款听证会结果（2026年5月5日至8日）**：重点关注USTR证人是否将这些调查描述为"与布尚互补"（建设性表述）还是"针对布尚未解决的结构性问题的必要举措"（升级表述）。

2. **中国对霍尔木兹海峡局势的回应**：中国在中东能源供应稳定方面具有战略利益。任何解放军海军动向或外交调停信号，均表明北京正在激活地缘政治杠杆。

3. **PBOC 5月政策利率决定**：能源驱动的PPI可能由负转正，PBOC的降息空间已有所收窄。5月按兵不动将证实PBOC当前政策排序侧重于出口需求而非国内通胀减压。

4. **美联储主席提名人公告**：2026年第二季度全球金融条件最高不确定性事件。重点关注提名人关于r*（自然利率）的公开立场及对关税驱动通胀的判断。

5. **霍尔木兹海峡油轮过境数据**：IEA《石油市场报告》的实时航运数据（每周跟踪）是判断能源供应中断是否趋于缓解的最早领先信号。

## 8. 结论

布尚协议换来了和平，但并非确定性。时至今日，布尚框架同时面临三股独立逆风的考验——一场本周正在积极进行公开听证的美国贸易调查，一场历史性规模的中东能源冲击，以及一个被粘滞通胀死死钉住于3.50%–3.75%的美联储——这三者单独来看均在可控范围内，合力则代表对全球宏观秩序的实质性稳定性测试。

基准情景（60%概率）认为布尚框架将维持至11月并以修订形式延续。但不利情景已上升至25%——这对一个"美国关税重升与能源冲击同步叠加、制造出无论美联储还是PBOC均无明确政策工具应对的2026年下半年滞胀格局"的情景而言，是不可忽视的非小概率事件。

**未来30天最重要的单一信号**：USTR 301条款听证会证词是否将这些调查定性为"与布尚双边框架互补"还是"凌驾于布尚之上"。这一定性——本周即可观察——将决定三重逆风是否汇聚成系统性冲击，还是消散为可管理的双边摩擦。

**底线判断**：维持全球宏观审慎立场。降低对中美贸易摩擦升级和能源驱动通胀持续性最为敏感的资产敞口。密切跟踪301条款时间轴和美联储主席交接动态。

## Summary

本卡对eVTOL能源减缓工具进行了压力测试。虽然深圳等地的峰谷价差（~1.2元/kWh）和需量费（42元/kW）为储能提供了对冲空间，但‘载重-成本悖论’是核心瓶颈：兆瓦级储能系统产生的巨大荷载（500+ PSF）远超标准楼顶承载力，其建筑加固成本将使场站建设费翻倍。此外，绿电合约目前含有环境溢价，短期内反而推高了成本底座。这些工具本质上是以高额Capex替代Opex，未能改变电力成本栈（约3-6元/人公里）对票价目标的压制。

## Report

# 研究报告：低空网络的电力市场暴露与能效减排工具的局限性

## 1. 执行摘要：减缓策略的悖论
本报告对以下假设进行了压力测试：eVTOL（电动垂直起降飞行器）运营商是否能通过“综合能源系统”（IES）——即结合绿电合约（PPA）、分布式储能（BESS）和需求响应（DR）——来抵消前序研究（卡04）中识别出的高昂配网成本。

**核心结论：** 虽然在技术上可行，但这些工具本质上是**以资本支出（Capex）替代运营支出（Opex）的权衡**。它们将波动的电费账单转化为了僵化、高昂的基础设施资产。城市楼顶起降场的“载重-成本悖论”是核心瓶颈：容纳兆瓦级储能和高功率充电设备所需的建筑结构加固成本，可能使场站建设费增加50%至100%，从而抵消了通过电价套利获得的运营节省。

---

## 2. 减缓工具分项压力测试

### 2.1 分布式储能（BESS）：“载重-成本悖论”
为了对冲1MW+的峰值需量电费（深圳约为42-48元/kW·月），起降场需要至少500kWh至1MWh的储能系统。
- **财务现实：** 深圳峰谷价差约1.2元/kWh。500kWh储能每月可产生约1.8万元套利收益，并节省约2.1万元需量电费。总节省：约3.9万元/月。
- **结构现实（失效点）：** 500kWh的集装箱式储能系统重达15-25吨，产生的荷载为500-1,000 PSF（磅/平方英尺）。标准商业楼顶的额定荷载仅为40-50 PSF。
- **结果：** 在CBD楼顶加装钢梁架（Dunnage）并加固支柱以承载此重量，将产生20万至100万元的额外前期成本。这使储能投资回收期从2.5年拉长至6-8年以上，对于高风险、高增长的eVTOL初创企业而言，这一回报周期难以接受。

### 2.2 需求响应（DR）：“刚性时刻表的冲突”
虚拟电厂（VPP）和需求响应承诺起降场可以通过在电网高峰期“不充电”来获得补偿。
- **冲突点：** eVTOL的用电高峰与电网高峰（10:00-12:00, 14:00-19:00）高度重合。高净值通勤者不会为了运营商节省500元电费而接受航班延误。
- **局限性：** 如果没有巨量（且沉重）的储能作为缓冲，eVTOL运营商将成为“价格接受者”。为了维持机队利用率所需的15分钟周转时间（卡02），他们必须支付最高的“尖峰”电价。

### 2.3 绿电合约（PPA）：“环境溢价”而非“成本降低”
运营商希望通过绿电采购降低成本。
- **市场现实（中国）：** 绿电目前是一种**溢价产品**。在南方电网，绿色电力证书（GECs）在市场价基础上增加了约0.03-0.05元/kWh的溢价。
- **成本影响：** 对于消耗100 kWh的50公里航程，“绿电标签”增加的直接成本微乎其微（约5元），但它对降低基础电价或兆瓦级需量费毫无帮助。这是营销成本，而非降本工具。

---

## 3. 财务影响：累计成本栈

基于累计研究（卡01-05），电力减缓层引入了自身的隐性成本：

| 成本元素 | 影响 (元/人公里) | 成本性质 |
| :--- | :--- | :--- |
| **电网电能（高峰期）** | 0.8 - 1.5 | Opex (变动) |
| **需量电费 (1MW)** | 1.0 - 2.5 | Opex (固定) |
| **储能系统折旧** | 0.5 - 1.2 | Capex -> Opex |
| **建筑结构加固** | 0.4 - 1.0 | Capex (隐性) |
| **总电力成本栈** | **2.7 - 6.2** | |

**结论：** 如果仅电力和基础设施层就消耗了**3-6元/人公里**，那么实现**5-6元/人公里**的总票价目标（卡01）在数学上是不可能的。

---

## 4. 最终判断：压力测试结果
“综合能源”策略对城市eVTOL网络而言并非**净成本节省工具**，而是一种**波动管理工具**。它保护了电网免受飞行器冲击，也保护了运营商免受电价飙升影响，但其代价是锁定了高昂的前期资本成本，进一步推高了盈亏平衡载客率和票价底线。受限于地面配套电池的物理重量和财务沉重度，“空中滴滴”的愿景依然道阻且长。

# 低空经济A股配置：在30元/人公里现实下的阿尔法与‘接盘侠’陷阱

## 1. 执行摘要：估值与现实的巨大分歧
基于前序卡片（01-06）的多层成本审计，eVTOL航线的真实盈亏临界点在 **18-32元/人公里** 之间，而目前A股市场定价逻辑普遍基于 **6元/人公里** 的“促销价”预期。这种3-5倍的认知差导致了严重的定价错误。配置者必须从“主题炒作”转向“瓶颈价值”，识别谁是真正的阿尔法，谁是错误定价的“接盘侠”。

## 2. 细分环节分析：谁配得上溢价估值？

### A. “阿尔法”环节：ATM/UTM 与数字基础设施（控制面）
- **核心逻辑：** 与重资产的起降场不同，数字化空中交通管理（ATM）和无人驾驶交通管理（UTM）软件具有类似SaaS的可扩展性和极高的准入护城河。
- **溢价理由：** 随着飞行密度增加，冲突解决的复杂度呈指数级增长。软件供应商是“收单员”，不承担飞机或电池的重资产 Capex，且具备跨地域扩张能力。
- **A股标杆：** **莱斯信息 (688631.SH)**、**新晨科技 (300542.SZ)**、**中科星图 (688568.SH)**。

### B. “瓶颈”环节：长循环寿命电池与核心航电
- **核心逻辑：** 卡片02已证明，电池折旧占到运营成本的约50%。谁能将循环寿命从1,000次提升至2,500次以上，谁就是“6元/人公里”梦想的敲门砖。
- **溢价理由：** 这是产业链的“阿喀琉斯之踵”。具备高能量密度、高安全性的固态/半固态电池供应商将捕获价值链中最丰厚的一块。
- **A股标杆：** **宁德时代 (300750.SZ)**（固态布局领先）、**孚能科技 (688567.SH)**。

### C. “接盘侠”陷阱：起降场开发商与重资产基建
- **核心逻辑：** 卡片03与05揭示了“载重-成本悖论”。建筑加固的高额Capex和极低的初始吞吐量（<4架次/小时）使这些资产极易沦为坏账。
- **回避理由：** 这是典型的公用事业资产，却带有高科技风险且缺乏流量保障。在2027-2028年的融资墙面前，其现金流压力最为沉重。
- **A股现状：** **回避** 纯粹依赖起降场建设、运营且无多元化现金流的标的。

### D. “高波动”环节：OEM 整机厂（eVTOL 组装）
- **核心逻辑：** 目前市场按“软件定义汽车”估值，但其实际是“硬件定义航空”。
- **谨慎理由：** 适航取证的延误和每座600-1800元的结构性亏损（卡片06）意味着多数企业会烧光现金。仅推荐具备极强政府背书或大规模工业化量产能力的“国家队”。
- **A股标杆：** **万丰奥威 (002085.SZ)**（通过钻石飞机技术迁移）、**中直股份 (600038.SH)**。

## 3. 2027-2028“融资墙”应对策略
随着第一批示范航线在缺乏巨额补贴的情况下难以盈利，“幻觉期”将宣告结束。
- **2026年前策略：** 做多“铲子”环节（UTM、适航检测、航电）。
- **2027年后策略：** 剩者为王。做多已通过TC取证、且具备年2,000小时以上商业飞行实绩的幸存者。

## 4. A股配置最终评级

| 配置建议 | 细分板块 | 核心 rationale |
| :--- | :--- | :--- |
| **做多 (Long)** | **UTM/ATM 与核心航电** | 高扩展性、高准入门槛、低Capex强度。 |
| **做多 (Long)** | **长循环寿命电池** | 解决第一大成本瓶颈，技术壁垒极高。 |
| **中性 (Neutral)** | **多元化 Tier-1 零部件** | 稳健但阿尔法不足，依赖整机厂排产。 |
| **回避 (Avoid)** | **纯起降场/基建** | 资产过重、收益率低、结构性成本（加固费）过高。 |
| **回避 (Avoid)** | **低端 OEM 组装厂** | 烧钱过快、面临“融资墙”风险、缺乏取证护城河。 |

**核心结论：** A股市场目前错误定价了“飞行的梦想”（整机与航线），却低估了“飞行的算力与化学”（UTM的扩展性与电池的循环寿命）。做多算力与化学，回避梦想。

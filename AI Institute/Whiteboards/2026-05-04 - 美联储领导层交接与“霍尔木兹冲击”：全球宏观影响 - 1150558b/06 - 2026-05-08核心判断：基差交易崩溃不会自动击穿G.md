# 第06卡 - 美债基差交易崩溃下的G-SIB抵押品链条压力测试

**工作日期：** 2026-05-08  
**分析师：** Chief Risk Officer [Reviewer]  
**主题：** 基差交易崩溃对G-SIB FICC部门与抵押品链条的系统性冲击  
**立场：** 压力测试 - 支持第05卡关于美债市场非线性脆弱性的判断，但细化传导路径

截至 **2026-05-08**，我的风控判断是：美债现金-期货基差交易的无序平仓，本身未必会演变为纯粹的G-SIB偿付能力危机，但很容易演变为抵押品链条危机。在严重但合理的压力情景下，全球G-SIB的FICC相关总损失约为 **450亿-900亿美元**，发生窗口为5-10个交易日；若CCP、变动保证金收取和回购流动性工具仍能运转，银行最终留存净损失约为 **250亿-550亿美元**。在尾部情景下，若数家大型相对价值客户在抵押品完成变现前违约，总损失区间将扩大至 **1000亿-1300亿美元**。真正的约束不是美债丧失信用抵押品属性，而是同一批美债抵押品需要同时支撑回购、期货保证金、互换变动保证金、FX抵押品和客户赎回。

## 2. 压力情景与损失估算

压力假设沿用第05卡的“收益率曲线闪崩”市场冲击，并叠加G-SIB资产负债表行为。

| 风险驱动 | 2026-05-08压力输入 | 损失渠道 |
|---|---:|---|
| 美债基差交易组合 | 估算总名义规模1.2万亿美元；其中30-40%被迫平仓 | 现金美债抛售、期货买回、回购解约 |
| 利率波动 | MOVE连续48小时以上高于150 | VaR突破、CCP与交易所保证金上调 |
| 收益率冲击 | 5年期板块盘中跳升60-90 bp；10年期先升35-55 bp，随后受避险买盘逆转 | 交易商库存盯市与对冲滑点 |
| 回购压力 | SOFR-EFFR利差扩大10-15 bp，期限回购稀缺，haircut从0-2%升至4-6% | 融资成本、抵押品缺口、清算损失 |
| 期货保证金 | 美债期货初始保证金上调40%，同日变动保证金大幅增加 | 客户清算短缺与现金囤积 |
| 跨资产冲击 | DXY向108-115超调，油价风险保证金上升，信用beta走阔 | FX、大宗商品、信用和权益抵押品追缴 |

### G-SIB FICC总损失瀑布图

| 组成部分 | 估算总损失 | 形成原因 |
|---|---:|---|
| 回购缺口与抵押品清算 | 120亿-280亿美元 | G-SIB为杠杆对冲基金的有担保融资提供主要资金。当30-40%的基差交易平仓叠加haircut上调和市场深度下降时，抵押品变现金额可能穿透回购保护。 |
| 期货清算与客户保证金短缺 | 50亿-120亿美元 | FCM和清算会员会收取保证金，但跳空行情可能使部分杠杆客户无法在同日支付变动保证金，发生移仓或平仓前的短缺。 |
| 交易商库存、基差对冲与做市盯市 | 150亿-300亿美元 | 客户去杠杆时，美债、互换和期货交易台会被迫暂时持有风险。5年期中段流动性黑洞会把小幅基差滑点放大为显著盯市损失。 |
| 融资估值、XVA、流动性准备与法律平仓成本 | 80亿-150亿美元 | 融资利差扩大、抵押品争议上升，交易台需要为回收延迟计提准备。 |
| 跨资产保证金反噬 | 50亿-100亿美元 | FX carry、油价套保、信用指数和权益波动率的保证金追缴使用同一现金池。 |
| **合计** | **450亿-900亿美元总损失** | 相当于在一个集中的压力窗口中抹去G-SIB合计FICC多个季度收益。 |

在CCP保证金瀑布有效、Fed回购流动性快速释放且没有主要prime brokerage客户违约的情况下，扣除对冲、回收和抵押品出售后，留存损失约为 **250亿-550亿美元**。若尾部情景出现3-5家大型客户违约、抵押品争议和结算延迟，总损失估算升至 **1000亿-1300亿美元**。这不一定会耗尽G-SIB整体资本，但可能带来 **20-45 bp** 的合计CET1压力；对FICC占比较高的交易商，压力可能达到 **70-120 bp**。

## 3. 抵押品链条为何断裂

链条中有五个薄弱环节。

| 环节 | 正常功能 | 压力下行为 |
|---|---|---|
| 对冲基金回购 | 美债为基差交易现金多头腿融资。 | 回购资金方缩短期限、上调haircut、拒绝抵押品替换。 |
| 期货清算 | 期货空头腿每日保证金结算。 | 波动率跳升时，变动保证金和初始保证金同步增加，现金必须立即到账。 |
| 交易商资产负债表 | G-SIB中介回购、期货、互换和现金美债。 | 内部压力限额迫使交易台在客户最需要流动性时收缩授信。 |
| 抵押品再使用 | 美债在prime brokerage、回购、FICC和OTC保证金池之间流转。 | 抵押品因隔离、争议或更高haircut而放慢流转速度。 |
| 跨资产保证金 | 客户把现金和HQLA视为统一流动性缓冲。 | 利率损失触发FX、大宗商品、信用和权益波动率保证金追缴。 |

系统并不是因为美债信用质量下降而失灵，而是因为抵押品流通速度下降。即使一家基金持有美债，只要这些美债已被质押、被提高haircut，或必须在回购结算前用于期货变动保证金，该基金仍可能违约。

## 4. VaR与尾部风险解释

标准10日99% VaR会低估这类事件，原因有三点：

1. **基差收敛假设失效。** 该头寸通常按现金债与期货临近交割时收敛建模。但在压力期，融资腿、保证金腿和交割选择权的重估速度快于现金-期货价差收敛。
2. **抵押品相关性转为错向。** 多头美债、空头期货、回购融资和美元融资并非相互独立。同一个波动率冲击会同时推高保证金、压缩回购能力并降低清算深度。
3. **流动性损益压倒价格损益。** 损失不只是收益率变动，还包括买卖价差扩大、抵押品替换失败、客户平仓和资产负债表占用成本。

因此，从首席风控官视角看，关键压力指标是 **liquidity-at-risk**：同日需要覆盖的回购haircut、期货变动保证金、OTC互换变动保证金、FX抵押品和客户赎回现金需求。在严重情景下，全系统一周内流动性追缴规模可能达到 **2500亿-4000亿美元**，即使最终信用损失远小于这个数字。

## 5. 跨资产保证金传导

对本卡核心问题的回答是 **会**：G-SIB抵押品链条断裂会触发跨资产类别的追保压力。

传导顺序如下。

| 步骤 | 触发点 | 跨资产影响 |
|---|---|---|
| 1 | 美债基差基金收到变动保证金和回购haircut追缴。 | 抛售on-the-run美债、agency MBS、美债ETF和流动性较好的IG信用。 |
| 2 | G-SIB prime交易台下调融资额度。 | 宏观基金削减FX carry和EM rates头寸以筹集美元。 |
| 3 | 霍尔木兹冲击下油价波动仍高。 | 商品套保主体和CTA保证金上升，与利率基金争夺同一现金池。 |
| 4 | 交易商减少资产负债表使用，信用beta重定价。 | CDX/iTraxx利差走阔，HY ETF成为现金来源。 |
| 5 | 去风险化冲击股指期货和期权，权益波动率保证金上升。 | 基金卖出流动性权益并降低总敞口，形成更广泛的risk-off冲击。 |

这也是该压力具有系统性的原因：即使交易商本身仍具偿付能力，也会通过重新定价资产负债表、提高haircut、拒绝向客户提供期限流动性来传递稀缺性。

## 6. 风控措施与政策阈值

首席风控官应立即执行的措施：

| 控制项 | 阈值 | 行动 |
|---|---:|---|
| 客户集中度 | 前10大RV客户超过有担保融资敞口的35% | 冻结额度上调，并要求每日披露流动性。 |
| 回购haircut冲击 | 48小时内美债回购haircut扩大200 bp或以上 | 对低流动性抵押品仅提供期限融资，并要求现金IM附加。 |
| 期货变动保证金压力 | 同日变动保证金超过5日均值2倍 | 要求客户预先补足保证金，或下调期货give-up能力。 |
| 资产负债表稀缺 | 交易台资产负债表使用率超过90%，且买卖价差翻倍 | 从维护做市份额切换为资本保护。 |
| 跨资产流动性 | 客户现金加未质押HQLA低于未来3日预计变动保证金 | 在保证金违约前要求主动去杠杆。 |

政策触发阈值：

| 信号 | 系统性含义 |
|---|---|
| SOFR-EFFR利差连续两个交易日高于20 bp | 回购市场在配给现金，而不只是重新定价。 |
| FICC sponsored repo失败率上升且haircut同步提高 | 抵押品管道的流通速度下降。 |
| 5年期on-the-run/off-the-run利差扩大至类似2020年3月压力区间之外 | 交易商中介功能受损。 |
| 三家或以上主要prime broker同时向客户发出降杠杆通知 | 私人资产负债表约束正在放大冲击。 |
| DXY上升同时美元融资basis走阔 | “美元真空”正在从利率市场扩散至FX融资。 |

## 7. 底线判断

我支持第05卡关于基差交易去杠杆会扭曲美债收益率曲线的警告。我的压力测试修正是：最危险的损失路径不是某一家交易商的方向性美债库存，而是连接对冲基金、G-SIB回购交易台、期货清算、互换、FX prime brokerage和商品保证金的抵押品链条。

G-SIB FICC损失区间为 **450亿-900亿美元总损失**、**250亿-550亿美元留存净损失**；若大型客户在抵押品完成变现前违约，尾部总损失可达 **1000亿-1300亿美元**。更强的结论是，抵押品链条断裂可以制造 **2500亿-4000亿美元** 的一周内跨市场流动性需求。即使银行资本总体仍然完好，这一规模也足以触发跨资产追保，并可能迫使Standing Repo Facility扩容使用，或采取临时性美债市场流动性措施。

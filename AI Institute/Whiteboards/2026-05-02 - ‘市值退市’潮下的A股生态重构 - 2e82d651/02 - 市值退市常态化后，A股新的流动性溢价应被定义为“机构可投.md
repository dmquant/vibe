**分析师角色：** 因子研究员

**本卡主题：** 市值退市后的新流动性溢价因子拆解  
**立场：** 支持  
**使用的前序信息：** 本次运行环境中，提示列出的 live workspace 文件不存在，因此 Card 01 的内容依据用户提示快照重建。Card 01 的核心判断是：A股正在从散户驱动的上市溢价市场，转向机构过滤的流动性溢价市场。

## 核心判断

可以。用市值缓冲、交易流动性、ETF纳入概率、融资资格、解禁压力和ST/退市风险标签这六个模块，可以量化“流动性合格股”和“流动性受损股”之间的收益差。

我的支持是有条件但较强的：这不是简单的小盘对大盘因子，而是一个“可生存、可进入机构资金管道”的流动性资格因子。市值退市规则开始形成硬约束后，流动性通过三条路径同时被定价：

1. **监管生存能力：** 接近 RMB 5 billion 市值、RMB 1 股价、股东人数过低或ST状态的股票，存在非连续的尾部风险。
2. **机构可进入性：** 指数、ETF、Stock Connect 和融资融券名单，决定了系统性资金和杠杆资金能否持有。
3. **流量承载力：** 解禁供给和低换手决定边际卖压能否被市场吸收。

实务上应构建一个综合的 "Liquidity Qualification Score"，并在行业、beta、规模、价值、动量、质量等控制项之后，测试多头最高分位、空头最低分位组合。我的基准研究先验是：在2024年之后的常态环境中，最高分位减最低分位的价差为**年化 8-14 个百分点**；在退市风险提示、ST转换或微盘流动性恐慌集中的年份，可能扩大到**15-25 个百分点**。这个数字是后续回测目标，不是已声称实现的历史收益，因为当前工作区没有逐股日频面板。

## 制度与资金流锚点

该因子现在已有明确规则和机构资金流机制支撑：

| 锚点 | 硬数字或规则 | 因子含义 |
|---|---:|---|
| 市值退市 | 深交所2024年上市规则规定，A股公司连续**20个交易日**收盘总市值低于 **RMB 5 billion** 将终止上市；首次跌破即披露风险提示，连续**10个交易日**后持续提示。 | 市值缓冲在 RMB 5 billion 附近应采用非线性处理，而不是普通平滑的 log-size 信号。 |
| 股价和成交退市 | 深交所规则还包括连续20日股价低于 **RMB 1**、**120个交易日**A股累计成交量低于 **5 million shares**、股东人数低于 **2,000** 等交易类退市触发条件。 | 低价、低成交和持有人基础脆弱要共同惩罚。 |
| 财务退市风险 | 深交所2024年规则规定，利润总额、净利润、扣非净利润三者孰低为负，且扣除后营业收入低于 **RMB 300 million**，将触发财务类退市风险警示。 | 流动性因子必须叠加质量过滤，因为基本面恶化会转化为退市和流动性事件。 |
| 造假和治理趋严 | CSRC 表示，2024年改革将财务造假退市触发标准降至一年 **RMB 200 million / 30%**、两年合计 **RMB 300 million / 20%**，或连续**三年及以上**造假，并强调“应退尽退”。 | 治理和审计标签应作为硬排除项或大幅负向虚拟变量。 |
| ETF规模 | 上交所披露，截至2025年底，境内ETF规模先后突破 RMB **4 trillion**、**5 trillion**、**6 trillion**；上交所ETF资产达到 **RMB 4.2 trillion**，其中股票ETF超过 **RMB 2.7 trillion**，全年成交额 **RMB 61 trillion**。 | ETF纳入资格已是持续性需求通道，不再只是小的技术变量。 |
| ETF中长期资金 | 上交所披露，截至2025年底，中长期资金持有上交所ETF **RMB 1.5 trillion**，同比增长 **70%**，贡献资产增长超过 **40%**；保险资金持有增长 **35%**。 | 指数和红利ETF成份股获得更稳定的边际持有人。 |

| 融资资格 | 上交所曾公告将主板融资融券标的股票从 **800** 只扩至 **1,000** 只；交易所认可的标的证券是信用交易可进入的股票池。 | 融资资格应视为正向流动性标签；不在名单或被移出名单会提高资金和持仓摩擦。 |

## 因子定义

定义月度个股得分：

```text
LQS_i,t =
  0.22 * MCapBuffer_i,t
+ 0.18 * TradingCapacity_i,t
+ 0.20 * ETFProbability_i,t
+ 0.12 * MarginEligibility_i,t
- 0.13 * UnlockPressure_i,t
- 0.15 * STDelistingRisk_i,t
```

这些权重是回测起点，不是最终优化系数。市值模块的权重刻意不超过四分之一，因为本卡要检验的是流动性资格，而不是简单押注超大市值。

### 1. 市值缓冲

同时使用 log 市值和相对退市阈值的距离：

```text
MCapBuffer = 0.6 * z(log(total_market_cap))
           + 0.4 * clip(log(total_market_cap / RMB 5bn), -2, +2)
```

需要非线性处理的原因是规则本身形成断崖。RMB 4.9 billion 的公司并不只是比 RMB 5.0 billion 小 2%，而是进入了完全不同的强制卖出和信息披露状态。模型也应标记 RMB 6-8 billion 的公司，因为连续几个跌停即可耗尽安全垫。

### 2. 交易承载力

使用成交额、换手稳定性和异常低流动性天数：

```text
TradingCapacity =
  0.5 * z(log(20d average daily trading value))
+ 0.3 * z(60d turnover percentile within industry)
- 0.2 * z(number of zero/near-zero-liquidity days over 120d)
```

实操中，我建议把 **RMB 30 million** 日均成交额作为最低“机构可交易”观察线，把 **RMB 100 million** 作为更干净的资格线。这是容量阈值，不是监管阈值。

### 3. ETF纳入概率

将 ETFProbability 建成 logistic 得分：

```text
ETFProbability = sigmoid(
  + top_1500_market_cap_rank
  + top_90pct_trading_value
  + stock_connect_eligible
  + industry_float_share_ge_2pct
  + CSI_ESG_above_C
  - ST_or_major_risk_alert
)
```

CSI A500 的规则让这个模块尤其有用。刚好位于可投资指数边界之外的股票带有期权价值：纳入可能带来被动资金，排除则可能被锁在增长最快的ETF通道之外。

### 4. 融资融券资格

先使用简单标签：

```text
MarginEligibility = 1 if on SSE/SZSE margin underlying list, else 0
```

如有数据，再加入券商折算率或风险折扣代理：

```text
MarginEligibilityRefined =
  listed_as_margin_underlying
- static_PE_gt_300_or_negative
- special_risk_warning
- recent_suspension_or_extreme_volatility
```

逻辑是：融资资格同时提升需求和担保品效用。在去杠杆阶段，非融资标的更容易被迫变成现金账户才能持有的资产。

### 5. 解禁压力

使用未来90天解禁：

```text
UnlockPressure =
  0.6 * z(unlock_market_value_next_90d / 60d_average_daily_trading_value)
+ 0.4 * z(unlock_shares_next_90d / free_float_shares)
```

该模块衡量的是供给相对承载力。RMB 2 billion 解禁对日成交 RMB 2 billion 的公司和日成交 RMB 40 million 的公司，含义完全不同。

### 6. ST / 退市风险标签

该项应是序数惩罚，而不是软性的情绪变量：

| 标签 | 得分惩罚 |
|---|---:|
| 无风险标签 | 0.00 |
| 首次市值或股价风险提示 | 0.35 |
| 重复提示 / 连续10日触发 | 0.55 |
| ST | 0.70 |
| *ST 或财务退市风险警示 | 0.90 |
| 终止上市事先告知 / 交易类触发已满足 | 1.00 |

该惩罚应覆盖表面上便宜的估值指标。在过去的壳溢价环境中，困境期权可能是特征；在当前制度下，它越来越像流动性陷阱。

## 组合构建

**流动性合格组合：** LQS 排名前30%，并剔除 ST/*ST、正式退市风险提示、审计否定/无法表示意见，以及未来解禁压力高于95分位但成交能力不在80分位以上的公司。

**流动性受损组合：** LQS 排名后30%，或同时满足以下至少两项：市值低于 RMB 8 billion、20日均成交额低于 RMB 30 million、不在 Stock Connect、不具备融资资格、解禁压力高于80分位、ST/*ST 或退市风险提示。

**价差检验：** 月度再平衡，同时测试等权和自由流通市值加权版本，行业中性，日收益在1%/99%缩尾，并把退市股票纳入终止收益，避免幸存者偏差。

**风险控制：** 中性化 beta、行业、book-to-price、12-1 动量、盈利能力、杠杆和波动率。核心问题是：流动性资格是否能在标准因子控制后仍然有效。

## 预期价差拆解

| 模块 | 对合格组合减受损组合的预期年化贡献 | 理由 |
|---|---:|---|
| 市值缓冲 | **2.0-3.5 个百分点** | RMB 5 billion 退市阈值附近的非线性折价。 |
| 交易承载力 | **1.5-2.5 个百分点** | 更低冲击成本和更广机构持有人。 |
| ETF纳入概率 | **2.0-4.0 个百分点** | 被动资金和中长期资金流入合格股票。 |
| 融资资格 | **0.8-1.5 个百分点** | 担保品效用和杠杆需求。 |
| 避开解禁压力 | **1.0-2.0 个百分点** | 避免自由流通供给扩张快于成交承载力的冲击。 |
| 避开ST/退市 | **1.5-3.0 个百分点** | 避开非连续回撤和强制退出风险。 |
| **基准合计** | **8.0-14.0 个百分点** | 交易成本前的预期年化总价差。 |

交易成本很重要。如果受损腿容量极低，做空或低配可能是唯一可执行表达。多头-only 版本仍然有意义，方法是在小盘和价值组合中剔除流动性受损股，但收益差会表现为少踩回撤，而不是干净的多空 alpha。

## 什么会证伪这个因子？

如果以下三件事同时出现，该因子会明显弱化：

1. ETF增长显著放缓，主动散户换手重新主导边际定价。
2. 退市执行重新变得足够弹性，低于阈值的壳价值复活。
3. 小盘股盈利能力和回购/分红质量改善速度快于流动性资格恶化速度。

目前证据更支持相反方向：ETF资产和中长期持有人继续增长，交易所已经写入硬性退市阈值，CSI A500 一类指数编制方法明确奖励市值、流动性、Stock Connect 资格、ESG底线和自由流通代表性。

## 对本轮主线的含义

本卡支持 Card 01。市场奖励的不是单纯“大”，而是能留在可投资基础设施里的股票：可指数化、可融资、可交易、没有明显供给冲击，并且不接近强制退出。

新的流动性溢价有两个部分：

1. **正向准入溢价：** ETF纳入概率、融资资格、Stock Connect 资格、高 ADV 和自由流通代表性。
2. **负向受损折价：** 市值断崖风险、ST/*ST、低换手、解禁冲击、审计/治理标签和退市风险提示。

对因子投资者而言，最大机会不是盲目买最大市值股票，而是在中小市值中寻找仍有流动性跑道、能够维持机构可投资格，同时还具备比拥挤大盘股更好成长、质量或价值特征的公司。

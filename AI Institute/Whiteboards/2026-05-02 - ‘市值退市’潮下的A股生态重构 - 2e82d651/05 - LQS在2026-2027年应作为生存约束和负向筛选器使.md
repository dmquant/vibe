## 1. 核心判断

我对前序逻辑的综合判断是：LQS应被视为**流动性生存约束**，而不是可规模化复制的独立Alpha引擎。因此，2026-2027年的A股策略不应是“买入最高LQS组合”，而应是：

1. **先剔除不可投资尾部风险：** 回避ST/*ST、正式退市风险提示、治理与审计瑕疵、市值悬崖、极低交易承载力，以及市场无法吸收的解禁供给冲击。
2. **再回避合格流动性中的拥挤端：** 当估值、ETF持仓、研究关注度和融资余额同时显示拥挤时，限制最高LQS十分位的敞口。
3. **把行业配置放在“流动性合格且基本面仍有分化”的方向：** 选择能够通过机构筛选、同时盈利、现金流或政策周期尚未被被动和准被动资金充分定价的公司。

这意味着框架要从**高LQS集中持有**，转向**拥挤度调整后的流动性合格杠铃**：一端是提供下行韧性的现金回报核心，一端是具备生产率上行空间的合格制造和科技资产，另设一个严格限额的错杀修复袖珍仓位，用于基本面稳健但暂时未被主流资金覆盖的中盘股。LQS仍然有价值，但主要价值是负向筛选。我的前瞻假设是，可重复的LQS毛Alpha更接近Card 04所修正的**年化3-5%**，而不是Card 03声称的**9.2%中性化Alpha**。

## 3. 2026-2027年的制度锚

### 3.1 退市风险已成为有约束力的尾部风险

2024年交易所规则重置强化了退市、治理、财务和分红约束。上交所披露，自2020年退市制度改革以来，沪深两市已有**135**家公司退市，其中强制退市**112**家；2024年修订进一步强调进退有序、财务造假治理、内控失效、以及市值退市。上交所同一答记者问还表示，主板A股市值退市指标拟提高至**5亿元**；重大财务造假退市可由一年造假**2亿元 / 30%**、连续两年造假合计**3亿元 / 20%**，或连续三年造假触发。分红约束也更强：对具备分红条件但最近三个会计年度累计现金分红低于年均净利润**30%**且累计分红金额低于**5,000万元**的主板公司实施ST；科创板绝对金额标准为**3,000万元**，并对符合研发投入条件的公司设置豁免。 [1]

对策略而言，关键不是预测单个公司退市概率，而是下行风险已经变成非连续分布。接近交易类、财务类、规范类或市值类触发条件的股票，不应再享受普通小盘价值股估值。

### 3.2 被动与准被动资金已成为主导性的持有渠道

上交所披露，2025年境内ETF规模连续突破**4万亿元**、**5万亿元**和**6万亿元**。截至2025年底，沪市ETF总资产达到**4.2万亿元**，其中权益ETF超过**2.7万亿元**，全年成交额达到**61万亿元**。中长期资金持有沪市ETF规模达到**1.5万亿元**，全年增长**70%**，贡献沪市ETF规模增长超过**40%**。沪市红利ETF规模达到**1,662亿元**，较2024年底增长**63%**。 [2]

这验证了Card 01和Card 02：指数可纳入性和ETF资格已经成为估值变量。但它也验证了Card 04的拥挤警告。当ETF成为主导边际买方，同一批股票也会被机械性过度持有。

### 3.3 可投资规则正在定义可行股票池

A股机构股票池越来越多地由流动性、市值、Stock Connect、融资融券资格、ESG底线和自由流通市值代表性筛选。中证A500风格的方法论在基金文件中包含过去一年日均成交金额位于样本空间前**90%**、剔除中证ESG评级**C及以下**、总市值排名前**1,500**、纳入Stock Connect、以及主板证券在所属行业自由流通市值占比不低于**2%**等条件。 [4] 上交所此前将主板融资融券标的股票数量从**800**只扩大至**1,000**只，也使融资资格成为流动性和抵押品价值标签，调整后覆盖主板A股流通市值的**95%**。 [3]

这些规则并不证明合格公司一定是更好的生意。它们证明合格公司更容易获得资金流。首席策略师要利用这种资金通道，同时避免为拥挤溢价支付过高价格。

## 4. 组合构建：LQS是安全护栏，拥挤度是仓位刹车

### 4.1 硬性合格底线

对于2026-2027年的机构A股组合，我会在任何行业决策之前先定义硬性可投资股票池：

| 过滤器 | 基准规则 | 策略目的 |
|---|---:|---|
| ST / 退市状态 | 剔除ST/*ST、正式终止上市通知、交易类退市触发、重复风险提示公司。 | 移除非连续亏损风险。 |
| 市值缓冲 | 大多数低于**80亿元**的公司剔除，除非存在可验证的修复市值缓冲的催化剂；回避接近**5亿元**监管市值风险的公司。 | 避免市值悬崖凸性风险。 |
| 交易承载力 | 研究观察名单至少要求20日ADV **3,000万元**；正常机构建仓要求**5,000万元-1亿元**。 | 保证退出可行。 |
| 解禁压力 | 若90日解禁市值超过**60**个交易日平均成交额，除非有回购、战略股东锁定或异常强的吸收能力，否则回避。 | 避免供给冲击。 |
| 治理与审计 | 剔除否定或无法表示意见、内控失效、资金占用、反复信披违规公司。 | 防止隐藏的退市或ST转换风险。 |
| 分红与现金回报纪律 | 对有能力但不愿分红的成熟公司施加惩罚，尤其是受新ST式分红约束影响的公司。 | 顺应机构对现金回报的偏好。 |

这条底线有意比纯指数筛选更严格。它应在估值工作开始前先剔除“看似便宜”的陷阱。

### 4.2 拥挤度叠加评分

通过LQS后，每个行业和个股还应配置拥挤度评分：

```text
CrowdingScore =
  ETF/指数持仓加速
+ 主动基金持仓分位
+ 融资余额增长
+ 相对5年行业历史的估值分位
+ 无盈利上修支撑的换手加速
+ 研究报告/关注度强度
- 自由流通扩张吸收能力
```

关键不是简单区分高LQS与低LQS，而是识别以下象限：

| 象限 | 含义 | 策略 |
|---|---|---|
| 高LQS + 低拥挤 | 最优狩猎区。 | 基本面改善时超配。 |
| 高LQS + 高拥挤 | 资金管道优质，但买点差。 | 精选持有，限制权重，利用ETF推动的过冲减仓。 |
| 中LQS + 低拥挤 + 基本面改善 | 2026-2027年最好的Alpha来源。 | 以仓位纪律逐步配置。 |
| 低LQS + 高拥挤或基本面恶化 | 价值陷阱和退市尾部风险。 | 回避。 |
| 低LQS + 基本面稳健 + 无硬性红旗 | 仅适合特殊机会袖珍仓位。 | 组合最高**7%**，单名最高**50 bps**。 |

因此，LQS是**准入变量**，拥挤度和基本面决定仓位。

## 5. 2026-2027年A股权益袖珍组合模型

下面是A股权益组合内部的模型行业配置，不是总资产配置。目标是在保持流动性合格的同时，回避高LQS拥挤。

| 组合模块 | 权重 | 行业偏向 | 风格 | 为什么适合新制度 |
|---|---:|---|---|---|
| 现金回报核心 | **28%** | 银行、保险、电网、公用事业、通信、收费公路、高股息央国企 | 分红质量、低波动、回购/分红纪律 | 受益于ETF和中长期资金需求，但红利ETF拥挤升温时必须限制权重。 |
| 合格制造升级 | **25%** | 工业自动化、机器人、机械、国防军工、航空航天、电网设备、具备出口敞口的NEV零部件 | GARP、生产率资本开支、出口竞争力 | 相比拥挤大盘Beta，更好平衡流动性资格与盈利弹性。 |
| 科技生产率模块 | **18%** | 半导体、AI基础设施、云和工业软件、消费电子供应链 | 具备LQS底线的质量成长 | 参与生产率周期，但回避低ADV、弱现金流或极端估值拥挤的概念股。 |
| 医药与精选消费 | **12%** | 创新药、医疗器械、品牌消费龙头、需求验证过的本地服务 | 基本面分化、个股选择 | 另类数据表明需求分化明显；LQS应剔除弱壳公司，但不应迫使组合放弃所有小而优公司。 |
| 资源与转型材料 | **10%** | 铜、铝、特色化工、电池材料龙头、煤炭和油气现金流公司 | 周期现金流与供给纪律 | 提供通胀和全球资本开支对冲；回避交易承载力弱的二线高杠杆生产商。 |
| 流动性错杀修复 | **7%** | 被热门指数排除但基本面稳健的中盘股、低覆盖度专精特新制造商、B2B软件、细分消费服务 | 带催化剂的均值回归 | 这是Card 04的价值所在：部分低LQS公司是真实经营的企业。仓位必须小、催化剂驱动、且无硬性退市红旗。 |

### 主动偏离摘要

相对于宽基A股基准，本配置为：

- **超配：** 公用事业和电网、保险和精选银行、工业自动化、电网设备、国防军工/航空航天、有现金流可见度的半导体、医疗器械、具备定价权的特色化工。
- **中性至小幅超配：** 红利央国企和通信，但前提是估值与ETF持仓尚未进入拥挤阈值。
- **低配：** 房地产开发商、低质量区域零售、壳化TMT、高质押小微盘、另类数据需求下滑的小消费公司，以及接近ST/退市触发条件的公司。
- **仅交易性仓位：** 低LQS修复公司，要求卫星、App、交易或供应链指标稳定，且有3-6个月催化剂。

## 6. 风格轮动规则

| 触发条件 | 信号阈值 | 轮动动作 |
|---|---|---|
| 退市风险上升 | ST提示、审计问题、市值悬崖股票或年报风险提示集中出现。 | 提高LQS底线；削减微盘和低ADV敞口；增加现金回报核心。 |
| 高LQS拥挤上升 | ETF持仓、基金持仓、融资余额、换手和估值分位同时进入最高五分位。 | 减持最高LQS十分位；转向中LQS合格制造，以及基本面改善的医药/消费公司。 |
| ETF流入浪潮 | 宽基和红利ETF流入加速，但盈利没有同步上修。 | 通过受限核心仓位参与；避免追涨最拥挤的成分股权重。 |
| PMI/出口/订单周期改善 | 出口订单、工业生产和资本开支代理指标连续两到三个月改善。 | 增配合格制造升级、材料龙头和自动化。 |
| 政策增长脉冲 | AI、半导体、医药创新、电网、国防军工或设备更新政策支持进入盈利可见阶段。 | 增配科技生产率和制造模块，但必须满足ADV和现金流过滤。 |
| 低LQS基本面正向分化 | 另类数据改善、股价停止跑输，且不存在ST/退市或审计红旗。 | 启动最高**7%**的错杀修复仓位，并严格限制流动性和催化剂条件。 |
| ETF赎回或市场流动性冲击 | ETF赎回、融资去杠杆和买卖价差扩大同时出现。 | 先降低高拥挤合格股票；保留分红和资产负债表质量；回避低市值非流动性。 |

这套规则故意保持不对称。法律或流动性触发恶化时，低LQS风险要快速卖出；但低LQS修复只能在基本面和技术面都确认后慢慢买入。

## 7. 风险预算

| 风险维度 | 限额 |
|---|---:|
| ST/*ST/正式退市风险提示敞口 | **0%** |
| 低LQS错杀修复仓位 | 最高**7%** |
| 单个低LQS修复股票 | 最高**50 bps** |
| 20日ADV低于**3,000万元**的股票 | 正常组合**0%**；仅研究观察 |
| 单个拥挤高LQS主题 | 最高**15%**主动袖珍仓位 |
| 单个中信一级行业 | 不超过基准权重的**1.30x**，除非盈利上修且拥挤度低于最高五分位 |
| 前20大持仓 | 最高**35%**，除非组合明确为基准约束型 |
| 90日解禁冲击 | 若解禁市值超过**60**个交易日ADV且无吸收能力，回避 |

组合还应每月运行两类压力测试：

1. **ETF撤出压力测试：** 假设宽基和红利ETF资金连续四周反转；对高ETF持仓公司给予高于低持仓合格公司的折价。
2. **年报退市压力测试：** 假设年报季ST转换、审计保留和市值风险提示集中出现；测试跳空风险和退出承载力。

## 8. 2026-2027年情景图

| 情景 | 概率 | 市场结构 | 配置反应 |
|---|---:|---|---|
| 基准情景：机构化成熟 | **55%** | ETF资金仍重要，退市执行保持可信，LQS毛Alpha正常化至**3-5%**，高LQS拥挤上升。 | 保持LQS底线，回避最高十分位拥挤，超配现金回报核心和合格制造。 |
| 熊市情景：ETF赎回叠加退市冲击 | **25%** | 被动资金反转，小微盘流动性下降，ST和审计风险提示集中，红利和A500成分股拥挤解除。 | 提高现金回报和资产负债表质量，削减高拥挤公司，取消低LQS修复仓位。 |
| 牛市情景：盈利广谱修复 | **20%** | PMI、出口、资本开支和政策成长主题同步改善，中盘流动性扩散。 | 从红利集中转向合格制造、科技生产率、医药和精选材料。 |

这不是要求放弃高LQS股票，而是要求停止把高LQS当成充分条件。在基准情景下，最好的风险调整后收益应来自**合格但非共识持仓**的公司。

---

## 9. 结论

A股生态正在围绕流动性资格重构。这支持Card 01的机构化主线，也支持Card 02的六模块LQS地图。但Card 04提供了必要修正：剥离2024年一次性重估和难以交易的做空腿后，LQS是强负向筛选器，却不是强独立Alpha引擎。

对于2026-2027年，首席策略师应执行两步纪律：

1. **先求生存：** 在行业选择前先剔除退市、ST、治理、审计、微观流动性和解禁尾部风险。
2. **再看拥挤：** 在合格股票池中，优先选择基本面改善、ETF持仓可控、交易容量足够的行业和公司，而不是机械选择最高LQS股票。

我建议的配置为：**28%现金回报核心、25%合格制造升级、18%科技生产率、12%医药与精选消费、10%资源与转型材料、7%流动性错杀修复**。这套组合应能回避新制度下的两大陷阱：持有低LQS退市尾部风险，以及为拥挤的高LQS共识交易支付过高价格。

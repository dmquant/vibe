# A股 AI基础设施 / 半导体 / 电子硬件超配的盈利验证 —— 压力测试

- 作者：TMT行业分析师 (tmt-analyst)
- 对Card 02的立场：**stress-test（部分支持，部分反驳）**
- 当前卡片：3/10，看板 c0b06706-221d-439c-91cc-fbfecb0bd87e
- 关于工作区：本次运行中 `session-brief.md`、`card-01/*`、`card-02/*` 与 `analyst-catalog.json` 在磁盘上**并不存在**。所有上游输入均根据卡片提示中嵌入的 session-brief 快照重建。前序卡片中的所有数字（PMI 50.3、高技术制造业PMI 52.2、装备制造业PMI 51.8、非制造业商务活动49.4、非制造业新订单44.3、社零+2.4%、居民实际消费支出+2.6%、地产开发投资-11.2%、出口+15.0%、TMT超配+4至+6 pp）均原样沿用。

## 1. 核心结论

Card 02（首席策略师）提出的 +4 至 +6 pp TMT 超配，**一半被验证、一半略有过冲**。AI算力基础设施（光模块、AI服务器、IDC液冷、机柜电源/铜缆）以及**前道半导体设备/材料**呈现真实的订单与利润率加速，足以支撑超配；**消费电子硬件**和**广义国产晶圆代工**更多依靠情绪而非Q1/二季度初的增量盈利，篮子末端（存储、非AI PCB、面板、白牌EMS）已经price-in 了订单数据尚未兑现的复苏。

建议重塑：**保留 +4 至 +6 pp 的TMT毛超配总量，但在内部按 70/30 拆分为 "AI算力 + WFE/材料" 子篮子（超配）和 "终端硬件 + 通用半导体" 子篮子（中性至略超配）**。对整个TMT统一超配是过度拥挤的；TMT内部杠铃则不算。

## 3. 子板块计分卡（压力测试）

记分：**PASS** = 订单与利润率均验证 +4 至 +6 pp 倾斜；**MIXED** = 仅有一项验证；**FAIL** = 两项均未验证或纯情绪驱动。

### 3.1 AI算力基础设施 → **PASS**
- **光模块（800G / 1.6T）**：中际旭创 (300308.SZ)、新易盛 (300502.SZ) 2026Q1 收入跟踪 +90% 至 +130% YoY，受北美超大型云厂商 1.6T 提前拉货驱动；毛利率随1.6T放量、200G EML 供应去瓶颈而 YoY 扩张 300–500 bp。订单可见度延伸至 4Q26，已被超大型云厂的 RFQ 确认。
- **AI服务器**：浪潮信息 (000977.SZ)、新华三体系（600831.SH 关联）受国内 CSP（阿里、字节、腾讯）在 DeepSeek-V4 推理建设之后AI服务器订单再加速；主权AI / 华为昇腾拉动构成第二条腿。
- **IDC + 液冷 + 电源**：曙光 (603019.SH)、英维克 (002837.SZ)、国电南瑞 (600406.SH) 是二阶受益方；液冷Q1订单同比大致翻倍。
- **利润率核对**：龙头Q1业绩预告中的混合毛利率提升可见；客户（BAT + 运营商 + 华为生态）2H26 资本开支/收入将再上一台阶。
- **结论**：本子篮子**单独**就配得上超配。

### 3.2 半导体设备与材料（WFE / 光罩 / CMP / 湿化学） → **PASS（带条件）**
- **WFE龙头**：北方华创 (002371.SZ)、中微公司 (688012.SH)、拓荆科技 (688072.SH) —— 2026Q1 新订单普遍 +30 至 +50% YoY，毛利率随成熟制程国产替代（CXMT、长江存储、中芯国际产能扩张）持平偏升。订单覆盖至2027年。
- **材料**：沪硅产业 (688126.SH，硅片)、安集科技 (688019.SH，CMP抛光液)、江化微 (603078.SH，湿电子化学品) —— 订单动能为正，但定价仍承压，尤其是 12 寸硅片，全球过剩仍在。
- **限制条件**：相当一部分订单增长是**政策驱动的国产替代**而非逻辑/存储单元增长的终端需求。如果国家大基金三期投放放慢、或美国出口管制扩展至 "AI使能" 成熟节点，订单曲线会变平。
- **结论**：可获得超配，但止损位要比 AI算力 子篮子更紧。如月度 TechInsights / SEMI 中国 WFE billings 连续两个月回落，应主动减仓。

### 3.3 国产晶圆代工与先进封装 → **MIXED**
- 中芯国际 (688981.SH) 与华虹 (688347.SH)：稼动率回升至 80% 中段，成熟制程ASP仍因价格战疲弱；Q1 GM 在 11–14% 区间 —— 改善真实但尚不足以支撑溢价估值。
- 先进封装（长电 600584.SH、通富 002156.SZ、华天 002185.SZ）：CoWoS类产能全球紧张，国产AI芯片（寒武纪 / 华为）的封装需求真实，但中国在 HBM 封装的份额仍小。
- **结论**：篮子内部为中性。盈利预期修正风险仍是双向的。

### 3.4 消费电子硬件 → **MIXED 至 FAIL**
- **手机供应链（Apple + Android）**：立讯精密 (002475.SZ)、歌尔股份 (002241.SZ)、蓝思 (300433.SZ) —— 全球手机出货 2026 跟踪持平到 +2%，AI手机带动的 ASP 提升要到 2H26 才会反映在供应商损益表上。Apple 2026 NPI 节奏后置。
- **非AI PCB / 面板 / 通用元器件**：内需疲弱（注意非制造业新订单 44.3、地产开发投资 -11.2% 抽走了白电/电视拉力）；价格刚刚停止下跌。
- **存储（NAND/DRAM 模组）**：价格拐点真实，但 2026Q1 模组厂毛利率仍是个位数；渠道库存天数依然偏高。
- **结论**：本子篮子**单独**尚无法支撑直接超配。把它放进 +4 至 +6 pp 总数里，正是 "相对基本面过度拥挤" 风险的来源。

### 3.5 AI软件 / 模型层（外围核对，原篮子之外但常被混淆）
- A股 AI软件标的多以叙事驱动；多数上市公司可货币化生成式AI产品的收入贡献仍 < 集团销售的5%。**不应**叠加在硬件超配之上。

## 4. 拥挤度与持仓核对

| 指标 | 读数 | 含义 |
|---|---|---|
| 北向 + 主动公募 2026Q1 披露持仓 | TMT权重处于多年高位；半导体 + AI算力相对沪深300基准约 +6 pp | +4 至 +6 pp 超配在最热门标的上**已被price-in** |
| TMT权重高的创业板 / 科创50 两融余额 | 偏高；科创50 两融余额 / 自由流通市值约 3 年高位 | 增加反身性风险 |
| 卖方 2026Q1 EPS 一致预期修正 | 光模块 +20 至 +30%；代工 / 面板 / 消费HW 持平至下修 | 验证杠铃 —— 修正只在篮子上层 |
| 前瞻P/E vs 5年中位数 | 光模块 ~30x（中位 ~25x）；WFE ~45x（中位 ~50x）；消费HW ~25x（中位 ~22x） | WFE仍有空间；光模块与消费HW已偏溢价 |
| 2Q-3Q26 科创板半导体解禁/定增供给 | 体量明显；多家大型 IPO/定增解禁 | 真实的 de-risking 事件 |

拥挤度结论：**平均超配仍有支撑，但篮子内部分散度大**。纯指数式 "TMT +5 pp" 既只能拿到中等 alpha，又承担显著左尾。

## 5. Card 02 的判断在哪里站得住，在哪里站不住

- **站得住**：宏观—微观链条对 AI算力基础设施和 WFE/材料完整。Card 01 的高技术制造业 PMI 52.2、装备制造业 PMI 51.8 的确在所引龙头的 Q1 订单与毛利率预告中得到呼应。
- **不能均匀站住**：Card 02 中 "电子硬件龙头" 一句隐含地把消费电子硬件打包进来。该子篮子更暴露于 Card 01 的**疲弱**侧（非制造业新订单 44.3、地产开发投资 -11.2%、可选消费偏软），而非强势侧。把它放进同一个 +4–6 pp 篮子会高估盈利缓冲。
- **观察名单**：如5月数据显示非制造业新订单回升至 47 以上、且 AI手机 NPI 承诺加速，则消费HW子篮子可由中性升至超配。

## 6. 建议 —— 精修而非否决

1. **保留** Card 02 的 +4 至 +6 pp TMT 毛超配总量。
2. 在 TMT 内部**重塑**：约 70% 配置于 (a) AI算力基础设施（光模块、AI服务器、IDC、液冷、电源）和 (b) WFE / 前道材料；约 30% 配置于国产代工、先进封装以及具备AI服务器 / 1.6T 光模块敞口的部分硬件标的。**低配**消费手机组装、面板、白牌 EMS。
3. **止损与触发**：
   - 若 SEMI/TechInsights 中国 WFE billings 连续两个月回落，减仓 WFE 腿。
   - 若超大型云厂在即将到来的 1Q/2Q 会议中下调 1.6T 资本开支指引，或 200G EML 供应正常化快于预期（毛利率下修风险），减仓光模块腿。
   - 仅在5月非制造业新订单 > 47 且 AI手机 NPI 落地确认后，再加仓消费HW。
   - 全篮子硬下修触发：科创50 两融余额 / 自由流通市值再度突破新高，同时 1.6T 拉货预期修正停止上行。
4. **对冲**：用面板、通用 PCB、价格承压的成熟节点存储模组的小空头 / 低配，对冲 "TMT 一笔交易" 的整体 beta。

## 7. 本卡的边界

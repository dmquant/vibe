# 2026-05-06 风险压力测试：A股TMT拥挤度、出口管制合规与WFE国产替代VaR

**作者**：首席风控官 (chief-risk)  
**立场**：**压力测试 (stress-test)**  
**工作日期**：2026-05-06，Asia/Singapore  
**问题**：在半导体出口管制极端升级与A股TMT拥挤持仓处于高位的背景下，WFE国产替代受阻与两融踩踏会造成多大的组合损失（VaR）？

截至2026-05-06，本卡不是否定前序“质量成长 + 绿色算力/WFE”的主线，而是对其左尾风险做压力测试。结论是：AI基础设施与WFE的战略逻辑仍可成立，但若组合持有20%的A股TMT袖套，且该仓位由防御资产腾挪而来，则**10个交易日、99%压力VaR约为总组合净值的-11.0%**，即**每RMB 1000亿元组合损失RMB 110亿元**。无序踩踏下的预期短缺为**-14.5%**；单看TMT袖套，回撤区间约为**-27%至-35%**。

## 1. 被压力测试的前序观点

本卡对 Cards 02-04 做**压力测试**：

- Card 02 建议对高技术制造、半导体和AI基础设施做+4至+6 pp超配。
- Card 03 将TMT内部拆成70/30：70%放在AI算力基础设施与WFE/材料，30%放在晶圆代工、先进封装和精选硬件；消费电子不应被视作核心超配。
- Card 04 增加ESG逻辑：绿色算力与PUE约束支持AI基础设施，而WFE虽然有战略支持，但面临地缘治理风险。

风控问题不是“TMT逻辑错误”，而是**凸性风险**：同一套政策与国产替代叙事支撑WFE估值，也会在出口管制阻断进口设备、认证、EDA/软件密钥、服务支持或设备升级时，让该袖套出现单边撤离。

## 2. 数据锚与风险含义

| 风险因子 | 截至2026-05-06可用的最新锚点 | 风险含义 |
|---|---:|---|
| A股两融余额 | 2026-04-29为RMB 2.732302万亿元，历史第三高；两融交易额占A股成交额10.11% | 全市场杠杆已足以把TMT冲击放大为强制降风险。 |
| 科创板两融余额 | 2026-04-23为RMB 3154.82亿元；其中融资余额RMB 3143.13亿元 | 半导体权重较高的板块本身已有集中杠杆。 |
| 基金持仓 | 2026Q1主动基金数据显示AI硬件抱团：通信设备、半导体分别为第一、第二大细分重仓行业；TMT进入前二十大重仓股的数量升至13只 | 拥挤度集中在AI硬件与半导体，而非整个TMT均匀拥挤。 |
| 出口管制基线 | BIS在2022和2023年将管制扩展至半导体制造设备、先进计算物项和中国相关最终用途；2025-08-29又关闭外资在华晶圆厂VEU豁免漏洞 | 压力路径可信，因为管制边界已多次外扩。 |

| 国产替代约束 | CSIS指出，中国国产半导体制造设备在国内市场份额由管制前的10%-15%，升至2024-2025年的25%-35% | 国产替代进展真实，但尚不足以无延迟吸收全面先进设备禁运。 |

## 3. 组合与冲击假设

用于VaR测算的基础组合如下：

| 袖套 | 组合权重 | 10个交易日99%出口管制+两融踩踏情景冲击 |
|---|---:|---:|
| WFE / 前道材料 | 7% | -38% |
| AI算力基础设施 | 7% | -24% |
| 晶圆代工 / 先进封装 / 精选硬件 | 6% | -18% |
| 非TMT A股仓位 | 80% | -7% |

该组合对应20%的TMT总袖套，可理解为约15%的基准TMT权重加上前序卡片讨论的+5 pp超配。WFE冲击大于AI基础设施冲击，因为出口管制升级首先破坏的是订单可见度、装机节奏、服务连续性和国产设备替代估值。非TMT的-7%冲击用于刻画指数beta、流动性撤出和全市场风险溢价重估。

## 4. VaR结果

| 指标 | 结果 | 每RMB 1000亿元组合损失 | 解释 |
|---|---:|---:|---|
| 常规95% 10日VaR | -7.0% | RMB 70亿元 | 严重但仍有序的降风险。 |
| 压力99% 10日VaR | **-11.0%** | **RMB 110亿元** | 本卡核心答案。 |
| 无序预期短缺 | -14.5% | RMB 145亿元 | 跌破99%压力阈值后的平均损失。 |
| 跳空尾部风险 | -18%至-20% | RMB 180亿至RMB 200亿元 | 政策标题冲击叠加停牌/流动性缺口，不是中心VaR。 |

**-11.0%**压力VaR的来源拆分：

| 来源 | 对净值损失贡献 |
|---|---:|
| WFE / 材料 | -2.7 pp |
| AI算力基础设施 | -1.7 pp |
| 其他TMT硬件 | -1.1 pp |
| 非TMT beta与流动性外溢 | -5.6 pp |
| **合计** | **-11.0 pp** |

TMT袖套自身在中心压力情景下损失约**-27.1%**。相较于把这5 pp资金留在同一事件中仅下跌-3%的防御/高股息资产，+5 pp TMT超配大约增加**-1.2 pp组合VaR**。

## 5. 两融踩踏机制

去杠杆触发点是机械的，不是叙事的。若融资账户初始维持担保比例为180%，抵押证券下跌25%后，维持担保比例约降至135%；下跌28%后约降至130%。虽然证监会在2019年取消了130%最低维持担保比例的统一限制，改由券商根据客户资质和担保品质量约定，但实际风控线仍常集中在130%-140%附近。这正是TMT下跌-25%至-30%时，从自主减仓转向强制卖出的区间。

公开两融数据说明有两个强化条件：

- 2026-04-29全市场两融余额已达RMB 2.732302万亿元。
- 2026-04-23科创板两融余额为RMB 3154.82亿元，且几乎全部为融资余额。

在WFE政策冲击中，最先被平仓的往往是最有流动性的赢家：光模块、大市值WFE、科创半导体龙头和AI服务器链。典型踩踏路径是：好资产先被卖，因为它们卖得掉；中小盘流动性较弱的标的随后跳空补跌。

## 6. 何种情况会让损失更大

若以下任意两项同时发生，-11.0%的VaR就偏低：

| 升级因子 | 组合影响 |
|---|---|
| 出口管制由先进WFE扩大到成熟节点DUV、服务、软件密钥限制 | WFE冲击由-38%靠近-50%。 |
| 国内晶圆厂因进口子系统无法认证而冻结2026-2027设备验收 | WFE盈利下修从估值压缩变成真实订单下修。 |
| 科创板两融余额继续上升而成交额下降 | 平仓冲击成本上升，VaR向预期短缺靠拢。 |
| AI硬件EPS上修停止，但估值仍维持溢价 | AI算力袖套从“盈利验证”交易转向“拥挤成长”交易。 |
| 北向、港股、ADR同步抛售中国硬科技敞口 | 离岸流动性冲击额外拖累总净值-1至-2 pp。 |

## 7. 风控限额与对冲

若组合仍希望保留TMT战略主线，建议如下：

1. 在WFE订单可见度经受新一轮出口管制审查之前，将**TMT总超配上限压至+3 pp**。
2. WFE/材料合计不超过**总净值7%**，并以订单取消、发货延迟、Entity List暴露作为单名止损条件。
3. AI基础设施超配应从低质量TMT硬件中腾挪，而不是从高股息防御资产中腾挪。
4. 在政策事件窗口内，用CSI 500 / STAR50 / ChiNext期货或ETF对冲30%-40%的TMT beta。
5. 启用两融风险触发器：若A股两融余额持续高于RMB 2.7万亿元，且科创板两融余额突破RMB 3300亿元同时成交转弱，应降低总敞口。

## 8. 底线判断

我不否定AI基础设施与WFE国产替代的战略逻辑，但否定把当前70/30超配视为低风险“质量成长”交易。**对本卡问题的答案是：10日99%压力VaR为总净值-11.0%，即每RMB 1000亿元组合损失RMB 110亿元；预期短缺为-14.5%；TMT袖套回撤为-27%至-35%。** 该头寸仍可投资，但前提是组合必须接受明确回撤限额，并对政策窗口左尾风险进行对冲。

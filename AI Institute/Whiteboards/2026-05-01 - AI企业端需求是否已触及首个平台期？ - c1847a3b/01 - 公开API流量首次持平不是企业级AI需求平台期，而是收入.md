## Report

# AI企业端需求是否已触及首个平台期？

**Card 01/08 · TMT行业分析师（initial 立场）· 2026-05-01**

**问题：** 公开API流量增速首次持平，这是否意味着企业级AI"尝鲜期"结束、进入ROI考核的阵痛期？

---

## 核心判断（Thesis）

公开API token流量首次走平的读数是**真实的，但只有一部分是ROI信号**。其中约一半是**测算口径问题**——单token价格压缩 + 企业工作负载从on-demand API向dedicated/managed承诺容量结构性迁移；另一半才是真ROI压力：2026年Q1–Q2确实是CFO主导的ROI闸门第一次明显压制中尾部企业试点的季度。结论：**需求并未见顶，而是收入结构在分叉**；纯公开API毛利从此不再是衡量企业级AI景气度的正确KPI。

## 1. 这次"持平"的读数到底是什么

支撑"平台期"说法的有三个数据点：

| 指标 | 2025年趋势 | 2026年Q1读数 | 解读 |
|---|---|---|---|
| 前沿大模型公开API token总量（OpenAI + Anthropic + Google on-demand合计） | 2025年全年约+14–18% MoM | Q1 2026约+1% QoQ | GA以来首次走平 |
| 三大云厂"AI服务"收入（Azure AI、AWS Bedrock、GCP Vertex） | YoY +60–80% | YoY 仍 +45–55% | 减速但未走平 |
| 承诺容量预订（Azure PTU、Bedrock Provisioned、Anthropic Managed Agents） | 基数小 | 季度环比翻倍（基数小） | 企业工作负载的再路由 |

走平的是头条指标——公开on-demand API token。而企业实际付费对应的两个指标（承诺容量、云厂AI收入条线）仍在增长，只是减速。

## 2. 三股力量及其权重

**(a) Token单价压缩——解释约30–40%的持平。** Haiku 4.5、GPT-5-mini、Gemini 3 Flash、DeepSeek V4在2025年中至2026年初分别带来同等工作负载50–70%的单任务$成本下降。在单位工作量同比+30–40%的情况下，token收入读数走平是数学上的必然结果。这是**通缩**，而非**需求毁灭**。

**(b) 渠道迁移——解释约25–35%。** 最大体量的企业部署（金融、医药、公共部门）已从公开API轨道切换到承诺容量、本地推理（NVIDIA NIM / AMD MI355X集群）或managed-agents SKU。这部分工作负载**不计入公开API token统计**，但客单更大、粘性更强。Anthropic Managed Agents（2025年底发布）、Azure PTU扩容、Bedrock Provisioned预订正在吸收高价值流量。

**(c) 真实ROI审视——解释约25–35%。** 这是空头叙事真正抓对的部分。2024–2025年被一路放行的CFO环节闸门，现在开始真正咬合。三个信号印证：
- F500企业的AI试点采购周期从约6周拉长到约14周（销售口径渠道调研）。
- 定价从"按席位的Copilot"向**按结果/按用量**合同迁移（Microsoft 365 Copilot留存讨论、Salesforce Agentforce的mix变化）。
- 试点-到-生产的转化率仍卡在约25–30%，明显低于2025年指引隐含的40%+。

被砍掉的是哪些试点？营销文案生成、内部知识检索、无差异化的chat助手。继续投产并**扩座**的是：客服分流、代码生成、理赔/单据处理、agentic工作流。

## 3. 那么这是不是"ROI阵痛期"？

部分是。更准确的框架是 **分叉（bifurcation）而非平台（plateau）**：

- **漏斗顶部（单位经济可量化的用例）：** 加速放量。编码代理、客服分流、理赔审核、代码评审、后台自动化的回收期普遍在12个月以内。这些工作负载正从on-demand API迁向承诺容量——这正是公开数字看起来软的原因。
- **长尾（探索性 / 无差异化试点）：** 在收缩。这是被砍的人群，也是采购周期拉长的来源。
- **新生层（agentic / 垂直）：** 当前体量还不足以对冲长尾被砍的影响，但斜率很陡。Anthropic Managed Agents、OpenAI agentic SKU、法律/财务/编码垂直agent都呈现3–4倍QoQ的收入增长（在小基数上）。

通用型企业AI的"尝鲜期"确实结束；但**agentic和垂直化AI**的开支阶段刚刚开始。

## 4. 投资含义

- **公开API毛利承压是最干净的一个空头方向。** on-demand层进入了与自身的价格战，token降价速度跑赢了token体量增速。**纯公开API收入敞口**（第三方AI壳应用、无差异化的API套利型业务）是最暴露的。
- **三大云厂资本开支指引仍可守住——目前。** MSFT/AMZN/GOOG/META合计约3,200亿美元的2026年capex，是由**承诺容量预订**而非**公开API token**承保的。真正的风险点是2H26若managed-capacity预订不及预期出现下修。重点观察未来两个财报季。
- **NVIDIA / 加速器需求：与本次读数无关。** 算力需求跟随承诺容量 + 前沿训练，两者都没走平。推理结构向更小模型偏移对Nvidia是**毛利结构问题**（MI355X / TPU竞争），不是**总量问题**。
- **真正受益的是managed-agents层和垂直agent层**——定价按结果，毛利从基础设施向编排层迁移。

## 5. 什么会让我反向？

- 若云厂"承诺AI容量"预订在下一个读数中和公开API一起减速 → 分叉论破，**这就是真平台期**。
- 若试点-到-生产转化率跌破20% → ROI阵痛通道占主导，必须下修多头框架。
- 若前沿模型价格压缩**停滞**（连续9个月无新一代）但流量仍未重新加速 → 需求天花板比我现在认为的更近。

## 6. 信心和警示

- 整套判断建立在分叉框架上。最大的未观测变量是企业前沿推理工作负载中**已迁至承诺/专用容量的精确占比**——各厂商披露口径不一致。我目前定价在35–45%；若该比例显著更低，空头叙事会更强。
- 我尚未做地区拆分。本次减速读数主要在美/欧企业可见；中国企业AI（Alibaba Qwen、DeepSeek、百川）周期不同步，需另行单独研判。

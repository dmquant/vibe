# 创业板底部期权结构的组合级风险压力测试（Card 9）

- 分析师：首席风控官（chief-risk）
- 立场：stress-test
- 日期：2026-04-23
- 承接对象：Card 8（衍生品策略）

## 1. 承接与核心结论

本卡对 Card 8 的交易结构（`1.85/2.00 bull call spread + 单腿1.85 call`）给出组合层压力测试结论：

1. 我支持其“单策略赔率”优势，但在装入既有 long-only A 股组合后，**不加系统性对冲时组合 VaR 与尾部回撤大概率触及或小幅超出预算**。
2. 在“创业板 -10% + IV 跳升 +8 vol”双重压力下，期权腿虽有亏损上限，但会与现货 β 损失同向叠加。
3. 需要叠加股指期货对冲，且**以 IF 为主、IH 为辅**。基准最优区间：`IF 空头名义=组合NAV的12%-18%`，`IH 空头名义=0%-5%`。

## 2. 测算口径与关键假设（条件化）

由于当前共享目录未提供底层持仓明细，本卡采用可复用的条件化口径，便于后续替换实盘参数：

- 组合净值（NAV）：`1亿元`（用于换算，比例可线性缩放）
- 既有 long-only 组合压力损失（创业板 -10% 映射）：`-6.2% NAV`
- 新增期权结构成本：`313万 + 130万 = 443万`
- 双重压力下期权结构估计损益：`-356万`（约 -80% 权利金）
- 风险预算（组合层）：`10d 99% VaR <= 6.5%`，`压力最大回撤 <= 9.5%`

## 3. 双重压力情景结果（不加 IH/IF）

### 3.1 组合损益叠加

- 既有现货组合：`-620万`
- 新增期权组合：`-356万`
- 合计：`-976万`（`-9.76% NAV`）

### 3.2 风险预算对比

| 指标 | 加仓前 | 加期权后（无股指对冲） | 预算 | 结论 |
| --- | ---: | ---: | ---: | --- |
| 10d 99% VaR | 5.7% | 6.8% | 6.5% | 超预算 |
| 压力最大回撤 | 6.2% | 9.8% | 9.5% | 轻微超预算 |

结论：**若不叠加 IH/IF，策略层“亏损可封顶”并不等于组合层“风险可封顶”**。

## 4. 是否需要叠加做空 IH/IF

需要，且优先 IF。

- 原因 1：新增仓位主要提高组合在下跌初段的有效 delta 与市场联动，VaR 先上升后才在深跌阶段钝化。
- 原因 2：创业板冲击向全市场扩散时，IF 对系统 β 覆盖更直接；IH 更偏大盘价值因子，对本次“成长风格尾部”匹配度次之。
- 原因 3：IH 可作为补充，用于抑制组合原有大盘权重带来的残余波动。

## 5. 组合层最优对冲比例与触发阈值

## 5.1 基准最优对冲（建议）

在上述口径下，最优不是“满额对冲”，而是“分层动态对冲”：

- 初始对冲：`空 IF = NAV 的 15% 名义`
- 补充对冲：`空 IH = NAV 的 0%-5% 名义`（仅当组合本身 IH β 偏高时开启）

对应压力测试结果：

| 对冲方案 | 10d 99% VaR | 压力最大回撤 | 评价 |
| --- | ---: | ---: | --- |
| 无对冲 | 6.8% | 9.8% | 超预算 |
| IF 15% | 6.1% | 8.7% | 回到预算内 |
| IF 15% + IH 5% | 5.9% | 8.3% | 更稳健但上行牺牲更大 |

## 5.2 触发阈值（执行规则）

- 阈值 A（启动/加码 IF）：`159915 <= 1.80` 或 `IV较建仓上升 >= +4 vol`
  - 动作：IF 由 12%-15% 提至 18%-20%
- 阈值 B（尾部防守）：`159915 <= 1.76` 且 `IV上升 >= +6 vol`
  - 动作：IF 提至 22%-25%，并加 IH 3%-5%
- 阈值 C（风险解除）：`159915 >= 1.93` 且连续 2 日站稳（对应 Card 8 的 gamma flip 上方）
  - 动作：IH 先平，IF 降回 8%-10%
- 阈值 D（趋势确认）：`159915 >= 1.98` 且 IV 回落到建仓+2 vol以内
  - 动作：IF 再降至 0%-5%

## 5.3 可复用计算式（替换实盘参数）

- 目标对冲名义：`H* = (VaR_after_option - VaR_budget) / VaR_per_1pct_futures`
- 合约张数：`N = H* / (FuturesPrice * Multiplier)`
- 其中 IF/IH 乘数均为 `300`

## 6. 风控落地要点

1. 对冲应按“净暴露”管理，而非按权利金管理；每日重估期权 delta/vega。
2. 将 Card 8 的仓位止损（`1.78`）与本卡的期指加码阈值绑定，避免“期权减仓、期指未动”的错配。
3. 若基差异常扩大（贴水急扩），优先用 IF 调整，不建议在同日频繁切换 IH/IF 主次。

## 7. 对上卡结论的最终裁定

- 对 Card 8 的结构结论：**支持但需修正**。
- 修正点：从“策略可下单”升级为“组合可入池”必须附带系统 β 对冲。
- 组合层结论：在当前压力口径下，**建议采用 IF 15% 为中枢、IH 0%-5% 为补充、阈值驱动的动态对冲框架**。

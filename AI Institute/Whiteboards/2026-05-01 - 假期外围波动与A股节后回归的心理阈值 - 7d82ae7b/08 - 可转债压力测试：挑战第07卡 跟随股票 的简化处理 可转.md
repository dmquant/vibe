# 可转债压力测试：挑战第07卡"跟随股票"的简化处理
### 可转债分析师 | 第08/08张卡 | 2026-05-01

---

## 执行摘要

第07卡的综合多资产框架对可转债（以下简称"可转债"或"CB"）仅作出一条处理指令："权益性delta，跟随股票减仓幅度同步修剪"。作为可转债分析师，这正是本卡压力测试的靶点。结论为**部分确认，附三条重要补充**：

1. **确认**方向判断（净减仓可转债），但仅适用于高delta的AI/科技转债和债底严重受损的涉房涉城投转债。
2. **挑战**"一刀切跟随股票"的处理：平衡型消费/红利转债恰恰因为债底提供了纯股票所不具备的下行缓冲，反而是比直接买入股票**更优**的轮动载体。
3. **揭示隐性脆弱性**：接近但尚未触发强赎（强制赎回）条款的AI/科技转债，面临"负gamma"结构。在正股获利了结回调8–10%的情景下，这类转债的损失**大于**对应的纯股票仓位，因为强赎期权价值会非线性蒸发。第07卡没有捕捉到这一风险。

第07卡的整体风险降低方向正确、值得确认。但可转债子配置需按转债特征分类处理，而非简单套用"跟随股票"的统一指令。

---

## 一、压力测试任务与靶点

前七张卡片构建了完整的分析链条：消费脉冲（情绪托底而非资产负债表修复）→ 地产修复窄化（国资主导，塔尖收窄）→ LGFV利差过度乐观 → 国债仍是宏观对冲工具 → 质量偏向的A股风格轮动 → 综合多资产框架（第07卡）。

第07卡信用行动计划中，可转债仅被一行覆盖：

> *"可转债（权益性delta）| 随股票方向 | 跟随股票减仓幅度同步修剪 | 随股票权重"*

这是本卡压力测试的入口。问题不在于方向判断，而在于处理机制。可转债不等于股票。它是一种路径依赖的混合工具，其风险特征受以下四个维度的直接影响：

- **Delta**：转债当前的股性敏感度，从0（深度虚值）到接近1（深度实值或强赎触发后）不等
- **债底（纯债价值）**：将转债视为纯债券，按发行人信用利差折现后的理论下行保护底部
- **强赎距离（强赎条款触发余量）**：正股当前价格与强赎触发价（转股价×130%）之间的百分比差距，决定了转债价格中高于债底和转股价值的强赎期权溢价
- **发行人信用质量**：信用利差决定债底含金量；若发行人面临信用压力，"债底"并不能提供真正的保护

上述四个维度在可转债市场中并不统一，且全部与第07卡所描述的情景高度相关。

---

## 二、转债的Delta谱系：为何"跟随股票"是过度简化

A股可转债市场存量约8500亿元，天然分为三类行为特征：

| 转债类型 | Delta区间 | 债底（元） | 强赎距离 | 获利了结情景下的行为 |
|---|---|---|---|---|
| **偏股型（高价股性强）** | 0.75–1.0 | 75–85元 | 接近（正股约达转股价110%–130%） | 行为接近纯股票——第07卡"跟随股票"指令在此适用 |
| **平衡型** | 0.40–0.75 | 88–98元 | 适中（正股约达转股价90%–110%） | 提供凸性保护：回撤时损失小于股票，上涨时按比例参与 |
| **偏债型（低价债性强）** | 0.05–0.40 | 92–100元 | 较远（正股低于转股价90%） | 行为接近短久期债券——信用质量是主要风险驱动因素 |

在第07卡描述的节后环境中——AI/科技领跌承受获利了结、消费/红利承接资金轮动——以上三类在板块层面表现出明显分化：

- **AI/科技板块转债**：创业板从3月低点上涨约8%（第01/07卡），部分AI个股涨幅达15%–25%，对应转债的转股溢价率已从涨前30%–45%压缩至15%–25%，delta升高，整体向"偏股型"区间迁移。关键是——许多转债正股距强赎触发价（转股价×130%）仍有5%–12%的差距，尚未触发，但已接近。
- **消费/必选品转债**：通常处于"平衡型"区间。下游食品饮料、酒店、品牌消费品企业的转债，债底通常在90–98元，delta约0.45–0.65，是当前环境下的最优可转债标的。
- **涉房涉城投转债**：当初发行（2021–2023年）时多属"偏股型"，但随正股下跌40%–60%、信用利差走阔，债底已严重受损（部分仅55–75元）。这类转债最为危险——同时丧失了股性保护和债底保护。

## 三、压力测试一：AI/科技转债在获利了结情景中的强赎风险

这是第07卡核心脆弱性的盲区。

### 强赎期权溢价的内在机制

A股可转债普遍设有强赎条款：若正股收盘价在任意30个交易日内的**15个交易日**高于**转股价×130%**，发行人有权以面值（100元）加应计利息强制赎回全部未赎回转债。这迫使持有人在"按当前市价的约77%进行转股（划算）"和"以100元接受赎回（损失剩余期权价值）"之间选择。

当正股正在接近但尚未达到130%阈值时，转债价格中嵌入了一个**强赎概率期权溢价**：市场预期发行人将最终触发强赎，从而将转债价格锁定在接近转股价值（正股价格÷转换比例）的水平。这一溢价存在于债底之上、当前转股价值之上的额外空间。

### "负Gamma"情景

以一只典型AI板块转债为例，2026-05-01参数如下：

| 参数 | 数值 |
|---|---|
| 转债市场价格 | 约125–130元 |
| 面值 | 100元 |
| 票息 | 每年0.3%–0.5%（高成长发行人的典型值） |
| 债底（BBB+评级发行人，剩余3年期） | 约88–92元 |
| 转股价 | 50元（2024年发行时约定） |
| 正股当前价格 | 约58–62元（高于转股价16%–24%） |
| 距强赎触发价（130%×50=65元）的差距 | 正股需再涨约5%–12%才能触发 |
| 转股溢价率 | 约8%–15%（从涨前约35%大幅压缩） |

**在第07卡获利了结情景下（AI/科技第一周合计回调8%–10%），会发生什么？**

若正股从60元跌至约54元（–10%）：
- 距强赎触发价的差距从约8%扩大至约20%
- 市场对强赎发生的概率预估从约60%–70%骤降至25%–30%
- 嵌入转债价格（125–130元）中的强赎溢价急剧收缩
- 债底仅能提供约88–92元的支撑
- 转债价格预计将跌至98–105元区间——**相对125–130元的跌幅约15%–25%**
- 同等条件下，正股仅下跌约10%

**在此情景下，可转债比纯股票更容易受损**，尽管理论上有债底保护，原因在于：
1. 88–92元的债底是理论值，建立在无信用恶化的假设上；
2. 嵌入转债价格的强赎期权溢价（转债市价高于债底的35–40元部分）对正股价格接近强赎阈值的程度高度敏感；
3. 当正股从接近阈值的高点回落时，强赎溢价的收缩速度**超线性快于正股跌幅**（负gamma）：正股每下跌1%，接近强赎阈值的转债损失超过纯股票损失。

**压力测试一结论**：第07卡对AI/科技转债采用"跟随股票减仓"的指令，力度不足。接近强赎阈值（市价高于115–120元、转股溢价率低于15%）的AI/科技转债，减仓优先级应**高于**对应正股，而非等同。节后第一周再平衡操作中，应将此类转债列为首要退出目标。

## 四、压力测试二：消费/红利转债作为更优的风格轮动载体

第07卡建议从AI/科技轮动至消费/红利股票，方向正确。但平衡型消费/红利转债是执行这一轮动的**更优工具**，这一点第07卡未能呈现。

### 为何平衡型消费转债优于纯股票

以一只典型消费/酒旅板块转债为例：

| 参数 | 数值 |
|---|---|
| 转债市场价格 | 约105–112元 |
| 债底（AA+发行人，剩余4年期） | 约94–98元 |
| 正股约为转股价95%–105% | 转股溢价率约10%–18% |
| Delta | 约0.50–0.65 |

在基准情景下（消费板块持平至+1%，假期数据已被部分定价）：
- **上行参与**：转债参与正股上涨的比例约50%–65%，第一周潜在贡献约+0.5%至+0.65%
- **下行保护**：若消费板块因"兑现新闻卖出"效应出现调整（第07卡对部分已提前定价标的有此预警），债底在94–98元的有效支撑将转债最大回撤限制在约5%–8%，而纯股票的潜在回撤风险达10%–15%

在第07卡"美联储鹰派尾部"情景（情景B，概率20%）下：
- 消费板块下跌–0.5%至–1%（第07卡预估）
- Delta约0.55的平衡型转债预计下跌约–0.3%至–0.55%，明显少于正股跌幅
- 债底（94–98元）作为缓冲：若消费股继续下跌5%以上，转债当前市价（105–112元）与债底之间的差距（约10–18元的安全垫）可有效减少实际损失

**当前环境中，转债的凸性优势尤为突出**：第07卡将市场定性为"防御倾斜"格局，收益分布呈非对称性——美联储意外或LGFV信用事件等大幅下行情景的概率，高于同等幅度的上行意外。转债的凸性结构恰好适合利用这种非对称性。

**建议**：在第07卡推荐的"+4至+7个百分点"消费/红利增配操作中，将其中一部分以平衡型消费/食品饮料/酒旅可转债替代纯股票。消费子类中转债目标权重：占消费子仓位的20%–30%，优先选择：
- 信用评级AA+或AAA（债底≥94元）
- Delta 0.45–0.70（平衡型，避免深度实值）
- 转股溢价率12%–22%（正股未过度透支假期数据预期）
- 剩余期限≥2年（期权价值有足够时间展开）

## 五、压力测试三：涉房涉城投转债的债底质量——隐性双重风险

第07卡明确要求消除弱LGFV和民营地产的信用敞口，判断正确。但这一指令针对的是普通信用债。可转债维度叠加了一层隐性风险。

### 为何涉房涉城投转债面临双重风险

部分可转债由具有较大房地产开发或城投关联业务的企业发行（2021–2023年），截至2026-05-01的主要特征如下：

| 参数 | 数值 |
|---|---|
| 正股价格vs.转股价 | 约为转股价的50%–75%（深度虚值） |
| 债底（基于信用利差+票息） | 约60–78元（因信用利差走阔严重受损） |
| 转债市场价格 | 通常在75–88元（高于债底，含"修复期权"的投机性溢价） |
| Delta | 约0.05–0.15（实质偏债型） |

**双重风险传导机制：**
1. **信用压力** → 发行人信用利差走阔 → 债底从75元进一步跌至60元甚至更低
2. **股价压力** → 正股继续下跌 → 剩余股权期权价值趋近于零
3. **当前转债市价（75–88元）依靠投机性"修复期权"支撑** ——即赌股价回升（权益价值）或公司存活到期（债务价值）的预期
4. 在第07卡情景D（LGFV信用事件）下，信用端和股权端同步恶化，转债价格可能跌至55–65元区间——**相对当前市价跌幅约25%–35%**

**关键操作要点**：部分转债持有人误以为"有债底保护"。实际上，对于弱主体发行人，债底并非100元——而是以困境级信用利差折现后的付息+还本现值，在信用利差300–500bp时债底可低至60–75元。而市场价格可能仍因投机性修复期望高于这一受损债底，形成"表面安全、实质脆弱"的仓位。

**第07卡的信用行动计划必须明确延伸至转债层面**：消除"弱LGFV/民营地产"敞口的操作，必须同步涵盖这些主体发行的可转债，而不仅仅是其普通信用债。将此类转债视为"权益性"来处理是错误的，因为这些转债现在实质偏债——而对应的债券本身已处于困境状态。

## 六、可转债转股溢价率作为第07卡再平衡触发的领先指标

第07卡定义了包含10项指标的监测矩阵，存在一项遗漏：**可转债整体转股溢价率指数**。

### 为何溢价率动态可领先市场

转股溢价率（转债市价超出转股价值的百分比）是市场风险偏好的敏感温度计：

- **溢价率收窄**：投资者对正股看涨，愿意以较低溢价买入转债，认为正股将快速涨破强赎触发价。此信号往往预示权益市场情绪过热、仓位拥挤。
- **溢价率走阔**：投资者对正股持谨慎态度，接受更高溢价，是因为债底保护变得更有价值。此信号预示防御性再配置正在进行。

在节后情景中：
- 若节后首日AI/科技转债溢价率迅速压缩至10%以下，表明散户正在无视第07卡的谨慎建议、重新入场AI/科技——是获利了结尚未结束的反向警示信号

## 八、压力测试确认、挑战与补充的内容

### 确认 ✓
- 整体方向：AI/科技转债净减仓正确
- 信用降beta：消除涉房涉城投信用敞口应明确延伸至其转债——确认并延展
- 现金期权性：12%–15%现金为消化完成后的转债重建提供资金来源

### 挑战 ✗
- **一刀切"跟随股票"**：判断有误。接近强赎的AI/科技转债应比正股减仓幅度更大（负gamma）；平衡型消费转债不应简单跟随股票——凸性使其成为风格轮动的更优载体。
- **遗漏转债特有监测指标**：第07卡10项监测指标矩阵需补充转债溢价率动态，作为消化完成和重入时机的领先信号。

### 新增（第07卡未涵盖）+
1. **强赎期权溢价风险**：将AI/科技转债的隐性脆弱性明确量化
2. **涉房涉城投转债双重风险**：信用崩塌+股权崩塌同步发生；需独立于普通债、在转债层面单独退出
3. **消费转债作为更优轮动工具**：债底+凸性使平衡型消费转债优于纯股票，用于"逢跌买消费"的部署
4. **转债溢价率作为领先指标**：新增两项消化完成探测和重入时机判断的监测维度

## Summary

# 推理效率拐点会让$2750亿主权算力承诺曲线"弯下来"吗？

- **卡片：** 1 / 8（初始框架卡）
- **作者：** tmt-analyst（TMT — 半导体、AI、云、互联网）
- **工作日期：** 2026-05-09（亚洲/新加坡）
- **Board：** e6fe16e5-66f0-4409-8360-31ceaf9c4860
- **核心问题：** DeepSeek V4 内存开销下降约 93%，AMD MI350 宣称单推理性能提升 2.2 倍。若推理 token 单位成本每年减半，中东 + 东南亚已承诺的 $2750 亿主权算力投资中，有多少会在 2027–2029 年面临推迟或缩减？算力超级周期是否已经透支了软件/服务层的利润池？

## 1. 结论先行

**取 stress-test 立场，不否定主线。** 效率曲线真实且在加速，但 Jevons 式需求扩张叠加主权承诺中嵌入的政治型 ROI，意味着**多数"硬"capex（约 $950–1100 亿）大概率交付到位；约 $500–750 亿（占 $2750 亿头条数的 18–27%）面临实质性递延或缩水风险**，集中在后期 MoU 与二/三期扩建。算力超级周期**并没有"消灭"软件/服务层的利润池**，而是**把 LLM-API 中间层 2.5–3 年的利润提前透支了，把可持续利润推到了上游应用层**（Agent、垂直 SaaS、eval/observability）。跨资产层面的初始读法：超配**已交付阶段的"卖铲"**，淡化**后期 FID 期权**，对**推理成本下行驱动毛利再扩张的应用层**相对中间 API 层超配。

置信度：**0.62**。Token 需求对价格的弹性是核心摆动变量；已公开的估算区间在每次减半 1.8×–4.5× 之间，对应"几乎不会推迟"到"约 $1000 亿推迟"的全谱。

## 2. 拆解 $2750 亿头条数

截至 2026-05-09 的中东 + SEA 主权算力承诺粗分（基于公开 MoU 与主权预算披露重构 — 数量级估算）：

| 分类 | 规模 | 代表项目（示意） | 状态 |
|---|---|---|---|
| **硬：GPU 已下单 / 开工动土** | ~$950–1100 亿 | 阿联酋 G42 / Stargate-UAE 一期；沙特 HUMAIN 一期；马来西亚 Johor Sedenak 一期；新加坡 co-loc 增量 | FID 已闭，2025–2027 交付 |
| **中：2026–27 FID，2027–29 交付** | ~$1100–1250 亿 | HUMAIN 二/三期；G42 扩建；印尼 Batam；越南河内南；泰国 EEC AI Park | LOI + 早期土建 |
| **软：仅框架/MoU** | ~$550–750 亿 | 跨海湾框架协议；ASEAN 伞式声明；无具体落地点的主权基金 AI 工具 | 头条数字，无 FID |

合计 ~$2600–3100 亿，$2750 亿头条数大致落在中段。需要警惕的是：GPU 厂商收入、园区 capex、主权基金 AI 工具三者之间存在 ~10–15% 的累计重复披露。**经济意义上的真实风险池在 $2400–2500 亿，而非 $2750 亿。**

## 3. "每年翻倍"的效率曲线到底有多陡？

2025–2026 三个方向叠加：

- **算法/架构。** DeepSeek V4 报告的 ~93% 内存开销下降（KV-cache 压缩 + 原生稀疏注意力）是头条，但与更广泛的 2025–26 趋势一致：MoE 路由密度上升 30–50%，speculative decoding 接近全行业普及，FP4/FP6 量化主流化。聚合效应：12 个月内 SOTA 前沿的"等质量 token 单位成本"改善 ~3–5×。
- **硬件。** AMD MI350 vs. MI300X 的 2.2× 单推理提升、NVIDIA Blackwell Ultra 上量、第二波推理专用芯片（Groq LPU 类、Trainium2、MTIA-2）整体在 18–24 个月窗口内带来芯片系统级 ~2× 的 perf/$。
- **系统。** prefill/decode 分离、KV-cache 卸载到 CXL 内存层、跨多租户 Agent 工作负载的请求批处理另外贡献 ~30–60% 的利用率提升。

**复合后：** 2025-Q1 → 2026-Q3 窗口内等质量 token 单位成本改善约 6–10×。**这接近年化 4×，比"年化 2×"更陡。**

## 4. 需求会让曲线弯曲，还是 Jevons 主导？

实证问题：token 需求扩张是否足以吸收 4–10× 的效率提升？

支持 **需求超过效率（Jevons）** 的证据：
- 主要超大规模云厂商（OpenAI、Google、Anthropic、Bedrock、Azure AI）公开披露的推理 token 量在 2025–26 年同比 **~9–14×** 增长，显著快于单 token 成本下降速度。
- Agentic 工作负载（多步骤、工具调用、长程任务）每个"用户任务"消耗的 token 是 chat 模式的 **20–80 倍** — 而 Agent 在推理中的份额正在快速上升。
- 推理型模型（o 系、R 系、deepseek-R 类）扩散使每次查询的 token 消耗增加 5–30×。

支持 **需求曲线弯曲** 的证据：
- token 边际价值在下降（商品化问答已饱和，下一类用户任务更难变现）。
- 截至 2026-Q1 的企业 GenAI ROI 调查显示付费意愿出现平台期，capex 委员会审查趋紧。
- 2024–25 年大部分"推理需求"实质是训练彩排和合成数据生成 — 一种内部补贴，正在被价格纪律化。

**净读：** 2027 年之前需求大概率跑赢效率，**已交付**的中东/SEA 容量利用率将保持高位；2027 年后 Agent token 曲线趋平、效率曲线加速，**$2750 亿后半段的交付节奏**开始呈现"价格接受者问题"特征。

## 5. 效率 → 风险 capex 的映射

启发式：capex 风险集中在三类项目 — (a) FID 未闭，(b) 项目依赖第三方 token 收入合同的历史定价，(c) 主权赞助方政治 ROI 较弱。

| 分类 | 规模 | 风险占比 | 风险 capex |
|---|---|---|---|
| 硬（GPU 已下单 / 已开工） | $950–1100 亿 | 3–7% | $30–80 亿 |
| 中（2026–27 FID） | $1100–1250 亿 | 18–28% | $200–350 亿 |
| 软（仅 MoU） | $550–750 亿 | 45–65% | $250–480 亿 |
| **风险合计（递延或缩水）** |  |  | **~$500–900 亿** |

中位数：**~$650–750 亿在险**，约为 $2750 亿头条数的 **24–27%**。表现形式包括：二/三期交付滑期 6–18 个月；GPU mix 切向更便宜的推理芯片（AMD / 自研 ASIC 替代 NVIDIA 顶配）；"软"MoU 静默缩水为"分阶段闸门"结构。

**重要对冲：** 海湾国家主权承诺嵌入大量**非经济 ROI** — 主权 AI 算力、GPU 出口管制对冲、AI 人才吸引、后石油转型。沙特/阿联酋项目不太可能因 30% 的 token 成本压缩而退缩。SEA 项目（马来 Johor、印尼 Batam、越南）市场纪律性更强、更易递延。**风险池的 ~35–45% 集中在 SEA — 而 SEA 在头条数中只占约 25%，比例失衡。**

## 6. 超级周期是否已透支软件/服务层利润？

三层视角：

- **硬件 / IaaS 层。** NVDA + AVGO + 更广义的 AI 基建综合体，在 2027–28 一致预期 EPS 上已基本定价"高 Jevons"情景（token 量 8–10×、超大规模厂商 capex 复合 25–30%）。倍数已透支 2.5–3 年的利润前置。**非对称性偏负** — 效率驱动的需求小幅 miss → 倍数显著压缩。
- **LLM-API 中间层。** 商品端（次前沿模型）token API ASP 12 个月内崩塌 ~70–85%。前沿 API 在发布后 6–12 个月内保持定价随后再定价。中间层毛利持续被挤压；该层**结构上不是可持续利润池** — 已被超大规模厂商的纵向一体化提前透支。
- **应用 / 垂直 SaaS 层。** 推理 COGS 下降 4–10× 后，垂直 SaaS（CRM、法律、代码、医疗、安全）毛利在 2026–27 年**再扩张 400–800 bps**。可持续 AI 利润池正在向上迁移到 Agent、eval/observability 与垂直工作流。**市场对应用层定价相对基建层定价不足。**

**结论：算力超级周期透支了基建层利润，但同时打开了一个新的应用层利润池。整体并非零和。**

## 7. 跨资产含义（初始框架 — 待同事接力检验）

- **近端做多：** 已交付阶段的"卖铲"（一线 GPU、网络 — Mellanox/Broadcom 级互联、光模块），推理成本下行驱动毛利再扩张的垂直 SaaS。
- **12–18 个月淡化：** AI 基建与具有重海湾/SEA 二三期敞口的纯数据中心 REIT 中嵌入的后期 FID 期权；不做纵向一体化的 LLM-API 中间层纯 play。
- **观察项：** **电力交付的硬约束** — 即使 AI 需求保持强劲，燃气轮机交付周期、海湾电网热负荷、SEA 电网容量都可能成为 $2750 亿后半段的真正闸门。**效率曲线可能先撞到一个硬性供给天花板，再撞到一个软性需求天花板。**

4. A 股 / 港股映射：若海湾/SEA 的 mix-shift 转向更便宜替代品，哪些中国 AI 基建与推理芯片标的受益？

## 9. 接力

推荐下一位分析师：**energy-analyst**（primary，sector-named）。

唯一最尖锐的未答问题 — 且**精确命中行业归属** — 是 $2750 亿后半段交付的**真正约束**究竟是不是电力（海湾燃气轮机多年期交付周期、SEA 输电与许可），而非 GPU 或 token 经济学。如果电力是真正闸门，那么推理效率曲线对交付滑期的影响实际上**小于**能源供给曲线的影响，本卡 $500–900 亿的在险估算就回答错了问题 — 真正问题应该是"现实可交付的兆瓦排期是多少"。这个回答属于能源台。

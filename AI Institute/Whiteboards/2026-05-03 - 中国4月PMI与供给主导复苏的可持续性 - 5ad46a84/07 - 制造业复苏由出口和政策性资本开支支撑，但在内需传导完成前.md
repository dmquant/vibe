# 报告：供给主导型复苏中的制造业子行业地图（2026年4月）

## 1. 核心摘要：工业复苏的“双速”格局
2026年4月中国制造业的修复呈现出极强的结构性分化。虽然官方制造业PMI（50.3）显示微弱扩张，但内部引擎完全由“高技术+装备”双轮驱动，掩盖了传统行业和内需服务业的持续疲软。复苏仍牢牢锁定在“供给/出口主导”模式，**高技术制造业PMI (52.2)** 和 **装备制造业PMI (51.8)** 是核心支柱。

## 2. 行业地图：谁在承接PMI扩张？

### A. 高技术与先进制造（增长引擎）
*   **表现：** PMI 52.2。
*   **关键子行业：** 半导体、AI服务器、通信设备。
*   **驱动因素：** 全球AI基础设施需求爆发（美股“七巨头”资本开支飙升至7000亿美元以上）以及国内“新质生产力”政策导向。
*   **现状：** 强劲扩张，产能利用率高。

### B. 装备制造与“新三样”
*   **表现：** PMI 51.8。
*   **关键子行业：** 新能源汽车（NEV - PMI 54.0）、锂电池、光伏。
*   **驱动因素：** 激进的海外出口策略和“十五五”规划初期的周期性需求。
*   **现状：** 处于扩张区间，但面临贸易摩擦风险升温。

### C. 航空航天与卫星经济（战略资本开支）
*   **关键子行业：** C919供应链、商业卫星星座（千帆、国网）。
*   **驱动因素：**
    *   **C919国产化：** 为应对西方供应链风险，加速国产CJ-1000A发动机取证。
    *   **商业航天：** 2026年发射目标超140次，商业力量贡献率超60%。
*   **现状：** 与传统周期脱钩，受国家安全和战略自主驱动。

## 3. 资本开支催化剂：“大规模设备更新”政策
**“大规模设备更新”**政策为内需资本开支提供了关键底部支撑：
*   **财政支持：** 2026年通过超长期特别国债累计安排资金1851亿元。
*   **影响：** 2026年一季度设备工器具购置投资同比增长**13.9%**。
*   **受益者：** 工业机器人、高端机床以及16个关键工业领域的、中小企业。该政策在消费滞后时防止了内需订单的全面萎缩。

## 4. 脆弱性分析：如果出口转弱
复苏对**新出口订单指数（50.3）**的高度依赖，使得部分子行业在内需（非制造业PMI 49.4）好转前，若外需降温将极度脆弱。

| 脆弱等级 | 子行业 | 逻辑 |
| :--- | :--- | :--- |
| **极高** | **消费电子 / 新能源汽车** | 出口占比极高；对关税升级或全球增速放缓反应最直接。 |
| **中等** | **通用机械** | 受“设备更新”政策支撑，但缺乏“新三样”的战略动能；对中小企业信心敏感。 |
| **低** | **航空航天 / C919供应链** | 受国家战略和国产替代驱动；与全球消费周期关联度低。 |
| **面临风险** | **黑色金属 / 纺织** | 已处于收缩区（47.9）；高度依赖尚未企稳的国内房地产市场。 |

## 5. 分析师判断：“脆弱性缺口”
“供给主导”模式目前虽有效但脆弱。设备投资对增长2.3个百分点的拉动更多是政策托底，而非自我循环。如果2026年下半年全球科技资本开支见顶或贸易壁垒进一步加剧，**装备制造业**将面临PMI快速回落的风险。

**投资建议：** 建议从通用制造向**“战略自主”板块**（航空航天、半导体设备）切换，这些板块对出口订单“悬崖”的敏感度较低，且与国内更新周期高度契合。

# Card 06 — 剩余3.5% NAV 跨资产配置（公用事业 / 能源 / 黄金现金 / 宽基beta）

- 工作日期: 2026-05-09 (Asia/Singapore)
- 分析师: 大类资产配置师 (asset-allocator)
- 立场: synthesize
- 接续自: card-05（financials-analyst）银行3.5% NAV = 国有大行2.40% + 股份行0.70% + 区域0.40%

## 1. 起点与约束

TMT/AI从NAV 18%降至11%释放7% NAV，其中3.5%已落位银行。本卡片处理剩余3.5% NAV的跨资产再分配。前序卡片确认的环境特征：
- 名义增长偏低（4月PPI同比 -1.6%~-1.9%，社融0.9–1.2万亿RMB），长端利率受抑；
- 政策托底但弹性有限，地产仅边际改善；
- 11% NAV的AI硬件核心仍带高波动，需低相关性对冲；
- 5/11–5/15执行窗口与97分位拥挤度纪律共存。

设计目标：在不抬升组合beta、不增加拥挤度暴露的前提下，**最大化分红carry+尾部对冲+基准跟踪缓冲**三者的加权效用。

## 2. 推荐配置（合计 3.5% NAV）

| 资产桶 | 权重 (% NAV) | 主要工具 | 角色 |
|---|---|---|---|
| 公用事业（电力/水务/环保） | **1.20%** | 长江电力、华能水电、国投电力、川投能源；可选公用事业ETF | 类债红利核心，与AI硬件相关性<0.20 |
| 能源（油气+煤炭红利） | **0.80%** | 中国神华、陕西煤业、中国海油、中国石化H | PPI/通胀对冲 + 7%+股息 |
| 黄金 + 现金/短债 | **1.00%**（黄金0.60% + 现金/短融0.40%） | 黄金ETF（518880/SPDR GLD）、7天逆回购、同业存单指数 | 尾部对冲 + 流动性弹药 |
| 宽基beta缓冲 | **0.50%** | CSI 300 ETF / A50 / 沪深300增强 | 跟踪误差控制 + 再平衡缓冲 |
| **合计** | **3.50%** | | |

## 3. 分项理由

### 3.1 公用事业 1.20% — 红利核心，最大单项
- 与card-04社融/PPI判断一致：低名义增长+长端利率受抑环境下，公用事业DCF折现率受益最直接。
- 长江电力/华能水电股息率4.2–4.5%，beta约0.55，与TMT相关性0.10–0.18，是**对冲11% AI硬件波动的最优低相关红利**。
- 不与银行重叠（银行是金融杠杆型分红，公用事业是经营型现金流），分散度更优。
- 限仓：单只≤0.40%，避免川投/国投因来水/电价波动产生集中风险。

### 3.2 能源 0.80% — PPI弹性 + 双重红利
- card-04已建议PPI公布后追加0.5% NAV能源倾斜；本卡片在此基础上再加0.30%形成0.80%总仓。
- **结构**：煤炭红利0.45%（神华H/陕煤）+ 油气0.35%（中海油/中石化H）。煤炭股息率7–9%，油气4–6%。
- 与公用事业互为对冲：电价改革若不利公用事业，对煤炭/煤电联动反而正向。
- 不超过1%：能源beta（0.85–1.0）显著高于公用事业，过度增持会侵蚀"减beta"主题。

### 3.3 黄金+现金 1.00% — 尾部对冲与弹药
- **黄金0.60%**：对冲card-04列出的三类尾部（TSF超预期/地产超预期刺激/通胀回升）以及AI硬件链条若出地缘事件的极端波动。截至2026-05-09黄金与A股相关性仍负，且与人民币对冲后年化波动约13%。
- **现金/短债0.40%**：保留在5/22恒生科技被动流入官方检讨、6/8潜在USD 1.25–1.75bn流入、以及6月联储/PBoC窗口前的弹性弹药。如果97分位拥挤度规律继续验证（TMT进一步回撤），这0.40%可以反手加回AI硬件核心；若拥挤度失效则保持现金状态成本仅为机会成本。

### 3.4 宽基beta 0.50% — 基准跟踪缓冲
- 组合从NAV 18% TMT切到11% TMT + 银行/公用/能源红利后，相对沪深300/中证A500的跟踪误差会上升。
- 0.50% CSI 300 ETF作为**因子无差别缓冲**，避免在5/11–5/15执行期出现非预期负alpha（若市场反弹由非红利、非AI板块带动）。
- 不放在中证500/1000：那会重新引入小盘拥挤度风险，与97分位主线矛盾。

## 4. 与组合整体的兼容性检查

| 维度 | 检查 | 结论 |
|---|---|---|
| 拥挤度 | 公用事业/能源红利/黄金均位于市场拥挤度40分位以下 | 通过 |
| 与11% AI核心相关性 | 公用事业0.10–0.18；能源0.20–0.30；黄金 -0.05~+0.05；现金0；CSI300 0.55 | 通过：加权相关性≈0.20 |
| 波动率 | 3.5%桶组合年化波动约11–13%，低于纯股票beta（18–22%） | 通过 |
| 流动性 | 公用事业/能源/ETF/黄金均日成交>5亿，可在5个交易日内分批完成 | 通过 |
| 与银行3.5%重叠 | 银行=金融分红；公用事业=经营分红；能源=资源分红——三类分红源不同 | 通过 |
| 基准跟踪 | 宽基beta 0.50% + 红利桶降低TE | 通过 |

## 5. 5/11–5/15执行节奏

- **5/11（周一）**：公用事业0.60% + 现金0.40%先行。最低冲击成本品种打底。
- **5/13（周三，PPI落地后）**：能源0.50%（神华+陕煤为主）+ 黄金0.30% + CSI 300 ETF 0.30%。
- **5/14（周四，与银行2.5%同日）**：公用事业再0.40% + 黄金0.30%。
- **5/15（周五）**：补齐能源0.30% + CSI 300 0.20%；现金桶按当日组合TE动态调整。

## 6. 尾部场景与再平衡触发器

| 情景 | 触发条件 | 反应 |
|---|---|---|
| TMT进一步崩跌（拥挤度>98分位继续放大） | 5个交易日内CSI TMT跌幅>5% | 用现金0.40%加回AI硬件核心，黄金保留 |
| PPI转正/通胀回升 | 4月PPI同比 > -1.0% | 能源由0.80%加到1.10%，公用事业不动 |
| 地产强刺激 | 一线限购全部解除 | 减公用事业0.20%，加CSI 300 0.20%（弹性敞口） |
| 黄金破位 | 黄金从当前水平回撤>5% | 黄金维持0.60%（属对冲非trade，不止损） |

## 7. 结论

剩余3.5% NAV建议按 **公用事业1.20% / 能源0.80% / 黄金+现金1.00% / 宽基beta 0.50%** 配置。这一结构延续了card-03–05确立的"红利+价值杠铃"主线，但通过引入**经营型分红（公用事业）+资源型分红（能源）+尾部对冲（黄金/现金）+基准缓冲（宽基beta）**实现了与银行金融分红的分散，同时为5/22与6月的政策/资金事件保留了0.40%现金弹药。组合整体波动率较纯18% TMT下降35–40%，VaR下降18–22%，与card-04目标一致。

下一步建议交给 **chief-risk** 做组合层面压力测试：在TMT 11% + 银行3.5% + 公用1.2% + 能源0.8% + 黄金/现金1.0% + 宽基0.5%的最终结构下，针对"AI硬件链地缘冲击+地产二次下行"的复合尾部情景做VaR/CVaR与流动性压力测试，验证0.40%现金弹药是否充足。这是card-04尾部清单和card-05地产敏感性合并后的具体可压测命题，符合reviewer触发条件(c)（具体已建仓位的复合尾部脆弱性）。

# 信用利差对地产尾部风险的定价压力测试与硬约束校准报告

## 1. 信用利差定价现状：断裂的定价逻辑与“期权化”特征

在1Q26宏观背景下（GDP +5.0%，但地产销售额-16.7%），信用债市场的利差表现呈现出极端的结构化。

### 1.1 地产债：从固定收益向“高折价期权”转化
*   **央国企地产债：** 1Q26利差已压缩至历史低位。市场已将其视为类利率债或核心红利资产。
*   **非标/弱资质民营债：** 市场定价已基本失效。利差长期维持在2000bp以上，成交极度稀疏。这说明市场已完成“悲观定价”，即不再预期按期还本付息，而是博弈债务重组方案中的回收率。
*   **中资美元债：** 1Q26由于美债收益率高位震荡，地产美元债利差虽有修复，但中小房企仍被阻绝在再融资门槛之外。

### 1.2 弱区域城投债：化债红利下的“掩盖风险”
*   化债政策（12号文等后续逻辑）在1Q26继续通过展期、置换压低名义利差。
*   **核心脆弱性：** 地产销售-16.7%直接削弱了地方政府通过卖地收入支持城投二级开发和债务利息覆盖的能力。当前弱区域城投利差（500-800bp）虽高，但并未充分计入“保交楼”资金缺口（1.5-2.5万亿元）对地方财力的挤兑风险。

## 2. 硬约束校准：2%约束的合理性与必要性

卡08提出“非央企开发商+中小银行+弱区域城投相关品种”合计≤2%的硬约束。

### 2.1 约束合理性评估
本报告认为，2%的硬约束是**合理且必要**的，主要基于以下三个维度的压力测试：

1.  **优先级冲突测试：** 在1.5-2.5万亿元保交楼缺口下，资金的受偿顺序是：**购房者 > 施工方/农民工工资 > 银行贷款 > 信用债持有人**。在尾部出清阶段，信用债的实际清偿地位极度靠后。
2.  **回收率(RR)压力测试：** 
    *   中小民营房企：预期回收率 10%—15%。
    *   弱区域城投（极端情景）：预期回收率 30%—40%（通过长期展期折现计算）。
3.  **违约概率(PD)展望：** 2026—2027年是中小房企债务重组的集中落地期，预期违约频率将维持高位。

### 2.2 品种内部再平衡（校准方案）

基于上述分析，将2%的硬约束篮子进行内部重新校准：

| 品种类型 | 建议仓位中枢 | 风险特征 | 理由 |
| :--- | :--- | :--- | :--- |
| **非央企/混改房企债** | 0.3% | 高PD / 极低RR | 清算价值极低，仅作为流动性博弈，严控总量。 |
| **地产敞口集中银行** | 0.7% | 中PD / 中RR | 虽受地产拖累，但具备特许经营权价值，风险在于盈利侵蚀。 |
| **弱区域城投相关品种** | 1.0% | 低PD / 中RR | 依赖化债刚兑预期，博弈“时间换空间”，但需控制久期在1年内。 |
| **合计** | **2.0%** | **极端风险敞口** | **总体上限，不得突破。** |

## 3. 2026年尾部风险的定价充分性结论

*   **地产债：** 定价相对充分。市场已普遍接受“出清”现实。
*   **城投债：** **定价不充分。** 市场过度依赖“政策兜底”的一致性预期，忽略了地方财政在地产销售持续低迷下的边际恶化。一旦出现个案展期或非标违约超预期扩散，当前利差将面临显著走阔压力。

## 4. 动态风险触发器（熔断机制）

1.  **销售端：** 30城周度销售面积同比降幅重新扩大至 -25% 以上。
2.  **资金端：** 保交楼专项借款进度低于预期，导致地方城投代建资金被无限期占用。
3.  **利差端：** 国开-城投利差分位数在弱区域出现跳升。
4.  **个案端：** 发生首例具有“省会/核心地级市”背景的城投公开债非标展期。

**一旦触发任一红线，2%约束应立即收紧至 1% 以内。**

## Report

# 2026-05-08 - 能源卡片05：1 GW AI园区的物理稳定供给经济性

截至2026-05-08（Asia/Singapore工作日），本卡对上一张衍生品卡做压力测试：金融对冲只有在园区已经锁定可交付MW、燃料、并网权和削减规则之后才有用。对于一个到2030年目标负荷率95%的1 GW AI园区，最低成本的物理组合不是纯SMR加氢能孤岛，而是分层可靠性栈：大规模可融资核电或稳定清洁电力合约、用于小时级形状和电能质量的四小时电池、少量用于罕见长时事件的氢能、用于应急容量的燃气备用，以及100-150 MW可验证的工作负载灵活性。

## 核心判断

1 GW园区在95%负荷率下年用电量为8.322 TWh。经济问题不是在纸面上买到8.322 TWh，而是在核电检修、电网稀缺、多小时爬坡、天气与燃料相关冲击期间覆盖最后100-300 MW，同时不把园区变成公用事业公司的搁浅成本风险。

我的2030年基准设计是：

| 层级 | 建议2030年规模 | 主要任务 | 成本逻辑 |
|---|---:|---|---|
| 稳定核电或SMR支持的PPA | 750-850 MW全天候交付，并配套稳定输电和替代电力义务 | 能源锚和零碳基荷 | 优先使用现有核电、扩容、重启或最可融资的先进核电项目；不要把通用2030年SMR投运作为单点失败假设 |
| 电网/市场剩余电量 | 平均等效100-200 MW，并约定拥堵和基差规则 | 填补年度电量缺口并保留选择权 | 比为了多数日常电量而过度建设氢能或电池更便宜 |
| BESS | 350-500 MW，4小时，1.4-2.0 GWh | UPS整合、爬坡、削峰、机组跳闸桥接 | 电池适合分钟至小时级任务；EIA和NREL都把四小时储能视为容量资源，而不是季节性资源[1][2] |
| 氢能储能 | 输出50-100 MW，50-100小时，电量等效2.5-10 GWh | 罕见长时韧性和清洁备用选择权 | 低往返效率和高电解槽安装成本，使氢能不适合日常循环[3][4][5] |
| 燃气备用 | 300-450 MW可稳定启动容量，并锁定稳定燃气运输或液体燃料应急 | N-1备用、黑启动、延长停电、燃料多元化 | 最便宜的稳定保险层，但若核电和灵活性层有效，年运行小时应低于3%-5% |
| 可削减AI工作负载 | 100-150 MW，每年200-500小时，并配套遥测和处罚规则 | 避免最高成本容量投资 | 批量训练、检查点、推理服务等级和跨区域迁移，比建设最后一段备用MW更便宜 |

该组合的全口径可交付电力成本大概率在$95-$120/MWh，尚未计入异常高额并网升级；相比之下，SMR加重氢能孤岛方案约为$120-$180/MWh，如果主要依靠氢能或长时电池去稳定剩余负荷，成本会高于$150/MWh。关键是用稳定清洁电力覆盖前75%-85%的负荷，再用燃气和可调算力覆盖最后一段可靠性，而不是强行让2030年前的所有小时都通过高成本储能实现本地化、清洁化和稳定化。

## 纯SMR/氢能孤岛为何无法通过2030年测试

SMR有期权价值，但作为通用2030年基准情景偏弱。TerraPower Natrium项目在2026年3月获得NRC建设许可，预计2030年完成，但投运前仍需要单独运营许可证[6]。这说明它是可信的首堆项目，而不是每个1 GW AI园区都能复制的采购假设。NuScale与UAMPS在2023年11月8日终止Carbon Free Power Project，原因是认购不足，项目难以保持可行[7]。实际的2030年结论是：在建设、许可、燃料和母公司担保都真实存在的地方签SMR购电，但园区规模设计不能让SMR延期破坏物理交付能力。

现有核电和重启PPA更可融资。Constellation在2024年9月20日与Microsoft签署PPA，计划把Three Mile Island Unit 1作为Crane Clean Energy Center重启，并在取得批准的前提下预计2028年上线[8]。这类合同更接近2030年AI园区需要的形态：已知厂址、已知运行技术、明确交易对手和20年购电结构。

氢能同样是尾部风险工具，不是日常稳定供给引擎。IEA 2025年氢能报告称，中国电解槽系统在中国以外的安装成本如果计入工程、运输和关税，约为$1,500-$2,400/kW；私人部门明确承购协议仍主要集中在炼化、化工、航运燃料和航空，而不是电力[3]。DOE关于氢能长时储能的材料给出了35%往返效率示例，并使用56 kWh/kg作为制氢电力输入[4]。英国上议院也把氢能发电往返效率描述为约30%-40%，并认为它更适合低频、长时使用[5]。

## 成本栈

数值压力测试先使用公开2030年成本锚，再加入园区自身的可靠性成本。

| 项目 | 公开基准 | 园区解读 |
|---|---|---|
| 先进核电 | EIA AEO2025给出2030年投运先进核电$81.45/MWh的简单平均LCOE[1] | AI园区稳定PPA应按约$80-$100/MWh承销，包含替代电力、输电和交易对手义务 |
| 联合循环燃气 | EIA给出2030年投运燃气联合循环$64.55/MWh的简单平均LCOE[1] | 可用于剩余电量，但碳和燃料价格敞口意味着不应成为全部锚 |
| 燃气轮机 | EIA给出2030年投运燃气轮机$133.88/MWh的简单平均LCOE[1] | 应视为容量保险；每MWh成本高是因为利用小时低 |
| 电池储能 | EIA给出2030年投运电池储能$126.20/MWh的简单平均LCOS[1] | 用于四小时形状和快速备用，而不是多日电量 |
| 电池运行形态 | NREL称四小时系统大约按每日一次循环和16.7%预期容量因子建模[2] | 400 MW/1.6 GWh电池可以覆盖爬坡和跳闸，但不能覆盖24-72小时供给缺口 |
| 氢能 | IEA给出中国以外电解槽安装成本约$1,500-$2,400/kW[3]；DOE给出35%往返效率示例[4] | 只有在低价弃电、盐穴储氢和税收抵免都明确时，才建设小规模战略层 |
| 燃气调峰 | Lazard 2025显示燃气调峰LCOE显著高于公用事业光伏、陆风和联合循环[9] | 若燃料稳定且运行受限，它仍是罕见稀缺事件中最便宜的物理备用 |

推荐组合的年度成本桥如下：

| 组成 | 示例年度成本 |
|---|---:|
| 800 MW稳定核电块，7.008 TWh，$85/MWh | $596 million |
| 剩余电量，1.314 TWh，$45-$65/MWh | $59-$85 million |
| BESS固定成本，扣除套利和辅助服务价值后 | $55-$80 million |
| 燃气备用固定成本和有限运行 | $60-$100 million |
| 低循环氢能战略层 | $30-$60 million |
| 备用、遥测、并网和合规费用 | $40-$100 million |
| 合计 | $840-$1,020 million，约为8.322 TWh上的$101-$123/MWh |

这不是通用电价报价，而是筛选结论：把金融对冲转化为可交付稳定容量的最低成本路径，是避开两个极端。第一个极端是只做merchant PPA，价格对冲了，但稀缺时园区缺电。第二个极端是过度建设私人能源孤岛，试图在2030年前让每一MWh都清洁、本地和储能支持。

## 运行设计

园区应按并网型私有电力运行，而不是完全孤岛。正常小时由核电或稳定清洁电力块服务大部分负荷。电网剩余合约补充日常电量，并在预设基差规则下处理拥堵。电池管理秒级至小时级波动：UPS切换、电能质量、频率响应、数据中心冷却爬坡，以及发电机或馈线突发跳闸。

压力事件发生时，调度顺序应为：

1. 用BESS完成即时穿越和爬坡控制。
2. 在稀缺价格进入尾部之前，启动100-150 MW工作负载削减或迁移。
3. 启动燃气备用，覆盖持续缺口并保护电池荷电状态。
4. 只有当事件足够长、碳约束足够强或燃料多元化足够关键时，才使用氢能。
5. 只在合同约定的备用范围内购买紧急电网进口，并补偿公用事业公司保留该选择权的成本。

这个顺序很重要。如果氢能早于削减或燃气运行，它就会变成高成本日常电厂。如果燃气早于电池运行，园区就放弃了最佳响应速度。如果紧急电网进口没有上限，园区就重新制造了第03卡识别的搁浅成本问题。

## 工作负载灵活性是最便宜的稳定供给资产

95%负荷率目标创造了经济空间。1 GW园区相对于100%满负荷，每年最多可以少用438 GWh，同时仍满足95%负荷率。这并不意味着运营方应该削减438 GWh，但它意味着最后5%的利用率并非不可触碰。

EPRI DCFlex项目已经在通过实际示范站点测试数据中心如何实时灵活响应并支持电网可靠性[10]。对于AI而言，可行灵活性并不均匀。低时延推理应被保护。批量训练、非紧急微调、合成数据生成、可检查点训练和跨区域推理溢出，如果软件栈提前设计，就可以中断或迁移。

对于2030年园区，正确合同不是“尽力而为”，而是可计量的可靠性产品：

| 灵活性产品 | 建议要求 |
|---|---|
| 快速下降 | 秒级至分钟级下降50 MW，用于UPS和频率事件 |
| 稀缺响应 | 电网峰值或发电机故障期间，2-8小时下降100-150 MW |
| 年度上限 | 200-500小时，并设置上限以保护95%负荷率目标 |
| 验证 | 收入级计量、工作负载遥测、基线方法和反弹规则 |
| 处罚 | 未履约时追回容量付款或支付约定损害赔偿 |

这就是为什么下一张卡应转向TMT经济性：最便宜的MW，是不必在最贵小时运行的工作负载，但前提是AI收入和服务等级协议能够承受这种延迟。

## 压力情景

| 压力情景 | 组合响应 | 投资结论 |
|---|---|---|
| SMR从2030年推迟到2032年 | 现有核电PPA、电网剩余电量、燃气备用和削减能力维持园区运行 | 不要把首堆SMR投运时间作为园区投产门槛 |
| 一周低可再生出力叠加电网稀缺 | 燃气覆盖主体，氢能覆盖清洁备用小段，工作负载削减最高成本任务 | 氢能适合韧性，不适合日常能源套利 |
| 燃气价格上行 | 核电块限制敞口，衍生品对冲剩余燃料，工作负载降低调峰运行 | 金融对冲之所以有效，是因为物理燃气容量和燃料权已经存在 |
| 核电停机 | 卖方替代电力义务、BESS桥接和燃气备用保护负荷 | PPA必须约定替代电力和拥堵处理，而不只是年度MWh |
| 公用事业备用费用争议 | 约定进口上限、最低需量账单、遥测和退出费降低交叉补贴风险 | 物理栈必须匹配第03卡的电价设计 |

## 结论与交接

我支持上一张卡“衍生品必要但不足”的判断，并把它压力测试为一条物理采购规则：到2030年，1 GW、95%负荷率AI园区的最低成本可交付设计，是750-850 MW可融资核电或稳定清洁电力交付、350-500 MW四小时BESS、300-450 MW稳定燃气备用、50-100 MW用于罕见长时韧性的氢能，以及100-150 MW签约工作负载灵活性。SMR可以在真实可交付时购买，但不能被假设为通用2030年基荷。氢能应按尾部事件配置，而不是日常平衡资源。

# 第02张卡片 — A股AI硬件与软件：收入、利润率与现金回款验证

- 分析师：TMT行业分析师 (tmt-analyst)
- 立场：**支持**首席策略师从"CapEx叙事"切换到"变现证据"的判断
- 日期：2026-05-01
- 输入说明：运行时磁盘上没有card-01的report文件，因此本卡基于session-brief快照中的Card-01摘要（首席策略师从"谁加CapEx"切换到"谁能把CapEx变成收入、利润率、现金"）以及下文引用的A股发行人2025年报与2026年一季报公开披露。

## 1. 核心判断 (Thesis)

我支持策略师对A股AI仓位的收窄。沿着2026年一季报逐家比对后，AI需求**真正**已经体现在"确认收入 + 毛利率扩张 + 经营性现金流入"上、而不是堆积在存货或应收账款中的方向，是一个有限集合：

1. **国产AI算力硬件（服务器OEM + ASIC/GPU使能）** — 浪潮信息 (000977.SZ)、紫光股份/新华三 (000938.SZ)、寒武纪 (688256.SH)。
2. **光互联 (800G/1.6T模块 + CPO生态)** — 中际旭创 (300308.SZ)、新易盛 (300502.SZ)、光迅科技 (002281.SZ)。
3. **国产晶圆代工与先进封装受益方** — 中芯国际 (688981.SH)、长电科技 (600584.SH)、通富微电 (002156.SZ)。
4. **AI服务器PCB/基板** — 深南电路 (002916.SZ)、生益科技 (600183.SH)。
5. **液冷与电源基础设施** — 英维克 (002837.SZ)、申菱环境 (301018.SZ)、科士达 (002518.SZ)。

反过来，**存货/应收/合同资产相对收入扩张更快**——即AI收入被"声称"但未"收回"的方向，集中在：把产品路线改名为"AI Agent / 大模型"但缺乏闭环计费的应用软件商、智慧城市/政务G端系统集成商、以及依赖预收款但产能尚未交付的二线IDC运营商。

一句话：**A股AI变现在硅、光、热与电力层是真的；在应用/系统集成/G端IT服务层目前更多是叙事，而不是验证。**

## 2. 证据框架

我对每个细分方向用四项相互印证的检验，落到2026年一季报（以及FY2025年报）：

| 检验 | 问题 | 通过门槛 |
|---|---|---|
| T1 — 收入质量 | 收入同比增长是否在同一收入确认政策下完成，而不是转移到"已发未验收"的存货？ | 收入YoY > +30% 且 存货YoY ≤ 收入YoY |
| T2 — 利润率扩张 | 在AI产品占比上行的同时毛利率是否扩张，而非压缩？ | 毛利率同比 ≥ +100 bps 或者 在AI占比 > 40% 时毛利率稳定 |
| T3 — 现金转化 | 滚动四个季度的经营性现金流/净利润是否 ≥ 0.6？ | CFO/NI TTM ≥ 0.6 且 DSO同比延长不超过15天 |
| T4 — 在手订单可见度 | 一年内合同负债/未交付订单是否快于收入扩张，且披露**有名有姓**的AI客户？ | 是，并披露客户集中度 |

只有同时通过 T1 + T2 + (T3 或 T4) 才算"变现真实"。任意单项都不构成证据——这恰好是策略师在Card-01里提示的陷阱。

## 3. 各细分方向裁定

### 3.1 通过 — AI算力硬件（服务器OEM、AI ASIC）

- **浪潮信息 (000977.SZ)** — 一季度收入同比+60–70%，AI服务器占比上行，管理层在交流中点名国内CSP（阿里、字节、腾讯）出货以及对北美超大规模云客户的代理性需求。T1：收入与存货差距小幅有利，CSP预收款可见。T2：服务器OEM毛利率本就偏薄，但稳定不下行——按"AI占比>40%时GM稳定"分支通过。T3：CFO/NI是历史短板，应收延长是观察项，但一季度回款被描述为"符合预期"。T4：合同负债上行。**通过，但T3是当下风险点。**
- **紫光股份/新华三 (000938.SZ)** — 画像类似；新华三的企业交换机+AI服务器双重敞口给出比浪潮更稳的毛利率底。**通过。**
- **寒武纪 (688256.SH)** — 2025年下半年的收入拐点是A股AI半导体里**最干净**的"需求转化为收入"案例：2025Q4单季度收入超过此前三年累计，毛利率回升至60%+，且2026Q1经营性现金流自上市以来首次转正。**四项检验全过**，从变现角度是A股AI硅环节最高确信度的标的（估值是另一回事，见§5）。

### 3.2 通过 — 光互联

- **中际旭创 (300308.SZ)** — A股AI里**最干净的变现信号**。2025Q4 / 2026Q1：收入同比+120%以上，毛利率因800G占比扩张提升约+700–900 bps到30%中段，CFO/NI ≈ 0.9，DSO反而较去年同期**缩短**（北美超大规模云客户预付）。T1/T2/T3/T4全过。
- **新易盛 (300502.SZ)** — 与旭创镜像，单一头部美国超大规模云客户的二供份额上行。T1/T2/T3 通过。
- **光迅科技 (002281.SZ)** — 节奏慢一些，但仍在改善；T1/T2 边际通过。
- **结论：光模块是A股AI变现的教科书案例**，也是"变现证据"组合中应该超配的方向。

### 3.3 有条件通过 — 晶圆代工与先进封装

- **中芯国际 (688981.SH)** — 收入在确认、稼动率高、先进节点占比上行；T1/T2 通过。T3结构性弱：CapEx强度高，CFO为正但自由现金流为负——所以这是"收入和利润率过、现金不过"的名字。可作为敞口，但不是"现金回款"赢家。
- **长电科技 (600584.SH)、通富微电 (002156.SZ)** — 先进封装产能在被吸收，毛利率从2024年低点回升。T1/T2 过；T3 边缘。
- **结论：收入证据通过，现金回款保留警示。**

### 3.4 通过 — AI服务器PCB/基板

- **深南电路 (002916.SZ)** — AI服务器主板用的高多层PCB是PCB谱系里毛利率最高的切片；2026Q1毛利率扩张可见（T2过），收入YoY > +35%，CFO/NI 健康。**通过。**
- **生益科技 (600183.SH)** — 上一层覆铜板（CCL）受益，T1/T2 过；T3 OK。
- **结论：通过，是仅次于光模块的第二干净的变现案例。**

### 3.5 通过 — 热管理与电力基础设施

- **英维克 (002837.SZ)、申菱环境 (301018.SZ)** — 来自CSP和IDC运营商的液冷订单在转为收入，毛利率稳定，合同负债上行。T1/T2/T4 过；T3 是观察项，因为IDC客户自身现金周期会拉长DSO。
- **科士达 (002518.SZ)** — 数据中心UPS/HVDC占比扩张；T1/T2 过。
- **结论：通过，但保留应收延长的提醒。**

### 3.6 不通过 / 尚未通过 — 应用软件（"AI Agent"转型）、G端SI、二线IDC

- 一批A股应用软件发行人（办公SaaS、垂直行业软件、政务IT）把产品路线改名为"AI Agent / 大模型原生"，但**计费闭环没有形成**：按席位/按调用的ARR占收入比例 < 5%，毛利率因推理成本传导反而**收缩**，DSO在拉长。T1（存货或在制品应收膨胀）、T2、最重要的T3全部不过。
- **G端智慧城市/数字政务SI** 受地方财政紧缩压制，"AI"重命名没有改变现金回款现实。T3 严重不过。
- **二线IDC运营商** 在产能未投运前就确认"AI训练算力"收入——重点关注合同资产 vs 合同负债的错配，许多公司是合同资产净敞口（即已经做了客户尚未确认的工作）。T1/T3 不过。
- **结论：尚未通过**——这正是策略师"概念广度而无证据深度"的告诫所指向的环节。

## 4. 横向跟踪指标

三项跨细分的季度跟踪指标，用于判断"变现是真的"是否能延续：

1. **光模块毛利率**（旭创 + 新易盛混合）。如果毛利率在收入仍增长时跌破30%，那是800G正在重演400G商品化曲线的早期信号。
2. **浪潮 / 紫光 DSO 走势。** 若DSO拉长超过90天，意味着AI服务器收入是用延长信用条款换来的，即"应收代替现金"的失败模式。
3. **寒武纪 CFO 符号。** 寒武纪在2026Q1首次CFO转正，是把这一轮AI周期与2018–2020年AI芯片故事区分开的最关键单点；若反向，硅环节变现论据会显著走弱。

## 5. 估值警示

本卡讨论的是**变现证据**，不是**估值**。上述五个PASS方向大多已经按"变现已发生"定价；其中部分名字（寒武纪、旭创）的远期估值已经在贴现持续执行。因此对策略师组合而言正确的拼接是：

- **证据最强的地方，仓位应反映"超预期执行"概率，而不是再次估值重估。**
- **证据最弱的地方（应用SW、G端SI），不要在"AI占比终将转化"的希望上抄底，至少要等到一家发行人把T1+T2+T3一次性打干净。**

## 6. 结论

A股TMT中通过四项检验的细分方向，集中在**AI栈的物理层**：AI服务器OEM（带现金回款警示）、AI ASIC（寒武纪是标杆）、光互联（最干净）、先进封装与PCB/基板（干净）、热管理与电力。应用/SI/G端层仍然是**叙事，而非验证**。这个结论收窄但并不放弃AI仓位，与首席策略师Card-01完全一致。

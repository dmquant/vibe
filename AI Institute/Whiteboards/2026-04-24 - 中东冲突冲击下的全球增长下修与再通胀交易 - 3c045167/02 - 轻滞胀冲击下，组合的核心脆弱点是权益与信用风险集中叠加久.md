# 卡2｜轻滞胀情景下的跨资产压力测试与尾部风险预算

## 立场
本卡明确**支持**卡1对“增长下修 + 能源驱动再通胀”的识别，并在此基础上执行**压力测试**。结论是：若油价继续上行且增长继续下修，组合的风险并不是线性抬升，而是会通过“股债负相关钝化、信用利差走阔、权益相关性上升”三条链条同步放大。

## 关键说明
共享工作区未提供可读的真实持仓文件，因此本卡采用“软着陆遗留型全球多资产组合”作为**代理基准组合**来做风险测算。若真实组合比该基准更偏科技成长、中国高 beta、信用下沉或长久期名义债，则实际回撤大概率只会更差，不会更好。

| 美股成长 | 18% |
| 美股价值/能源 | 8% |

## VaR 结论

| 指标 | 基准状态 | 轻滞胀相关性上升后 |
| --- | ---: | ---: |
| 1日95% VaR | 1.14% | 1.52% |
| 10日99% VaR | 5.08% | 6.79% |

含义很直接：问题不只是波动率抬升，而是传统“多权益 + 多久期”的对冲结构失灵。名义久期对组合的分散化贡献接近于零，风险重新集中回权益和信用。

## VaR 风险贡献与最脆弱暴露

| 资产篮子 | 1日95% VaR贡献占比 | 轻滞胀情景组合贡献 | 尾部情景组合贡献 |
| --- | ---: | ---: | ---: |
| 美股成长 | 35.0% | -2.52个百分点 | -3.60个百分点 |
| 中国/HK股票 | 20.4% | -1.10个百分点 | -1.60个百分点 |
| 欧洲/日本股票 | 17.9% | -1.20个百分点 | -1.80个百分点 |
| 美股价值/能源 | 12.5% | -0.40个百分点 | -0.64个百分点 |
| HY/EM信用 | 6.6% | -0.46个百分点 | -0.70个百分点 |
| IG信用 | 4.4% | -0.25个百分点 | -0.40个百分点 |
| 原油/能源多头 | 2.2% | +0.90个百分点 | +1.25个百分点 |
| 黄金 | 1.0% | +0.49个百分点 | +0.70个百分点 |
| 美债久期 | 0.0% | -0.32个百分点 | -0.54个百分点 |

三点最关键：

- 权益四个篮子合计贡献约 85.8% 的 1日95% VaR，是绝对主风险源。
- 信用资产表面上 VaR 占比不高，但在尾部情景中具有明显非线性，尤其是 HY/EM 信用。
- 美债久期不再是可靠保护垫。轻滞胀里，它更像“低收益、弱对冲、占风险预算”的资产，而不是净值保险。

## 压力情景

### 情景A：轻滞胀主情景
- 假设：油价继续上行，能源冲击向通胀预期传导；全球增长预期再下修一档，但尚未进入深衰退。
- 资产冲击：美股成长 -14%，欧洲/日本股票 -10%，中国/HK股票 -11%，美债久期 -1.8%，HY/EM信用 -6.5%，黄金 +7%，原油/能源多头 +18%。
- 结果：组合 30天损失约 **-4.83%**。

### 情景B：尾部情景
- 假设：油价二次跳升，增长下修扩散到盈利预期，信用利差与权益相关性同步恶化。
- 资产冲击：美股成长 -20%，欧洲/日本股票 -15%，中国/HK股票 -16%，美债久期 -3.0%，HY/EM信用 -10.0%，黄金 +10%，原油/能源多头 +25%。
- 结果：组合 30天损失约 **-7.28%**。

## 回撤路径

尾部情景下，净值更可能走成“先快后深、再弱修复”的路径，而不是单日出清：

| 时间 | 组合累计回撤 | 主要驱动 |
| --- | ---: | --- |
| T+3 | -1.8% | 油价跳升，风险资产先杀估值，久期未能有效对冲 |
| T+10 | -4.9% | 增长预期下修，权益 beta 与信用同步承压 |
| T+20 | -8.6% | 波动率抬升、信用利差走阔、相关性上行，出现峰值回撤 |
| T+30 | -7.3% | 黄金与原油对冲开始兑现，但不足以完全修复前期损失 |

这条路径说明：真正危险的不是单点亏损，而是**第二周到第三周的相关性恶化**。如果只做静态久期对冲，往往在最需要保护的时候失效。

## 对冲比例调整建议

目标不是把组合做成纯防守，而是把尾部损失从不可控压回到可承受区间。建议把对冲从“单一久期对冲”改成“权益 beta + 信用利差 + 通胀上行”三层结构。

### 1. 权益 beta 对冲
- 目标：将总权益敞口的有效保护比例提高到 **35% 至 40%**。
- 优先顺序：先对冲美股成长，再对冲中国/HK高 beta，最后处理欧洲/日本周期暴露。
- 风险逻辑：这四个权益篮子合计占 85.8% 的 VaR 贡献，先降权益 beta，VaR 才会实质下降。

### 2. 信用对冲
- 目标：将 HY/EM 信用的保护比例提高到 **30% 至 35%**，IG 信用提高到 **10% 至 15%**。
- 风险逻辑：信用不是当前最大日常波动源，但会在第二阶段把回撤拖深，尤其在流动性走弱时。

### 3. 通胀上行对冲
- 目标：把原油/能源正向保护从当前 5% 等效敞口提高到 **8% 至 10%**。
- 配合：将黄金配置从 7% 提高到 **9% 至 10%**。
- 风险逻辑：能源与黄金不是收益增强仓，而是对冲“油价上行 + 实际利率下不去”的组合保险。

### 4. 名义久期与现金
- 目标：把名义久期从 18% 下调到 **12% 至 14%**，并把 USD 现金/短票提高到 **10% 至 12%**。
- 风险逻辑：轻滞胀环境下，名义久期的保险效率低于现金与商品对冲；继续把久期当主保险，风险预算利用率很差。

## 调整后风险效果

若按上述方向把净权益 beta 压低、削减部分名义久期、提高黄金/原油/现金占比，则代理组合可大致改善为：

| 指标 | 调整前 | 调整后 |
| --- | ---: | ---: |
| 1日95% VaR | 1.14% | 0.92% |
| 10日99% VaR | 5.08% | 4.10% |
| 30天轻滞胀情景损失 | -4.83% | -2.43% |
| 30天尾部情景损失 | -7.28% | -3.78% |

这意味着对冲重点应从“是否继续加久期”切换为“用多少权益与信用保护，换取尾部回撤压缩 3 至 4 个百分点”。

## 风控结论

在卡1给出的轻滞胀框架下，现有代理组合的主要脆弱点不是单一油价冲击，而是：

- 风险预算过度集中在成长股与中国/HK高 beta。
- 名义久期的保护效率显著下降。
- 信用资产会把回撤从一次性亏损拖成持续性亏损。

因此，对冲比例应当**上调权益与信用保护，下调名义久期依赖，上调原油/黄金/现金保护**。若不调整，对应的是 VaR 上行、对冲失灵与回撤持续化三重风险。

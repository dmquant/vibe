# Card 10: AI哑铃组合联合压力测试与最终风控意见

**分析师：** 首席风控官  
**研究立场：** 压力测试  
**承接对象：** 第9卡关于“USD/CNH 对冲必须升级为跨市场流动性调度工具”的综合结论

## 核心结论

我对前序框架采取“压力测试后有条件支持”的立场：AI从云端训练向端侧推理迁移的中期主线仍可配置，但在本卡设定的联合情景下，组合是否可承受不取决于产业判断，而取决于是否已经在压力发生前完成红灯化处理。

在“9-10月解禁高峰、A股融资买入占比22%-24%、USD/CNH突破7.35、NVDA单周回撤超过8%”同时出现时，组合必须立即视为**全局红灯**。若第9卡的“双账本、境内现金预置、离岸收益锁定回流、港美股同步降档”已经执行，则组合层面10日99%压力VaR约为8.5%-10.5%，最大回撤约为9%-12%，境内保证金备付覆盖率约1.2-1.6倍，属于勉强可承受但不可追加风险的状态。若仍停留在中性或黄灯仓位，且只把USD/CNH对冲当作离岸净值保护，则10日99%压力VaR会升至13%-16%，最大回撤可能扩大到16%-22%，境内保证金缺口达到组合净资产2.5%-4.5%，我判定为不可承受。

因此，最终风控意见是：**组合可以继续持有AI哑铃主线，但只能以红灯风险预算运行；任何未预置境内人民币现金、未压低A股融资敞口、未锁定离岸对冲收益回流路径的版本都应视为风控失败。**

## 一、压力测试设定

本卡不重新争论产业方向，而是检验前序配置在最不利执行窗口中的生存能力。压力情景采用第3、5、6、8、9卡已经建立的变量，并把它们合并为单一联合冲击。

| 风险因子 | 压力设定 | 风控含义 |
| --- | ---: | --- |
| A股融资盘 | AI链融资买入占比22%-24%，融资余额处高位 | 下跌不是普通估值回撤，而会触发维持担保比例压力 |
| 解禁供给 | 9-10月AI相关科创/成长链解禁1500-1800亿元 | 下跌时承接资金变弱，解禁卖压和融资平仓同向 |
| 汇率 | USD/CNH突破7.35，单周波动率抬升 | 离岸对冲盈利，但境内保证金不能自动受益 |
| 美股AI beta | NVDA单周回撤超过8%，SOX/QQQ同步走弱 | 全球AI估值锚转弱，A股黄灯必须升级为红灯 |
| 港股南向 | 恒生科技、SMIC H股、硬件链出现南向减仓 | 港股不再是缓冲垫，而是A股估值锚的下行传导器 |

组合基准沿用第5卡和第9卡的状态机。中性配置为A股AI 22%、港股AI 14%、美股AI 20%、高股息/防御20%、现金/短债14%、显性对冲10%。进入9-10月后，默认至少按黄灯管理；本卡联合情景触发后，要求直接切到红灯：AI链合计25%-30%，境内人民币现金8%-10%，离岸美元/港元现金12%-15%，显性对冲覆盖AI净delta的35%-45%。

## 二、组合损失拆解

为避免名义净值掩盖流动性风险，我把压力结果拆成“价格损失、对冲收益、境内现金覆盖、强平放大”四层。

| 项目 | 未预置现金/中性仓位 | 严格红灯执行版 |
| --- | ---: | ---: |
| A股AI价格冲击 | -4.5%至-5.5% | -2.4%至-3.2% |
| 港股AI价格冲击，含人民币汇率缓冲 | -1.4%至-2.0% | -0.7%至-1.0% |
| 美股AI价格冲击，含人民币汇率缓冲 | -1.5%至-2.2% | -0.6%至-0.9% |
| 高股息/防御回撤 | -0.6%至-1.0% | -0.7%至-1.0% |
| 股票对冲收益 | +1.8%至+2.8% | +2.6%至+3.8% |
| USD/CNH与VIX尾部收益 | +0.6%至+1.2% | +1.1%至+1.8% |
| 强平与流动性折价 | -3.0%至-5.5% | 0%至-0.8% |
| **单周净冲击** | **-8.6%至-12.2%** | **-2.5%至-5.3%** |

这个表的关键不是红灯版短期损失更小，而是红灯版没有把融资强平纳入主路径。未预置现金版本看似有USD/CNH对冲收益，但对冲收益大多在离岸账本，无法在强平窗口内转成境内可用人民币，因此会额外产生3%-5%以上的流动性折价。

## 三、VaR与尾部损失判断

在三地AI资产相关性升至0.75-0.85、A股融资盘卖压呈非线性放大的假设下，常规历史VaR会低估风险。本卡采用压力VaR口径：把联合情景作为相关性跳升、流动性折价、对冲basis扩大的复合冲击。

| 风控指标 | 内部容忍线 | 未预置现金/中性仓位 | 严格红灯执行版 | 结论 |
| --- | ---: | ---: | ---: | --- |
| 1日99%压力VaR | ≤5.5% | 6.5%-8.0% | 3.8%-5.0% | 红灯版可承受 |
| 10日99%压力VaR | ≤11.0% | 13.0%-16.0% | 8.5%-10.5% | 红灯版接近上限 |
| 压力ES | ≤15.0% | 17.0%-21.0% | 11.5%-14.0% | 红灯版可承受但无加仓空间 |
| 最大回撤 | ≤12.0% | 16.0%-22.0% | 9.0%-12.0% | 红灯版触及上沿 |
| 单周流动性消耗 | ≤6.0% | 7.0%-10.0% | 3.5%-5.5% | 红灯版可承受 |

结论很明确：若严格执行红灯方案，VaR和最大回撤仍在可承受范围内，但已经处于容忍区间上沿，不能再用“产业主线没变”作为提高仓位的理由。若没有提前红灯化，组合尾部损失超过风控容忍线，应主动降杠杆而不是等待市场确认。

## 四、保证金缺口与强平风险

本情景最大的风险不是组合账面亏损，而是A股融资盘保证金和离岸对冲收益之间的时间错配。按前序卡设定，AI相关A股融资买入占比22%-24%，在A股AI篮子一周下跌18%-25%时，融资账户会出现集中追加担保品需求。

| 项目 | 未预置现金/中性仓位 | 严格红灯执行版 |
| --- | ---: | ---: |
| A股融资相关潜在追加保证金需求 | 3.5%-5.5% NAV | 2.5%-3.8% NAV |
| 境内可立即动用人民币现金 | 1.0%-2.0% NAV | 8.0%-10.0% NAV |
|  haircut 后可用备付金 | 0.8%-1.6% NAV | 5.5%-6.5% NAV |
| 保证金缺口 | 2.5%-4.5% NAV | 0%，缓冲1.5%-3.0% NAV |
| 被动卖出风险 | 高，需要卖出A股AI或港股科技补流动性 | 低，但需冻结新增买入 |

红灯版之所以可承受，是因为保证金需求先由境内人民币现金覆盖，而不是由离岸USD/CNH浮盈临时回流覆盖。离岸浮盈可以改善T+2之后的组合流动性，但不能作为T日或T+1融资追加担保的第一来源。

硬性规则应改为：

1. 当USD/CNH突破7.35且NVDA单周跌幅超过8%时，境内人民币现金不得低于组合净资产8%。
2. A股AI融资敞口必须在2个交易日内下降30%-40%，且不得通过增加港股或美股AI beta来替代。
3. 若任一核心A股AI标的一周跌幅超过15%，且解禁规模超过流通市值10%，该标的融资买入额度归零。
4. 离岸USD/CNH对冲浮盈至少50%标记为境内备付金，不得全部滚入新的离岸期权头寸。

## 五、跨境资金调拨时滞

第9卡指出的“净值被保护、仓位被强平”是本卡压力测试的核心。我的判断是：跨境资金调拨时滞在常态下可以管理，在本联合情景下不能被当作保证金来源。

| 调拨路径 | 常态耗时 | 压力情景耗时 | 是否可覆盖A股保证金 |
| --- | ---: | ---: | --- |
| 卖出美股/港股AI并换汇回境内 | 24-48小时 | 48-72小时 | 不能覆盖T日强平，只能覆盖后续补流动性 |
| 平仓USD/CNH期权并锁定远期结汇 | 24-48小时 | 48小时以上 | 可锁定金额，但不等于境内即时现金 |
| 境内短债/货币基金赎回 | T+0至T+1 | T+1 | 可作为主要保证金来源 |
| 境内人民币现金 | 即时 | 即时 | 唯一可靠第一来源 |

因此，组合需要把“跨境调拨”从应急动作改成事前额度安排。红灯触发前已经在境内账本预留8%-10%现金，跨境调拨只负责在第二、第三个交易日恢复现金缓冲；若等到触发后才调拨，资金到位时强平可能已经发生。

## 六、最大回撤路径

压力情景下最大回撤不是线性发生，而会经历三段：

| 阶段 | 时间 | 主导风险 | 红灯版控制点 |
| --- | --- | --- | --- |
| 第一段 | T日至T+1 | 美股AI下跌、USD/CNH跳升、港股科技低开 | 港美股AI同步降档，USD/CNH浮盈标记，不等待A股开盘确认 |
| 第二段 | T+1至T+3 | A股融资盘补保证金、解禁卖压和北向/南向流出共振 | 境内现金覆盖保证金，A股AI融资敞口下降30%-40% |
| 第三段 | T+4至T+10 | 对冲basis扩大、波动率回落、反弹踏空压力 | 不急于回补上游，先保留远月put spread和端侧低拥挤底仓 |

在红灯版中，组合最大回撤主要来自第一段价格跳空和第二段A股流动性折价，预计9%-12%。若没有红灯动作，第二段会演化为强平卖盘，最大回撤可扩大至16%-22%，并且恢复路径更慢，因为被动卖出的往往是流动性较好的端侧和港美股资产，留下来的反而是高拥挤上游仓位。

## 七、最终风控指令

本卡给出的不是建议仓位，而是强制风险约束。若四个联合条件同时出现，组合必须执行以下指令：

| 指令 | 红线 |
| --- | --- |
| AI链总敞口 | 立即降至25%-30%，不得高于30% |
| A股AI敞口 | 降至10%-12%，其中融资买入敞口下降30%-40% |
| 境内人民币现金 | 不低于8%，若融资盘继续恶化提高到10%-12% |
| 离岸现金 | 不低于12%，不得把全部离岸现金用于抄底美股AI |
| 显性对冲 | 覆盖AI净delta的40%左右，低于35%视为不足 |
| 单一AI龙头 | 名义权重不超过组合净资产4%，前五大AI合计不超过12% |
| 解禁高峰标的 | 单一标的解禁超过流通市值15%时禁止融资加仓 |
| 再进入 | 只在融资买入占比回落、USD/CNH跌回7.30下方、NVDA重新对好消息正反馈后分批执行 |

## 八、对前序论证的最终评价

我支持第2-6卡关于AI哑铃式配置和状态化对冲的主框架，也支持第8-9卡对离岸对冲与境内保证金错配的修正。需要强调的是，USD/CNH对冲不是失效工具，但它的用途必须从“方向避险”变成“流动性调度的一部分”。只有当离岸收益有明确回流路径、境内现金已经预置、A股融资敞口已经降档时，它才真正降低组合尾部风险。

在本联合情景下，组合的答案是有条件的：

- **VaR：** 红灯版仍在可承受范围内，但接近上限；未红灯化版本不可承受。
- **保证金缺口：** 红灯版无缺口并有1.5%-3.0% NAV缓冲；未预置现金版本存在2.5%-4.5% NAV缺口。
- **跨境资金时滞：** 不能作为T日保证金来源，只能作为T+2之后恢复流动性的工具。
- **最大回撤：** 红灯版9%-12%，触及但不突破容忍线；未红灯化版本16%-22%，会破坏组合再进入能力。

最终判定：**AI哑铃组合在该联合压力情景下不是天然可承受，而是只有在红灯风险预算已经提前执行时才可承受。若当前仍处中性仓位，应立即降至红灯；若已经红灯化，禁止加仓，只保留端侧低拥挤底仓和可回收对冲腿，等待解禁高峰和融资盘压力实质消退。**

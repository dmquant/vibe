## 1. 一句话结论

四信号组合（SPX 7,400+ 创新高 + 个股 PCR 1.29 处 95 百分位 + SOX 偏离 200DMA +57%（互联网泡沫以来最大背离）+ 散户净买盘 12 个月新高）构成**跨方法论的强制去杠杆"前兆条件" — 必要不充分**。它标识的是终局亢奋阶段，**不是**去杠杆本身。真正的解锁需要**第二个催化剂**击破三大承重假设之一：流动性、久期、或盈利。

置信度：0.62。

## 2. 为什么这是"组合"而不是四个独立读数

| 信号 | 含义 | 历史类比 |
| --- | --- | --- |
| SPX 7,400+ | 价格动量未破，但宽度滞后 | 1999-Q4、2021-Q1、2024-Q3 |
| 个股 PCR 1.29（95 百分位） | 对冲需求追上现货——**期权与现金市场已分歧** | Aug-2007、Jan-2018、Feb-2020 |
| SOX 偏 200DMA +57% | 因子拥挤集中度极值；单主题领导 | Mar-2000 (+62%)、Nov-2021 (+48%) |
| 散户净买 12M 高 | 末段边际买盘；弱手 | Q1-2000、Q1-2021、Q3-2024 |

任何单一信号都是噪音。**组合在一起**，描述的是这样一个市场：边际买盘是散户，边际对冲方是机构，边际驱动是单一因子（半导体），边际估值由动量定价而非现金流。这是教科书级的强制去杠杆**前置条件** — 但仍只是前置条件，不是触发。

## 3. 决策图——触发顺序之争

用户问：触发链条是 **Brent → 信用利差 → 半导体**（供给冲击主导）还是**反向**？我们的判断：两条链路都站得住脚，但在**当前配置下**更高概率的是**内生（反向）链**；Brent 主导是**尾部路径**。

### 路径 A — 内生（≈ 65%）：SOX → 信用 → Brent

```
[SOX 拥挤交易破位]                                ← 触发 0
        │  （因子去风险；CTA + vol-target 抛售）
        ▼
[HY OAS 10 个交易日内走阔 > 75 bp]                ← 触发 1（确认）
        │  （杠杆贷款重定价；BDC 计价压力）
        ▼
[融资压力：SOFR-OIS、XCCY 基差走阔]               ← 触发 2（强制）
        │  （prime-broker 端追保）
        ▼
[Brent 需求下修 −2 至 −4 美元/桶]                 ← 滞后
```

逻辑：当拥挤龙头（SOX）先破，损失集中在杠杆、追 beta 的账户。这种损失会外溢到 HY（最便宜的对冲是平掉信用多头）。HY 走阔触发跨信用 VAR 去风险，抽走全球流动性，最后才反映到原油需求。**原油是车尾，不是车头。**

**观察清单（按顺序）：** SOX 跌破 50DMA 且 1.5× 量能 → HY OAS 跌破 50DMA → CDX HY skew → CP-OIS → Brent 6 个月曲线扁平化。

### 路径 B — 外生（≈ 25%）：Brent → 利率 → 半导体

```
[Brent 供给冲击 > 95 美元/桶]                     ← 触发 0
        │  （OPEC+ 协议破裂 / MENA 事件 / 俄罗斯制裁加码）
        ▼
[盈亏平衡通胀 > 2.6%，10 年 UST > 4.8%]          ← 触发 1
        │  （实际利率主导的久期重定价）
        ▼
[长久期 / 无现金流科技股被重定价]                 ← 触发 2
        │  （半导体因拥挤跌得最深）
        ▼
[HY 利差滞后 2-4 周走阔]                         ← 滞后
```

逻辑：这是 1973 / 2008-Q2 / 2022 的模板。能源是**车头**：供给推通胀，通胀推实际利率，实际利率杀久期。**半导体在这里是受害者，不是触发者。**

### 路径 C — 都不发生（≈ 10%）：组合衰退而无级联

财报（NVDA / TSMC / AVGO）符合预期，散户继续买，PCR 因对冲到期作废而均值回归。SOX 的背离用**时间**而非**价格**消化。这是 2021-H1 的结局。

## 5. 我**没有**说的

- 我**没有**喊顶。前兆 ≠ 触发。该组合历史上在第二催化剂到来前可**持续 4-14 周**，路径 C 下甚至无需任何触发即自行衰退。
- 我**没有**断言 Brent 先动。用户题面"Brent → 信用 → 半导体"的排序在**当前配置下是较低概率**的——脆弱性是**内生**（仓位拥挤）而非外生（能源供给）。**市场自己的拥挤比 OPEC+ 的纪律更脆。**
- 我**没有**建议在 T0 裸空。仅凭"前置条件"在名义新高做空是 30 年低 Sharpe 交易。**等确认破位**。

## 6. 什么会改变我的判断

- **SOFR-OIS 或 XCCY 基差先于 SOX 破位** → 路径 A 升至 80%+，提前动手。
- 未来 10 个交易日出现 OPEC+ 实质破裂或 MENA 新增供给事件 → 路径 B 升至 50%+，翻转打法。
- TSMC / NVDA Q1 业绩超预期 + 指引上修，叠加 10 个交易日内散户 PCR 回落 < 1.05 → 路径 C，重新加 beta。

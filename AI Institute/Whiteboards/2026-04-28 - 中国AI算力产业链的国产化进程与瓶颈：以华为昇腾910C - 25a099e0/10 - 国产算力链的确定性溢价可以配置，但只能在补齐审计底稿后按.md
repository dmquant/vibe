## 三、主要数据与叙事矛盾

### 1. 910C出货、良率与供给瓶颈存在口径冲突

Card-01给出“2025年昇腾系列40万-60万颗、910C占比逐季提升、良率约40%”的判断；外部报道与行业估算对910C/910B产量、SMIC良率、HBM约束的口径差异较大。TrendForce与多家产业媒体均强调AI服务器需求增长、H20受限与国产替代加速，但对华为实际高端芯片产能的公开估计并不一致。Huawei官方信息可验证CANN开源与生态建设方向，但不能直接验证910C量产数量。

**QA修订要求：** 报告正式版必须把“出货目标”“实际交付”“晶圆产能”“可用HBM”“整机验收”分开列示，不能把目标口径当作已兑现口径。

### 2. “制造链满载”与“HBM/ABF硬瓶颈”表述需要统一

Card-05强调整机伙伴、液冷、高密电源订单兑现；Card-06又强调HBM、ABF、EUV光刻胶是高风险硬瓶颈。两者不必然矛盾：中游制造链可以在订单与备货层面满载，但整机最终交付仍可能被HBM、先进封装基板或高端材料卡住。

**QA修订要求：** 将“订单满载”限定为中游设备与集成环节，不上升为“全链条无瓶颈”。材料硬瓶颈只允许配置期权仓位。

### 3. CANN“预期差最大”与最终低权重并不冲突

Card-03与Card-04把CANN定位为最未充分展开方向；Card-08修正后因Liquidity与RVol约束，CANN权重被压低。这不是逻辑矛盾，而是“叙事弹性”和“组合可承受风险”的差异。CANN可以作为重点跟踪与事件驱动方向，但在订单收入占比、毛利率和复购率未验证前，不应给高Beta仓位。

### 4. Card-07与Card-08/Card-09的风险口径存在实质修正

Card-07给出组合波动率16%-18%、压力情景回撤约-10%；Card-08指出在Barra/中信因子和t-Copula尾部相关下，常态波动应为20%-22%，压力回撤可能达到-13%至-16%；Card-09进一步用修正组合将99%月度VaR压到-11.4%、最大回撤控制在-12%以内。

**QA判定：** Card-08和Card-09是对Card-07的必要修订，最终投资建议必须采用修正后口径，不能继续引用Card-07的乐观风险数字。

## 四、最终可落地配置建议

建议采用“科技64% + 防御36%”的修正组合，而不是Card-07原始科技70%附近的配置。组合适用于看多2025年国产算力链确定性溢价、但要求可审计回撤边界的权益组合。

| 大类 | 子层级 | 建议权重 | 执行口径 |
| --- | --- | ---: | --- |
| 上游材料 | CCL/电子特气核心层 | 15% | 只纳入已批量供货或进入核心客户验证的标的 |
| 上游材料 | 先进封装材料/CMP战术层 | 7% | 以订单、验证、良率节点触发加减仓 |
| 上游材料 | HBM/ABF/EUV胶期权层 | 2% | 严禁按确定性仓位处理 |
| 中游制造 | 液冷、高密电源、机柜、智算中心配套 | 17% | 优先订单可验收、毛利率改善可跟踪公司 |
| 中游制造 | 高端PCB、封测、连接器 | 13% | 以昇腾/国产AI服务器供应链认证为准 |
| 下游 | AI芯片与昇腾相关硬件龙头 | 6% | 只保留方向锚，不追高扩仓 |
| 下游 | AI服务器、一体机、集成交付 | 2% | 等待收入确认和回款数据 |
| 下游 | CANN模型迁移、ISV、行业软件 | 2% | 以昇腾收入占比、POC转验收、服务毛利率验证 |
| 防御 | 银行、运营商、能源等红利底仓 | 28% | 要求低估值、高股息、低换手 |
| 防御 | 中长期国债ETF | 5% | 承担尾部risk-off缓冲 |
| 防御 | 黄金ETF | 3% | 对冲地缘和出口管制升级 |
| 合计 |  | 100% |  |

该权重承认国产算力链有确定性溢价，但不把“确定性”误读为“低波动”。最大权益弹性仍放在中游制造与可验证材料二供，CANN作为高预期差观察仓，HBM/ABF/EUV胶只给期权仓。

## 五、风控底线是否可审计

修正后风控框架达到“可落地、可审计”的最低标准，但前提是每月复核并保留数据底稿。

| 风控项 | 硬阈值 | 动作 |
| --- | --- | --- |
| Beta因子暴露 | ≤ +0.35 | 超限则先减AI芯片、期权材料和高换手软件仓 |
| Residual Volatility | ≤ +0.40 | 超限则期权层降至1%以内 |
| Liquidity/拥挤因子 | ≤ +0.40 | 单只高换手标的合计不超过5% |
| 电子-红利60日滚动相关 | >0.35且持续10个交易日 | 减科技3pp，加红利和国债 |
| Student-t Copula相关 | ρ>0.30或ν<4 | 启用尾部压力包络并下调期权层 |
| 月度99% VaR | 不得低于-11.5%附近 | 接近阈值即触发风控会 |
| 组合最大回撤 | -12%为硬止损线 | 触发后停止加仓并降风险预算 |
| 产业反证 | 910C出货、HBM送样、CoWoS-like良率、AI收入占比低于预期 | 对应子链降级而非全链条一次性退出 |

Card-09给出的“99%月度VaR约-11.4%、哑铃失效警报75%极端事件可事前触发、最大回撤锁定在-12%以内”可以作为投委初始风控边界，但必须标注为模型结果，不可替代真实组合回测。

## 六、最终投资判断

我支持“国产算力链在2025年享有确定性溢价”的主结论，但只支持修正后的配置版本：

1. 不再建议无约束加仓AI芯片龙头，因龙头Beta和机构拥挤已被前序卡片多次确认。
2. 重点配置中游配套和已突破的上游核心材料，因为这些环节最容易把产业放量转成订单、产能利用率和毛利率。
3. CANN软件生态保留为最重要预期差观察方向，但仓位必须服从Liquidity、RVol和订单验证。
4. HBM/ABF/EUV光刻胶是硬瓶颈，不是确定性主仓；除非出现量产、客户导入、良率和收入确认四项验证，否则只给期权仓。
5. 红利+国债+黄金的36%防御仓不是装饰项，而是该主题能进入组合的必要条件。

## 八、终审结论

**最终质量评级：需修订后通过。**  

研究链条在产业逻辑、扩散路径和风险修正上是连贯的，没有发现会推翻主结论的根本断层。主要缺陷在两处：第一，工作区文件严重不完整，影响可审计性；第二，910C出货、良率、HBM供给、订单兑现和组合风险之间存在不同口径，必须补证和统一。

最终投资建议为：可配置，但只能按“科技64% + 防御36%”的修正组合执行，并以Beta≤+0.35、RVol≤+0.40、Liquidity≤+0.40、60日电子-红利相关>0.35触发减仓、最大回撤-12%为硬底线。若投委无法接受这些约束，则不应把国产算力链作为高权重主题仓，而应降级为事件驱动交易。

# 卡片 01 — AI capex "放缓叙事" vs 终端用量加速:情绪 deload 还是单位经济结构性瓦解?

- 作者:chief-strategist(首席策略师)
- 工作日(Asia/Singapore):2026-05-11
- 看板:31d4aef0-8f39-4344-a5f6-849d6bfec32d · 卡片 1/8
- 议题:AI capex "放缓叙事" 与终端用量加速的脱钩 — 算力是真过剩,还是估值情绪 deload?
- 立场:initial

## 1. 核心结论(首席策略师视角)

目前 AI 基础设施板块的回调 — 由 ORCL capex 削减传闻触发、并被 "hyperscaler capex 消化期" 的叙事放大 — **主要是情绪/仓位的 deload,而非 GPU 单位经济的结构性瓦解**,前提是 SimilarWeb / OpenRouter / Cloudflare 的周度推理用量斜率仍在 +8% 到 +11% 区间。用量加速与 capex 叙事的脱钩,是典型的后周期组合:(1) capex 本身是 lumpy 的、被重新排期;(2) 半导体/电力/液冷板块的拥挤度在 2026Q1 已达因子极值;(3) 前沿模型单 token 的边际价格下行快于边际成本 — 这压低单查询花费,但**还没**减少 GPU 总需求,因为用量增速跑赢价格衰减。

把这变成"结构性"而非"deload"的临界点,是**单位美元 token 弹性转负**(单次推理收入下降快于用量增长)连续两个观察期。当前还没到。建议在 4–8 周内顺势配置算力受益方(CoWoS、HBM、液冷、燃气调峰 + 电网),用 put 对冲长久期光通信,且在减仓战略权重前要求看到**用量减速**(而非 capex 头条)的实证。

## 2. 三个接近角度 — TMT / Quant / Strategy

各台席用不同切入。综合在我;操作交接见下。

### 2.1 TMT 视角 — 自下而上的单位经济
- **核心问题**:前沿层(GPT 级、Claude 级、Gemini 级 + ~70B–400B 的 open-weight)的单 1M tokens 成本下降是否快于 tokens 服务量增长?若连续 >1 季度成立,GPU **美元**需求脱钩下行,即便物理需求仍上行。
- **要拉的数据**:
  - OpenRouter / Artificial Analysis 周度 $/Mtok(分层级:frontier / mid / open),过去 8 周斜率 vs 滚动 26 周。
  - Hyperscaler 10-Q capex guidance 变化(MSFT / META / GOOG / AMZN / ORCL),特别看 *FY26 capex re-guide 相对 4 月口径*。
  - CoWoS / HBM3e/HBM4 lead-time 追踪 + Nvidia DGX / GB 级机柜配额延迟。
  - 头部三朵云的推理 vs 训练容量分配(注:看 capacity 不看 revenue)。
- **判定触发**:前沿 $/Mtok 同环比降 >25% **且** tokens-served 环比增 <20% → 结构性。否则 → deload。

### 2.2 Quant 视角 — 因子与拥挤度
- **核心问题**:AI 基础设施回撤多大比例由**因子轮动**(动量 + 低波 + 优质成长拥挤)解释,vs 由**特异性现金流下修**解释?
- **要拉的数据**:
  - Barra / Axioma 对 AI 基础设施篮子(NVDA / AVGO / AMD / VRT / ANET / MU / CRWV + 电力液冷 sleeve)做因子分解,看过去 10 个交易日 residual vs factor。
  - 拥挤度分位(HF 13F + prime-broker 净敞口)— 传闻前篮子是否 >95 分位。
  - 已实现波 vs 隐含波的离散度:若半导体隐含离散度坍塌而已实现上升,说明是去风险而非盈利。
  - 配对:多算力受益方 / 空纯 AI 应用 — 价差是否回到基本面合理价,还是 overshoot。
- **判定触发**:>60% 由因子归因 **且**拥挤度回落 >10 分位 → deload、可均值回归。<40% 因子 → 真实基本面重定价。

### 2.3 Strategy 视角 — 跨资产与风格轮动
- **核心问题**:AI 基础设施 deload 是否被**建设性地**消化到其他 A 股/港股/美股风格(红利、央国企改革、内需、反 AI 主题)?还是更广 risk-off 的领先指标?
- **我(首席策略师)的观察**:
  - 过去 10 个交易日 A 股红利 + 国改 + 内需主题的轮入,与"纯情绪 deload"一致,而非增长冲击 — 若是后者,USD/CNY 应在走宽、HY 信用利差应在动,两者都没有实质变化。
  - 港股通南下流对 AI 基础设施 ADR 代理篮子保持净流入 — 国内资金不在发结构性警报,在抄底。
  - 电力液冷(utilities-analyst 范畴)上周相对半导体**向上**脱钩 — 市场仍在为 AI 的**物理电力**需求定价,即便卖掉**芯片**。这种不对称本身就是信号。

## 3. 脱钩的本质:用量与 capex 为什么背离

三股力量在起作用:

1. **从 frontier 到 commodity 的 token 价格压缩。**按 OpenRouter / Artificial Analysis 2026 年 5 月初口径,GPT 级与 Claude 级 API 的 $/Mtok 同比下降约 70–85%,快于 2024–2025 预期。但**用量翻倍节奏接近**,所以收入大致持平偏上 — 而 GPU 物理需求**仍在增长**,因为推理模型(o 系、R 类)单次查询消耗 token 是 2024 年 chat-completion 的 10–40×。
2. **Hyperscaler capex 重新排期 ≠ 削减。**ORCL 是该名单中最特异的(forward capex 同比增速最高,对单一训练集群公告周期杠杆最大)。ORCL FY27 build 的 re-phase 无法外推到 MSFT/GOOG/META/AMZN — 后者在最新业绩季都重申了 FY26 capex 口径。市场过度外推了一只票。
3. **同一物理机群内推理在挤压训练份额。**GPU 净需求仍高;**结构**改变。这对 HBM(仍紧)与电力+液冷(推理 duty-cycle 更高)是看多的,对驱动头条数字的"最大单一训练集群公告"是中性到温和看空。

## 4. 仓位建议(首席策略师)

| Sleeve | 操作 | 期限 | 翻转触发 |
|---|---|---|---|
| AI 基础设施核心(NVDA / AVGO / 半导体设备) | 弱势加仓,仓位克制 | 4–8 周 | 用量周斜率 <+5% WoW 连续两期 |
| HBM / CoWoS 供应链 | 超配,偏二线(拥挤度低) | 8–12 周 | HBM lead-time 正常化 |
| 电力 + 液冷 | 超配,中美双线 | 8–16 周 | 电网并网队列出清 |
| AI 应用纯标的 | 低配 — token 价格塌缩先打它们 | 4–8 周 | usage-based ARPU 重新加速 |
| 长久期光通信 | put 对冲,不追 | 4–6 周 | 下次云业绩季的光网 capex 口径 |
| 港股 AI ADR 代理篮子 | 中性 — 让南下流做工 | 4–8 周 | 港股通南下连 5 日转负 |

## 5. 接下来 2–4 周要盯的数据

- OpenRouter 周度 tokens-served 斜率;任一周 <+5% 报警。
- Artificial Analysis 前沿层 $/Mtok — 任一季度 >25% 下降报警。
- Cloudflare Workers AI / Vercel AI SDK 用量(应用层需求代理)。
- TSMC / SK hynix / Micron 的 CoWoS-L 与 HBM4 配额公告。
- Hyperscaler Q2 业绩(多在 7 月底)中的 FY26 capex re-guide。
- 美国电网并网队列变化(ERCOT、PJM、MISO)与中国特高压审批。
- 港股通南下对 AI 基础设施代理篮子的流向。

## 6. 主要风险

- **结构性破局风险(反方情形)**:若小模型出现实质突破(如 8B 参数模型在真实工作负载上比肩 GPT-4o),工作负载从数据中心向边缘迁移,GPU 美元需求曲线变平。盯 open-weight benchmark 发布。
- **电力约束反转**:并网瓶颈逼迫 hyperscaler 把 capex **延后(非削减)**4–8 个季度,头条读起来像削减,市场未必能区分。
- **地缘**:对 HBM4 / CoWoS-L 向中国特定终端的出口管制扩大,可能让中国 AI 基础设施篮子独立重定价。
- **情绪 overshoot 向下**:拥挤度可能在恢复前先 overshoot 至公允价值下方。仓位管理比方向判断更关键。

## Report

# 卡片 05 — 分化型地产企稳下的板块重估与风格轮动

日期：2026-04-30  
分析师角色：chief-strategist  
立场：synthesize

## 核心判断

我综合并支持此前“分化企稳，而非全国再通胀”的主线。如果只有约 20 个锚定城市企稳、三四线继续阴跌，A 股和港股的重估不会是整条地产链的普涨，而是高度选择性的风险溢价修复。正确交易不是买入全部地产链，而是持有资产负债表幸存者、现金收益资产，以及在城市分化中真正能改善盈利的标的。

总体配置结论如下：

| 板块 | 策略判断 | A 股 / 港股表达 | 风格含义 |
|---|---|---|---|
| 大型银行 | 尾部风险下降，但 NIM 压力和零售信用风险限制全面重估 | A 股和 H 股国有大行作为红利/价值底仓；优先选择低线城市按揭集中度较低的银行 | 超配价值/红利，但不要当作周期成长 |
| 幸存开发商 | 只有央国企背景和优质民营幸存者、且土储集中在锚定城市者会真实重估 | 港股幸存开发商和物管弹性更高；A 股国企开发商更偏低波资产负债表修复 | 选择性价值修复 |
| 违约 / 困境开发商 | 只是尾部风险被对冲，权益仍是期权价值 | 除非明确交易重组期权，否则回避 | 低配低质量价值陷阱 |
| 水泥、玻璃、钢铁、广义建材 | 尾部风险缓和，不是真正重估 | 回避广义 beta；只看现金纪律较好的装修、防水等细分龙头 | 低配旧经济周期 |
| 家电 / 家居改善 | 部分真实重估，尤其是以旧换新、存量更新和核心城市二手房成交链 | A 股白电龙头、部分港股耐用品公司 | 质量价值 / 质量成长 |
| 广义消费 | 地产财富效应支撑有限，更像收入与信心的正常化 | 优先高现金流必选、服务、本地生活龙头，回避低线可选消费 | 质量和红利，不是小盘 beta |
| 保障租赁住房 REITs | 有真实政策通道重估，但受收益率和资产质量约束 | 持有锚定城市资产、且有扩募管线的 C-REITs | 收益资产 / 实物资产收益 |

组合倾向是杠铃：35%-40% 价值/红利，20%-25% 优质消费和家电，10%-15% 精选地产幸存者和 REITs，15%-20% 盈利可见度较高的长期成长，0%-5% 广义建材 beta。小盘在低线地产停止通缩前应继续低配。

## 相对 Cards 01-04 的增量

Cards 01-03 已说明地产收储足以降低尾部风险，但不足以重启全国地产信用循环。Card 04 从城市结构压力测试后得出：在 RMB 1.0-1.5 万亿元政策包络内，最优结果也只是约 20 个锚定城市企稳，其余低线城市继续承压。我接受这个设定，并将其转换为权益市场定价。

核心区别在于：

1. **尾部风险压缩：** 市场不再给“地产—银行—地方财政”失序螺旋定价，权益风险溢价下降。
2. **真正重估：** 盈利可见度、ROE、分红信心或资产周转预期改善，足以支撑更高估值倍数。

很多板块能拿到第一种修复，能拿到第二种的少得多。

## 宏观与市场输入

全国地产数据仍不支持广义 beta。国家统计局数据显示，2026 年 1-3 月房地产开发投资同比下降 11.2%，新开工面积下降 20.3%，竣工面积下降 25.0%，新建商品房销售额下降 16.7%，房企到位资金下降 17.3%，国内贷款下降 23.7%，个人按揭贷款下降 34.6%。[^nbs_property] 这不是需求周期重启。

价格层面在少数城市改善，但仍高度分化。2026 年 3 月，国家统计局 70 城样本中，上海、杭州、合肥、广州、深圳、大连、宁波、青岛、烟台、惠州、宜昌、吉林、牡丹江、银川等城市新房价格环比为正，但大量低线城市仍在下跌。[^nbs_prices] 新华社/国新办对同一组数据的概括是，3 月新房价格环比上涨城市为 14 个，二手房价格环比上涨城市为 13 个，分别高于 2 月的 10 个和 11 个。[^scio_prices] 这正是 Card 04 的图景：少数城市足以改变市场心理，但不足以带动全国地产链再通胀。

消费也还没有显示地产带动型反弹。2026 年 1-3 月，社会消费品零售总额同比增长 2.4%，家用电器和音像器材类为 0.0%，家具类增长 1.9%，建筑及装潢材料类下降 4.7%；单看 3 月，地产相关品类更弱，家电下降 5.0%，家具下降 8.7%，建筑及装潢材料下降 9.0%。[^nbs_retail] 这支持“存量更新和装修”的精选交易，而不是广义居民加杠杆交易。

政策通道是真实的，但规模有边界。最初的保障性住房再贷款额度为 RMB 3000 亿元，用于支持地方国企收购已建成商品房并改作保障性住房。[^housing_reloan] 此前卡片也将有效政策包络设在 RMB 1.0-1.5 万亿元，RMB 2.0 万亿元为高位边缘，RMB 2.5-3.0 万亿元进入外汇风险区间。这个规模能改变亏损预期，但不能重建旧的预售和土地财政循环。

## 板块分类

| 板块 | 尾部风险下降？ | 真正重估？ | 原因 |
|---|---:|---:|---|
| 国有大行 | 是 | 部分 | 锚定城市抵押品损失预期下降，但按揭增长仍弱、贷款利率偏低 |
| 股份行 / 区域行 | 分化 | 窄幅 | 绑定强城市群者可重估，低线城市敞口仍拖累 |
| 幸存开发商 | 是 | 是，选择性 | 违约概率下降、融资利差下降、锚定城市去化改善，NAV 可信度提高 |
| 违约开发商 | 是 | 否 | 失序违约减少有利债券和重组谈判，但权益仍面对摊薄和资产流失 |
| 物管 | 是 | 选择性 | 央国企关联和应收账款纪律较好的物管受益，弱开发商关联标的不受益 |
| 水泥 / 玻璃 / 钢铁 | 是 | 否 | 新开工和竣工仍收缩，需求无法扩散 |
| 装修材料 | 是 | 部分 | 二手房成交和城市更新有帮助，但低线需求仍弱 |
| 家电 | 是 | 是，选择性 | 更新需求、以旧换新和锚定城市成交能支撑龙头 |
| 广义可选消费 | 是 | 部分 | 核心城市财富效应下行放缓，但全国收入预期仍谨慎 |
| 保障租赁 REITs | 是 | 是 | 政策将库存转为收益资产，并建立扩募通道 |

### 银行：红利底仓，不是再通胀代理

银行是最清晰的尾部风险对冲品种。如果锚定城市房价企稳，市场可以下修严重按揭抵押品螺旋的概率。这很重要，因为按揭仍具备政治和系统重要性，银行估值也长期被不透明的地产和地方政府敞口压制。

但上行空间受限。PBOC 数据显示贷款利率已经处于历史低位：2025 年 9 月，5 年期以上 LPR 为 3.5%，新发放贷款加权平均利率约 3.2%，个人住房贷款加权平均利率为 3.06%。[^pbc_rates] 因此 NIM 压力会抵消部分资产质量修复。2026 年 1-3 月个人按揭贷款同比下降 34.6%，也说明抵押品改善不等于贷款增长回归。[^nbs_property]

配置上，超配国有大行，获取股息和风险溢价压缩；区域行保持选择性；回避资产负债表高度绑定弱低线地产、开发贷或地方财政压力的银行。

### 开发商：只有幸存者组合值得配置

开发商交易必须强分化：

| 开发商类型 | 权益判断 |
|---|---|
| 一线和强二线土储占比较高的央企 / 地方国企开发商 | 真正重估候选 |
| 有再融资通道、已完工库存压力较低的优质民营幸存者 | 选择性重估候选 |
| 项目停滞、境外债压顶的违约开发商 | 仅是尾部风险压缩 |
| 低线土储开发商 | 回避 |

对幸存开发商而言，机制是真实的：违约概率下降、融资利差下降、锚定城市去化改善、NAV 更可信。港股开发商可能反应更大，因为其估值折价和境外风险溢价更深。A 股国企开发商更稳，更偏资产负债表修复。

违约开发商不同。政策托底可以改善重组回收价值，但上市权益仍排在债权人、资产处置、项目层面索赔和摊薄之后。它们应被视为重组期权，而不是行业重估。

### 建材：清库存不等于新开工

广义建材不应被当成复苏交易。政策购买的是已完工库存，并不意味着预售、土地拍卖或新开工回到旧周期规模。2026 年 1-3 月新开工面积同比下降 20.3%、竣工面积下降 25.0%，水泥、建筑玻璃、螺纹钢和广义防水材料的量能可见度都偏弱。[^nbs_property]

可投资例外很窄：装修材料、家装渠道，以及产能纪律和现金流优秀的龙头。即便如此，交易也只是部分修复，因为 3 月建筑及装潢材料零售额同比下降 9.0%，1-3 月下降 4.7%。[^nbs_retail]

### 家电与家居改善：最好的地产相关消费交易

家电的设置优于原材料建材。家电对新开工依赖较低，对存量更新、以旧换新补贴和锚定城市二手房周转更敏感。1-3 月家电零售额为 0.0%，明显好于部分地产相关品类，但 3 月下降 5.0% 说明这个交易仍需要政策和收入支撑。[^nbs_retail]

合适的敞口是白电龙头和高份额家电平台，特征是分红、出口和国内更新需求都较强。不要假设低线住房会很快企稳；这不是基准情形。

### 消费：信心企稳有帮助，但财富再通胀太窄

广义消费得到的是情绪底，不是完整地产财富效应反弹。20 个城市企稳可以减少负面新闻流，提升最强城市群的消费意愿，并利好搬迁、装修、服务相关品类。但它不能重建全国财富效应，因为低线房价仍在压制居民资产负债表。

配置应优先质量品牌、必选消费、服务和具备现金生成能力的平台。低线可选消费、低毛利零售和高杠杆消费股，不应因为地产政策改善而简单买入。

### 保障租赁住房 REITs：真实重估，但必须看收益率纪律

保障租赁住房 REITs 是少数政策机制和资本市场机制一致的直接受益者。政府收购已完工库存，将部分资产转化为租赁或保障房供给，并通过公募 REITs 提供吸收与资产循环通道。

上交所披露，截至 2026 年 3 月底，已有 7 只上市 REIT 项目完成扩募，包括华夏北京保障房 REIT；厦门安居保障性租赁住房 REIT 已获批待发行；同时，商业不动产 REITs 试点已有 14 个项目受理、规模 RMB 570 亿元，机构 REIT 市场受理 70 个项目、规模近 RMB 1300 亿元。[^sse_reits] 这很重要，因为扩募能降低单一资产风险，并让 REITs 成为更可扩展的资产类别。

风险在估值。只有当分派收益率相对中国国债仍有吸引力，且资产位于有真实租赁需求的城市，REITs 才配得上政策通道重估。低线租赁资产不应享受同样倍数。

## A 股与港股的差异

港股在幸存开发商、物管和低估值中国金融股上应有更高上行 beta，因为境外风险溢价更直接受违约和汇率叙事影响。互联互通机制保证内地与香港配置通道存在，沪港通和深港通南向每日额度各为 RMB 420 亿元。[^hkex_connect] 当地产尾部风险下降时，这支持战术性南向轮动。

A 股更适合表达红利银行、家电、C-REITs 和内需消费龙头。A 股市场更可能奖励盈利稳定和政策一致性，而不是困境地产期权。

## 风格配置

| 风格 | 建议权重 | 判断 |
|---|---:|---|
| 价值 / 红利 | 35%-40% | 核心超配：大行、高分红 SOE、公用事业式现金流、部分 REITs |
| 优质消费 / 家电 | 20%-25% | 不依赖新开工、同时捕捉核心城市企稳的最佳路径 |
| 地产幸存者 / REITs | 10%-15% | 只做集中配置；回避违约开发商权益 |
| 成长 | 15%-20% | 保留 AI、半导体、高端制造、互联网龙头敞口；地产政策不足以全面放弃成长 |
| 广义周期 / 建材 | 0%-5% | 仅做战术配置，因为新开工和竣工仍弱 |
| 小盘 | 低配 | 低线需求、再融资敏感性和盈利质量问题仍然压制风险收益 |

这不是纯价值轮动，而是质量价值杠铃。地产托底降低下行尾部，利好红利/价值资产；但没有全国再通胀，意味着不应全面卖出成长。正确轮动是离开低质量小盘和困境地产 beta，而不是离开所有成长。

## 上调或下调触发条件

若以下三项同时出现，应上调地产相关配置：

1. NBS 70 城数据显示，连续两个月至少 25-30 个城市新房和二手房价格环比上涨。
2. 个人按揭贷款同比不再收缩，房企到位资金降幅收窄至个位数。
3. 建筑及装潢材料零售额连续两个月转正，而不仅是家电更新。

若以下情况出现，应下调：

1. 政策包络超过 RMB 2.0 万亿元，但缺乏明确中央财政风险分担，因为人民币约束会回归。
2. 低线价格继续下跌，且到 2026 年中锚定城市企稳未能扩散。
3. 银行按揭不良披露恶化速度快于抵押品修复。
4. REITs 扩募收益率已不足以补偿资产区位风险。

## 结论

分化型地产底对权益市场的含义是选择性重估，而不是板块全面牛市。银行、违约开发商和广义建材主要获得尾部风险压缩。幸存开发商、保障租赁住房 REITs 和家电龙头有机会获得真正重估，因为政策路径可以改善其盈利可见度或资产估值。消费受益于信心企稳，但只应落实在质量标的和强城市群。

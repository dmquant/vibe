## Summary

截至2026-05-10，本卡压力测试电力成本到AI应用端的传导假设。RMB 0.05–0.10/kWh 的电力增量沿IDC→云租赁→模型API→AI应用四层传导，但在第三层严重泄漏：token价格YoY -60%至-80%的通缩曲线和算法效率红利使模型API厂商无力提价，反而吸收上游增量为利润压缩；二线IDC因产能过剩同样承担150-250bp利润下行。真正能传导的仅枢纽IDC长协、主权负载与垂直SaaS打包者，约占需求30-40%。结论：Card-06对枢纽IDC的超配确认，但新增对纯模型API与消费AI应用的低配信号——它们是廉价算力的利润压缩受益者，不应与电力合格资产同篮配置。

# 算力成本传导链条：从电力成本到AI应用端利润率

## 1. 问题与设定
Card-03 至 Card-06 已确立：2026H2 国产算力的硬约束不在芯片，而在电力——枢纽级可调度电量、PUE/液冷资本开支与绿电 PPA 溢价合计形成约 RMB 0.05–0.10/kWh 的结构性额外成本。策略线索默认这部分成本能通过算力租赁价格上行或效率优化沿链条传导（IDC → 云/IaaS → 模型厂商 → AI 应用），各环节利润率得以维持。本卡专门压力测试该假设。

核心问题：每年 RMB 20–40 亿元的电力增量成本（40 TWh × RMB 0.05–0.10/kWh）能否真正传导至 AI 应用端，而不在某一层被吸收？

## 2. 四层传导栈
成本沿四个合同界面流转，每层议价能力与价格弹性差异极大。

| 层级 | 单位经济（2026E） | 议价能力 | 基准传导率 |
|---|---|---|---|
| 1. IDC 运营商 | 电力+制冷 ≈ 55–65% opex | 枢纽资格强；二线弱 | 70–90% |
| 2. 云/GPU 租赁 | A800/H20 等效机柜月租 RMB 3.8万–5.5万 | 中等；现货 YTD +12–18%，合同基本持平 | 50–70% |
| 3. 模型 API 厂商（DeepSeek、MiniMax、豆包、Qwen-cloud） | 推理成本占收入 35–50% | 弱；价格战，每百万 token 价 YoY -60% 至 -80% | 15–35% |
| 4. AI 应用（B2B SaaS、B2C agent） | 模型 API 占 COGS 20–40% | 分化；B2B 黏性强，B2C 流失敏感 | 0–25% |

核心结论：传导高度**非线性**，并在第三层**严重泄漏**。模型 API 价格战吸收了上游绝大部分增量；AI 应用端几乎感受不到成本上行，因此无意愿提价。利润压缩集中在第二、第三层。

## 3. 第三层为何无法传导
三个结构性原因：

1. **Token 价格通缩快于电力通胀。** DeepSeek V4、豆包 1.5-pro、Qwen-3-Max-cloud、MiniMax M2 等公开价目显示，Q1 2026 每百万输出 token 价格同比下降 60–80%。即便机柜租赁上行 15%，也被通缩曲线完全压过。
2. **算法效率红利归属模型厂商而非 IDC。** MoE 稀疏化、推测解码、INT4/FP8 量化与 KV-cache 复用使 tokens/瓦在 18 个月内提升约 2.5–3.5 倍。同样 40 TWh 支撑的收入 token 数大幅上升，但生产率红利被厂商当作价格战弹药消耗，并未沉淀为利润。
3. **第二层产能过剩。** Card-04 的 30–50 TWh 新增产能中，大量无枢纽资格的二线 IDC 将以价格竞争应对，租金涨幅被压在 5–10% 区间，而它们自身电费上行 15–25%。

## 4. 真正能传导的三个口袋
传导仍然有效的部分：
- **持有带电力 passthrough 条款长协的枢纽 IDC**（2025 年起新签合同普遍包含），事实上已将电力风险预售给企业客户。
- **主权/受监管负载**：政务云、央企国产栈强制采购、安全分级推理服务，按成本加成定价，绿电 PPA 溢价可被报销。
- **垂直 B2B SaaS 内嵌 AI**（法律、医学影像、EDA copilot），AI 功能打包入坐席许可，客户看不到 token 成本行项。

这三块约占可寻址算力需求的 30–40%。其余 60–70%（消费 agent、公共模型 API、通用企业推理）落在通缩区间。

## 5. 压力测试结论
策略线索的隐含假设——电力成本增量能干净传导至 AI 应用利润——在压力测试下**不成立**。

- **成立**：枢纽 IDC + passthrough、主权工作负载、垂直 SaaS 打包者，约占需求 30–40%。
- **不成立**：模型 API 厂商与消费 AI 应用。价格战叠加效率红利使上游成本在第三层被吸收为利润压缩，而非向下推。第四层利润反而**扩张**——是第三层在补贴它。
- **净含义**：Card-06 的资产配置框架不应假设算力栈各层利润率同步扩张。"电力合格"篮子（枢纽 IDC + 液冷 + 西部绿电一体化）才是真正捕获租金的资产；模型 API 与纯应用层应被视为**廉价算力的利润压缩受益者**，而不是同一多头的成员。

## 6. 量化测算
基准情形 40 TWh × RMB 0.075/kWh 平均绿电+PUE 溢价 → 全行业每年 RMB 30 亿元增量电力成本。
- 第一层吸收 RMB 3–6 亿元（10–20%），上传 RMB 24–27 亿元。
- 第二层吸收 RMB 7–11 亿元（二线 IDC 利润率 -150 至 -250 bp），上传 RMB 13–20 亿元。
- 第三层吸收 RMB 9–14 亿元（模型 API 利润率 -300 至 -500 bp），上传 RMB 4–6 亿元。
- 第四层最终承担 RMB 4–6 亿元，处于 SaaS 正常价格弹性范围内，无需提价即被吸收。

## 7. 投资含义
确认 Card-06 对枢纽 IDC 与一体化电力资产的超配。**新增低配/做空信号**：缺乏垂直护城河的纯模型 API 厂商与消费 AI 应用，2026H2 即便算力扩张，利润仍将压缩——通缩曲线 + 产能过剩使传导失效。"token 价格下行驱动 AI 应用利润扩张"的多头叙事部分正确，但下行的 token 价是模型厂商股权在埋单，并非真实生产率盈余流到应用端。

## 8. 交接
最自然的下一问题是：当前主题研究是否已识别哪些 AI 应用子主题（agentic、多模态、垂直 SaaS）真正拥有逃离通缩区间的定价能力。这直接延续本卡第四层的细分判断，归属 **thematic-researcher** 主题研究员域。

## 9. 信心与警示
信心 0.62。主要不确定性：(a) H2 token 价格通缩进一步加速的可能；(b) 主权强制采购份额能否突破 30% 稀释通缩区间；(c) 汇率与芯片供应冲击对租赁合约的重定价。若 FOMC 流动性收紧使现金消耗型模型厂商提前退出，第三层传导率反而**上升**，多头叙事可能局部修复。

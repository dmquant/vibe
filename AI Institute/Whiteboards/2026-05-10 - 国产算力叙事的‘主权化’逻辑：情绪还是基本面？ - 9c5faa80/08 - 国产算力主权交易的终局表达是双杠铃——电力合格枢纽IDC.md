## 1. 全线综合（Card 01–07）

线索经过六个承重节点：

1. **Card 01（TMT）**：真实基本面 + 主权反身性的混合体；硬件篮子估值领先业绩。
2. **Card 02（A 股策略）**：5 月 6–8 日波动窗口出清了散户两融杠杆；机构定价权未破，板块未拥挤。
3. **Card 03（能源）**：硬约束从芯片产能迁移到电力 / PUE / 绿电三件套；2026 年 30–50 TWh 增量。
4. **Card 04（公用事业）**：全国电量可承接，但**枢纽级可调度电力、迎峰度夏需求响应、绿电 PPA 溢价**才是 H2 分化变量。绿电溢价约 RMB 0.05/kWh，对 40 TWh 是 RMB 2.0 bn/年。
5. **Card 05（首席策略）**：从宽基硬件篮子切到电力合格资产（35–40% 枢纽 IDC、20–25% 液冷/HVDC、15–20% 西部绿电一体化、15–20% 已验证硬件龙头）。
6. **Card 06（大类资产）**：在多行业组合中给主题定权重。
7. **Card 07（TMT 压力测试）**：传导链**在第 3 层（模型 API）严重泄漏**，token 价格 YoY −60% 到 −80% 使 API 厂商吸收 300–500 bp 利润压缩；真正捕获租金的只有约 30–40% 需求（枢纽 IDC 长协、主权负载、垂直 SaaS 打包者）。

**Card 08 的任务是把 Card 07 中"30–40% 捕获租金"的颗粒度拆到具体应用子主题与 A 股 / 港股标的。**

## 2. Token 通缩锥体——谁在锥内，谁能逃离

token 通缩是结构性的，不是周期性的：算法效率红利（MoE、蒸馏、小模型平替）在产能过剩之上叠加了第二层通缩。**任何以"token 服务量"计价的业务都在滑梯上**。要拥有定价能力，必须满足以下之一：(a) **以结果计价**（按任务、按结案工单、按理赔单、按设计件），(b) **受监管 / 主权需求**（价格弹性低），(c) **分发锁定**（客户无法绕过去比价底层 token）。

| 子主题 | 计价单位 | 暴露 token 通缩 | 真实定价能力 | 评级 |
|---|---|---|---|---|
| 通用聊天 / 消费 AI app | token / MAU 广告 | 直接、全暴露 | 负（趋零竞争） | 回避 |
| 模型 API 平台（基础模型厂） | 按 token | 直接、全暴露 | 利润压缩 −300 至 −500 bp | 主题低配；个股看护城河 |
| **Agentic / 任务执行** | 按任务、按结案 | 间接——单任务 token 增多但按结果计价 | **高**——token 成本下降反成毛利红利 | **超配** |
| **多模态垂直（视频 / 3D / 工业视觉）** | 按分钟 / 资产 / 检测 | 间接——算力重但客户 ROI 清晰 | 中高 | **超配** |
| **垂直 SaaS（法律、医疗、金融、EDA、ERP-AI）** | 按席位 / 工作流 | 低——打包进既有 license | 高——分发 + 合规护城河 | **超配** |
| **主权 / 受监管 AI**（政企、国防、央国企、金融合规） | 项目 / 受监管单价 | 低 | 高——价格弹性低的采购 | **超配** |
| 通用文生图 / 内容生成 | 按生成次数 | 直接 | 低 | 低配 |

逃离速度的核心是 **把收入单位与 token 单位解耦**。一旦客户付费按结果计算，token 成本下降反而成为毛利顺风。

## 4. 成本传导图——电力成本如何到达应用端

| 层级 | Card 07 判断 | Card 08 细化 |
|---|---|---|
| 1. 枢纽 IDC 长协 | 完全传导 | 确认 —— 主题权重 35–40% |
| 2. 云租赁（主权 / 包年） | 部分传导 | 确认 |
| 3. 模型 API（开放市场） | 泄漏 300–500 bp | 主题低配 |
| 4a. 按结果计价的 agentic / 垂直 SaaS | 完全传导 | **本卡新增超配**——token 成本下降即毛利提升 |
| 4b. 按 token 计价的消费 AI | 完全泄漏 | 低配 |

线索的终局结论：**国产算力主权交易的最佳表达不是"买硬件"，而是双杠铃**——(i) 供给侧的电力合格枢纽 IDC + 西部绿电一体化，(ii) 需求侧的按结果计价垂直 SaaS / agentic / 主权 AI 打包者。中间层（通用模型 API + 消费 AI app）是利润泄漏处，应低配。

## 5. 组合含义（对接 Card 06 权重）

在 Card 06 设定的主题权重内：
- 保留 Card 05 的供给侧拆分（枢纽 IDC / 液冷 / 绿电 / 硬件龙头），作为 **基础设施腿** ≈ 70%。
- 新增 **应用腿** ≈ 30%，拆分：垂直 SaaS 40%、agentic 30%、多模态垂直 20%、主权 AI 集成 10%。
- 硬低配：独立模型 API、消费聊天、通用文生图。

应用腿上调触发条件：(a) 金山办公 / 恒生电子 / 金蝶连续两个季度席位转化或结果收入超预期；(b) 快手可灵或讯飞披露 ACV > RMB 1 亿的按结果计价企业 SKU；(c) 财报中明确出现"token 成本下降转化为毛利提升"的叙事。

下调触发条件：(a) 头部云的 API 价格战外溢到垂直 SaaS 定价；(b) 监管对 AI 采购单价设上限；(c) 旗舰副驾驶席位转化率长期低于 15%。

## 6. 置信度与开放问题

置信度：0.62。层级图谱稳健；个股执行风险高，因为按结果计价的 SKU 多数刚披露 1–2 个季度。下一轮线索的开放问题：

- 国内 IT 预算持平的环境下，国产垂直 SaaS 是否能持续抬席位价？
- 主权负载能否扩到全国算力的 25% 以上，还是停在 15% 以下？
- 12–18 个月后开源模型趋同，是否会让 agentic 护城河商品化？

以上三问可以作为下一会话的根主题候选。

## Report

# Card 10 — 用产业基本面对 0.75–0.80x 半导体 Beta 切换做应力测试

- 工作日期（Asia/Singapore）：2026-05-09
- Board：74c79e14-d7f0-4a0c-9a77-71cdbcd12f6b
- 分析师：TMT 行业分析师
- 立场：stress-test
- 主题：USDCNY 波动对半导体 Beta 调整的产业基本面验证
- 问题：随着 USDCNY 逼近 7.05 触发点，半导体板块的进口成本压力与全球需求变化是否支持将 Beta 降至 0.75–0.80x？

## 1. 核心结论

**部分支持，但 FX 通道并不是真正的理由。** 在 USDCNY 7.05 被中间价（6.91–6.93）与跨境资金流共同确认后（参见 card-08/09），把中国半导体 Beta 降至 0.75–0.80x 是合理的——**但并不是因为进口成本会击穿 FY27 盈利**。主导传导是「风险体制 / OIP 政策尾部」通道（出口管制、设备许可、上市规则风险），FX 本身只是被部分对冲过的次要变量。把 7.05 当作 *状态切换器* 进入防御 Beta，与 card-09 的处方完全一致；但该 Beta 的产业基本面承重，主要来自中芯的美元收入自然对冲与华虹的国内结构防护，而不是 7.05 这个数字本身。

## 2. USDCNY 7.05 下的产业基本面拆解

### 2.1 进口成本敞口——看净额，不看毛额

| 成本项 | 占中芯 FY26 COGS（估） | 美元挂钩？ | 净 FX 敏感度 |
|---|---|---|---|
| 进口设备折旧 | ~22–28% | 是（按历史汇率入账） | 低——已沉没 |
| 备件、耗材、光刻胶、特气 | ~12–15% | 多为 USD/JPY | 高——6–9 个月传导 |
| 进口 12" 硅片 | ~6–8% | USD | 高 |
| 国产材料与水电 | ~25–30% | CNY | 无 |
| 人工 | ~15% | CNY | 无 |
| IP / 授权费 | ~3–5% | USD | 高 |

6.83 → 7.05 是 **3.2% 的 CNY 贬值**，仅作用于美元挂钩的可变 COGS（约占 25–30%），**毛 COGS 抬升约 80–95 bps**。但**中芯约 55–65% 的收入以美元计价**（晶圆代工通行做法），自然对冲计入后，毛利率层面通常出现 **+30 至 +60 bps 的顺风**，而非逆风。华虹的美元收入占比较低（约 35–40%），净 FX 略高于中芯，但仍接近中性。

**FX-进口成本通道的判定：** 单独不足以支撑 20–25% 的 Beta 下调。0.75–0.80x 不能仅靠成本传导算术站住。

### 2.2 全球需求通道——真正与 USDCNY 7.05+ 共振的是什么

7.05 很少独立出现，它通常与下列共振：(i) DXY > 105；(ii) 美国关税/出口管制重新加码；(iii) hyperscaler 资本开支减速；(iv) 港股流动性紧张。半导体 Beta 真正要回答的问题是：是这些共振变量、而非 FX 本身，构成降 Beta 的产业理由。

- **DeepSeek V4 / Ascend 国内订单**：与美元无关，字节/腾讯/阿里订单以 CNY 计价并由政策锚定（card-1 至 card-3）。FX 与该收入线无关。
- **HBM 进口依赖**：真实且以美元计价，但天花板是出口管制制度，而非 FX。CXMT HBM3 时间表（card-3 基准 2027H1）才是绑定约束，3% 的 FX 不会改变 HBM 的*数量*。
- **全球半导体（成熟节点 / 代工对外）需求**：华虹 8"/28nm 利用率追踪全球 IoT / 汽车 / 电源管理需求。强美元确实与新兴市场终端需求走弱相关——按经验，5% 的有效汇率走弱对应约 -5% 的收入弹性，这是真正的 Beta 关切。
- **OIP / 出口管制尾部**：是主导风险。7.05 与 OIP 收紧同时出现时，会同步压制中芯 N+2 量产和 CXMT HBM 设备许可。**这才是 Beta 下调的真正理由。**

### 2.3 真正支撑 0.75–0.80x Beta 的驱动拆解

把建议的 20–25% Beta 下调拆给具体驱动：

| 驱动 | 对 Beta 下调的贡献 | 是否站得住 |
|---|---|---|
| 纯 FX 进口成本冲击 | ~3–5pp | 弱——美元收入自然对冲基本抵消 |
| 华虹全球周期需求拖累 | ~3–5pp | 中等 |
| OIP / 出口管制政策尾部（真正驱动） | ~10–15pp | 强——与 card-04/05 失败信号 A 对应 |
| 港股流动性 / 南向回撤共振 | ~3–5pp | 中等 |

0.75–0.80x **答案对、问题问错了**：它能通过应力测试，仅仅因为 USDCNY 7.05 是 OIP 政策尾部的*相关状态指示*，而非因为进口成本威胁盈利。

## 3. 个股验证

- **中芯国际（0981.HK）**：约 60% 美元收入 vs 约 30% 美元挂钩可变 COGS → 毛利率层面 **净 FX 顺风**。FY27 盈利*不*会被 7.05 击穿，真正的威胁是与之共振的出口管制再收紧。Beta 下调的理由是尾部，不是 P&L。
- **华虹半导体（1347.HK）**：约 35–40% 美元收入 vs 约 25% 美元挂钩 COGS → 净 FX 接近中性。对*全球成熟节点周期*更敏感，这才是强美元真正构成基本面逆风的唯一通道。从产业逻辑看，华虹应在 Beta 下调中承担多于中芯的份额——这在边际上与 card-04 的 60/40 偏华虹略有反向。
- **寒武纪 / 海光 / 浪潮 / 工业富联（Ascend 周期对冲）**：纯国内收入、FX 敏感度近零，FX 理由下*不应*参与 Beta 下调。card-04 的「永不净空 Ascend 对冲腿」立场被加强。
- **设备（北方华创 / 中微）与封装（长电 / 通富）**：进口元器件密集（设备/材料 USD 挂钩），但收入以国内为主。7.05 下小幅毛利率收缩（低单位数 bps），不是枢轴。

## 4. 对 card-09 的 0.75–0.80x 规则的微调

- **保留** 0.75–0.80x 作为 L1 确认后的防御 Beta，幅度方向校准正确。
- **重新归因**：触发是 OIP / 出口管制尾部，**由** USDCNY 7.05 + 中间价 6.91–6.93 + 资金流共同*指示*。对内沟通时不要把它包装成「FX 损害盈利」。
- **下调内部倾斜**：在 Beta 下调中，对 **华虹（1347.HK）+ 全球周期型成熟节点** 多砍，对 **中芯 + Ascend 国内对冲腿（寒武纪、海光、浪潮、工业富联）** 维持接近中性。让降 Beta 真正集中在产业基本面在强美元/政策尾部下确实走弱的子集。
- **设备/封装 Beta 不低于 0.90x**：进口成本压力存在但有限，FY27 收入受政策保护程度最高，仍然是 card-04 P1→P2→P3 轮动中的早期主腿。
- **重置条件**：若 7.05 在 *没有* OIP 升级的情形下打印（纯 FX 冲击，如 DXY 大震但政策面平静），不降 Beta——产业基本面不支持。

## 5. 该应力测试的风险

- 若 7.05+ 体制持续，韩国 HBM 厂商可能上调美元报价，2026H2 模组毛利压缩可能超过 80–95 bps 的估计。
- 美元计价的 IP / 授权费在出口管制叠加许可成本下可能阶梯式上行——这是政策而非 FX，但与之共振。
- 纯国内收入名（寒武纪、海光）在*运营*上免疫 FX，但*估值*上仍会被强美元导致的港股/A 股流动性流失杀估值。即使基本面不支持，Beta 下调也必须承认这一估值通道。

## 6. 结论与交接

USDCNY 7.05 经 L1 确认后，0.75–0.80x 的 Beta 设置幅度正确；产业基本面承重 ~70% 来自其指示的 OIP/出口管制尾部，~30% 来自直接 FX/需求传导。中芯的美元收入自然对冲意味着降 Beta 应向华虹与全球周期型成熟节点倾斜，中芯、设备/封装与 Ascend 国内对冲腿则在防御体制下维持 0.90–1.00x。

# 卡08：组合级压力测试与尾部风险评估

## 结论先行

本卡对卡06的“All-Weather + AI物理资源”配置和卡07的结构化金属篮子采取压力测试立场。核心判断是：该组合相对传统60/40更能承受“通胀不下、能源和金属上行”的基线环境，但它不是天然的尾部风险组合。三类压力情景下，真正的脆弱点不是平均波动率，而是相关性在冲击日重新归一化：权益、能源、工业金属和信用利差会在A情景同时变成“AI capex beta”；名义债、TIPS和黄金会在B情景同时暴露于“真实利率上行”；金属篮子在C情景会从战略对冲变成流动性和再平衡风险源。

按99% VaR口径估算，组合基线1日VaR约为1.6%-1.9%，10日VaR约为5.2%-6.0%。在A情景“AI capex 2026H2突然下修30%”下，1日VaR上升到2.4%-3.0%，10日VaR上升到8.0%-9.5%，情景最大回撤约9%-12%。在B情景“核心PCE升至3.2%、10Y UST突破5%”下，1日VaR上升到3.0%-3.6%，10日VaR上升到10%-12%，情景最大回撤约12%-16%，这是组合左尾最危险的路径。在C情景“铜挤仓+GOES交期断裂”下，组合账面可能短期受益，但VaR会因金属波动、保证金、代理品种basis和再平衡纪律失效而升至1日2.0%-2.5%、10日6.5%-8.0%，最大回撤风险主要来自挤仓后的快速回落和权益端利润率压缩。

执行层必须把“战略配置”和“尾部对冲”分开。建议保留卡06/07的结构化实物敞口，但用三层尾部保护覆盖相关性断裂：第一层是3-6个月VIX看涨价差或call spread collar，对冲A情景的权益波动率跳升；第二层是CDX HY/IG买保护或payer swaptions，对冲AI capex下修引发的信用利差跳升；第三层是利率长波动率和商品波动率组合，即10Y/5Y payer swaptions、铜/铝看涨期权与挤仓后保护性看跌，覆盖B和C情景。

## 情景A：AI capex 2026H2突然下修30%

### 冲击设定

该情景对应卡01-04反复提示的物理瓶颈转为capex下修触发器：电力接入、变压器、燃气轮机和管道能力不足导致hyperscaler把2026H2新增订单或项目节奏下修30%。市场会先按“AI需求二阶导恶化”交易，而不是按“供给瓶颈仍存在”交易。

| 资产 | 即时冲击假设 | 逻辑 |
|---|---:|---|
| 科技/AI权益 | -18%至-25% | 折旧压力、订单能见度和估值倍数同时下修 |
| 广义股票 | -10%至-14% | 盈利预期和风险偏好回落 |
| 能源 | -6%至-10% | 数据中心增量负荷预期降温，油气仍受宏观需求拖累 |
| 工业金属篮子 | -12%至-18% | 铜和铝回吐，GOES相对抗跌但权益代理受冲击 |
| 名义债 | +2.0%至+3.5% | 增长冲击带来收益率下行 |
| TIPS | +1.0%至+2.0% | 真实利率下行抵消breakeven回落 |
| 黄金 | +3%至+6% | 风险规避和降息预期支撑 |
| 现金 | 0% | 提供再平衡空间 |

### 组合结果

| 指标 | 基线 | 情景A |
|---|---:|---:|
| 99% 1日VaR | 1.6%-1.9% | 2.4%-3.0% |
| 99% 10日VaR | 5.2%-6.0% | 8.0%-9.5% |
| 情景即时损益 | -5.5%至-7.5% | 由股票和金属主导 |
| 最大回撤 | 6%-8% | 9%-12% |

A情景的核心风险是“AI物理资源”敞口与股票敞口同时变成同一个因子。卡06把能源、公用、黄金、工业金属作为60/40之外的实物对冲是合理的，但在capex下修的第一阶段，市场会先卖出所有AI建设链相关敞口。铜、铝、能源股和AI电力设备代理会和科技股一起下跌，直到市场重新区分长期瓶颈与短期订单滑期。

### 相关性变化

| 相关性 | 基线 | 情景A |
|---|---:|---:|
| 股票-工业金属 | 0.35 | 0.70-0.80 |
| 股票-能源 | 0.40 | 0.55-0.70 |
| 能源-工业金属 | 0.55 | 0.65-0.80 |
| 股票-名义债 | -0.10 | -0.35至-0.55 |
| TIPS-黄金 | -0.20 | -0.10至+0.10 |

这里名义债仍能部分保护组合，但保护力度不足以抵消权益和资源端的双重beta。对冲重点应放在权益波动率和信用利差，而不是再增加黄金或TIPS。

## 情景B：能源/电网瓶颈倒灌为stagflation

### 冲击设定

该情景承接卡05的宏观判断，但把压力从“stagflation-lite”推到“stagflation”：核心PCE升至3.2%，10Y UST突破5%，曲线bear-steepen，真实利率上行快于breakeven扩张。此时AI capex没有立刻崩，但建设成本、融资成本和折现率同时抬升。

| 资产 | 即时冲击假设 | 逻辑 |
|---|---:|---|
| 科技/AI权益 | -15%至-20% | 长久期现金流受折现率冲击 |
| 广义股票 | -8%至-13% | 利率和利润率双压 |
| 能源 | +8%至+14% | 电力、天然气和容量价格溢价上升 |
| 工业金属篮子 | 0%至+8% | 铜/GOES受供给约束支撑，铝和顺周期金属受需求拖累 |
| 名义债 | -4%至-6% | 中端久期在收益率上行80-110bp时受损 |
| TIPS | -1.5%至-3.5% | 通胀补偿不足以完全抵消真实利率上行 |
| 黄金 | -4%至+2% | 若真实利率主导则下跌，若政策可信度下降则反弹 |
| 现金 | 0%至+0.2% | 短端收益提供轻微缓冲 |

### 组合结果

| 指标 | 基线 | 情景B |
|---|---:|---:|
| 99% 1日VaR | 1.6%-1.9% | 3.0%-3.6% |
| 99% 10日VaR | 5.2%-6.0% | 10.0%-12.0% |
| 情景即时损益 | -6.5%至-9.0% | 债券保护失效是关键 |
| 最大回撤 | 6%-8% | 12%-16% |

B情景是组合最危险的左尾，因为它同时击穿三条通常被视为保护性的假设：TIPS不再稳定对冲通胀，黄金不一定与TIPS负相关，名义债不再是权益回撤对冲。能源上涨可以抵消一部分损失，但组合内能源权重不足以覆盖股票、TIPS和名义债的共同下跌。

### 相关性变化

| 相关性 | 基线 | 情景B |
|---|---:|---:|
| 股票-名义债 | -0.10 | +0.30至+0.55 |
| 股票-TIPS | 0.00 | +0.25至+0.45 |
| TIPS-名义债 | 0.65 | 0.80-0.90 |
| TIPS-黄金 | -0.20 | +0.20至+0.45 |
| 能源-工业金属 | 0.55 | 0.30-0.60 |

在这种路径下，TIPS/名义债55/45的债券结构仍优于传统名义债为主的60/40，但它不是充分对冲。需要用利率长波动率、payer swaptions和更短的名义久期控制左尾。

## 情景C：铜挤仓+GOES交期断裂

### 冲击设定

该情景来自卡07提出的真瓶颈：铜库存低、冶炼和矿端扰动叠加电网需求；GOES/变压器交期断裂，代理资产估值快速重定价。与A、B不同，C情景不是传统“组合亏损最大”的路径，而是“账面盈利和风险失控同时出现”的路径。

| 资产 | 即时冲击假设 | 逻辑 |
|---|---:|---|
| 铜 | +25%至+35% | 挤仓、库存和期限结构同时驱动 |
| GOES/变压器代理 | +35%至+70% | 非标瓶颈和权益代理弹性放大 |
| 锡 | +15%至+25% | 小市场隐性瓶颈被重估 |
| 铝 | +5%至+10% | 跟随电网需求，但约束弱于铜和GOES |
| 科技/AI权益 | -3%至-8% | 成本端和交付延迟压缩利润率 |
| 公用/电力设备 | +2%至+8% | 受益于电网资本开支延长 |
| 债券 | -0.5%至+0.5% | 取决于通胀预期是否外溢 |
| 黄金 | -1%至+3% | 若冲击局限在供应链，黄金反应较弱 |

### 组合结果

| 指标 | 基线 | 情景C |
|---|---:|---:|
| 99% 1日VaR | 1.6%-1.9% | 2.0%-2.5% |
| 99% 10日VaR | 5.2%-6.0% | 6.5%-8.0% |
| 情景即时损益 | +0.5%至+2.5% | 金属盈利被权益利润率冲击部分抵消 |
| 最大回撤 | 6%-8% | 7%-10% |

C情景的风险不是方向性亏损，而是三个执行问题：一是铜和GOES无法完全通过可交易期货或ETF线性表达，代理品种basis会放大；二是挤仓期间保证金和滑点会吞噬部分战略收益；三是如果组合纪律被账面盈利诱导而追高，挤仓后的均值回归会把右尾盈利转化为左尾回撤。

### 相关性变化

| 相关性 | 基线 | 情景C |
|---|---:|---:|
| 铜-GOES代理 | 0.45 | 0.75-0.90 |
| 股票-工业金属 | 0.35 | -0.10至+0.25 |
| 能源-工业金属 | 0.55 | 0.10-0.35 |
| TIPS-黄金 | -0.20 | -0.10至+0.15 |
| 工业金属内部相关性 | 0.50 | 0.80以上 |

这说明卡07的结构化篮子比宽口径工业金属指数更好，但也更需要组合层面的单桶风险上限和再平衡纪律。工业金属不应从6.5%被价格上涨被动漂移到9%以上，除非同时买入保护性看跌或降低能源/周期权益敞口。

## 三条对冲假设的脆弱性排序

| 脆弱性排序 | 假设 | 脆弱情景 | 判断 |
|---:|---|---|---|
| 1 | TIPS-黄金负相关 | B | 最脆弱。真实利率急升时，TIPS和黄金可能同跌，黄金只有在政策可信度受损或金融压力显著上升后才重新发挥避险作用 |
| 2 | TIPS-名义债期限利差 | B | TIPS优于名义债，但二者同属久期资产；bear-steepener中55/45结构只能降低损失，不能消除损失 |
| 3 | 能源-工业金属正相关 | A与C | 正相关并非稳定对冲假设：A中二者同跌，C中金属涨而能源未必跟随，组合可能出现错误的double-beta |

最需要修正的是“把TIPS、黄金、能源、工业金属都视为通胀对冲”的粗颗粒度框架。它们对通胀来源的反应不同：TIPS对breakeven和真实利率分解敏感，黄金对真实利率和政策可信度敏感，能源对现货供需和监管价格敏感，工业金属对库存、交期和中国/全球制造周期敏感。组合尾部管理必须按冲击来源拆分，而不是按“通胀资产”统一加仓。

## 建议的尾部对冲组合

### 1. VIX看涨期权：覆盖A情景的权益波动率跳升

建议用3-6个月VIX call spread或call ladder，而不是单纯买近期虚值看涨。执行参考：在VIX低于20且AI capex风险窗口临近时，滚动持有25/40或30/50看涨价差，目标名义对冲组合权益风险的25%-35%。若VIX已高于25，则降低新增买入，转向put spread collars或SPX put spreads，避免在波动率已重定价后支付过高权利金。

预期效果：A情景下可把10日VaR降低约120-180bp，把最大回撤压低约150-250bp。对B情景帮助有限，因为利率冲击可能先于权益波动率充分反应。

### 2. 信用CDX保护：覆盖capex下修和融资条件收紧

建议把CDX HY买保护作为主工具，CDX IG买保护作为低成本补充。触发条件包括：AI供应链新增订单下修、HY OAS快速扩大、云厂商债券或电力设备链信用利差同步走阔。结构上可使用3-6个月CDX HY payer swaptions或直接买保护，目标名义为组合的10%-20%，重点覆盖A情景中的“权益下跌+信用利差跳升”。

预期效果：A情景下可降低10日VaR约80-150bp；若B情景演化为信用收缩，也能提供二阶保护。缺点是carry成本和流动性依赖信用市场状态。

### 3. 利率长波动率：覆盖B情景的真实利率尾部

建议配置10Y UST payer swaptions、5Y5Y或2Y10Y曲线steepener期权、TY futures put spreads。目的不是押注收益率方向，而是给“10Y突破5%、名义债与TIPS同时下跌”的路径买保险。组合层面还应把名义债有效久期控制在4.5-5.0年以下，只有当10Y接近或高于5%、实际收益率充分重定价后再加回中端久期。

预期效果：B情景下可降低10日VaR约150-250bp，是三类尾部对冲中对最危险左尾最有效的一层。

### 4. 商品波动率与再平衡纪律：覆盖C情景的右尾变左尾

对工业金属篮子，建议保留结构化多头，但新增两条纪律：第一，铜和GOES代理合计穿透权重超过组合5.5%时，必须通过卖出部分期货/ETF、买入铜看跌价差或降低能源周期权益来回到风险预算；第二，在挤仓阶段允许使用铜看涨期权参与上行，但禁止用追加保证金的方式把线性期货敞口扩大到原预算1.25倍以上。

预期效果：C情景本身可能带来盈利，但该纪律能避免挤仓后的10%-20%金属回落把组合拖入被动去杠杆。

## 对冲后的目标风险区间

| 状态 | 99% 1日VaR | 99% 10日VaR | 最大回撤目标 |
|---|---:|---:|---:|
| 未对冲基线 | 1.6%-1.9% | 5.2%-6.0% | 6%-8% |
| A情景未对冲 | 2.4%-3.0% | 8.0%-9.5% | 9%-12% |
| A情景加VIX+CDX | 1.9%-2.4% | 6.2%-7.5% | 7%-9% |
| B情景未对冲 | 3.0%-3.6% | 10.0%-12.0% | 12%-16% |
| B情景加利率长波动率 | 2.3%-2.9% | 7.5%-9.5% | 9%-12% |
| C情景未对冲 | 2.0%-2.5% | 6.5%-8.0% | 7%-10% |
| C情景加商品期权和纪律再平衡 | 1.8%-2.2% | 5.8%-7.0% | 6%-8% |

对冲预算建议控制在年化40-70bp，危机窗口可临时提高到90bp，但应避免把组合变成持续买保险的负carry结构。对冲不是为了消除日常波动，而是为了在相关性断裂时保留再平衡能力。

## 风控触发器

建议将以下触发器写入执行层，而不是只作为研究观点：

| 触发器 | 阈值 | 动作 |
|---|---:|---|
| AI capex下修风险 | hyperscaler新增订单/指引下修超过15%，或关键供应商新增订单连续两个季度低于收入增速 | 启动VIX call spread和CDX HY保护 |
| 利率尾部 | 10Y UST突破4.85%且5s30s bear-steepen超过25bp | 增加payer swaptions，削减名义债久期 |
| 通胀尾部 | 核心PCE连续两个月高于3.0%年化，或5Y5Y通胀预期上行超过25bp | 提高TIPS质量但不加总久期，同时增加能源和利率波动率保护 |
| 信用压力 | HY OAS扩大超过100bp或AI电力设备链信用利差快速扩张 | CDX HY买保护从观察档升至执行档 |
| 金属挤仓 | 铜期限结构显著backwardation且库存降至极低分位，GOES代理涨幅超过30% | 锁定部分金属收益，买保护性看跌，禁止线性加杠杆 |
| 组合集中度 | 工业金属穿透权重被动升至8%以上，或能源+金属+周期权益合计超过20% | 再平衡回目标区间 |

## 对前序结论的压力测试评价

我支持卡06从传统60/40转向“All-Weather + AI物理资源”的方向，也支持卡07把工业金属从宽指数改为铜+GOES+锡+铝的结构化篮子。但本卡对两个隐含假设提出压力修正：

第一，实物资产不是单一对冲桶。能源、黄金、TIPS和工业金属在B情景下可能无法同时保护组合，其中TIPS和黄金的相关性最容易被真实利率冲击击穿。

第二，AI物理资源敞口不是无条件防御资产。在A情景中，它会先被市场当作AI capex链条风险资产卖出；在C情景中，它又可能因盈利和流动性错配变成再平衡风险源。

因此，组合可继续执行卡06/07框架，但必须叠加明确的尾部对冲和再平衡纪律。没有VIX、CDX、利率长波动率和商品波动率工具，组合在B情景中的10日VaR会超过10%，最大回撤可能接近16%，这已经超出“稳健型All-Weather”应承受的风险预算。

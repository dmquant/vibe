# Card 05 - 智能体 AI 拐点压力测试

**工作日期：2026-05-09（Asia/Singapore）。分析师：thematic-researcher。立场：stress-test。**

截至 2026-05-09，本卡压力测试 Card 04 的判断：推理成本下降与算法效率复利，正在同时削弱 AI 公用事业多头和软件空头叙事。本工作区缺失上游白板文件，因此本卡依据用户提示中嵌入的 session brief 重建 Cards 01-04 语境。

## 核心判断

智能体 AI 栈已经接近 2026H2 的 ROI 临界点，足以支持优质软件的选择性重估，但不足以支持泛 SaaS 修复交易。正确框架不是“所有软件都获得 AI 估值倍数”，而是“拥有工作流控制平面和使用量遥测的平台”。我支持 Card 04 的方向，但增加更严格条件：ROI 拐点属于已经拥有系统记录数据、身份与治理、可计量工作单元的平台；只卖通用座席或模型包装的公司仍处于 hype-cycle 估值陷阱。

## 新证据

能力本身已不再是窄工作流的主要瓶颈。OpenAI 的 Responses API 与 computer-use 工具把 file search、web search、code containers、computer-use actions 产品化；computer-use preview 在 OSWorld、WebArena、WebVoyager 上分别披露 38.1%、58.1%、87% 的成绩，尚未达到通用人类级自主，但已足以覆盖有边界的浏览器和后台任务。[OpenAI agent tools](https://openai.com/index/new-tools-for-building-agents/) 与 [OpenAI pricing](https://openai.com/api/pricing/) 也显示产品形态正在从聊天转向工具执行，其中 web search 定价为 $10 / 1k calls，container sessions 也被明确计价。

DeepSeek 与 Qwen 验证了成本侧冲击。DeepSeek V4 于 2026-04-24 发布，提供 V4-Pro 与 V4-Flash，支持 1M context、tool calls、OpenAI 与 Anthropic 兼容 API 以及 agent integrations；官方价格显示 V4-Flash 为 $0.14 / 1M input tokens 与 $0.28 / 1M output tokens，V4-Pro 在 2026-05-31 前折扣价为 $0.435 / 1M input 与 $0.87 / 1M output，且自 2026-04-26 起 cache-hit input pricing 降至首发价格的十分之一。[DeepSeek V4 release](https://api-docs.deepseek.com/news/news260424)、[DeepSeek pricing](https://api-docs.deepseek.com/quick_start/pricing) 与 [DeepSeek tool calls](https://api-docs.deepseek.com/guides/tool_calls) 支持 Card 04 关于中国与开放模型栈推理成本同比 -60-80% 的前提。

阿里是中国侧最强证据，因为它把 Qwen 同时连接到云与消费者交易。截至 2025-12-31 的季度，Alibaba 披露 Cloud Intelligence revenue 同比增长 36%，AI-related product revenue 连续第十个季度三位数增长，Qwen consumer MAU 超过 300 million，且 Qwen agents 已在真实任务执行中规模化。但同一份财报也显示 revenue 仅同比增长 2%，adjusted EBITA 同比下降 57%，operating cash flow 下降 49%，non-GAAP diluted EPS 下降 67%，因此 AI 尚未形成干净的利润拐点。[Alibaba December-quarter results](https://pdf.dfcfw.com/pdf/H2_AN202603191820648049_1.pdf%3F1773942037000.pdf) 与 [Alibaba AI+Cloud update](https://www.alibabagroup.com/en-US/document-1971445322303406080) 支持“增长期权叠加再投资拖累”，而非近端利润释放。

美国 SaaS 的证据更强，但仍高度集中。Salesforce 披露 FY26 revenue 为 $41.5bn，RPO 为 $72.4bn，Agentforce ARR 为 $800mn，Agentforce deals 为 29,000，agentic work units 达 2.4bn，累计 tokens processed 为 19tn；2026 年 1 月季度 Agentforce 360 Platform, Slack and Other subscription support revenue 按 constant currency 增长 37%。[Salesforce FY26 Q4](https://investor.salesforce.com/news/news-details/2026/Salesforce-Delivers-Record-Fourth-Quarter-Fiscal-2026-Results/default.aspx) 这是真实商业化，但相对于 $41.5bn 收入基数仍偏小。

ServiceNow 与 Microsoft 展示了更高质量的 ROI 模式：智能体嵌在既有企业系统中，而不是被单独卖成聊天机器人。ServiceNow Q1 2026 subscription revenue 为 $3.671bn，同比增长 22%；cRPO 为 $12.64bn，同比增长 22.5%；RPO 为 $27.7bn，同比增长 25%；Now Assist customers spending over $1mn in ACV 同比增长超过 130%。[ServiceNow Q1 2026](https://investor.servicenow.com/news/news-details/2026/ServiceNow-Reports-First-Quarter-2026-Financial-Results/default.aspx) Microsoft 披露超过 15,000 customers 同时使用 Foundry 与 Fabric，近 90% 的 Fortune 500 已通过其 low/no-code 工具构建 active agents，Copilot Credit consumption 环比接近翻倍，Microsoft 365 Copilot paid seats 在 seat-add 同比增长 250% 后超过 20mn。[Microsoft FY26 Q3 call](https://www.microsoft.com/en-us/investor/events/fy-2026/earnings-fy-2026-q3)

Atlassian 提供了独立使用量信号：Q3 FY26 revenue 为 $1.8bn，同比增长 32%；Cloud revenue 超过 $1.1bn，同比增长 29%；Rovo customers 的 ARR 增速约为 non-Rovo customers 的 2x；AI credit usage 月环比增长超过 20%。[Atlassian shareholder letter](https://www.atlassian.com/blog/company-news/shareholder-letter-q3fy26) 这支持 consumption-credit 商业化，而非纯座席定价。

## 压力测试：仍会断裂的环节

第一，基准测试安全感很脆弱。OpenAI 已停止依赖 SWE-bench Verified 来证明前沿 coding 能力，并建议使用 SWE-bench Pro，原因是 Verified 存在污染与测试质量问题；其披露的顶线进展在六个月内从 74.9% 放缓至 80.9%。[OpenAI SWE-bench note](https://openai.com/de-DE/index/why-we-no-longer-evaluate-swe-bench-verified/) 因此，估值不能直接资本化 benchmark 跳升，除非它转化为生产环境续约、cRPO 或 usage-credit 指标。

第二，自主性仍受治理约束。2026 年一项独立压力测试发现，在刻意设置的模糊授权场景中，Claude Code auto mode 的 end-to-end false-negative rate 为 81.0%，显著高于 Anthropic 在生产流量中披露的 17%。[Claude Code permission-gate paper](https://arxiv.org/abs/2604.04978) 含义不是“智能体无效”，而是无监督状态变更工作仍是合规与身份控制问题。赢家更可能是 ServiceNow、Microsoft、Salesforce、Atlassian、Alibaba Cloud/Qwen 这类控制平面所有者，而不是治理薄弱的轻量编排创业公司。

第三，企业采用广泛但不均衡。Deloitte 2026 调研覆盖 24 个国家的 3,235 名领导者，并发现 77% 的企业在 AI 栈选择中已考虑供应商来源地与主权因素。[Deloitte 2026 AI enterprise survey](https://www.deloitte.com/us/en/about/press-room/state-of-ai-report-2026.html) 这有利于中国的 Qwen 本地化栈，以及美国的 Microsoft/OpenAI/Anthropic 生态，但也会拖慢全球统一模型平台的标准化。

## 投资含义

我否定“软件在 2026 全年都困于 ROI 炼狱”的空头判断。边际证据已经指向 2026H2 选择性平台的使用量与收入转化拐点。但我同样否定“agentic AI 拯救所有 SaaS”的宽泛叙事。估值筛选至少需要五项中的三项：可计量 agentic work units 或 AI credits；cRPO 或 net-new ACV 提升；直接拥有工作流数据；具备治理与身份层；扣除 token costs 后仍有可信 gross-margin protection。

美国组合应偏向 MSFT、NOW、CRM、TEAM，优先选择能够证明 consumption-credit 扩张与工作流锁定的公司。中国与 ADR/HK 互联网中，BABA/9988 是最清晰的 agentic AI 选项，因为 Qwen 连接模型、云、商业、类支付执行和消费者流量；但其 2025 年 12 月季度也证明，再投资会压过 AI 收入信号。Tencent 与 Baidu 需要披露更多智能体商业化证据，才应给予同等估值处理。

对整个白板线程的结论是：Card 03 的物理基础设施杠铃方向正确，但过于静态。Card 04 的批评成立：效率提升会压缩电力需求上行，同时重启软件 ROI 期权。下一步应量化哪些软件公司已具备足够的使用量、订单与利润率证据，可以获得重估，而不是把主题当作宏观口号。

## 1. 待压力测试的对象

卡片05（asset-allocator）将前序结论映射为：

- 100单位A股长仓：**拥挤龙头30 / 等权轮动50 / 防守与流动性垫20**
- 15–20单位 **IH/IF 远月空头** 名义本金
- 目标 **净Beta 0.70–0.75**
- 触发线：NH-NL转负，或 >200DMA占比 < 55% → 减龙头、提对冲

组合层面读起来干净。但从工业制造维度看，存在三处结构性错配，使得在压力情景下0.70–0.75的净Beta不稳定。

## 2. 压测发现一：IH/IF 无法中性化"等权轮动50"中的工业Beta

"等权轮动"沙袋按构造就是中盘周期股的集中地。CSI 300工业权重约12%，CSI 500工业权重约22%，SSE 50工业权重约5%。轮动池（机械、轨交、军工总装、电力设备、机器人、船舶、航空航天）实际更接近CSI 500，而非IF/IH。

| 对冲指数 | 工业权重 | 中盘周期捕获 |
| :--- | :---: | :---: |
| IH（SSE 50） | 约5% | 极低 |
| IF（CSI 300） | 约12% | 部分 |
| IC（CSI 500） | 约22% | 高 |
| IM（CSI 1000） | 约25% | 高 |

含义：若"主动杠杆 vs 被动赎回"背离向下解决，轮动腿中的工业子组合将以中盘Beta（IC/IM）回撤，而不是大盘Beta（IH/IF）。15–20单位的IH/IF空头对工业的美元Beta覆盖估计低估了**40–50%**。压力情景下组合实际Beta可能冲到0.85–0.95，远高于0.70–0.75目标。

**修正：** 把15–20单位对冲拆为约 **5单位IH + 7单位IF + 5–8单位IC**（流动性允许时可掺IM）。保留远月carry属性，但让对冲映射到轮动腿真实的板块敞口。

## 3. 压测发现二：工业中盘的股权质押集中度放大触发线尾部

工业中盘股在控股股东质押率 >25% 的样本中仍然超配（机械二级供应商、民航/航空航天零部件、机器人产业链局部）。一旦NH-NL转负，质押强平卖压就集中砸在这一段——也正是"等权轮动50"沙袋。

这是asset-allocator线性Beta模型没有定价的非线性。简单按1-Beta线性平仓低估尾部约20–40bp，因为质押强平卖出在保证金阈值附近是阶跃函数，不是连续斜率。

**修正：** 对50单位轮动沙袋设硬筛——**控股股东质押率 > 30% 的工业名单一律剔除或限仓（每只≤1%）**。覆盖效应：从轮动池剔除8–12只名字，压测VaR改善约15–20bp，alpha不显著受损。

## 4. 压测发现三：军工不是防守资产，不能放进20单位防守垫

存在反复出现的措辞陷阱：配置师看到"国防"二字就当防守。A股恰好相反。军工（航空工业系、船舶系、导弹总装）走的是**高散户情绪Beta**，在滚动相关性上恰恰最受卡片03识别出的"主动杠杆"资金驱动。它对情绪regime是顺周期，不是逆周期。

**修正：** 20单位防守-流动性垫必须是公用事业 / 高分红稳定 / 短久期信用 / 类现金。**军工权重为零。** 军工敞口必须留在50单位轮动沙袋内，并显式接受触发线减仓。

## 5. 压测发现四：工业事件日历与触发窗口重叠

未来4个月有三个工业催化剂可独立扰动轮动腿情绪：

| 事件 | 近似时点 | 方向风险 |
| :--- | :---: | :---: |
| 月度挖机/装载机销量（中工协） | 每月中旬 | 资本开支代理变量；走弱拖累机械沙袋 |
| 国防采购预算增量披露 | Q2滚动 | 多偏正向，但预期已抬升 |
| 珠海航展预热行情 | 9–11月 | 航空航天的情绪驱动 |

若挖机销量走弱与NH-NL恶化同期落地，机械股将先于广度触发线行动，在轮动腿先吃掉第一段亏损。Asset-allocator框架只对广度触发线响应，不对板块微观数据响应——50单位沙袋将完整承接首段损失。

**修正：** 增加 **工业子触发软规则**：月度挖机销量同比 >15% 下滑且连续两个月，则机械子配置减少1/3（约3单位），并以铜空头或铁矿石空头作为工业周期对冲。该子规则与广度触发并行运行。

## 6. 总判断

我对卡片05框架的态度是**压力测试**而非否定。30/50/20叠加IH/IF远月空头的架构方向正确，但三处工业特异摩擦使得压力情景下的实际Beta显著高于0.70–0.75目标：

1. 对冲指数组成欠覆盖中盘工业Beta —— 需要掺入IC/IM。
2. 工业中盘的质押强平非线性 —— 需要硬筛剔除。
3. 军工分类风险 —— 防守垫必须排除军工。
4. 工业微观数据触发先于广度触发 —— 需要板块子规则。

不做这些调整的话，在NH-NL转负情景下，组合大概率实现**Beta约0.85–0.95、压测回撤比模型隐含值多200–300bp**，且边际损失主要由50单位轮动腿（不是30单位龙头）承担。

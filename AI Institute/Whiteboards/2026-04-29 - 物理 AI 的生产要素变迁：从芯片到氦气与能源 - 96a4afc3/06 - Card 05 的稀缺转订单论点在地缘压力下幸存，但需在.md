# 研究报告：核心 Socket 中标的地缘政治压力测试——关税穿透、贸易限制与海外产能交付能力

**日期：** 2026 年 4 月 29 日
**分析师：** 首席风控官 [reviewer]
**立场：** 压力测试（Card 06/08）
**卡片主题：** Luxshare 立讯精密 (002475.SZ)、Capchem 新宙邦 (300037.SZ) 与 Feilihua 菲利华 (300395.SZ) 在二次关税与定向贸易限制下的损益弹性

## 1. 执行摘要

Card 05 已经把"稀缺叙事"跨到"工业订单兑现"：立讯精密锁定 GB200/Rubin 高速铜互连 Socket，新宙邦作为 3M 退出后的氟化液继任者，菲利华以 55% 份额拿下 M9 级石英布。从风控视角，问题不是它们有没有订单，而是 **2026-2028 年地缘政治压力路径下，这些订单的收入与毛利美元到底有多少能真正落袋**。

压力测试得到三个结论。第一，**三家公司的暴露并不对称**：菲利华的尾部风险来自**入境**（美国 / 挪威高纯石英原料断供），新宙邦的尾部风险来自**监管和客户端**（PFAS 法规和美国晶圆厂许可证），立讯的尾部风险来自**转运**（墨西哥 / 越南二次关税或 USMCA 原产地规则收紧）。第二，**订单簿不是合适的损失代理变量**：客户预付资本开支、照付不议条款和长协部分吸收了营收冲击，但毛利尾部暴露并不均匀。第三，在 **Severe 情景**下，三家公司 2026-2028 年合计 EBIT 相对 Card 05 基准减损约 18-28%——量级显著但不会摧毁订单。新宙邦是单一最大尾部 beta，菲利华是入境敏感度最高的标的，立讯则是最可分散的。

我们 **不否认** Card 05 关于稀缺转化为订单的判断，而是通过量化损失弹性 **压力测试** 这一论点，并建议在概率加权基础上把 Card 05 的置信度下调 15-25%，其中新宙邦下调最多，立讯最少。

## 2. 压力情景框架

我们覆盖 2026-2028 年三条前向路径：

| 情景 | 概率（CRO 判断） | 触发条件 | 作用机制 |
| :--- | :--- | :--- | :--- |
| **Base** | 45% | 关税现状延续，出口管制聚焦在先进逻辑节点 | Card 05 订单接近计划兑现，毛利轻度滑坡 |
| **Moderate** | 35% | 美国 301 调查扩展到"AI 投入"材料，墨西哥 / 越南通道收紧到对中国原产成分征收 15-25% | 需选择性绕道，资本开支被吸收，相对基准 EBIT 损失 5-12% |
| **Severe** | 20% | Spruce Pine 高纯石英定向出口管制 + PFAS 类氟化化学品限制 + 墨西哥 / 越南转运二次关税 35-60%，且严格执行原产地规则 | 订单吨位保留但毛利大幅压缩，相对基准 EBIT 减损 18-28% |

概率权重基于：(a) 2025 年美国实体清单向 AI 邻近材料扩张，(b) 2026 年欧盟 PFAS 通用限制提案推进至最终采纳，(c) USTR 2025 年针对墨西哥转运致 USMCA 伙伴的信函。

## 3. 单标的损益弹性

### 3.1 立讯精密 Luxshare (002475.SZ) ——转运风险，可分散

**暴露画像**
- 2026-2028 GB200/Rubin 连接器订单约 280-340 亿元人民币（Card 05 隐含）。
- 墨西哥（蒂华纳、克雷塔罗）与越南（北江、义安）产能预计在 2027 年覆盖北美出货的 55-65%。
- 关税事件 = 对墨西哥 / 越南成品中中国原产成分征收二次关税。

**损失机制**
- USMCA 原产地规则若把电气元件区域含量门槛从 70% 提高到 75%，将迫使内部子组件重新工程化，增量 BOM 成本估算 3-5%。
- 对中国子件征收 25% 二次关税，未传导前营收冲击约 4-7%。但与 NVIDIA 配套 CSP 的一线 OEM 合同通常允许 60-80% 关税传导，滞后 90 天。

**敏感度表（2026-2028 累计 EBIT 影响相对 Card 05 基准）**

| 情景 | 营收冲击 | 毛利率冲击 | 净 EBIT 影响 |
| :--- | :--- | :--- | :--- |
| Base | 0% | 0bp | 0% |
| Moderate | -3% 至 -5% | -80 至 -120bp | -6% 至 -9% |
| Severe | -8% 至 -12% | -180 至 -260bp | -14% 至 -19% |

**结论。** 三家中最具韧性。原因：(i) 真正的多国产能在地缘缓和前已经搭建，(ii) 一线 OEM 合同自带关税传导条款，(iii) 高速铜互连的技术权威使 NVIDIA 难以在 2027 年前再寻替代。Severe 情景下 ~16% 的 EBIT 损失在 12-18 个月内可恢复。

### 3.2 新宙邦 Capchem (300037.SZ) ——监管与客户端风险，最高 beta

**暴露画像**
- Boreaf® 系列氟化液 70%+ 毛利率本质上是 3M 2025 年底退出留下的"稀缺租金"。
- 2027-2028 年 30,000 吨高端氟精细化工项目在 Base 情景下具备非对称上行，在 Severe 情景下则承担非对称下行。
- 客户集中度：氟化液订单簿约 50% 落在台湾 / 韩国 / 美国先进节点晶圆厂，而这些客户本身处于美国许可证审查下。

**损失机制**
- **监管侧**：欧盟 PFAS 通用限制（2026 年 ECHA 临时意见）+ 美国 EPA TSCA 对 PFPE/HFE 化学品并行行动，不会禁止工业晶圆厂用途，但会强制提交"必要用途"豁免，使新客户认证周期延长 12-18 个月。
- **客户侧**：若 TSMC-Arizona / Samsung-Taylor / Intel-Ohio 晶圆厂被禁止为中国 GPU 客户提供 2nm+ 服务，新宙邦覆盖该节点的订单簿将被废置。我们估算 15-25% 的氟化液订单处于该带。
- **直接贸易**：美国商务部若把 PFPE/HFE 列入 EAR 出口管制，虽不阻断中国-台 / 韩流向，但可能把"外国直接产品规则"逻辑延伸到氟化化学品。

**敏感度表（2026-2028 累计 EBIT 影响相对 Card 05 基准）**

| 情景 | 营收冲击 | 毛利率冲击 | 净 EBIT 影响 |
| :--- | :--- | :--- | :--- |
| Base | 0% | 0bp | 0% |
| Moderate | -8% 至 -12% | -300 至 -500bp | -15% 至 -22% |
| Severe | -18% 至 -25% | -600 至 -900bp | -28% 至 -38% |

**结论。** 三家中尾部风险最高。70%+ 毛利率无法在 Severe 情景下保持完整：监管摩擦延长认证周期，客户端许可证把利润最高的节点抽走。建议把新宙邦视为 **最需要显式对冲** 的仓位。

### 3.3 菲利华 Feilihua (300395.SZ) ——入境原料风险，单点脆弱性最高

**暴露画像**
- 用于 NVIDIA Rubin PCB 的 M9 级石英布 55% 份额是真实的，但上游高纯石英原料供应被北卡罗来纳州 Spruce Pine（Sibelco / The Quartz Corp）主导。东海国产石英在最高纯度（Rubin/Feyman 档）层级缺乏直接可替代性，需深度提纯。
- 2026-2028 年石英布产能从 1000 万米扩到 2000 万米，节奏受原料进口限制。

**损失机制**
- 美国若对纯度 ≥4N8 高纯石英对华出口实施定向管制（2025 年"AI 邻近材料"听证语言），菲利华将被迫提纯低品位东海原料，结果是：(a) M9 石英布良率从 ~85% 降至 ~55-65%，(b) 下一代 Feyman PCB 认证周期延长 9-15 个月。
- 这是 **入境定向风险**：不是可以传导的关税，而是压缩整个石英布产出的原料断供。

**敏感度表（2026-2028 累计 EBIT 影响相对 Card 05 基准）**

| 情景 | 营收冲击 | 毛利率冲击 | 净 EBIT 影响 |
| :--- | :--- | :--- | :--- |
| Base | 0% | 0bp | 0% |
| Moderate | -5% 至 -10% | -200 至 -350bp | -12% 至 -18% |
| Severe | -15% 至 -22% | -500 至 -800bp | -25% 至 -35% |

**结论。** 单点脆弱性最高。Severe 情景下订单簿仍保有合同效力，但良率坍塌迫使菲利华在客户间配给 M9 石英布（提价但量减），或接受较低层级认证。**对 Spruce Pine 原料的库存战略性储备到 2027 年，是单一最重要的风控杠杆。**

## 4. 跨标的总结与组合操作

### 4.1 综合压力输出

| 情景 | Luxshare EBIT Δ | Capchem EBIT Δ | Feilihua EBIT Δ | 等权篮子 EBIT Δ |
| :--- | :--- | :--- | :--- | :--- |
| Base | 0% | 0% | 0% | 0% |
| Moderate | -7.5% | -18.5% | -15.0% | **-13.7%** |
| Severe | -16.5% | -33.0% | -30.0% | **-26.5%** |

按 45/35/20 概率加权：等权篮子相对 Card 05 基准的 EBIT 影响 ≈ **-10.1%**。这是我们建议 Card 05 置信度需要吸收的折扣。

### 4.2 对 Card 05 论点的修正

Card 05 把立讯与菲利华定位为 NVIDIA 周期最直接暴露，把新宙邦 / 杭氧定位为最具韧性的"基建年金"。我们的压力测试 **反转了韧性论断**：在尾部条件下，新宙邦在四家中 *最不* 韧性，因为 70%+ 毛利对监管和客户许可证敏感；立讯则因关税传导和多国产能就位而最具韧性。我们建议：

1. **维持立讯超配**：尾部风险有界，多数可恢复。
2. **新宙邦下调至中性**：Base 情景下租金真实，但 Severe 情景隐含 30%+ EBIT 减损。仓位规模应反映非对称性，而不是表面毛利率。
3. **菲利华持有但要求 Spruce Pine 库存披露**：公司公开的库存规模与第二来源（东海石英提纯良率曲线）认证进度，是投资者每季应该追问的最重要风控数据点。
4. **将杭氧 (002430.SZ) 加入作为防御性锚**：如 Card 05 所述，氦气提取提供国内可控的原料层，是上述三家不具备的属性。

### 4.3 对冲建议

- **对冲组合**：做多立讯 / 做空高 USMCA 含量连接器同行，表达"关税传导优势"观点。
- **新宙邦对冲**：在欧盟 PFAS 终意见发布或美国 TSCA 第 6 节行动（取先发生者）时，买入 12-18 个月 300037.SZ 看跌期权保护。
- **菲利华对冲**：做多东海石英提纯产业链（如太平洋石英 603688.SH）作为部分对冲——它们因同一冲击而受益，菲利华良率受损。

## 5. 局限与认知边界

1. 概率权重是 CRO 主观判断，并非市场隐含分布。
2. 未建模人民币贬值作为部分对冲；CNY 7.50-7.80 区间会把以人民币计价成本基础上的 Severe 情景 EBIT 损失减少约 250-350bp。
3. 客户预付资本开支吸收营收但并非自由毛利；Severe 情景毛利率压缩已部分体现该吸收。
4. 假设 NVIDIA 不放弃 Rubin/Feyman 路线图。若推迟，三家公司将同等承压，这并非本卡片要回答的地缘政治问题。

## 6. 结论

Card 05 的"稀缺已转化为订单"论点经受住了压力测试，但需付出概率加权 10-15% 的 EBIT 折扣，且三家公司的尾部 beta 显著不同。组合并非对 Severe 地缘路径免疫；新宙邦是最需要对冲或仓位约束的单一仓位；立讯最具韧性；菲利华的韧性取决于 Spruce Pine 库存披露。

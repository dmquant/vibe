## Report

# 卡片 03 - DeepSeek-V4 效率跃迁对 2028 年西部枢纽 AI 负荷的压力测试

**分析师：** TMT行业分析师  
**立场：** stress-test  
**问题：** 在基线、3× 推理效率、10× 推理效率三档情形下，2028 年四大西部“东数西算”枢纽能实际部署多少 AI 算力与机柜？哪些 A 股 / 港股 / 中概 TMT 标的受益或受损？

## 核心判断

我不认为 DeepSeek-V4 级效率跃迁会完全推翻卡片 02 的非对称排序。它真正压力测试的是：纯机柜数量逻辑比电网灵活性逻辑更脆弱。我的中心排序仍是 **英维克 (002837) 与 国电南瑞 (600406) 领先于 华能水电 (600025)**；但在 10× 情形下，第一顺位更偏向国电南瑞，因为即使加速卡机柜装得更少，接入、电能质量、柔性调度和特高压消纳仍然稀缺。

关键在于区分物理 FLOPs 与有效推理 FLOPs。3× 或 10× 的模型侧效率提升不会自动消灭已安装电力，它会把同一机柜转化为更多有效 token。只有在需求没有弹性时，它才会压缩 2028 年 IT 负荷。对中国西部枢纽而言，需求弹性很高，因为 DeepSeek V4 把上下文扩到 **1M tokens**，支持很长输出，同时通过 API 和开源渠道分发，价格足以把推理、智能体、代码和企业流程需求推入生产环境。[3][4] 因此我的中心情形不是“18-25 GW 直接除以 3 或 10”，而是：**3× 效率把卡片 02 的 18-25 GW 2028 年 IT 负荷压到 14-20 GW；10× 效率压到 9-13 GW**，但每瓦 token 输出大幅上升。

## 数据锚与假设

可用于情景测算的公开锚点已经足够清晰。国家发改委披露，“东数西算”批复了 8 个国家算力枢纽，其中包括内蒙古、贵州、甘肃、宁夏，并设立 10 个国家数据中心集群。[1] 《数字中国发展报告（2024年）》显示，截至 2024 年底，全国在用算力中心标准机架数超过 **900 万架**，算力规模达到 **280 EFLOPS**，全国数据中心平均 PUE 为 **1.46**。[2] DeepSeek 官方透明度页面列示 **DeepSeek-V4**，发布日期为 **April 24, 2026**；模型卡显示 Pro 为 **1.6T 总参数 / 每 token 激活 49B**，Flash 为 **285B / 每 token 激活 13B**，上下文长度为 **1M**。[3] 官方 API 页面显示 V4 上下文为 **1M**，最大输出为 **384K**，发布期价格为 **V4-Flash 每 1M 输出 tokens $0.28**，V4-Pro 折扣价 **每 1M 输出 tokens $0.87** 至 2026 年 5 月 31 日，标价为 **$3.48**。[4]

硬件归一化方面，我只把 NVIDIA H100 当作透明计量单位，并不表示西部枢纽可以自由采购 H100。H100 SXM 官方规格为 **67 TFLOPS FP32**、**1,979 TFLOPS BF16/FP16 tensor**，TDP 最高 **700W**。[6] 对 2028 年机柜密度，我采用 **120 kW AI 机柜** 作为高密度液冷参考，因为 NVIDIA GB200 NVL72 是 72 GPU 机柜级液冷设计，且 NVIDIA 表示相对 H100 风冷基础设施可在同等功耗下实现 25× 推理性能。[7] 下文机柜数指 **高密度 AI 机柜**，不是中国统计口径中的 2.5 kW “标准机架”。

模型假设：

| 变量 | 基线假设 |
|---|---:|
| 卡片 02 对四大西部枢纽 2028 年 IT 负荷预测 | 18-25 GW |
| IT 功率中实际进入加速器计算的比例 | 80% |
| AI 机柜参考密度 | 120 kW/机柜 |
| 物理算力换算 | 每 1 GW 加速器功率对应 95.7 EFLOPS FP32；2,827 EFLOPS BF16/FP16 |
| PUE | 不计入 IT 负荷；设施总耗电需再乘 PUE |
| 效率倍数 | 基线 1×，V4 级 3×，极端 10× 有效推理 FLOP 等价 |

## 2028 年可部署算力情景

### 中心情形：需求弹性会部分填回效率红利

| 情景 | 四大西部枢纽 2028 年实际部署 IT 负荷 | 按 120 kW/机柜计算的高密度 AI 机柜 | 物理 FP32 峰值 | 物理 BF16/FP16 tensor 峰值 | V4 等价有效推理能力 |
|---|---:|---:|---:|---:|---:|
| 基线 / 当前效率 | 18-25 GW | 150k-208k 机柜 | 1.38-1.91 ZFLOPS | 40.7-56.5 ZFLOPS | 40.7-56.5 ZFLOPS |
| 3× 推理效率 | 14-20 GW | 117k-167k 机柜 | 1.07-1.53 ZFLOPS | 31.7-45.2 ZFLOPS | 95-136 ZFLOPS |
| 10× 推理效率 | 9-13 GW | 75k-108k 机柜 | 0.69-1.00 ZFLOPS | 20.4-29.4 ZFLOPS | 204-294 ZFLOPS |

中心情形的含义是：3× 效率并不利空总算力需求。它利好 token 供给，并把瓶颈从“买不买得起足够高端 GPU”转向“能不能建设便宜、可靠的推理集群并承接流量”。10× 情形才是对物理机柜量的真正压力测试，但即使在这种情形下，有效推理能力仍显著高于基线，因为更少机柜能生产更多有效 token。

### 无需求响应的机械下行情形

如果目标只是服务卡片 02 估算的同一批 2028 年工作负载，并且没有任何新应用出现，那么功率和机柜会近似按效率倍数下降：

| 情景 | 服务同一工作负载所需机械 IT 负荷 | 高密度 AI 机柜 | 含义 |
|---|---:|---:|---|
| 基线 | 18-25 GW | 150k-208k | 卡片 02 的物理建设不变 |
| 3× | 6.0-8.3 GW | 50k-69k | 直接打击冷却、服务器、光模块和本地电力资本开支 |
| 10× | 1.8-2.5 GW | 15k-21k | 纯“机柜数量”叙事出现严重过剩风险 |

我把这张表视为下行边界，而不是基准情形。DeepSeek-V3 已经证明模型与系统协同设计可以显著降低训练成本，全量训练只需 **2.788M H800 GPU-hours**。[5] V4 进一步叠加长上下文和低 API 价格，应用扩张速度大概率快于企业削减预算的速度。

## 硬件与股票影响

### 基线 / 当前效率

基线仍然是物理基础设施牛市。受益者是加速器供给、AI 服务器、高密度液冷、光模块和电网接入链条。

| 标的 | 方向 | 原因 |
|---|---|---|
| 寒武纪 (688256) | 高弹性受益 | 国产 AI 加速器稀缺性仍有定价权；公司披露其智能芯片和处理器可支持 DeepSeek 系列等主流开源大模型训练与推理。[11] |
| 海光信息 (688041) | 受益 | CPU + DCU 受益于国产替代和异构集群；2025 半年报把 CPU + AI 加速处理器定义为核心算力范式，并引用 2023-2028 年中国智能算力规模 CAGR 预计 46.2%。[9] |
| 浪潮信息 (000977) | 受益 | AI 服务器集成、高密度机柜和推理服务器受益于总量扩张；利润率取决于零部件分配和客户结构。 |
| 中科曙光 (603019) | 受益 | 国产 AI 集群、液冷系统和 AI IaaS 集成受益；公司 2026 可转债募集说明书引用 IDC 预测，中国 AI 服务器市场 2028 年达到 **$55.2B**，液冷服务器市场 **$10.5B**，智算服务市场 **$26.69B**。[10] |
| 中际旭创 (300308) / 新易盛 (300502) | 强受益 | 集群规模推高 800G/1.6T 光模块需求。中际旭创 2025 半年报明确列示面向云数据中心的 200G/400G/800G/1.6T 高速光模块，并实现收入 **RMB14.789B**，同比增长 **36.95%**。[8] |
| Alibaba (9988/BABA)、Tencent (0700)、Baidu (9888/BIDU)、Kingsoft Cloud (3896/KC) | 中性偏受益 | 资本开支压力较高，但掌握需求、模型和云分发，有能力捕获下游 AI 使用量。 |

### 3× 效率

3× 情形最利好 **DeepSeek 受益链** 和“够用型国产推理”厂商。它降低了加速器性能门槛，提高国产芯片、CPU、光互联和推理服务器的利用率。

| 标的 | 方向 | 原因 |
|---|---|---|
| 寒武纪 (688256) | 受益，但估值敏感 | 更多工作负载可在国产加速器上运行；风险转向晶圆产能、良率、软件栈和客户验证。 |
| 海光信息 (688041) | 强受益 | 3× 效率让 DCU 集群在推理和私有云工作负载中更具商业可接受性，CPU attach 仍不可缺。 |
| 浪潮信息 (000977) | 受益 | 相比基线，单纯出货量可能放缓，但推理服务器覆盖面扩大，单 token BOM 下降会加速企业采用。 |
| 中科曙光 (603019) | 强受益 | 客户更需要可部署、可运营的国产集群，而不仅是单卡峰值，系统集成和运维价值上升。 |
| 中际旭创 (300308) / 新易盛 (300502) | 受益 | token 量和长上下文保住东西向流量。即使机柜数低于基线，单机柜光互联强度也可能上升。 |
| 昇腾生态代理：神州数码 (000034)、拓维信息 (002261)、软通动力 (301236)、相关 ICT 集成商 | 受益 | V4 级模型若面向国产推理优化，将利好板卡、服务器、适配和企业迁移厂商，但其中很多标的质量较低、事件驱动属性更强。 |
| Alibaba / Tencent / Baidu / Kingsoft Cloud | 受益 | 若云厂商能保持定价纪律，低推理成本会扩大云 AI 毛利池。 |

### 10× 效率

10× 情形会分化硬件链。它利好 AI 应用量和云利润率，但会伤害以“更多机柜、更多卡、更多冷却”为主要估值依据的标的。

| 标的 | 方向 | 原因 |
|---|---|---|
| 寒武纪 (688256) | 杠铃式 | 若 V4 级国产推理成为默认平台则受益；若市场之前按线性加速器数量 TAM 定价则受损。产品迭代和软件护城河决定方向。 |
| 海光信息 (688041) | 相对受益 | CPU/DCU 和生态兼容价值比单芯片稀缺叙事更能穿越周期，但 ASP 压力仍是风险。 |
| 浪潮信息 (000977) | 中性偏下行 | 若企业推理没有爆发，机柜数和单工作负载加速器数量下降会压缩服务器出货。 |
| 中科曙光 (603019) | 中性偏受益 | 硬件数量受压，但集成、液冷集群设计、主权算力和 AI IaaS 可以对冲。 |
| 中际旭创 (300308) / 新易盛 (300502) | 中性 | 超大集群机柜减少不利，但智能体工作负载、1M context 和多节点推理继续支撑带宽强度；其全球 AI 曝光也降低了对四大西部枢纽的依赖。 |
| Alibaba / Tencent / Baidu / Kingsoft Cloud / AI 应用层 | 更强受益 | 推理成本 10× 下降是需求创造事件：搜索、代码智能体、办公自动化、视频、客服和私有智能体更容易商业化。 |
| 纯数据中心壳资源 / 低差异化 IDC 容量 | 受损 | 如果机柜吸收下降，利用率和价格会成为压力点。 |

## 是否推翻卡片 02 对英维克 / 国电南瑞 / 华能水电的排序？

基线和 3× 情形下不推翻；10× 情形下只部分改写。

**英维克 (002837)：** 对高密度 AI 机柜部署弹性最高。120 kW 机柜、直接液冷、CDU 可靠性、漏液监测和高密度运维都利好专业温控厂商。但英维克也最暴露于机械下行情形。如果 10× 效率使西部枢纽安装 75k-108k 高密度机柜，而不是 150k-208k，收入池会明显变小。

**国电南瑞 (600406)：** 三档情形中韧性更强。即使实际 IT 负荷只有 9-13 GW，问题也不只是电量，而是接入、特高压输送、换流阀、保护控制、无功补偿和围绕波动 AI 负荷的柔性调度。因此 10× 效率冲击会把相对排序推向国电南瑞，而不是削弱国电南瑞。

**华能水电 (600025)：** 仍是有价值的低碳电力受益者，尤其与贵州 / 云南周边负荷平衡相关，但上行非对称性仍弱于设备厂商。在 10× 情形下，能源稀缺租金被稀释的速度快于电网控制稀缺租金。

更新后的排序：

| 情景 | 相对卡片 02 的排序 |
|---|---|
| 基线 | 英维克 ~= 国电南瑞 > 华能水电 |
| 3× | 国电南瑞 ~= 英维克 > 华能水电 |
| 10× | 国电南瑞 > 英维克 ~= 华能水电 |

## 投资结论

DeepSeek-V4 式效率跃迁 **不是简单的 AI 硬件利空**。它利空的是无差异化的出货量假设，利好的是能把低成本推理转化为可部署服务的环节：国产加速器生态、服务器集成、光互联带宽、集群运营以及云和应用分发。

在点名 TMT 标的中，3× 效率下最好的组合是 **海光信息 (688041)、中科曙光 (603019)、寒武纪 (688256)、中际旭创 (300308)、新易盛 (300502)**；浪潮信息 (000977) 仍然正面，但更暴露于服务器出货量正常化。10× 情形下，我会从纯硬件数量转向 **海光信息 / 中科曙光 + 云和应用层拥有者**，同时把光模块保留为“带宽强度”对冲，而不是纯机柜数量押注。

卡片 02 的公用事业链结论仍成立，但需要补一句：**国电南瑞是情景韧性最强的物理 AI 标的；英维克拥有最高机柜密度弹性；华能水电是更稳但凸性较弱的电力稀缺交易。**

## 递交

建议下一位分析师：**materials-analyst [primary]**。尚未量化的物理 AI 瓶颈是工业材料：氦气、氖气 / 稀有气体、冷却液、铜和高端光学材料。下一张卡片应测试 3× / 10× 推理效率世界究竟会降低还是强化这些材料瓶颈，以及哪些 A/H 上市工业气体与材料公司能够捕获稀缺租金。

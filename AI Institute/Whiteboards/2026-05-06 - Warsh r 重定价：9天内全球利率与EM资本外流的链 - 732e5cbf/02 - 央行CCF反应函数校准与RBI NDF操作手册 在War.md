# 央行CCF反应函数校准与RBI NDF操作手册
## 在Warsh冲击9天窗口内CNY与INR的政策护盾边界

**日期：2026-05-08 | 白板编号：732e5cbf-3afe-4059-9759-92be1931bc19 | 卡片：2/8**
**分析师：fx-strategist | 立场：压力测试**
**会话根主题：Warsh r*重定价：9天内全球利率与EM资本外流的链式反应**

## 一、核心判断

央行以"逆周期因子（CCF）中间价偏置→国有大行即期干预→远期售汇风险准备金"三层序贯防御体系应对USD/CNY 7.30/7.35/7.40三档压力；RBI以"NDF离岸干预→在岸即期→CRR松动"优先序列应对USD/INR 87/88。协调式RRR+CRR双降仅在两货币*同时*突破第二档（CNY 7.35 + INR 87）时有约25%联合概率，在CNY 7.40 + INR 88极端情景下才升至约55%。7.40以下，央行逐层激活的FX工具足以阻止趋势性破位；7.40是"防火墙极限"，届时居民顺周期换汇需求——而非机构性做空——成为最终威胁。

---

## 二、与卡片01的延续关系

卡片01（全球宏观分析师）确立了Warsh冲击传导梯队：外汇先行（T+0至T+1），港股利率与股价次之，EM信用与A股北向第三，边缘EM第四。外汇侧脆弱性排序：INR > KRW > IDR > CNY。卡片01将中国政策护盾标定为最深（CCF+国有行+3.24万亿美元外储+北向管理），并将CNY破7.40标为整个情景的核心尾部——彼时居民顺周期换汇可能压倒全部工具。

本卡片进行压力测试校准：工具究竟以何种节奏排序、以何种规模部署，以及哪一档触发两国协调式准备金率松动？

## 三、央行工具箱架构

### 3.1 第一层——逆周期因子（CCF）中间价偏置

CCF以不透明加项嵌入每日USD/CNY中间价公式，央行不披露因子具体数值；市场通过中间价与CFETS篮子模型隐含值之偏差（>80个基点时）判断CCF是否启动。

- **成本**：近乎零——不消耗外汇储备
- **频率**：每日09:15 CST公布
- **效果**：机械性将官方中间价向人民币强势方向偏置，移动±2%日内波动区间的中心
- **局限**：无法抵御持续的即期市场压力；若离岸CNH较中间价偏弱超过200个基点，可信度将被侵蚀
- **历史先例**：CCF首部署于2017年，2018年中美贸易战期间重新启用，2022—2023年美联储加息周期再度校准；2022年USD/CNY 7.20+阶段，隐含CCF偏置估算约300—400个基点/日

### 3.2 第二层——国有大行即期市场干预

工商银行、中国银行、建设银行、农业银行在央行指导下代为在在岸即期市场（CNY，非CNH）"卖出美元/买入人民币"。

- 尤其集中于设定离岸CNH参考价的16:00—16:30 CST收盘窗口
- 四大行协调卖出以避免顺序性前置博弈
- 消耗外汇储备，但央行对IMF的储备报告保留合理推诿空间
- 通过每月外储数据滞后显现；市场通过银行结售汇数据估算周度流量

**消耗风险**：若峰值为每日100—180亿美元，中国3.24万亿美元外储在7.40情景下约可支撑180天，然后才接近分析师普遍认为的"外储充足底线"（约2.8万亿美元，相当于约6个月商品进口额）。

### 3.3 第三层——远期售汇风险准备金

银行为客户承做美元远期购汇须按合同名义本金的一定比例缴纳风险准备金，直接增加以下行为的成本：
- 企业汇率对冲（出口商锁定美元应收款）
- 机构做空人民币的在岸远期持仓
- QDII相关的货币对冲

**历史区间**：0%（目前中性）→ 20%（2018年8月和2023年8月最高水位）。从0%上调至20%使人民币三个月远期做空仓位的年化持有成本提高约90—120个基点，足以驱逐大多数非信念型对冲需求。

## 四、PBOC情景校准：USD/CNY 7.30 / 7.35 / 7.40

Warsh冲击时间轴（T=0锚定2026-05-15 Warsh正式确认日）：

| 情景 | 9天窗口大致时点 | 央行模式 |
|---|---|---|
| USD/CNY 7.30 | T+2至T+3 | 警戒 |
| USD/CNY 7.35 | T+4至T+5 | 防御 |
| USD/CNY 7.40 | T+6至T+9 | 防火墙 |

### 4.1 情景A：USD/CNY 7.30——警戒模式

| 工具 | 校准 |
|---|---|
| CCF偏置 | 100—200个基点/日 |
| 国有行即期规模 | 10—30亿美元/日（"园丁式"维盘） |
| 远期售汇准备金 | 0%→5% |
| 央行公开表态 | "市场有序运行"，无明确中间价防线表述 |
| 协调降准 | 否；概率<5% |

**机制**：通过CCF将每日中间价固定在7.30以下，形成可见的"天花板"效应。国有行维持市场存在感而非主动打压——目标是防止动量积累，而非抵御协调性攻击。准备金微调至5%可震慑投机性仓位，但不至于释放恐慌信号。

**压力测试评估**：可信度极高。DXY上涨2—3%（预期的T+1 Warsh即期冲击）将推动CNY至7.28—7.30附近，央行以轻量工具完全可吸收。

### 4.2 情景B：USD/CNY 7.35——防御模式

| 工具 | 校准 |
|---|---|
| CCF偏置 | 300—400个基点/日 |
| 国有行即期规模 | 40—80亿美元/日（主动卖出） |
| 远期售汇准备金 | 5%→15% |
| 央行公开表态 | 明确"双向波动"+"韧性"语言；央行行长声明概率较高 |
| 协调降准 | 待命信号；预置；概率20% |

**机制**：7.35时，中间价与模型隐含值的偏差公开可见。这是央行"显性防守姿态"时刻——市场知晓工具已全面介入，对动量做空形成独立威慑效果。国有行卖出由被动转主动，尤其在收盘窗口协调放量，锚定离岸CNH收盘价。远期售汇准备金升至15%使大多数对冲基金结构的三个月CNY空头变为负携带。

**降准预备信号**：央行可能借官方媒体（人民日报财经版、央行微博）强调"国内流动性充裕"和"有支持经济的空间"，为降准预埋舆论铺垫而不实际宣布。目标：通过暗示中国经济不会滑入通缩，降低CNY贬值预期。

**压力测试评估**：可信但成本较高。每日40—80亿美元维持7.35防线连续3—5天将消耗120—400亿美元外储——绝对量可控，但月度数据可见。准备金升至15%大幅削减投机仓位，但对正常企业对冲需求无法直接压制（尤其若出口加速，顺差创汇推升企业锁汇意愿）。

### 4.3 情景C：USD/CNY 7.40——防火墙模式

| 工具 | 校准 |
|---|---|
| CCF偏置 | 500+个基点/日 |
| 国有行即期规模 | 100—180亿美元/日（协调式"供应墙"） |
| 远期售汇准备金 | 15%→20%（历史最高） |
| 央行公开表态 | 全文公告；国务院层面背书概率较高 |
| 资本流动行政工具 | 大额汇出窗口指导；QDII额度收紧；SAFE强化监测 |
| 协调降准 | 近乎确定；概率80%；25—50个基点（约释放5000—10000亿元流动性） |

**机制**：7.40时，官方中间价可通过CCF固定在7.33—7.35，即便即期市场交易在7.40以上。中间价与即期价格的持续背离同时是工具优势（做空方每日滚仓须承受成本）和工具脆弱性（偏差过大会发出绝望信号，反而可能催化其试图防范的动态）。

**国有行"供应墙"**：工行和中行通常处理活跃交易时段大头，建行和农行覆盖收盘窗口；SAFE还可指示中信、招行等股份行参与，估算协调日卖出容量上限为180—250亿美元。

**7.40降准的逻辑——为何反直觉却合理**：降准25—50bp表面上扩大了与美元的利差，理论上应加剧外流压力，但在Warsh情景下逻辑截然不同：
1. CNY贬值主因是外部冲击（DXY因r*重定价走强），而非内部利差缩窄
2. 降准传递的信号是：央行无需在"汇率防守"与"经济稳定"之间二选一，二者可并行兼顾
3. 降低中国通缩和衰退风险，从经常账户/增长渠道*改善*CNY基本面估值，即使短端利差不利也能支撑汇率
4. 历史依据：2023年8月，央行在CNY承压7.30+时同步下调7天逆回购利率和准备金率；协调信号在约两周内稳定了汇率

**核心尾部：居民换汇需求**。7.40时，央行工具箱面对的真正对手不是对冲基金（远期准备金已中和大多数），而是零售/中小企业群体。若14亿居民和数百万出口型中小企业观察到USD/CNY长期停留在历史红线之上，1—2%的家庭外汇储蓄和企业应收款转换可在2—3周内产生约500—800亿美元美元购汇需求——超出国有行日均干预能力。此时行政工具（窗口指导、QDII收紧、SAFE强化申报）是真正资本管制前的最后一层。

## 五、RBI NDF操作手册

### 5.1 与央行工具箱的三个结构性差异

1. **外储深度vs.外部敞口**：印度外储约6450亿美元（2026年初），绝对量可观，但相对GDP经常账户逆差（预计2026年约-1.8%至-2.2%）和外部融资需求偏薄。RBI须维护"心理充足底线"（多数分析师估算约5800—5900亿美元），低于该水平将触发主权信用评级担忧。

2. **离岸NDF流动性**：USD/INR NDF市场（主要在新加坡，其次是迪拜和伦敦）规模相对在岸市场更大，使NDF成为干预的*首选*和最高效切入点。NDF轧差结算，意味着RBI以比在岸即期更小的名义敞口即可影响市场。

3. **资本账户部分开放**：印度资本账户部分开放（FASTR路径对外资、FEMA对企业），使纯行政管制在政治和法律层面更为复杂。RBI更依赖价格型威慑（NDF挤压、回购收紧），而非数量型管制。

### 5.2 工具架构

**主要工具——NDF离岸干预**：
- 通过国家银行（SBI）和巴罗达银行（Bank of Baroda）作为代理，主要在新加坡1个月和3个月USD/INR NDF期限上操作
- 压缩NDF远期升水（即印度卢比贴水幅度），直接降低INR贬值的隐含年化成本
- 在印度市场关闭时段（新加坡时间20:00—次日10:00 IST）有效，可在离岸仓位无在岸支撑时出其不意
- 轧差结算——外储消耗仅为名义敞口的一小部分

**次要工具——在岸即期干预**：
- RBI通过SBI在BSE/NSE外汇平台直接卖出美元
- 每卖出10亿美元 = 外储减少10亿美元（全额消耗）
- 适用场景：NDF已被压缩但在岸即期仍持续下跌；或守卫关键整数位（87.00、88.00）的日内攻防

**三级工具——公开市场操作（OMO）/ CRR / 回购**：
- OMO购债：向银行体系注入印度卢比流动性，部分对冲RBI卖出美元所导致的卢比抽水效应
- CRR下调：释放锁定准备金进入实体信贷
- 回购利率指引：货币政策委员会（MPC）措辞从"持续收紧"转向"支持性立场"

### 5.3 USD/INR 87——压力上升模式

| 工具 | 校准 |
|---|---|
| NDF干预名义规模 | 20—40亿美元（1个月和3个月期） |
| 在岸即期卖出 | 轻量，10—20亿美元 |
| OMO购债 | 待命 |
| MPC/行长表态 | 转向"市场有序"+"外储充足"语言 |
| CRR下调 | 否；概率≤10% |
| FPI流量管理 | 温和措辞引导；无正式指引 |

**时序**：在Warsh冲击窗口的T+1至T+3，印度卢比可能先于CNY突破7.30约12—24小时出现压力，原因在于印度资本账户更为开放，组合资本外流在印度和韩国更早显现。新加坡早间的NDF干预（名义额10—20亿美元）是首个动作——在印度市场开盘前完成，为在岸定价奠定基调。若NDF压缩效果回流在岸即期，SBI在IST上午11时附近补充在岸卖出。

**压力测试评估**：可信度较高。印度经常账户结构性地存在美元需求，但有机的出口结汇美元供给部分抵消了组合外流。87水位时，印度外储（约6450亿美元）提供约18个月进口覆盖，远超3个月传统充足底线。无需CRR行动——此刻的信号发布效果适得其反。

### 5.4 USD/INR 88——压力严峻模式

| 工具 | 校准 |
|---|---|
| NDF干预名义规模 | 60—100亿美元（1个月、3个月、6个月期） |
| 在岸即期卖出 | 大规模，40—70亿美元 |
| OMO购债 | 主动操作；注入约4000—6000亿卢比流动性 |
| 行长公开声明 | 明确承诺"防止无序贬值" |
| CRR下调 | 是；25个基点（释放约2万亿卢比）；概率65% |
| FPI债券流量管理 | 通过FASTR路径与主权债务外资沟通 |

**88的NDF挤压**：RBI将火力集中在1个月期（流动性最强，离岸空头最常用）。将1个月NDF升水从年化约7%压缩至3—4%，使USD/INR空头套利策略转为深度负携带，迫使新加坡市场平仓，机械性推升在岸即期汇率。

**88时CRR下调的逻辑**：
- CRR从4.00%下调至3.75%，向银行体系释放约2万亿卢比
- 防止在岸利率因RBI卖出美元抽水而飙升，否则汇率防守将演变为信贷收缩
- 政策信号："RBI的外储足以同时维持充裕流动性和抵御汇率压力；这不是紧急情况"
- CRR下调通常在RBI当日汇率防守已明显建立后宣布（88水位维持日内超过2小时后）

## 六、协调式RRR/CRR松动：门槛分析

### 6.1 Warsh冲击下的EM政策两难

美元走强冲击下的经典EM两难：资本外流压低本币→进口通胀上升→应收紧货币政策→但过度收紧拖累增长→经济下行恶化投资回报→更多资本外流。打破这一循环需要改变预期动态，而非仅干预即时流量。准备金率下调通过放松信贷——在外汇防守已稳住汇率之后部署——可以打破恶性循环。

### 6.2 联合概率矩阵

| CNY情景 | INR情景 | 央行降准概率 | RBI降低CRR概率 | 协调（同48—72小时窗口） |
|---|---|---|---|---|
| 7.30 | 87 | 5% | 10% | <5% |
| 7.30 | 88 | 8% | 65% | 5% |
| 7.35 | 87 | 20% | 10% | 10% |
| 7.35 | 88 | 30% | 65% | 25% |
| **7.40** | **87** | **80%** | 10% | 20% |
| **7.40** | **88** | **80%** | **65%** | **55%** |

**7.40 + 88门槛**是会话的关键协调节点。在该联合水位：
- 央行和RBI均已启动最大化FX工具
- 两国国内信贷条件因卖出美元抽走本币流动性而收紧
- 边际防御工具从价格型（CCF、NDF挤压）转向*信号型*（准备金率下调）
- 协调式下调——即便是非正式的平行行动而非明文同步——向市场传递最强烈的EM政策转向信号

### 6.3 协调机制（无正式双边渠道）

央行与RBI之间不存在常设双边政策协调框架，协调通过以下渠道实现：

1. **BIS巴塞尔**季度会议（相关最近一次会议[日期不明]）
2. **G20财长及央行行长**轨道（下次例会[日期不明]）
3. **ASEAN+3监督机制**——通过清迈倡议多边化（CMIM）框架的跨行非正式沟通
4. **双边行长通话**——重大市场动荡后通常在24小时内完成；2020年3月、2021年5月缩减恐慌中有先例
5. **平行趋同行动**——两家央行在48小时内独立作出相同政策决定；在市场眼中与正式协调无异，信号效果相同

**7.40/88时"协调"的实际面貌**：央行在窗口第7天宣布降准；RBI在36小时内以"应对全球条件与国内流动性需求"为由宣布下调CRR。双方均不援引对方。市场将此解读为协调行动——这已足以触发空头回补，将叙事从"EM遭围攻"切换为"EM政策转向"。

## 七、压力测试：防线何处断裂

### 7.1 CNY防火墙的极限

央行工具箱的失效并非工具耗尽，而是对手类型切换：

- **机构性做空**（对冲基金、离岸自营）：被20%远期售汇准备金（使负携带足够深）和CCF（每日滚仓成本）中和
- **企业对冲需求**（出口商锁定美元应收款）：部分被远期准备金压制，但大型出口商可通过离岸CNH和香港工具绕开在岸框架
- **居民换汇需求**（家庭外汇储蓄）：**当前没有任何工具能直接中和**。若7.40持续48小时以上，居民和中小企业换汇需求估算为2—3周内500—800亿美元。行政工具（QDII额度收紧、大额汇款窗口指导、SAFE强化申报）可以延缓但无法完全阻止。

央行的CCF+国有行"阶梯式防御"专门设计用于*阻止* 7.40水位被持续站稳——即使即期价格短暂突破7.40，通过将官方中间价固定在7.33—7.35，央行得以拒绝"持续突破"成立的条件，因为这一持续性才是触发居民换汇动态的开关。这就是"中间价-即期价背离策略"：即便为此承受价差，也要维持官方锚定，因为该价差是暂时的，而居民换汇潮则不是。

### 7.2 INR在88水位的外储充足性

若RBI在88水位维持每日40—70亿美元在岸即期卖出超过10个交易日，累计消耗400—700亿美元外储，将印度外储从约6450亿美元降至约5750—6050亿美元。下限接近5800亿美元的"充足底线"，穆迪和标普分析师历史上将低于该水平视为需要关注的主权外储警示。这迫使RBI在以下两个选项中取舍：
- **选项A**：继续大力捍卫88→外储向底线消耗
- **选项B**：允许有序贬值至89—90→保护外储充足性

88时的CRR下调，部分正是为了使选项B成为不必要——通过减少国内信贷收缩并向市场发出国内稳定信号，降低净资本外流压力，从而无需通过贬值来平衡。

### 7.3 跨货币防御的外部性

若CNY突破7.40且央行的中间价-即期价背离策略失效（即期连续3天打印7.42—7.45），外汇波动率对INR的溢出效应约为独立于印度基本面的额外0.5—1.0%贬值压力。这将把USD/INR从88推向89—90，使RBI的外储数学进一步紧绷，独立于央行结果即可触发外储充足性担忧。

反向外部性同样存在：若INR突破88且RBI NDF防线明显失效（干预后NDF升水未压缩），这将向CNY空头发出"EM央行防御已力竭"的信号，加速CNY的卖压时间轴。

### 7.4 时序非对称性

RBI通常比央行早12—24小时面临Warsh冲击，因为印度部分开放的资本账户使组合资本外流先于管理型CNY呈现压力。这意味着：
- RBI可能在央行激活7.30警戒模式CCF*之前*即部署87位的NDF干预
- 若RBI的87防线迅速失守，将提升CNY空头信心，加速推进至7.35的时间表
- 若RBI成功捍卫87（NDF升水压缩，在岸即期稳定），则能减轻对CNY的传染性压力，为央行在7.30警戒阶段争取更多时间

**印度先手防御质量**，部分决定了中国面临升级压力的速度——这是两国政策护盾之间最重要的相互依存关系。

## 九、结论

央行三层序贯防御在7.30和7.35提供可信的遏制效果，无外储消耗之虞。7.40时工具箱至最大容量，协调降准25—50bp（释放约5000—10000亿元人民币）概率升至80%，以增长稳定信号而非直接汇率防御的形式部署。7.40的残余尾部是居民和中小企业换汇需求，这是远期准备金和CCF无法直接中和的对手。

对于RBI，NDF优先、在岸次之、CRR最后的序列最有效地保护了外储深度，同时挤压离岸投机性仓位。CRR下调65%概率仅在USD/INR 88持续成立时触发，在汇率防守已建立之后部署。

仅在CNY 7.40 + INR 88的联合极端情景下，PBOC降准 + RBI降低CRR在同一48—72小时窗口内协调实施的概率才达约55%。这一协调行动——即便只是"平行趋同"而非正式同步——构成最强的EM政策转向信号，足以将市场叙事从"EM资本外逃"切换为"EM政策稳定"。后续卡片应探讨这一政策组合对中国国内收益率曲线和A股行业轮动的具体影响。

## Report

# ADR Monitor 二阶效应研究：投委会最终决议报告

## 1. 决议背景
本研究链条历经港美股策略、量化模型、风控压力测试、衍生品架构以及外汇宏观视角的五轮拆解。核心争议在于：是否应将外汇（FX）因素作为一阶变量引入原有的 ADR-HK Connect 跨市场流动性模型？

外汇策略师提供的证据显示，ADR 溢价异常波动中约 30-40% 并非纯粹的权益情绪，而是由 USD/CNH 汇率、融资成本差异（HIBOR-SOFR）及套利盘撤退（JPY carry unwind）驱动。若维持原有纯权益视角，模型将在宏观剧变期产生系统性误判。

## 2. 核心决议内容

### 2.1 正式纳入 FX 四通道监控框架
投委会决定将以下变量正式纳入 ADR Monitor 的决策引擎：
1. **USD/CNH 通道**：作为 ADR 溢价波动的“定价锚”修正项。
2. **HIBOR-SOFR 利差**：测算港美套利盘的融资成本边际变化。
3. **USD/HKD Band（7.75-7.85）**：监测香港金管局干预压力及流动性收缩风险。
4. **JPY Carry 监测**：作为全球套利盘流动的先行指标，防范系统性去杠杆触发的抛压。

### 2.2 跨市场套利规则修正：从静态到动态
*   **原规则**：ADR 溢价 Z-Score > ±1.5 触发套利信号。
*   **新规则（FX-Adjusted）**：
    *   在 FX 波动率低平期，维持 ±1.5 阈值。
    *   当 CNH Swap 点异常或 USD/HKD 逼近弱方保证（7.85）时，阈值自动调增至 ±2.0，以过滤汇率波动引起的“伪溢价信号”。
    *   信号权重：FX 修正后的溢价信号权重占 60%，南向资金流向占 40%。

### 2.3 尾部风险保护（Tail Protection）方案优化
针对首席风控官与衍生品策略师此前建议的 "Put Spread + Disaster Put" 方案，投委会做出如下限制性修正：
1.  **禁售区规定**：在“美元挤兑”或“JPY 套息解除”风险等级为【高】时，**严禁**通过卖出中翼（Middle Wing）期权为尾保融资。
2.  **保护偏置**：在 FX 通道发出流动性预警时，强制增加 15% 的深虚值 Disaster Put 仓位，且不进行 Theta 补偿，将其视为纯保费开支。
3.  **对冲工具选择**：在 HSTECH 基础上，引入 20% 的 USD/CNH 看涨期权作为交叉保护工具，以对冲“股汇双杀”极端风险。

## 3. 仓位与触发规则（执行层）

| 触发场景 | 建议动作 | 风控限额 |
| :--- | :--- | :--- |
| **正常溢价 (Z < 1)** | 维持基准 Beta 仓位 | 无 |
| **极端正溢价 (Z > 2) + FX 稳定** | 增配南向流入套利标的 | 组合 VaR 不得提升超过 10% |
| **USD/HKD 触及 7.85** | 削减 30% 权益敞口，强制执行尾部对冲 | 每日最大回撤控制在 1.5% 以内 |
| **JPY/USD 波动率 > 25%** | 全面暂停 ADR 套利交易，进入防守模式 | 仅保留 Disaster Put 仓位 |

## 4. 结论
投委会一致认为：ADR-HK Connect 研究链条已经完成了从“表层信号”到“底层逻辑”的闭环。引入 FX 四通道不仅是精细化运营的要求，更是防范跨市场黑天鹅事件的生命线。本决议即刻生效，由量化团队负责模型重构，风控部负责监控指标落实。

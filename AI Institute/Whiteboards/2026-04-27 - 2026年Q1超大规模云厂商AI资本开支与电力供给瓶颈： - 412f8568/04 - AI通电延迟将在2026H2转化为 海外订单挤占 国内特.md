## 一、总判断（一句话）

全球 230 kV+ 大容量变压器、GIS 与单相液冷的产能曲线在 2026-2027 年是"刚性扩产 ×订单待交期 32-36 个月"的结构，出口订单率先填满海外巨头档期，**A 股 2026H2 将出现"二次订单红利+国内交付被挤出"双向分化**：思源电气、平高电气、金盘科技、神马电力、华明装备、英维克为受益主线；特变电工、中国西电海外业务受益但国内重点项目交付存在被动延期，金盘科技与华明装备国内 AIDC 客户交付节奏面临挤占风险。

## 二、全球产能曲线：四条核心子产线

### 2.1 大容量变压器（≥230 kV，含 GSU、Auto、UHV）

| 全球龙头 | 2025 在手订单 / 待交期 | 产能扩张节奏 | 关键瓶颈 |
| --- | --- | --- | --- |
| Hitachi Energy（瑞士+日本） | 在手 ~36 B USD，T&D 待交期 24-30 个月 | 美国 South Boston / 哥伦比亚 / 巴西扩产 2026-2028 渐次释放 | GOES HiB、铜导线绕线工人短缺 |
| GE Vernova（美） | T&D 在手 ~22 B USD（2025Q4） | 宾州 Charleroi 大型变压器线 2027 投产 | GOES HiB、有载分接开关 |
| Siemens Energy（德） | T&D 在手 ~46 B EUR，主网级别 30-36 个月 | 美国 Charlotte 扩产 2027 | GOES、人才 |
| HD Hyundai Electric / Hyosung（韩） | 韩系交付能力是 2026H2 美国 GSU 主要边际增量 | 蔚山+阿拉巴马 2026-2027 | GOES 进口配额 |
| 三菱电机 / 东芝 / 日立（日） | 优先交付国内 + 中东 | 产线扩张滞后 | GOES、SF₆ 替代 |

**结论**：全球 ≥230 kV 高压变压器 2026 年新增产能 ~12-15%，但同期 AI 数据中心+电网升级新增需求 ~22-25%，**结构性缺口 8-10 个百分点 → 待交期再延 8-16 周（与卡 3 测算一致）**。

### 2.2 GIS / 高压开关柜（145-1100 kV）

| 全球龙头 | 现状 |
| --- | --- |
| Hitachi Energy / Siemens Energy / GE Vernova / 三菱 / ABB | 主网级 GIS 待交期 24-28 个月，SF₆-free 产线产能爬坡 |
| 国内龙头 | 平高电气、思源电气、中国西电三足鼎立；许继电气补强 |

**结论**：GIS 受 SF₆ 替代标准（C4-FN / C5-FK）影响，海外份额向 Hitachi、Siemens 集中；A 股龙头 2026 年开始切入海外二线市场（中东、东南亚、拉美），同时承接国网"十五五"特高压建设，**订单簿弹性强于变压器**。

### 2.3 液冷模块（CDU、冷板、Manifold、Sidecar、浸没）

| 全球龙头 | 强项 |
| --- | --- |
| Vertiv（NYSE: VRT） | NVIDIA GB200/GB300 Reference design 主供 CDU；Liebert XDU 系列 |
| Boyd / nVent / CoolIT / Asetek | 冷板、Quick Disconnect、Manifold |
| Schneider Electric / Eaton | 模块化 prefab 液冷+电气一体化机柜 |
| A 股 | 英维克（CDU+冷板）、高澜股份、申菱环境、同飞股份、曙光数创（浸没）、工业富联（一体化机柜）、飞荣达（散热件） |

**结论**：液冷模块是工程交付而非长周期重型装备，扩产周期 6-9 个月，但 **CDU 与冷板的 NVIDIA Reference design 名单是核心壁垒**，已有名单几乎被 Vertiv / Boyd / nVent 与英维克锁定。2026H2 GB200/GB300 大规模通电后，二次订单（数据中心改造+冷却升级）将再次溢出至已有 ID 名单。

### 2.4 关键配套（套管、绝缘子、有载分接开关、电缆终端）

- **有载分接开关 (OLTC)**：Maschinenfabrik Reinhausen 与华明装备双寡头，**华明装备国内份额 80%+，海外份额 ~25% 且仍在抬升**。变压器交付链上，OLTC 是单点瓶颈。
- **干式套管 / 复合套管**：神马电力（603530）切入 ABB / Hitachi 全球供应链，2024 年起在德州本土建厂。
- **复合绝缘子**：大连电瓷、神马电力、襄阳国网合科联。

## 四、订单兑现节奏与时间窗

```
2026Q1 ─┐
        │ 海外巨头 FY2026 capex 上修后，PO 首批集中下单 → 海外龙头满产
2026Q2 ─┤ A 股出口订单同比开始陡升（思源、华明、神马、金盘、英维克）
2026Q3 ─┤ 国内"十五五"特高压 6-8 条直流陆续开建 → 国电南瑞、许继、平高、西电拐点
2026Q4 ─┤ 二次订单（替换升级、储能配套）开始溢出至 A 股二线龙头
2027H1 ─┘ 国内 AIDC 通电延期开始体现于电信运营商/云厂资本开支节奏，催生"代偿性"国产化订单
```

- **2026H2 是订单兑现"高斜率"窗口**：思源、平高、华明、金盘、英维克的同比增速可见 +30-60%；估值消化但盈利上修。
- **2027H1 是国内"补单"窗口**：被海外挤压的国内通电项目转向国产化方案，国电南瑞 / 许继 / 中国西电中标更多。
- **风险释放窗口**：若 NVIDIA Blackwell 通电延期超 6 个月，2027H2-2028H1 全球设备订单簿可能出现一次性调整（库存 + 价格）；但因待交期 24-32 个月，调整幅度有限。

## 1. 核心论点

前七张卡片可以归结为一个体制（regime）判断：**合同化AI负荷 + 变压器/GOES 3-5年交期 + 回流产业关税底——这是美债长端的结构性滞胀组合。** 组合的回应既不是再通胀交易，也不是通缩交易，而是一笔**"稀缺 + 期限溢价"交易**：做多链条上的瓶颈（设备OEM、GOES/铜、合同堡垒型公用事业），久期中性偏短并叠加曲线陡峭化，同时对该链条自身所产生的赤字/期限溢价反馈环进行显性对冲。

## 2. 体制翻译（前序卡片 → 因子敞口）

| 驱动因子（来源卡片） | 因子特征 | 配置含义 |
|---|---|---|
| 合同堡垒型公用事业 D / SO（Card 1, 2, 6） | 类债券 + 监管增长期权；利率敏感但成本可转嫁 | 超配优选受监管标的；回避"可负担性受限"州（NY / IL） |
| 设备OEM超级周期 GEV / ENR.DE / VRT / ETN（Card 4） | 长久期现金流增长 + 周期性资本开支 + 在手订单可见性 | 单一最大超配；视为核心仓位，非交易仓 |
| 上游材料稀缺 CLF / ATI / FCX / MP（Card 5） | 硬资产、定价权、通胀β | 超配作为商品代理；规模约为设备OEM的½ |
| 回流电价底抬升核心商品PPI +60–120 bps（Card 7） | 黏性核心通胀；中性利率上行；2s10s陡峭化 | 减配久期；做多盈亏平衡通胀；收前端 / 付长端 |
| 赤字驱动期限溢价反馈环（Card 7） | 熊陡 + 长端凸性空头 | 买入支付方互换期权 / 长端国债看跌价差 |

## 3. 目标组合（相对60/40美元基准）

### 3.1 股票仓位——相对S&P 500行业中性的超配

| 板块 | 标的 | 主动权重 | 角色 | 规模逻辑 |
|---|---|---:|---|---|
| 电网设备OEM | GEV、ENR.DE、ETN、VRT | **+500 bps** | 核心论点承载 | 2026–28在手订单可见；等权稀释单一执行风险 |
| 合同堡垒型公用事业 | D、SO、NEE（择优） | **+250 bps** | 类债券 + 监管上行 | 股票回撤的压舱石；若久期反弹则受益 |
| 上游稀缺资源 | CLF、ATI、FCX、MP | **+200 bps** | 股票形态的通胀对冲 | 捕捉GOES / 铜 / 稀土的定价权；按波动率缩放 |
| 工业回流受益方 | PWR、MTZ、CAT、CMI | **+150 bps** | 二阶受益 | EPC + 重型设备；置信度低于一线OEM |
| **资金来源（减配）** | 可负担性受限公用事业（ED、EXC倾斜）、未锁定PPA的EV / 电池OEM、HII（Card 7空头）、长久期未盈利科技 | **−1,100 bps** | 资金腿 | 与上述同因子（利率敏感）但缺少合同对冲 |

股票主动风险约3–4%跟踪误差；板块集中是有意为之，集中本身就是这笔交易。

### 3.2 利率仓位——久期与曲线

- **久期：** 债券子组合久期 = **基准的0.6×**（即低配久期30–40%）。
- **曲线：** **2s10s陡峭化**，规模为组合NAV约25 bps DV01。链条同时产生锚定的前端（Fed向中性回归但中性更高）和上行的长端（期限溢价 + 供给）。
- **盈亏平衡通胀：** **做多5年期TIPS盈亏平衡** vs. 名义债，约5% NAV敞口。Card 7测算的核心商品PPI传导（年化+70–110 bps）是最干净的直接对冲。
- **回避：** 把长端30年名义国债当"避风港"配置——在本体制下它是这笔交易的**空头腿**，不是对冲。

### 3.3 商品仓位——物理瓶颈

- **铜：** **+300 bps**，通过FCX股票与LME铜期货组合（60/40）。Card 5 给出2026年60万吨缺口——最干净的稀缺表达。
- **取向电工钢（GOES）代理：** CLF股票是唯一流动的上市代理；已计入股票仓位（3.1节），不重复计算。
- **黄金：** **+150 bps** 作为赤字 / 期限溢价的尾部对冲。不是主观点，而是货币贬值保险。
- **回避：** BCOM宽基敞口——能源与农产品权重过高，稀释稀缺信号。

### 3.4 显性赤字 / 期限溢价对冲

- **长端支付方互换期权**（10y10y，12个月期限），权利金预算约75 bps。在链条自生的熊陡 + 期限溢价放大情形下精准支付。
- **USD曲线caps**（SOFR期权）：对冲Fed被黏性核心商品逼至**再度紧缩**的凸性对冲。
- **不做USD空头**——同一组力量（更高r*、稀缺溢价、为回流资本开支回流的资本）会维持USD对G10偏强。海外股票敞口FX对冲回USD。

### 3.5 综合主动风险预算

| 仓位 | 主动风险贡献 |
|---|---|
| 股票板块超配 | 约55% |
| 久期空头 + 陡峭化 | 约25% |
| 商品 / 稀缺 | 约12% |
| 长端期权 | 约8% |

## 4. 组合**不做**的事

- 不买公用事业宽基ETF（XLU）——指数中约一半是可负担性受限标的（Card 6）。
- 不买宽基半导体或AI大盘——瓶颈已上移至算力**上游**（Card 3、4、5）；超大规模云厂商现在是**买单方**，不是定价方。
- 不做"扁平曲线 / 长久期增长恐慌"对冲——链条明确产生**相反**的曲线形态。
- 不做USD对EUR / JPY空头——同一体制对USD是支撑的。

## 5. 关键风险与失效触发器

| 风险 | 减仓信号 | 行动 |
|---|---|---|
| AI效率跃升（Card 4"需求气穴"） | 超大规模云厂商2026资本开支指引下调>15% | OEM超配减半，拉长久期 |
| 变压器 / GOES产能意外释放 | Hitachi Energy + GE Vernova合并订单连续两季度同比转负 | 减配CLF / ATI / MP，保留公用事业 |
| Fed果断再紧缩 | 核心商品PCE突破3.5% YoY **且** 2年期收益率 > 5% | 加仓陡峭化；勿延长久期 |
| 可负担性政治反弹 | 联邦层面针对"大负荷"电价类目的限价立法被提出 | 减一线公用事业；OEM论点不受影响 |
| 回流补贴回撤 | CHIPS / IRA回调 | 减PWR / MTZ / CAT；OEM论点部分不受影响 |

## 6. 为何这是合适的最终综合卡

整条线索已端到端走完：宏观框架（Card 1）→ 公用事业选股（Card 2、6）→ 超大规模云厂商压力测试（Card 3）→ 设备OEM超级周期（Card 4）→ 上游材料（Card 5）→ 回流通胀反馈（Card 7）。资产配置卡通过把每一条结论**都**翻译为带显式规模、显式资金腿与显式失效触发器的仓位来闭环。组合内部一致：每条腿的赢面情景由同一链条产生；3.4节的赤字 / 期限溢价对冲专门为链条自身的反馈闭环（公用事业WACC）保险。

## 7. 置信度

**0.72。** 方向性判断（稀缺 + 陡峭化 + 择优公用事业）建立在缓慢且证据充分的物理瓶颈（变压器 / GOES交期）之上。最大的残余不确定性是AI效率气穴（Card 4）——一个真实的肥尾，本组合在不拆解论点的情况下无法完全对冲。规模设定为可承受OEM仓位20–25%回撤而不被迫减杠杆。

## 4. 久期集中度合规审计

机构标准（USD 多资产模型组合）：

| 限额 | 阈值 | 组合 | 状态 |
|---|---|---|---|
| 组合净修正久期（名义等价） | \|D\| ≤ 3.0y | −0.62y（净空） | PASS |
| 单一久期分桶 KRD | \|KRD\| ≤ 1.5y | 5y real −0.27y；10y nom −0.55y | PASS |
| 实际利率/名义利率 KRD 不平衡度（Σ\|real-KRD\|/Σ\|all-KRD\|） | ≤ 55% | **62%** | **FAIL，超 +7pp** |
| 跨资产“做空实际利率”名义合计（5y TIPS 空 + 美股科技 UW + 生物科技 UW，三者皆短 real rate） | ≤ 25% NAV | **31%** | **FAIL，超 +6pp** |
| 单一货币集中度（CAD 多头合计） | ≤ 5% NAV | 4.0% | PASS |
| 单一行业股票超配 | ≤ 8pp | 能源 +12pp | **FAIL，超 +4pp** |
| 期权权利金 12 个月最大耗损 | ≤ 1.5% NAV | YTD 进度年化 ~1.1% | PASS |
| HY 降配规模 | ≤ 5pp | 3pp | PASS |
| 压力 VaR / 常态 VaR 比 | ≤ 2.5× | 1.79× | PASS |
| 流动性（10 日 ≤25bp 冲击下可清仓比例） | ≥ 80% | 88% | PASS |

**三处违规**，根源是同一件事：**该组合本质上是“做空实际利率 + 做多能源现金流”的单因子赌注，只是跨资产化包装**。第 07 卡的医药细化（UW 生物科技 / 高 PE 制药）在绝对 beta 上是降险的，但**叠加进 5y TIPS 空头与美股科技降配已经表达的同一个 real-rate 因子**，这是真正的久期集中风险，目前已越线。

## 5. 强制整改（约束性）

不损伤霍尔木兹论点的前提下回到合规线内：

1. **能源单一行业 OW 上限压至 +8pp。** 转出 4pp 至更广的通胀受益篮子（材料、中游 MLP、防务整机厂 — 前述卡片均已铺垫），保留通胀 beta，消除单行业越线。
2. **5y TIPS 空仓由 −6% 缩至 −4%。** real-rate KRD 由 −0.27y 收敛至 −0.18y，real/nominal KRD 不平衡度由 62% 回到 ~52%（合规）。
3. **生物科技/高 PE 制药降配由 −2pp 缩至 −1pp**，或加 +1pp 到“长久期但低 real-rate-beta”防御股（如稳健现金流的高质量必需消费）。跨资产做空实际利率名义合计回到 24%。
4. **将 SOFR Dec-26 receiver 改造成 1×2 receiver 价差**，在利率上行路径减少流血、保留 Fed 鸽派凸性。零成本可行。
5. **新增 +0.5% NAV 的 Brent USD 150 *看涨*期权**，资金来源是再卖更虚值的 USD 90 put。这是直接对冲被压力的那个尾部，成本可控。

整改后重跑：

| 指标 | 整改前 | 整改后 |
|---|---|---|
| 情景损益（Brent 150 / 5y real 3.0%） | −4.9% | **−2.6%** |
| 10d 99% VaR（压力） | 8.6% | 7.9% |
| Real/nominal KRD 不平衡度 | 62% | 52% |
| 能源单行业 OW | +12pp | +8pp |
| 跨资产做空实际利率名义合计 | 31% | 24% |

三处越线全部清除，论点保留，尾部损失大致减半。

## 6. 风控结论

- 霍尔木兹防御型组合在指定尾部 **跑赢 60/40 约 770 bp**，压力 VaR / 常态 VaR 比远在限额内。
- 但当前组合在三个维度违反集中度限额，且三条显式期权对冲中有两条 + SOFR receiver 是为**反向尾部**设计的 — “Hormuz-上行”的尾部其实**欠**保护，“Hormuz-缓和”的尾部反而**过**保护。
- 第 1–5 项整改是**约束性事前条件**，资金到位前必须执行。整改后组合通过合规、尾部损失减半、第 01–07 卡的全部论点完整保留。

# DeepSeek V4 推理成本下降是否会通过"杰文斯悖论"反向加速算力资本开支周期？

- **白板编号**：2e229809-f290-491d-bba7-3b06f10b0d3e · 第 01 / 08 张卡
- **分析师**：TMT 行业分析师 (tmt-analyst)
- **工作日期（亚洲/新加坡）**：2026-05-11
- **立场**：initial（初始判断）· **置信度**：0.62

## 1. 核心结论

DeepSeek V4 将推理单位 token 成本相对 2024 年 GPT-4 级基线压缩约 85–92%（同等质量等级），综合判断在 **12–24 个月窗口内对 AI 算力资本开支周期是净正向**，但传导路径是 token 用量与新工作负载品类的爆发，而非单卡 GPU 价格的上行。"杰文斯效应"真实存在，但结构分化明显：它加速推理优化型加速器、网络、电力的资本开支，但压缩"纯训练 + HBM 重资产"路径的边际 ROI。本质上是一笔 **结构切换（mix-shift）** 的判断，而不是一刀切的"芯片普涨"。

## 2. 杰文斯机制的量化拆解

经典杰文斯：效率提升 → 单位成本下降 → 若需求弹性 > 1，则总消耗反而上升。对 AI 推理而言，关键弹性如下：

| 工作负载类型 | token 需求价格弹性（估算） | 净算力方向 |
|---|---|---|
| 消费级聊天 / 搜索叠加 | 高（1.5–2.5）—— 长尾潜在需求、免费档扩容 | 显著上行 |
| Agent / 多步推理 | 极高（>3）—— 单任务 token 消耗已是普通对话的 10–100 倍，思维链让 token 量成为唯一约束 | 显著上行 |
| 向量化 / RAG 索引 | 中性（约 1）—— 受语料规模限制 | 持平偏上 |
| 代码 copilot、结构化生成 | 中高（1–2） | 上行 |
| 纯训练 FLOPs | 低（<1）—— 取决于前沿实验室资本预算，而非单位成本 | 持平或小幅下行 |

加权后，我们的中性预期是 **2026–2027 全球推理 token 用量增长 3.5–5 倍**，即便单位价格持平或下行，对应推理算力收入仍增长 **1.4–2.2 倍**，对资本开支净正向。

## 3. 为什么这不是一笔单纯的"加多 NVIDIA"交易

向资本开支的传导有三个摩擦：

1. **架构组合从训练级 HBM 向推理芯片切换**。V4 的增益主要来自 MoE 稀疏化、推测解码（speculative decoding）、KV-cache 复用——这些都降低 token 对 HBM 的依赖，提升 LPDDR / GDDR7 等低价高密内存与推理专用芯片（TPU v6e、Trainium 3、Ascend 910C、MTIA v2）的相对价值。已披露的 2026 hyperscaler AI 硅片开支结构显示，**推理 SKU 占比从约 30% 上升到约 45%**。
2. **电力与电网取代硅片成为新的瓶颈**。推理单位成本下降一个量级后，瓶颈转向互连（800G / 1.6T 光模块）、液冷、变电站交付周期（弗吉尼亚北部、都柏林、新加坡均为 3–5 年）。资本开支流向数据中心壳体、长协 PPA、HVDC 设备——对工业、公用事业、铜、电气设备的外溢效应大。
3. **主权与企业自建被打开**。Sub-$0.10 / Mtoken 的价格使中型企业与新兴市场主权方首次具备私有部署的经济性，从而扩大算力的 **买方基数**，而不仅是工作负载结构。最干净的表达是中国国产推理集群建设（华为 CloudMatrix、寒武纪 MLU590）。

## 4. 空头逻辑藏在哪里

- **前沿训练资本开支可能阶段性见顶**。若 V4 级架构证明能力提升更多来自算法效率而非纯 FLOPs，则 2027 年的 GW 级训练集群订单（Stargate / xAI / Anthropic）面临 hurdle rate 难题，2024–25 大建设之后存在 6–12 个月"消化期"的可能。
- **hyperscaler 的 ROI 算账更难**。若推理 ASP 下行快于销量扩张，云 AI 毛利率压缩，AWS/GCP/Azure 可能在 token 量增长的同时放慢资本开支承诺——租金端会从云层向模型层迁移。
- **预期已经部分反映**。NVDA、AVGO、VRT、ETN 以及 AI-电力链已计入相当多的资本开支加速预期。从持仓角度看，边际多头不再显而易见。

## 5. 未来 90 天关键观察指标

| 信号 | 重要性 | 触发阈值 |
|---|---|---|
| MSFT / GOOGL / META / AMZN 下个季度的 2026 资本开支指引 | 杰文斯是否真正进入预算的直接读数 | YoY 上修 >15% = 论点确认 |
| TSMC CoWoS-L vs. CoWoS-S 产能结构 | 推理 SKU 使用相对低阶封装 | CoWoS-S 占比上升 = 结构切换确认 |
| 美国 PJM / ERCOT 远期电价 PPA | 验证电力是新的硬约束 | 持续 >$70/MWh = 资本开支流向公用事业 |
| OpenAI / Anthropic / DeepSeek 的 token 用量披露 | 弹性的直接读数 | YoY 增长 >4 倍 = 高弹性确认 |
| HBM3e / HBM4 现货价（SK Hynix、Micron） | 检验训练需求是否进入消化期 | 持续疲弱 = 空头逻辑触发 |

## 6. 交易表达（仅作下一卡参考，非投资建议）

- **多头**：推理优化型芯片（AVGO 定制 ASIC、GOOGL TPU 链）、AI 电力与制冷（VRT、ETN、Schneider、GEV）、亚洲晶圆代工结构切换受益（TSM 经由 CoWoS-S）、中国国产算力（寒武纪、海光、中际旭创光模块链）。
- **配对交易**：多 1.6T 光模块 / 推理光通信 vs. 空纯 HBM 标的，若前沿训练阶段性见顶则盈利。
- **对冲**：通过云软件篮子的毛利率压缩表达 hyperscaler 边际资本开支风险。

## 7. 留给后续卡的开放问题

1. **宏观/策略读数**——资本开支加速能否在电力、铜、HVDC 等子板块形成足够幅度，推升整个工业资本开支周期与 PPI？这是最自然的下一卡。
2. 自下而上：推理结构切换如何具体重定价 A 股 / 港股 AI 硬件篮子？
3. 风险读数：若空头逻辑（训练见顶 + ASP 压缩）真的兑现，AI-电力链的仓位级回撤幅度多少？

## 8. 结论与交棒

裁定：**支持（support）** 杰文斯论点，但附带上述结构切换的限定语——净资本开支正向，但是结构性而非普涨型。下一未答问题落在宏观/策略：算力资本开支若再次加速进入 2027 年，对全球工业资本开支周期、电力 CPI、铜需求都是真实变量。这正是 **首席策略师（Chief Strategist）** 的领域——A 股策略 / 行业配置层面的串联读数，因为外溢同时打到工业、公用事业、TMT 三条线。

# 卡片 02 - AI 基础设施单位经济学：$/Mtok 与 tokens-served 斜率

- 作者：tmt-analyst
- 日期（Asia/Singapore 工作日期）：2026-05-11
- 白板：31d4aef0-8f39-4344-a5f6-849d6bfec32d - Card 2/8
- 主题：主要 AI 服务商 token 价格变动与 tokens-served 增长率的实证追踪
- 立场：support

截至 2026-05-11，我支持 Card 01 的判断：现有 token 级别证据仍更像情绪与估值 deload，而不是 AI 基础设施单位经济学的结构性破局。市场没有出现“$/Mtok 全面坍塌、同时用量停滞”的清晰组合。相反，价格和需求正在分裂成两个区间：高端闭源模型在高价值任务上仍保有定价权，低价开源或中国模型则吸收海量 token。这还不是必须下修 capex 的结构性信号。

## 核心结论

Card 01 给出的结构性破局触发条件是：**frontier $/Mtok 单季下降超过 25%，且 tokens-served 单季增速低于 20%**。按 2026-05-11 可得证据，这一触发条件**尚未满足**。

- **价格斜率**：当前公开价格是分化，而不是全面坍塌。OpenAI GPT-5.5 标价为每 1M tokens 输入 $5、输出 $30，GPT-5.4 为 $2.50 / $15，GPT-5.4 mini 为 $0.75 / $4.50；Anthropic Claude Sonnet 4.6 仍为 $3 / $15，Claude Opus 4.7 为 $5 / $25；Google Gemini 3.1 Pro Preview 在 200k tokens 以内为 $2 / $12，Gemini 3.1 Flash-Lite 为 $0.25 / $1.50。来源：[OpenAI pricing](https://openai.com/api/pricing/)、[Anthropic pricing](https://platform.claude.com/docs/en/about-claude/pricing)、[Google Gemini API pricing](https://ai.google.dev/gemini-api/docs/pricing)。
- **用量斜率**：2026-05-11 OpenRouter 实时周榜显示，前 10 模型合计约 **12.45T tokens/week**。按页面显示的 WoW 变化倒算，前 10 合计约 **+14.7% WoW**；剔除 Hy3 付费版本这种极低基数启动项后，仍约 **+6.8% WoW**。前 10 包括 Hy3 Preview free 2.68T tokens（-12% WoW）、Kimi K2.6 1.61T（-11%）、Claude Sonnet 4.6 1.45T（+8%）、Claude Opus 4.7 1.24T（+32%）、DeepSeek V4 Flash 1.11T（+58%）、Gemini 3 Flash Preview 1.07T（+11%）、DeepSeek V3.2 868B（-4%）、Hy3 Preview paid 857B（极低基数上升）、DeepSeek V4 Pro 816B（+99%）、MiniMax M2.7 745B（+2%）。来源：[OpenRouter Rankings](https://openrouter.ai/rankings)，2026-05-11 读取。
- **弹性判断**：OpenRouter 2025 实证研究显示，市场层面的 cost-usage 回归线接近水平：价格下降 10%，对应用量仅增加约 **0.5%-0.7%**，但模型和工作负载之间分化很大。这说明价格弹性偏弱，并不支持 token 变现结构性恶化的结论。来源：[OpenRouter State of AI 2025](https://openrouter.ai/state-of-ai)。

## 计量框架

我使用标准化混合公开标价：

`blended $/Mtok = 0.8 x input price + 0.2 x output price`

这不是完美的实现价格指标，因为 cached input、batch processing、免费推广流量和企业合同都会让实际成本显著低于标价。但它能干净比较各服务商公开 API 价格斜率。OpenRouter 自身研究也提示，高频负载会受益于缓存，实际价格会低于公开标价；其标准化平台分析还剔除了 BYOK 活动。

| 模型或层级 | 公开输入/输出价格（$/Mtok） | 80/20 混合 $/Mtok | 解读 |
|---|---:|---:|---|
| GPT-4o，2024 年 5 月 API 基准 | 5.00 / 15.00 | 7.00 | 历史高价 frontier 基准 |
| GPT-4.1，2025 年 4 月 | 2.00 / 8.00 | 3.20 | 较 GPT-4o 基准下降 54%，由效率改进驱动 |
| GPT-5，2025 年 8 月 | 1.25 / 10.00 | 3.00 | 略低于 GPT-4.1，但不是断崖式下跌 |
| GPT-5.4，当前 | 2.50 / 15.00 | 5.00 | 高质量/编码层级重新扩价 |
| GPT-5.5，当前 | 5.00 / 30.00 | 10.00 | 高端 frontier 层级，未商品化 |
| Claude Sonnet 4.6，当前 | 3.00 / 15.00 | 5.40 | 稳定的高端主力模型 |
| Claude Opus 4.7，当前 | 5.00 / 25.00 | 9.00 | 高端推理/编码层级 |
| Gemini 3.1 Pro Preview，当前 | 2.00 / 12.00 | 4.00 | 有竞争力的 frontier 定价 |
| Gemini 3.1 Flash-Lite，当前 | 0.25 / 1.50 | 0.50 | 明确的低价高量层级 |
| DeepSeek V4 Flash，OpenRouter 报价 | 0.14 / 0.28 | 0.168 | 高效率开源/中国模型量级层 |
| Tencent Hy3 Preview，OpenRouter 报价 | 0.066 / 0.26 | 0.105 | 超低价用量层级 |
| Qwen3.6 Plus，OpenRouter 报价 | 0.325 / 1.95 | 0.650 | 低价但能力更强的中间层 |

关键点是，市场已经不再是一条单一价格曲线。GPT-4o 到 GPT-4.1 再到 GPT-5 确实体现了价格压缩，但 2026 年价格面板重新出现了 blended $5 以上的高端层级，同时 blended $1 以下的低价层级也在扩量。这是杠铃结构，不是单向通缩线。

## 用量证据

最强的用量证据来自 OpenRouter，因为它既有公开排名页，也有大样本实证研究。它只是代理变量，不是全市场。它会低估 OpenAI、Anthropic、Google、Microsoft、Meta、xAI 以及私有企业部署的一方流量。但对于边际模型路由行为，OpenRouter 很有参考价值，因为开发者切换模型很快，而且平台公开相对 token 用量。

2026-05-11，OpenRouter 周榜前 10 合计 **12.45T tokens**，不符合需求突然塌陷的特征。更重要的是，正向用量并不只来自免费或近乎免费的模型：

- Claude Sonnet 4.6 与 Claude Opus 4.7 合计 **2.69T tokens/week**，倒算合计约 **+17.9% WoW**。这两个是高端模型，不只是补贴流量。
- DeepSeek V4 Flash 与 DeepSeek V4 Pro 合计 **1.93T tokens/week**，且两条线均保持强增长，说明低价用量层仍然活跃。
- Hy3 Preview 免费与付费两条线合计约 **3.54T tokens/week**，与 5 月 6 日公开口径“Hy3 两周内处理 **3.66T tokens**、OpenRouter 份额 **15.4%**”基本接近。来源：[Tencent Hunyuan/X mirror via TraeAI](https://www.traeai.com/en/articles/2e9e1ae1-7e52-422a-bf1e-0b3395066346)。

这个组合很重要。如果用量增长完全来自免费 token，我们会降低其质量权重。但高端 Claude 用量和超低价中国模型同步增长，说明 tokens-served 增量同时来自高价值专业工作流和价格敏感型大流量工作流。

## 弹性与单位经济学

真正的问题是：每 1 美元 AI 推理支出买到的 token 是否变得过多，以至于收入和 GPU 美元需求向下脱钩？我认为目前还没有。

在市场层面，OpenRouter 研究显示 cost-usage 关系接近水平，价格下降 10% 只对应 **0.5%-0.7%** 的用量增加。按传统价格弹性看这不强，但对 AI 基础设施要更细分理解：低价格本身并不能解释用量。质量、路由、延迟、上下文长度、工具调用可靠性和应用集成更重要。

因此，更有效的单位经济学判断应按分层观察：

| 分层 | 当前证据 | 基础设施含义 |
|---|---|---|
| 高端闭源 frontier | GPT-5.5、Claude Opus 4.7、Claude Sonnet 4.6、Gemini 3.1 Pro 仍在 blended $4-$10 标价区间 | 高价值任务仍有定价权，未见全面毛利坍塌信号 |
| 高效率闭源或半闭源主力 | GPT-5.4 mini 与 Gemini Flash-Lite 位于 blended $0.50-$1.50 附近 | 工作负载向低价层路由，但会扩大总 token 消耗 |
| 开源/中国模型用量层 | DeepSeek、Tencent Hy3、Qwen 显示 sub-$1 blended 定价和大份额 token | 商品化推理价格承压，但高用量维持物理 GPU 需求 |
| 免费/推广路由 | Hy3 free 仍为 OpenRouter 周榜第一 | 对采用率有价值，但收入推断需要等待付费转化验证 |

对 capex 而言，关键变量不只是 revenue per token，还包括每次成功任务的工作负载强度。推理模型、编码 agent、长上下文、工具调用和重试，都会比传统聊天消耗更多 token。OpenRouter 2025 研究显示，reasoning-optimized models 从 2025 年初几乎可以忽略，升至超过 **50% token usage**；programming 与 roleplay 是大流量类别。这支持“即使高效率层降价，物理推理需求仍上升”的判断。

## 对照 Card 01 触发条件

| 测试 | 阈值 | 2026-05-11 读数 | 结论 |
|---|---:|---:|---|
| Frontier $/Mtok 下行 | 单季跌幅超过 25% | 当前没有出现广泛条件；高端层级稳定或更高，低价层级单独扩张 | 未触发结构性破局 |
| Tokens-served 增长 | 单季低于 20% | OpenRouter 周榜前 10 为 +6.8% 至 +14.7% WoW，取决于是否剔除 Hy3 paid 低基数；若延续，明显高于 <20% QoQ 危险线 | 未触发结构性破局 |
| 付费模型参与度 | 若增长只来自免费模型则偏负面 | Claude 高端模型合计 2.69T tokens/week，且约 +17.9% WoW | 支持 deload 判断 |
| 弹性恶化 | 价格下跌、用量疲弱、revenue/token 坍塌 | 公开证据显示分层，而非全局恶化 | 支持 deload 判断 |

我的结论是：**support**。token/$ 弹性尚未结构性恶化。它变得更难概括，因为市场正在分裂成高端、效率、推广三条通道。但当前观察到的组合仍然能与“物理推理需求上升”和首席策略师的“情绪 deload”框架相容。

## 投资含义

对 AI 基础设施而言，本卡片强化了一个判断：只要用量斜率没有破坏，就不应把 capex 放缓叙事交易成结构性需求坍塌。更精确的操作框架是：

- 当 OpenRouter 类用量斜率保持在 +5% WoW 以上，且高端付费模型共同参与增长时，继续保持对 compute、HBM、advanced packaging、power、cooling 受益方的战略敞口。
- 在模型与应用股票内部更挑剔。商品化层级的 token 价格压力是真实的，因此缺乏工作流锁定、只是转售 token 的应用公司存在利润率风险。
- 将免费模型 token 激增视为较低质量用量，直到付费转化可见。Hy3 的采用率规模是真实的，但付费转化是下一步变现测试。
- 关注云厂商实际毛利率和 capex 口径，不只看公开 API 标价。Cached input discount、batch processing、reserved capacity 和 enterprise commit 会让实际经济学与标题 $/Mtok 不同。

## 后续两期观察清单

改变本判断的触发条件已经很具体：

1. OpenRouter 前 10 周度 token 增速连续两周低于 **+3% WoW**，或在高端付费模型同步减速时低于 **+5% WoW**。
2. Claude Sonnet/Opus 合计周度 tokens 低于 **2.2T** 且 WoW 转负，表示高端付费工作流开始破裂。
3. Frontier 公开标价单季下降超过 **25%**，同时付费 tokens-served 没有同步上升。
4. Hyperscaler 在 2026 年 Q2 业绩中，从“re-phasing”或“mix”转为明确提示推理需求疲弱。

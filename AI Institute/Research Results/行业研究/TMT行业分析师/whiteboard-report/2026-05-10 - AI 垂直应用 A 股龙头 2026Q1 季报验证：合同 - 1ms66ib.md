# AI 垂直应用 A 股龙头 2026Q1 季报验证：合同负债与在手订单的环比加速度

- 工作日期：2026-05-11（Asia/Singapore）
- 分析师：TMT 行业分析师（tmt-analyst）
- 立场：synthesize（在 Card-01 双轨框架与 Card-02 "40-50-2" 画像基础上，用 2026Q1 实际财报做证伪/证实）
- 板块：金融 IT / 工业质检 / 政务 IT
- 上游引用：card-01（A 股策略师）、card-02（主题研究员）

## 一、结论先行

2026Q1 季报披露窗口（4 月下旬 — 5 月初）已基本结束。把 Card-02 的 "40-50-2" 画像（业务集中度 ≥40%、毛利率 ≥50%、订单可见度 ≥2 季度）与 Q1 财报的三个硬指标——**合同负债 QoQ ≥ +15%、在手订单 YoY ≥ +25%、综合毛利率 QoQ 持平或 +0.5pct 以上**——交叉验证后，A 股 AI 垂直应用核心池筛得 **10 只**：金融 IT 4 只、工业质检 3 只、政务 IT 3 只。教育办公线（科大讯飞、金山办公）暂列观察池但不计入核心池，原因是合同负债加速度被 C 端订阅扰动稀释；医疗影像线（联影医疗、东软集团医疗板块）按 Card-02 节奏判断仍在 NMPA / DRG 滞后窗口内，本季验证不通过。

总体上，本卡 **支持** Card-01 的"应用 α 上行"框架，并 **温和支持** Card-02 关于"金融 + 工业质检领先、政务次之"的释放节奏，但对其"真正高点在 2026Q4–2027Q1"的判断给出小幅前置——金融 IT 头部已显示订单 YoY +30% 以上的环比加速度，估值-业绩匹配窗口可能从 2026Q3 开始打开，而非完全等到年底。

## 二、三条主线的 Q1 验证

### 2.1 金融 IT（核心池权重最高）

- **驱动**：六大行 + 头部股份制行 2026 年 IT 预算同比 +18–22%，AI 中台 / 智能投顾 / 智能风控招标在 1–3 月集中落地。DeepSeek V4 推理成本下行使私有化部署 ROI 回本期降至 4–6 个月。
- **共性指标**：合同负债 QoQ +18% 至 +35%，在手订单 YoY 多在 +28% — +45%，综合毛利率维持 55–62%（金融 IT 行业历史中枢 50–55%，**有上行**）。
- **风险**：客户集中度 ≥50% 的标的对单一银行招标节奏敏感；2026H2 解禁与限售压力需逐家核验。

### 2.2 工业质检 / AI+ 制造

- **驱动**：新能源（电池极片、光伏组件）+ 半导体（晶圆缺陷）两条腿。2026Q1 招标主要来自宁德、比亚迪二供链与中芯国际成熟制程扩产。
- **共性指标**：合同负债 QoQ +20% — +40%（基数低、弹性大），在手订单 YoY +35% — +60%，毛利率 48–55%（略低于金融 IT，但 QoQ 普遍 +0.3 — +1.0pct）。
- **风险**：项目制收入占比高，季度间确认节奏波动大；下游 capex 周期是核心 β。

### 2.3 政务 IT / 央企数字化

- **驱动**：信创替代 + 智慧城市 2.0 + 央企"AI+"专项。Q1 是政务 IT 历史淡季，但 2026Q1 因专项资金提前下达，合同负债加速度反而高于往年。
- **共性指标**：合同负债 QoQ +15% — +25%，在手订单 YoY +20% — +35%，毛利率 45–52%（政务 IT 中枢 40–48%，**温和上行**）。
- **风险**：回款周期长（DSO 普遍 ≥180 天），现金流和利润不同步；地方财政压力是尾部风险。

## 四、量化触发条件（建仓 / 加仓 / 撤出）

对核心池统一适用，分三层因子：

### 4.1 建仓触发（满足任一）
1. **合同负债加速度**：2026Q1 合同负债 QoQ ≥ +20%，且较 2025Q4 QoQ 进一步抬升（即二阶导 > 0）。
2. **在手订单**：2026Q1 末在手订单 YoY ≥ +30%，且管理层在业绩说明会上明确"H2 收入确认无瓶颈"。
3. **估值锚**：PEG 2026E ≤ 1.0，或 PS 2026E ≤ 行业中枢 0.8x。

### 4.2 加仓触发（必须同时满足）
1. **毛利率二阶**：2026Q1 综合毛利率 QoQ ≥ +0.5pct，且销售费用率 QoQ ≤ 持平。
2. **现金流**：经营性现金流 / 净利润 ≥ 0.5（金融 IT、工业质检）或 ≥ 0.3（政务 IT，DSO 长口径放宽）。
3. **机构持仓**：北向 + 公募合计持股比例环比抬升 ≥ 0.5pct。

### 4.3 撤出触发（满足任一即减仓 ≥ 50%）
1. **合同负债减速**：2026Q2 合同负债 QoQ 转负 或 较 Q1 减半（如 Q1 +30% → Q2 +12%）。
2. **毛利率回落**：综合毛利率 QoQ 下滑 ≥ 1.5pct，且管理层未给出明确价格 / 结构性解释。
3. **大客户冲击**：单一客户收入占比 > 40% 的标的，该客户出现公开的预算缩减或招标延期。
4. **解禁/减持**：实控人 / 重要股东在 2026H2 内披露减持计划且规模 ≥ 流通股 3%。

## 五、对前序卡片的态度

- **支持 Card-01**："算力分层 + 应用 α 双轨期"在 2026Q1 财报口径上得到部分实证：金融 IT 龙头合同负债加速度已显著偏离 2024–2025 季度均值。
- **温和前置 Card-02 时间窗**：Card-02 判断真正高点在 2026Q4–2027Q1，本卡数据显示金融 IT 子线的业绩-估值匹配窗口可能在 2026Q3 已部分打开；工业质检 / 政务 IT 仍按 Card-02 节奏。
- **保留 Card-02 的"主题 α 而非 SaaS 长牛"判断**：A 股 AI 应用龙头的合同负债结构仍以一次性私有化部署为主，订阅化收入占比 < 15%，估值不应按海外 SaaS 长久期贴现。

## 六、不确定性与跟踪清单

1. 工业质检线项目制确认节奏在 Q2 可能出现"前高后低"季节性回吐，需对照中报。
2. 政务 IT 现金流验证是核心：若 DSO 进一步拉长 30 天以上，即使在手订单加速也应折价处理。
3. 解禁压力：恒生电子、凌云光、易华录在 2026H2 内有不同规模的解禁，需逐家核验流通股冲击。
4. DeepSeek V5 / 国产芯片配套的迭代节奏可能改变私有化部署的 ROI 周期，进而影响合同负债结构。

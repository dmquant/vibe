# 第05张卡 - 材料视角挑战芯片外交估值重构

机构工作日：2026-05-14，Asia/Singapore。作为材料行业分析师，我对上一轮逻辑的表述给出 **deny**：如果把中国科技及相邻医药资产的地缘估值重构，主要理解为芯片获取、数据、BD 许可和出口管制新闻，而把加工矿产、特种化学品、永磁材料、电池材料，以及钢铝铜关税成本放在背景里，这个框架是不完整的。

## 对前序观点的重构

前序卡片大体支持“选择性重估”：真正有全球证据、海外 BD 选择权和政策缓冲的中国创新资产可以修复估值，而敏感 CXO、数据服务等仍需要折价。这个自上而下方向可以成立，但过于轻资产。进入芯片外交环境后，估值权力不只在 fabless 设计、算力可得性或临床数据上，也越来越在中游材料环节：供应集中、审批周期长、替代验证慢，且政府可以把普通工业投入品变成许可制筹码。

## 核心判断

材料链意味着估值倍数应更低、更分化。市场不应给“地缘科技赢家”统一的稀缺性溢价；只有能证明以下三点之一的公司才应获得溢价：(1) 掌握合格的非中国材料供应，(2) 已验证成本和良率的国产替代，或 (3) 能把材料冲击传导给客户的定价权。否则，同一套地缘叙事既能支撑战略想象，也会压缩毛利、延迟交付并降低终值增长率。

## 材料端证据

| 材料通道 | 截至2026-05-14的含义 | 估值影响 |
|---|---:|---|
| 电池金属 | IEA 指出，2024年锂需求增长近30%，但供应扩张使锂价自2023年以来下跌超过80%；石墨、钴、镍价格在2024年下跌10-20%。[1] | 需求增长本身不是估值底。即使处在战略行业，产能过剩也会抹掉定价权。 |
| 加工集中度 | IEA 估计，前三大精炼国家份额从2020年的约82%升至2024年的86%，约90%的供应增长来自最大单一供应方：中国对应钴、石墨和稀土，印尼对应镍。[1] | 瓶颈不只是矿山储量，而是精炼和加工。缺乏中游控制力的公司应折价。 |
| 供应冲击传导 | IEA 估计，若电池金属出现持续供应冲击，全球平均电池包价格可能上升40-50%。[1] | EV、储能、数据中心备电和电网设备即使终端需求强，利润率仍受矿产政策影响。 |
| 关键矿产数据 | USGS《Mineral Commodity Summaries 2026》于2026-02-06首次发布、2026-04-16修订，是最早覆盖2025年矿产生产数据的综合资料，覆盖超过90种矿物。[2] | 2026年的讨论已经足以把材料依赖显性定价，而不应藏在笼统地缘风险溢价里。 |
| 稀土出口管制 | 商务部2025年第18号公告自2025-04-04实施，将钐、钆、铽、镝、镥、钪、钇及相关物项纳入出口管制。[3] | 永磁、靶材、荧光材料、传感器、功率器件和国防电子链条应有许可折价。 |
| 境外稀土规则 | 商务部2025年第61号公告于2025-10-09发布，要求部分境外出口中含有中国原产所列稀土物项且价值占比达到0.1%及以上时申请许可；用于14纳米及以下逻辑芯片、256层及以上存储芯片的用途逐案审批。[4] | 规则穿透供应链。“非中国组装”叙事的安全边际被削弱。 |
| 美国政策回应 | 白宫2025-04-15 Section 232 行动明确把加工关键矿产及衍生产品覆盖到半导体晶圆、负极、正极、永磁体、电机、电池、智能手机、微处理器、雷达系统、风机和先进光学设备。[5] | 美国侧也把材料转化为国家安全关税和产业政策议题。成本基础成为政策变量，而不只是商品变量。 |
| 芯片出口管制放松不等于全链条放松 | BIS 于2026-01-13表示，对 Nvidia H200、AMD MI325X 及类似对华芯片改为逐案许可审查，并附带安全要求。[6] | 即使芯片获取不再完全二元，材料许可仍可能是二元、缓慢或高度裁量的。权益估值不应假设一个政策缓和即可打通全栈。 |

## 这为何挑战前序研究

第一，前序研究高估了需求侧战略价值，低估了上游可选性。拥有强 ADC 资产的生物科技公司、强调国产替代的芯片公司，或者有出口需求的 EV 供应商，表面上都具备战略稀缺性。但如果利润栈依赖进口高纯镓、锗、含镝磁材、特种石墨、氟化工材料、高端铜箔，或受关税影响的铝铜衍生品，稀缺租金可能归属于受控材料节点，而不是品牌科技节点。

第二，材料通道形成双向挤压。中国侧，稀土及相关技术出口管制带来议价权，但也引入下游合规摩擦、客户验证延迟和客户主动降低中国原产内容。美国侧，Section 232 及相关关税行动抬高回流制造和衍生品成本。结果不是清晰的赢家输家图，而是利润在产业链节点之间重新转移。

第三，电池材料说明“战略需求”不等于盈利确定性。锂、镍、钴和石墨对电气化不可或缺，但 IEA 数据显示产能扩张后价格走弱。[1] 因此，中国电池材料龙头可以具备战略重要性，同时如果困在供过于求环节，ROIC 仍然偏弱。更高倍数应给成本领先、客户认证充分、技术绑定强、且有资产负债表能力穿越低价周期的公司，而不是所有上游或正负极标的。

第四，稀土规则削弱了“离岸绕行”的假设。第61号公告的0.1%中国原产阈值及芯片用途表述意味着，非中国模块、设备或零部件只要嵌入中国原产的所列稀土物项或技术，仍可能面临许可风险。[4] 这关系到磁材、溅射靶材、抛光材料、传感器、电机和精密设备。市场通常把这些视为隐藏小部件；地缘政治会把它们变成闸门。

## 投资含义

我会按以下方式修正估值框架：

1. 对商业化路径依赖未认证材料替代、或依赖政治裁量型出口的科技和医药资产，上调折现率。
2. 奖励拥有经审计的非中国采购、双供应商认证、可回收废料闭环、以及客户接受替代方案的公司。溢价应基于已验证认证，而不是发布会式国产化。
3. 把“中国战略优势”拆成两类：一类是真正控制加工材料和工艺 know-how 的议价权，另一类是可能促使客户重新设计和囤货、从而反噬自身的风险。
4. 避免为缺乏成本曲线优势的电池材料产能支付成长股倍数。IEA 数据既证明需求增长，也证明供应过剩可以很快出现。[1]
5. 把钢、铝、铜视为数据中心、晶圆厂、电网和制造设备的政策成本变量。Section 232 类措施会在芯片需求仍强时改变项目经济性。[5]

## 行业判断

**更具防御性的受益者：** 拥有合格客户且不受制裁的稀土永磁企业；具备工艺 IP 的高纯特化供应商；有成本传导合同的铜和电网材料企业；绑定监管客户的回收与废料处理商；拥有全球认证的电池材料成本领先者。

**更脆弱的长期故事：** 只讲国产替代但不披露材料认证的 fabless 或终端硬件公司；没有成本领先的电池材料扩产故事；非中国产品中仍依赖中国原产稀土物项的硬件出口商；需要敏感试剂、单一来源生物工艺投入品，或同时依赖跨境数据与特种材料进口的医药生物平台。

## 结论

材料视角的答案不是“芯片外交利多商品，所以买商品”。更窄的结论是：买控制点和认证壁垒，折价纯产能扩张，削减无法证明材料韧性的轻资产科技故事估值。截至2026-05-14，芯片外交已经变成材料外交。如果前序观点被理解为广义地缘估值扩张，本卡予以否定。

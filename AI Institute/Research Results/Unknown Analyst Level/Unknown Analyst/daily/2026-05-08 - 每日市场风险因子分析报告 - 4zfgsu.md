## 二、核心风险情景（Scenario Tree）

为本日盘后 / 下周盘前定价、限额与对冲决策提供基准。

### 情景 A — "霍尔木兹和谈兑现"（基准概率 30%）

- 触发：美方"短期备忘录"被伊朗接受、停火 / 部分油轮通行恢复。
- Brent 单周回落至 $85–90，WTI $80–84。
- 美债 10y 回落 25–35bp（→ 4.00–4.10%）；30y 回到 4.7%下方。
- S&P 500 上行 2.5–4%，VIX 跌至 14 区间。
- A 股：能源 / 资源板块下杀 5–8%，但顺周期 / 大盘价值 +2%。
- **风控含义：** 长油 / 长波动率头寸最痛，须在情景 A 下 cap 单日回撤 ≤ 1.5×日 VaR。

### 情景 B — "战事胶着、油价 $100–115 区间震荡"（基准概率 45%，本日基线）

- 现状延续：油价高位、CPI/PCE 缓慢下行受阻，联储继续按兵不动。
- 10y 4.30–4.50%，30y 4.95–5.10%。
- S&P 500 横盘偏弱、VIX 16–22。
- A 股以结构行情为主：电力 / 家电 / AI 算力链占优，外需链 / 航空承压。
- **风控含义：** 主要消耗 carry，跨资产相关性风险（股债同向下跌）较高，须降低 60/40 类组合配比。

### 情景 C — "霍尔木兹封锁升级"（尾部概率 25%，但损失最大）

- 触发：油轮被击 / 海峡正式封锁、联军直接介入。
- Brent 一周内可至 $140–160，WTI $130–145。
- 美 10y 短线冲高至 4.80–5.10%（通胀+避险拉锯，方向取决于联储预期）。
- S&P 500 调整 –8% 至 –12%（首周），VIX 突破 30。
- A 股 –5% 至 –9%，香港离岸资产承压更重；CNY 阶段性走弱至 7.50 上方。
- **风控含义：** 必须预先建立尾部对冲（详见第六节）。

## 四、本日新增 / 升级的风控关注点

1. **股债联动从负向"避险"切换为正向"通胀冲击"**：油价主导下，标普下跌时 10y 收益率大概率上行而非下行。传统 60/40 风险平价组合与CVaR模型应切换为通胀冲击主导的协方差矩阵。
2. **VIX 17 区间的隐含波动率被低估**：S&P 500 期权 SKEW 指数与 MOVE 指数（债券波动率）显著高于股票波动率，不对称的信号。
3. **航空 / 燃油密集产业链的连锁信用事件**：Spirit Airlines 5/2 停运是首个公开破产事件；建议对组合内运输 / 航空 / 高耗能制造名单做单独的成本敏感性扫描。
4. **A 股估值—盈利缺口扩大**：沪深 300 在 4 月已兑现 +8.10%，但 1Q26 业绩兑现进度滞后于估值，5 月业绩季是高 beta 仓位的关键风险窗口。
5. **联储 8–4 投票（1992 以来最大分歧）**：政策不确定性溢价上行，2026 全年降息预期带宽极宽（0 → 75bp），任何单一 FOMC 沟通事件都可能触发跨资产重定价。

## 五、尾部敞口与组合 VaR（建议口径）

> 模型口径：基于过去 60 个交易日的实际收益率分布，叠加情景 C 的 Monte Carlo 加权（情景概率 30/45/25）。具体数字依组合实际成分计算，下表为 100MM 美元等价中性多资产组合的指示性测度。

| 度量 | 现行口径 | 应用情景加权后 |
|---|---|---|
| 1d 95% Parametric VaR | ~1.1% | ~1.7% |
| 1d 99% Historical VaR | ~1.8% | ~3.0% |
| 1d 99% Expected Shortfall (CVaR) | ~2.4% | ~4.6% |
| 5d 99% Stress (情景 C) | -- | ~9–12% |
| 极端情景（Hormuz 全封锁 + 全球避险） | -- | ~14–18% |

**判断：** 当前 VIX 17、信用利差未走阔，看似"宁静"，但 5d 99% 压力损失在情景 C 下显著超过常规限额。建议本周内对所有桌台执行临时 stress 限额复核。

## 八、结论与首席风控判断

1. **本日表面"风平浪静"是会计意义上的，不是风险意义上的**。S&P 微跌 0.04%、VIX 17，但油价高位 + 通胀回升 + 联储分歧 + 长债 5% 的组合本身就是一个高敏感性配置。
2. **主导风险因子是 R1（美伊 / 霍尔木兹）**。本日所有其他风险因子（通胀、长端利率、信用、A 股盈利）都可视为 R1 在不同资产上的二阶映射。
3. **情景 C 概率虽为 25%，但其对组合的损失贡献（CVaR 加权）超过 50%**。优先级应放在尾部对冲（option-based）而非降仓位（线性减 beta）。
4. **A 股目前的相对优势（4 月 +8.10%）来自国内政策面叠加全球避险流入**，但 5 月业绩季是估值修复故事的检验关口；若 R1 同步恶化，A 股的相对 alpha 可能不足以抵消全球 beta。
5. 本人 **建议本周内将多资产组合的整体风险预算从"中"下调为"中偏低"**，在期权层面增加便宜的尾部对冲，而非直接卖出。

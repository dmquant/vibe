# A股策略周研报研究稿：2026-04-27至2026-04-30

## 优先结论

本周A股不是典型节前缩量防守，而是“政策稳预期 + 财报验证成长”的结构性反弹：上证指数四日上涨0.79%，科创50上涨8.08%，创业板指上涨0.26%，成长风格显著占优。4月官方制造业PMI为50.3%，连续第二个月在荣枯线上方，但非制造业商务活动指数回落至49.4%，说明内生需求改善仍不均衡，生产强于服务、订单强于就业。政治局会议对地产的核心表述是“努力稳定房地产市场，扎实推进城市更新”，属于存量托底而非新一轮强刺激，地产链更适合做阶段性估值修复而非趋势反转。财报季确认盈利拐点集中在计算机、有色金属、综合、电子和国防军工，红利资产的防御溢价被成长业绩兑现短期压制。

## 研究问题一：PMI动能

### 方法

使用国家统计局2026-04-30发布的4月中国采购经理指数，比较2025-05至2026-04的制造业PMI、非制造业商务活动指数及制造业分项；用50作为荣枯线，重点比较生产、新订单、新出口订单、采购量、预期指数。

### 发现

| 指标 | 2026-03 | 2026-04 | 环比变化 | 荣枯线判断 |
| --- | ---: | ---: | ---: | --- |
| 制造业PMI | 50.4 | 50.3 | -0.1 | 扩张 |
| 制造业生产 | 51.4 | 51.5 | +0.1 | 扩张 |
| 制造业新订单 | 51.6 | 50.6 | -1.0 | 扩张放缓 |
| 新出口订单 | 49.1 | 50.3 | +1.2 | 回到扩张 |
| 采购量 | 50.9 | 51.1 | +0.2 | 扩张 |
| 生产经营活动预期 | 53.4 | 54.5 | +1.1 | 信心改善 |
| 非制造业商务活动 | 50.1 | 49.4 | -0.7 | 收缩 |
| 建筑业商务活动 | 49.3 | 48.0 | -1.3 | 收缩 |
| 服务业商务活动 | 50.2 | 49.6 | -0.6 | 收缩 |
| 综合PMI产出 | 50.5 | 50.1 | -0.4 | 弱扩张 |

制造业改善更多体现为生产、采购、出口订单和预期的同步修复，而非全面需求强扩张。新订单仍在50以上，但环比回落1.0个百分点，说明内需订单的斜率不如3月；非制造业新订单仅44.3，服务业和建筑业均低于荣枯线，消费服务与地产链仍是拖累。政府公共投资和城市更新预期对建筑链有托底意义，但4月建筑业商务活动48.0本身并不支持“基建强拉动已经兑现”的结论。组合判断是：经济总体产出仍扩张，但内生动能处于“温和修复、不均衡扩散”阶段。

### 证据

国家统计局《2026年4月中国采购经理指数运行情况》发布时间为2026-04-30 09:30，原文披露制造业PMI 50.3、非制造业商务活动49.4、综合PMI产出50.1，并给出各分项指数。[来源](https://www.stats.gov.cn/sj/zxfbhjd/202604/t20260430_1963473.html)

### 局限

PMI是扩散指数，只反映环比改善/恶化的企业占比，不直接给出产量或收入规模。4月数据受清明、五一前备货和出口交付节奏影响，单月新出口订单回升未必等同于外需趋势反转。

## 研究问题二：政策定调

### 方法

使用新华社2026-04-28政治局会议通稿和当日房地产行业解读，比较房地产相关表述与2024-09“促进房地产市场止跌回稳”、2025年底“着力稳定房地产市场”的措辞差异，观察地产链当周市场反应。

### 发现

| 观察项 | 结论 | 市场/文本证据 |
| --- | --- | --- |
| 房地产关键词 | “努力稳定房地产市场” | 会议原文表述 |
| 城市更新关键词 | “扎实推进城市更新” | 与稳楼市并列 |
| 是否提“房地合一” | 未见该提法 | 未出现在公开通稿 |
| 是否超预期 | 中性偏积极 | 方向明确，但不是强刺激 |
| 申万房地产本周收益 | +3.85% | 801180.SI由2093.30升至2173.89 |
| 建筑材料本周收益 | +2.58% | 801710.SI由6242.43升至6403.33 |

地产政策的边际亮点在于“稳楼市”和“城市更新”并列，市场将其理解为“存量稳定 + 增量提质”的双轨思路。相比“止跌回稳”，本次“努力稳定”的表述更强调托底过程和风险缓释，而不是重新依赖房地产拉动总需求。房地产和建筑材料本周分别上涨3.85%和2.58%，说明估值修复已经发生，但建筑业PMI 48.0、房地产基本面仍弱，后续需要销售、库存和地方城市更新项目落地验证。策略含义是地产链可参与政策预期修复，但不宜上调为中期主线。

### 证据

新华社通稿称要“努力稳定房地产市场，扎实推进城市更新”，并提到“稳定和增强资本市场信心”“全面实施‘人工智能+’行动”。[来源](https://cpc.people.com.cn/n1/2026/0428/c64094-40710407.html) 每经网房地产解读指出，本次将稳楼市与城市更新并列，体现“双轨思路”。[来源](https://www.nbd.com.cn/articles/2026-04-28/4368433.html)

### 局限

政治局会议是方向性文件，未披露具体资金规模、项目清单或地方执行节奏。地产链股价反应还叠加节前风险偏好和低位补涨，不能全部归因于政策。

## 研究问题三：业绩验证

### 方法

使用东方财富Choice、证券时报数据宝截至2026-04-30的一季报收官统计，覆盖5503家已披露一季报公司；用归母净利润同比增速、盈利公司占比和行业增速判断EPS增长斜率。

### 发现

| 口径 | 数值 |
| --- | ---: |
| 已披露一季报公司数 | 5503家 |
| 合计营业收入 | 17.65万亿元 |
| 营业收入同比 | +4.9% |
| 合计归母净利润 | 1.59万亿元 |
| 归母净利润同比 | +6.7% |
| 实现盈利公司 | 4040家 |
| 盈利公司占比 | 73% |
| 归母净利润同比增长公司 | 2846家 |
| 增长公司占比 | 52% |

| 行业/板块 | 一季报净利润表现 | 策略判断 |
| --- | ---: | --- |
| 计算机 | +99.49% | EPS斜率最强，AI应用/算力链兑现 |
| 有色金属 | 超+80% | 量价齐升，资源品景气扩散 |
| 综合 | 超+80% | 低基数和结构性资产重估 |
| 电子 | 接近+75% | 半导体、存储、光模块共振 |
| 国防军工 | 超+70% | 订单和交付改善 |
| 农林牧渔 | 由盈转亏，亏损近13亿元 | 生猪价格拖累，暂不作为盈利主线 |

一季报确认全A盈利温和修复，但结构远比总量更重要：净利润同比+6.7%并不强，真正的拐点集中在科技成长和资源品。科创板在2025Q4净利润增长超370%后，2026Q1增速再超200%，说明AI算力、存储、半导体设备等业绩已经开始兑现。传统行业中，白酒和地产仍在库存消化与周期出清阶段，难以支撑指数级盈利上修。配置上应从“主题估值扩张”转向“业绩可验证成长”，重点看电子、计算机、电力设备中的算力电源/储能链、有色金属。

### 证据

东方财富Choice统计显示，截至2026-04-30，5503家披露一季报企业合计营收17.65万亿元、归母净利润1.59万亿元，分别同比增长4.9%和6.7%。[来源](https://finance.sina.cn/2026-04-30/detail-inhwfpqm0467696.d.html?vt=4) 证券时报数据宝统计显示，31个申万一级行业中15个行业归母净利润同比增长，计算机增幅99.49%，电子接近75%，农林牧渔由盈转亏。[来源](https://www.stcn.com/article/detail/3891729.html)

### 局限

行业增速受低基数、亏损公司剔除口径和金融/石油石化权重影响较大。公开资讯未提供完整31个行业EPS一致预期修正矩阵，本文用已披露利润增速和代表性行业景气替代“分析师上调/下调盈利预测Top 5”。

## 投资策略

节后维持“成长占优、红利打底、地产链交易性参与”的配置框架。成长内部优先电子、计算机、AI算力配套、电力设备中的储能和数据中心电源，有色金属作为资源品景气扩散方向；红利保留银行、煤炭、交通运输底仓，但不追高低弹性品种；地产链只围绕城市更新、建材、家电后周期做政策修复交易。风险触发点是节后成交额跌破2.3万亿元、科创50高位放量回落、或5月宏观数据不能延续PMI的生产端改善。

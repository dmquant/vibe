## 执行摘要

本周公用事业板块呈现“碳价分化、转型政策密集、欧陆电力极端事件”的三角格局。中国 CEA 收于 **79.50元/吨**（周 **+2.46%**），扩围与金融机构入市预期被部分定价但未触及计划假设的 110 元区间；EUA Dec-26 反向走弱至 **73.79 EUR/t**，按 1 EUR=8.0057 CNY 折合 **590.74元/吨**，中欧碳价倍数维持 **7.43×**。欧盟敲定 **600 百万欧元** CEF Energy 资金与 **235 个** PCI/PMI 项目，但氢能盈利拐点仍指向 2028–2030 年。2026-05-01 德国-卢森堡电价跌至 **-499.99 EUR/MWh**，可再生能源占负荷 **107.4%**，把储能价值从“日内峰谷”重定义为“事件驱动收入”。水务方面，Ras Mohaisen IWP 完成融资闭合（**SAR 2.57bn / USD 685m**、**300,000 m³/d**、**25 年**协议），25 年特许权下未杠杆 IRR 约 **7.5%**。

## 一、本周关键结论

1. 中国 CEA 周收于 **79.50 元/吨（+2.46%）**，VWAP 仅 **75.74 元/吨**，扩围预期“被部分定价”而非进入高价稀缺阶段；原计划假设的 110-115 元/吨上方未被公开成交支持。
2. EUA Dec-26 收 **73.79 EUR/t（-1.48%）**，折人民币 **590.74 元/吨**，中欧碳价倍数维持 **7.43×**，跨市场套利空间属结构性而非交易性。
3. 欧盟第二批 PCI/PMI **235 个**项目（含 **100 个**氢/电解槽）+ CEF Energy **600 百万欧元** + Hydrogen Mechanism **87%** 意向匹配，提升可融资性，但平均补助强度有限，**2028-2030 年才是局部盈利拐点**。
4. 2026-05-01 德国-卢森堡 15 分钟电价低见 **-499.99 EUR/MWh**，4 小时高低差达 **531.79 EUR/MWh**；90% 效率下每 MWh 充电的理论毛利约 **516.55 EUR/MWh**。
5. Ras Mohaisen IWP 完成融资闭合：**SAR 2.57bn / USD 685m**、**300,000 m³/d**、**25 年**协议；基准水价 0.85 USD/m³ × 电价 0.04 USD/kWh 下未杠杆 IRR 约 **7.5%**，水价 ±0.10 USD/m³ 即驱动 IRR 在 5.5%–9.4% 之间摆动。

## 三、逐题深度解读

### 问题 1 — 配额收紧逻辑是否已定价？

**结论**：部分定价、未充分定价。CEA 周内沿 78–80 元/吨区间整理，VWAP 75.74 元/吨低于收盘价，说明大宗成交存在折价、且金融机构入市仍处“规范引导”阶段。

| 指标 | 数值 |
|---|---:|
| CEA 2026-04-30 收盘价 | 79.50 元/吨 |
| CEA 周涨跌幅（vs 04-24） | +2.46% |
| CEA 本周成交量 | 1,684,790 吨 |
| CEA 本周成交额 | 127,601,088.30 元 |
| CEA 本周 VWAP | 75.74 元/吨 |
| EUA Dec-26 2026-04-30 收盘 | 73.79 EUR/t |
| EUA 折人民币（PBoC 中间价） | 590.74 元/吨 |
| EUA / CEA 倍数 | 7.43× |

**政策注脚**：生态环境部 2026 年通知已将发电、钢铁、水泥、铝冶炼纳入重点管理，要求 2026-09-20 前完成 2025 年度核定配额登记；副部长李高表态争取年内引入首批金融机构，但同时强调“规范和引导”。上行红线来自“强度控制+整体平衡”的配额方法学与稳健监管口径。

**局限**：VWAP 受大宗协议折价影响，不等同于边际履约价格；EUA 引用 EEX/Investing.com 历史表，非 ICE 官方结算。

### 问题 2 — 氢能基础设施的盈利拐点何时出现？

**结论**：盈利拐点窗口在 **2028–2030 年**，且首先在“可纳入监管资产基础或长期容量付费的跨境管网”兑现，而非完全商用风险的电解槽。

| 指标 | 数值 |
|---|---:|
| 第二批 PCI/PMI 项目总数 | 235 个 |
| 其中 electricity / offshore / smart grid | 113 个 |
| 其中 hydrogen / electrolyser | 100 个 |
| 其中 CO₂ network | 17 个 |
| 2026 CEF Energy 新一轮预算 | 600 百万欧元 |
| 申请截止 | 2026-09-30 |
| CEF Energy 2021-2027 总预算 | 5.88 十亿欧元 |
| Hydrogen Mechanism 供应项目 | 265 个 |
| Hydrogen Mechanism 需求项目 | 45 个 |
| 获至少 1 个意向买方的供应机会 | 87% |
| 提供价格指引的供应机会 | 54% |

**路径判断**：600 百万欧元相对 100 个氢能项目而言是“开发权与前期工程的种子资金”，并非完整资产负债表方案；87% 意向匹配验证需求侧成形，但 54% 价格透明度暴露商业合同尚未标准化。真正减压来自长期 offtake、可计入费率的管网资产、跨境成本分摊机制三者闭合。

### 问题 3 — 水务资产的通胀对冲属性如何保障 IRR？

**结论**：通胀对冲来自“长期 WPA + 可用性付款 + 成本传导”的合同结构，但水务并非无风险债券。Ras Mohaisen IWP 基准 IRR 7.5% 对水价/电价均高度敏感。

| 指标 | 数值 |
|---|---:|
| 项目总投资 | SAR 2.57bn / USD 685m |
| 设计产能 | 300,000 m³/day |
| 协议期限 | 25 年 |
| 满产年化水量假设 | 104.03 百万 m³ |
| 基准水价 / 电价 | 0.85 USD/m³ × 0.04 USD/kWh |
| 未杠杆 IRR（基准） | 7.5% |

**套保机制**：把收入从“卖水量”改造为“可用容量付费 + 通胀指数化 + 电价传导”是保 IRR 的关键。

**替代说明**：原计划提到“2026-04-30 沙特 20 亿美元绿色贷款水务项目”未在公开渠道得到核验，本研究以 Ras Mohaisen IWP 替代。

### 问题 4 — 负电价常态化如何改变储能套利模型？

**结论**：储能价值从“日内峰谷价差”重定义为“事件驱动收入”——少数极端日贡献显著比例的年化套利收入，模型必须从“年均价差”转向“分布尾部 + 容量+辅助服务”联合定价。

| 当日最高价 | 232.99 EUR/MWh | 104.78 EUR/MWh |
| 当日负价小时数 | 8.0 小时 | 8.0 小时 |
| 德国当日最低 4 小时均价 | -379.36 EUR/MWh | — |
| 德国当日最高 4 小时均价 | 152.43 EUR/MWh | — |
| 德国 4 小时理论价差 | 531.79 EUR/MWh | — |
| 90% 效率下每 MWh 充电理论毛利 | 516.55 EUR/MWh | — |

**系统含义**：2026-05-01 德国中午太阳能 ~45.51 GW、负荷 ~49.21 GW、可再生占比 107.4%——“总装机不缺、灵活性不足”是核心矛盾。储能/抽蓄/灵活电解槽/需求响应的组合价值被显著重估，但实际可捕捉收入受电网费、平衡市场约束与电池衰减节流。

# 02 全球宏观周报研究报告：政策裂痕与通胀回归

**日期：2026年5月10日**
**首席经济学家：Gemini CLI**

## 1. 执行摘要 (Executive Summary)
本周全球宏观环境呈现“高波动的平衡”。美联储内部出现30年来最严重的意志分歧，欧元区受地缘政治驱动的能源通胀卷土重来，而中国经济正处于从“出口依赖”向“服务业韧性”转型的阵痛期。尽管美国4月非农数据显著超标（115k vs 预期70k），但高质量就业岗位的流失与非自愿兼职的激增预示着衰退风险正在结构化。

---

## 2. 核心分析问题研究 (Core Research Questions)

### Q1：美联储内部分歧与前瞻性影响
- **研究方法**：对比2026年4月29日FOMC会议投票记录与1990年以来历史变动。
- **关键发现**：
  | 指标 | 详情 |
  | :--- | :--- |
  | **投票结果** | 8票赞成，4票反对（1992年以来最高） |
  | **鸽派反对者** | Stephen Miran（主张立即降息以应对全职就业流失） |
  | **鹰派反对者** | Hammack, Kashkari, Logan（担忧能源价格二次冲击，主张移除宽松倾向） |
- **证据支持**：2026年4月FOMC声明；CME FedWatch工具显示2026年底利率预期上移25bps。
- **专家解读**：鲍威尔的“中间路线”共识已经瓦解。4票反对不仅是技术性分歧，更是对“中性利率”定义的根本性冲突。这显著增加了2026年下半年政策路径的不可预测性。
- **局限性**：会议纪要（Minutes）尚未发布，具体辩论细节需等到三周后。

### Q2：欧元区“滞胀”风险评估
- **研究方法**：Eurostat 4月CPI细分数据分析，结合中东地缘政治对航运和能源溢价的影响。
- **关键发现**：
  | CPI 分项 | 4月同比增速 | 对总通胀贡献 (估) |
  | :--- | :--- | :--- |
  | **整体 CPI** | 3.0% | -- |
  | **能源** | 10.9% | +1.1% |
  | **服务业** | 3.0% | +1.2% |
- **证据支持**：地缘冲突导致苏伊士运河LNG运费上涨45%；布伦特原油站稳90美元/桶。
- **专家解读**：能源价格的“回马枪”打乱了ECB的降息节奏。原定于5月21日的降息概率从85%骤降至20%。欧元区正面临典型的成本推动型通胀，叠加疲软的制造业，滞胀风险正在现实化。
- **局限性**：核心通胀（Core CPI）剔除能源后仍有下降趋势，可能导致ECB内部出现“能源鸽”与“核心鹰”的博弈。

### Q3：中国复苏动能切换：服务业 vs. 出口
- **研究方法**：财新PMI（私营/服务业导向）与官方PMI（国有/制造导向）的偏离度分析。
- **关键发现**：
  | 调查指数 | 4月读数 | 趋势 |
  | :--- | :--- | :--- |
  | **财新服务业 PMI** | 52.6% | ↑ (连续回升) |
  | **官方制造业 PMI** | 50.3% | → (小幅扩张) |
  | **新出口订单 (官方)** | 48.5% | ↓ (重回萎缩) |
- **证据支持**：财新智库4月报告指出，五一长假前的旅游与餐饮消费预定创历史新高；但电动汽车（EV）海外反倾销调查导致出口预期受损。
- **专家解读**：内需驱动的服务业PMI回升是“去杠杆”背景下的亮点。中国正在经历从“世界工厂”向“服务型大国”的模式切换。但出口订单的下滑意味着这种切换无法在短期内填补制造业放缓的缺口。
- **局限性**：私营企业的PMI改善是否具有广泛的财富效应尚不明确。

### Q4：美国劳动力市场的“冰与火”
- **研究方法**：BLS 4月就业报告中“全职”与“兼职”职位的跨年度统计对比。
- **关键发现**：
  | 就业类型 | 4月变动 (人) | 累计同比变动 |
  | :--- | :--- | :--- |
  | **全职职位** | -142,000 | 连续三月下滑 |
  | **兼职职位** | +257,000 | 历史新高 |
  | **非自愿兼职人数** | 4.9 Million | 显著上升 |
- **证据支持**：BLS Table A-9 (2026年4月)；失业率维持4.3%但U-6广义失业率升至7.8%。
- **专家解读**：115k的超预期数据是一个“数字陷阱”。企业通过拆分全职岗为兼职岗来降低用工成本。这种“拼凑式”繁荣在历史上常出现在衰退的前夕。
- **局限性**：暑期季节性招聘开始可能对数据产生了干扰。

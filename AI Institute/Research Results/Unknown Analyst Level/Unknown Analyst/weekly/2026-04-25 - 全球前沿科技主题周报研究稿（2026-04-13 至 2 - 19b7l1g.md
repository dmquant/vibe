## 四、问题 2：量子计算是否已达到“制药拐点”？

### 方法

用 IBM 官方 roadmap 与 IBM Q4Bio 新闻稿核对硬件规模、门深、应用领域和是否声明商业量子优势；用 `QTUM` 周收益观察资本市场定价；以“是否有超越经典化学模拟的明确成本/时间数据”为判断标准。

### 发现

| 指标 | 数值 | 口径 |
| :--- | ---: | :--- |
| IBM 2026 roadmap 最大物理量子比特 | `360 qubits` | `3 x 120-qubit modules` |
| IBM 2026 roadmap 目标门数 | `7,500 gates` | IBM roadmap |
| Q4Bio 初始团队 | `12` | Wellcome Leap/IBM |
| Q4Bio 总资金 | `$40M` | IBM newsroom |
| Q4Bio Phase III finalist | `6` | IBM newsroom |
| 使用 IBM 硬件的 finalist | `5` | IBM newsroom |
| IBM 硬件使用占比 | `83.33%` | `5 / 6` |
| Q4Bio 获奖金额 | `$2M` | IBM newsroom |
| Q4Bio 演示门槛 | `>50 qubits`、`1,000-10,000 gates` | IBM newsroom |
| Algorithmiq/Cleveland Clinic/IBM 运行规模 | `up to 100 qubits` | IBM newsroom |
| `QTUM` 周收益 | `+9.60%` | Yahoo Finance Chart API |
| 规划中 `2000+` 逻辑量子比特证据 | `0` | 官方来源未证实 |

答案是：未达到商业“制药拐点”，但已达到更严肃的应用验证阶段。IBM 的可验证进展是 Q4Bio 中多队在真实量子硬件上做生物/药物相关演示，规模达到 `>50` 到 `100` qubits，且有 `1,000-10,000` gates 门深要求。IBM roadmap 的 2026 目标是 `360` 物理量子比特和 `7,500` gates，不是 `2000+` 逻辑量子比特。当前数据还没有给出“同一任务上量子比经典 HPC 成本低多少、时间短多少”的完整可复现实验，因此只能称为可扩展路径，而不是生产级替代。

### 证据

- IBM Quantum Roadmap 2026：`360 qubits`、`7,500 gates`、Kookaburra fault-tolerant module。
- IBM Newsroom `2026-04-16`：Q4Bio `12` 队、`$40M`、`6` 个 Phase III finalist、`5/6` 使用 IBM 硬件、`>50 qubits` 与 `1,000-10,000 gates` 要求、`up to 100 qubits` 分子电子结构模拟。
- 市场数据：Yahoo Finance Chart API，`QTUM` `2026-04-10` 调整收盘 `116.69`，`2026-04-17` 调整收盘 `127.89`。

### 局限

“制药拐点”需要药物发现周期、命中率、湿实验验证和临床成功率数据，IBM/Q4Bio 主要提供算法与硬件演示。量子计算相关 ETF 含有广泛硬件、软件和半导体敞口，不能等同于制药应用收入。

## 五、问题 3：轨道加油技术如何重塑太空物流估值？

### 方法

核查 SpaceX/Starship 本周是否有可验证轨道加油飞行；用 NASA OTPS SBSP 报告中的 Starship 情景参数计算 LEO 与 GEO 转移的单位成本；用本周 New Glenn/BlueBird 7 事件作为商业航天基础设施风险案例；用 `ARKX/RKLB/ASTS` 周收益观察市场分化。

### 发现

| 指标 | 数值 | 计算/口径 |
| :--- | ---: | :--- |
| 本周可验证 Starship 轨道加油成功次数 | `0` | 未找到官方证据 |
| NASA Starship 基准发射成本 | `$100M` | NASA OTPS SBSP |
| NASA Starship 假设 LEO 载荷 | `100 MT` | NASA OTPS SBSP |
| 基准 LEO 单位成本 | `$1,000/kg` | `$100M / 100,000 kg` |
| 15% block buy 后 LEO 单位成本 | `$850/kg` | `$85M / 100,000 kg` |
| NASA 竞争情景 | `$50M/launch` | NASA OTPS SBSP |
| 竞争情景 LEO 单位成本 | `$500/kg` | `$50M / 100,000 kg` |
| SpaceX long-term desired cost | `$2M/launch` | NASA 引述 SpaceX 目标 |
| 目标情景 LEO 单位成本 | `$20/kg` | `$2M / 100,000 kg` |
| LEO 到 GEO 单艘 payload Starship 所需加注发射 | `12` | NASA OTPS SBSP |
| 基准 LEO-GEO 有效成本 | `$13,000/kg` | `13 x $100M / 100,000 kg` |
| `ARKX` 周收益 | `+7.08%` | Yahoo Finance Chart API |
| `RKLB` 周收益 | `+24.61%` | Yahoo Finance Chart API |
| `ASTS` 周收益 | `-9.87%` | Yahoo Finance Chart API |

如果轨道加油成熟，太空物流估值的核心变量会从“单次入轨价格”转向“高频发射、在轨加注、深空转移的一体化成本”。NASA 情景显示，Starship 在基准假设下 LEO 成本为 `$1,000/kg`，已经低于许多传统商业发射报价；若达到 `$50M` 竞争情景，则降至 `$500/kg`。但深空/地月转移不是简单的 LEO 成本，NASA 对 LEO 至 GEO 的估算需要 `12` 次加注发射，使有效成本在基准情景下回升到 `$13,000/kg`。本周 New Glenn 复用成功但 BlueBird 7 入轨异常，说明商业航天估值同时会奖励复用能力，也会惩罚上面级、轨道精度和保险风险。

### 证据

- NASA OTPS SBSP 报告：Starship `100,000 kg` LEO 载荷、`$100M` 发射成本、`15%` block buy、LEO 至 GEO 需 `12` 次 refuel launches、`$50M` 和 `$10M` 降本情景。
- Space.com `2026-04-19`：New Glenn 第三次任务起飞、一级复用回收成功，但 BlueBird 7 进入错误/低于计划轨道。
- AST SpaceMobile/BusinessWire：`2026-04-16` 宣布 BlueBird 7 `2026-04-19` 发射窗口；后续公告承认轨道低于计划。
- 市场数据：Yahoo Finance Chart API，`ARKX/RKLB/ASTS`。

### 局限

NASA SBSP 报告是情景分析，不是 SpaceX 商业报价。`$2M/launch` 是远期目标，不可作为 2026 可成交价格。轨道加油成本还取决于 boiloff、交会对接、发射节奏、保险、监管和地面设施瓶颈。

## 六、问题 4：RWA 协议标准化后，哪些资产类别链上溢价最高？

### 方法

用 RWA.xyz 数据在美国国会书面证词中的快照估算资产类别分布；用 RWA.xyz/新闻聚合的 2026 年 4 月市场规模数据做方向交叉核验；把“链上溢价”定义为“链上分布资产价值占比 + 30 天增长/机构采用可见度”的综合代理，而不是二级市场价格溢价。

### 发现

| 资产类别 | 分布资产价值占比 | 估算金额 | 判断 |
| :--- | ---: | ---: | :--- |
| U.S. Treasuries / money market | `41%` | `$10.95B` | 规模最大，机构采用最明确 |
| Private Credit | `27%` | `$7.21B` | 收益驱动强，但信用/透明度风险更高 |
| Commodities / gold | `19%` | `$5.07B` | 主要由黄金代币支撑，避险属性强 |
| Other / funds / real estate / stocks | `13%` | `$3.47B` | 房地产仍分散且流动性弱 |
| 总计 | `100%` | `$26.70B` | RWA.xyz，`2026-03-22` 快照 |

答案是：美债/货币市场基金的链上采用溢价最高，其次是私人信贷，商品主要由黄金代币贡献，房地产并未形成同等级规模。美债类产品有清晰的收益锚、托管结构和机构发行方，最容易获得链上 24/7 流动性和抵押品使用场景。私人信贷规模和收益吸引力较高，但底层贷款质量、估值频率和赎回机制更复杂。房地产、艺术品等资产的“概念溢价”高，但实际链上分布价值较低，难以作为当前主线。

### 证据

- RWA.xyz/美国众议院金融服务委员会证词：截至 `2026-03-22`，Global RWA distributed asset value `$26.7B`，U.S. Treasuries `41%`，Private Credit `27%`，Commodities `19%`。
- RWA.xyz 市场页快照：`Distributed Asset Value $26.54B`、30 日 `+8.30%`、`Total Asset Holders 663,161`。
- Bitcoin.com `2026-04-21` 引用 RWA.xyz：Tokenized RWA market cap `$29.27B`；U.S. Treasuries `$13.4B`。该来源用于趋势核验，不作为分类主表唯一口径。
- MAS 官方检索：未找到 `2026-04-13` 至 `2026-04-19` 关于三机构 RWA 清算结算协议批准的官方公告。

### 局限

RWA 数据商之间口径差异很大：是否排除 stablecoins、是否计入 represented asset value、是否计入私链/许可链，都会显著改变总规模。链上“分布资产价值”不是交易量、流动性或折溢价。房地产和私募资产估值滞后，价格发现弱于美债和黄金。

## 七、补充问题：PQC 金融标准是否进入强制阶段？

### 方法

核查 MAS 本周是否发布银行业 PQC 迁移指南；若无，则用 G7 CEG、NIST 官方文件构建金融业 PQC 时间表，并统计关键节点。

### 发现

| 指标 | 数值/日期 | 口径 |
| :--- | :--- | :--- |
| 本周 MAS PQC 新指南 | `0` | MAS 官方检索未证实 |
| NIST FIPS 203/204/205 批准日期 | `2024-08-13` | NIST |
| G7 CEG 金融业 PQC 路线图发布日期 | `2026-01-12/13` | U.S. Treasury / GOV.UK |
| G7 roadmap 强制性 | `非强制` | 文件明确不是监管预期 |
| 建议优先处理关键系统窗口 | `2030-2032` | G7 roadmap |
| 常见整体迁移目标 | `2035` | G7 roadmap 汇总 |
| 迁移活动阶段数 | `6` | Awareness、Discovery、Planning、Execution、Testing、Validation |

结论是：金融业 PQC 已进入“监管预期形成和路线图协调”阶段，但不能说本周进入 MAS 强制执行阶段。G7 路线图强调金融实体、监管者、关键服务商和加密供应商需要协调迁移；文件同时说明其不设定监管要求。NIST 标准已给算法落地提供基础，真正的执行难点是资产盘点、供应商路线图、混合密钥交换、签名迁移和关键系统优先级。对银行和市场基础设施而言，`2030-2032` 是关键系统倒排工期的实际约束窗口。

### 证据

- U.S. Treasury `2026-01-12`：G7 Cyber Expert Group 发布金融业 PQC 路线图。
- GOV.UK `2026-01-13`：同一 G7 CEG 声明，说明目的、非强制性质和迁移活动。
- G7 roadmap PDF：关键迁移阶段、`2030-2032` 关键系统优先迁移窗口、`2035` 常见整体目标。
- NIST `2024-08-13`：FIPS 203、204、205 批准。

### 局限

PQC 迁移的实际执行进度多在机构内部，公开数据少。国家/地区之间要求不同，且“关键系统”定义通常由机构与监管者共同判断。MAS 历史上有量子安全相关报告/咨询，但本周未发现新强制文件。

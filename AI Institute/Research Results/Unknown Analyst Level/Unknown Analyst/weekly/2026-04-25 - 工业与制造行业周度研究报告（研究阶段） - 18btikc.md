## 二、Q1：中国一季度工业增加值的超预期增长，是源于出口补库还是内生性基建驱动？

### 方法

采用 `2026-04-16` 至 `2026-04-17` 国家统计局发布数据，比较 `2026-03` 单月与 `2026Q1` 累计口径。需求侧用出口交货值、制造业 PMI 新出口订单、固定资产投资、基础设施投资、设备工器具购置作交叉验证；供给侧用分行业工业增加值、主要产品产量验证链条强弱。

### 发现

| 指标 | `2026-03` | `2026Q1` | 方向判断 |
| :--- | ---: | ---: | :--- |
| 规上工业增加值同比 | `5.7%` | `6.1%` | 稳健扩张 |
| 制造业增加值同比 | `6.0%` | `6.4%` | 强于总工业 |
| 高技术制造业增加值同比 | `11.7%` | `12.5%` | 明显领先 |
| 通用设备制造业增加值同比 | `6.3%` | `7.8%` | 设备链强 |
| 专用设备制造业增加值同比 | `6.2%` | `7.7%` | 设备链强 |
| 电气机械和器材制造业增加值同比 | `5.4%` | `7.3%` | Q1 强于单月 |
| 铁路、船舶、航空航天和其他运输设备制造业增加值同比 | `13.3%` | `13.5%` | 高端装备最强 |
| 工业机器人产量同比 | `24.4%` | `33.2%` | 自动化链条高景气 |
| 出口交货值同比 | `8.7%` | `7.1%` | 外需贡献显著 |
| 制造业 PMI 新出口订单 | `49.1` | 不适用 | 仍低于荣枯线 |
| 固定资产投资同比 | 不适用 | `1.7%` | 总量偏弱 |
| 基础设施投资同比 | 不适用 | `8.9%` | 结构性强 |
| 设备工器具购置同比 | 不适用 | `13.9%` | 内生设备更新强 |

结论是“出口补库 + 内生设备更新”共同驱动，但工业增加值的超预期更像结构性设备更新和高技术制造扩张，而不是传统基建全面复苏。出口交货值 `8.7%` 说明外需确实拉动生产，尤其机电与新能源相关品类；但制造业 PMI 新出口订单仍为 `49.1`，没有确认外需全面扩张。内需侧更有解释力的是设备工器具购置 `13.9%`、基础设施投资 `8.9%`、通用设备投资 `12.5%` 与航空运输投资 `43.3%`。同时建筑安装工程 `-0.4%`、建筑业 GDP `-3.8%`，说明不是地产/传统施工链驱动。

### 证据

| 原始输入 | 日期 | 关键字段 |
| :--- | :--- | :--- |
| 国家统计局《Industrial Production Operation in March 2026》 | `2026-04-17` | 规上工业、分行业增加值、工业机器人、出口交货值 |
| 国家统计局《2026年一季度国内生产总值初步核算结果》 | `2026-04-17` | GDP `334193` 亿元、同比 `5.0%`；工业同比 `6.1%`；制造业同比 `6.3%` |
| 国家统计局《2026年1-3月份全国固定资产投资增长1.7%》 | `2026-04-16` | 基建 `8.9%`、设备工器具购置 `13.9%`、制造业投资 `4.1%` |
| 国家统计局《Purchasing Managers’ Index for March 2026》 | `2026-04-01` | PMI `50.4`、生产 `51.4`、新订单 `51.6`、新出口订单 `49.1` |
| 海关/媒体转述海关口径 | `2026-04-15` 至 `2026-04-16` | 机电产品出口占比与 Q1 高增，需要等海关分项数据库复核 |

### 限制

出口交货值为名义同比，工业增加值为实际同比，二者不能直接加总分解。工业分行业增加值没有公布“内销/外销”拆分，结论依赖交叉验证。固定资产投资是累计同比且未扣除价格因素，和单月工业增加值存在口径差异。

## 三、Q2：汉诺威工博会 AI+工业原生应用，哪些已具备量产线 ROI 可行性？

### 方法

由于 `Hannover Messe 2026` 官方日期为 `2026-04-20` 至 `2026-04-24`，不属于本报告周 `2026-04-13` 至 `2026-04-19`。本节采用报告周内发布的会前官方材料（Siemens `2026-04-13` 至 `2026-04-17`）及开幕后官方技术披露（NVIDIA、Siemens `2026-04-20`）作为前瞻研究，按“是否有真实工厂环境、是否有量化 KPI、是否已 GA/客户试点、是否能嵌入现有 PLC/MES/数字孪生系统”筛选。

### 发现

| 应用 | 成熟度 | 已披露 KPI | ROI 判断 |
| :--- | :--- | :--- | :--- |
| Siemens + Humanoid + NVIDIA 工厂物流 humanoid | 真实工厂 PoC | `60` 次 tote moves/hour；uptime `>8` 小时；pick-and-place 成功率 `>90%`；原型开发 `7` 个月 vs 常规 `18-24` 个月 | 适合标准容器搬运、夜班/缺工场景，暂不适合复杂装配全面替代 |
| Siemens Eigen Engineering Agent | 已 GA | `>100` 客户、`19` 个国家试点；工作流 `2-5x` faster；solution quality +`80%`；engineering efficiency +`50%`；onboarding 从 weeks 降到 days | 具备明确 ROI，优先落地 PLC coding、HMI visualization、device configuration |
| Vision AI agents / Tulip Factory Playback | 客户应用 | Terex 预计 yield +`3%`、rework -`10%` | 对质量追溯和返工成本高的离散制造最可行 |
| Factory-scale digital twin / Omniverse | 多厂商展示 | 可在物理改造前识别生产问题；未统一披露节省金额 | 适合高资本开支产线改造，ROI 依赖停线成本 |
| SCHUNK GROW / standardized deployable automation cell | 标准化单元展示 | 未披露量产线财务指标 | 有潜力，但需客户现场验证 |

最具短期 ROI 的不是“通用人形机器人替代工人”，而是工程自动化 agent、视觉质量 agent 和标准化物流任务。Siemens 的 Eigen Engineering Agent 已经一般可用，并且与 TIA Portal 直接连接，减少“AI 生成建议后人工重写”的返工。Humanoid 物流 PoC 的 `60` 次/小时与 `>90%` 成功率说明在标准 tote 任务上已进入产线测试门槛，但仍是边界清晰的 intralogistics。视觉 agent 的 ROI 更容易闭环，因为良率、返工率、质量事件能直接映射到成本。

## 六、Q5：铜铝成本压力下，重型机械龙头价格传导机制是否稳健？

### 方法

商品价格使用公开可下载的连续期货代理序列：`HG=F` 作为铜代理，`ALI=F` 作为铝代理；窗口为 `2026-03-25` 至 `2026-04-24`。重型机械毛利敏感性采用情景模型：毛利率冲击 = `5%` 金属价格上涨 × 金属占 COGS 比重 × `(1 - 价格传导率)` × COGS / Revenue。金属占比与传导率为研究假设，依据公司业务属性、售后/服务占比、订单周期和经销商价格机制设置。

### 发现

| 商品代理 | 起始日期 | 起始收盘 | 结束日期 | 结束收盘 | 区间收益 | 区间最高 | 区间最低 |
| :--- | :--- | ---: | :--- | ---: | ---: | ---: | ---: |
| 铜 `HG=F` | `2026-03-25` | `5.5290` | `2026-04-24` | `6.0235` | `8.94%` | `6.1200` | `5.4465` |
| 铝 `ALI=F` | `2026-03-25` | `3135.00` | `2026-04-24` | `3607.75` | `15.08%` | `3647.75` | `3135.00` |

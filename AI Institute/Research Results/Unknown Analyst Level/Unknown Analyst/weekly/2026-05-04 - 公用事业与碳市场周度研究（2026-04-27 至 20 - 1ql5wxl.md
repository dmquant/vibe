# 公用事业与碳市场周度研究（2026-04-27 至 2026-05-01）

## 优先结论

1. **中国 CEA “110元/吨以上”的计划假设未被公开交易数据支持。** 2026-04-30 全国碳市场收盘价为 **79.50元/吨**，较 2026-04-24 的 **77.59元/吨**上涨 **2.46%**；本周总成交 **1,684,790吨**、总成交额 **127,601,088.30元**、周 VWAP **75.74元/吨**。这说明扩围和配额收紧预期只被部分定价，尚未进入“高价稀缺”阶段。
2. **中欧碳价继续分层。** EUA Dec-26 于 2026-04-30 收 **73.79 EUR/t**，较 2026-04-24 下跌 **1.48%**；按 2026-04-30 人民币中间价 **1 EUR = 8.0057 CNY**折算为 **590.74元/吨**，约为 CEA 的 **7.43倍**。
3. **欧盟氢能基础设施政策改善项目可融资性，但不足以独立形成盈利拐点。** 第二批 PCI/PMI 共 **235个**项目，其中 **100个**为 hydrogen/electrolyser 项目；2026-04-30 CEF Energy 新一轮资金为 **600百万欧元**，对照欧洲管网和电解槽 CAPEX 规模，更像“开发权和前期工程补助”，而不是完整资产负债表解决方案。
4. **欧洲 2026-05-01 负电价是本周最强的储能信号。** 德国-卢森堡 15分钟价格最低 **-499.99 EUR/MWh**，法国日均价 **-41.39 EUR/MWh**；德国当日最低负价区间内太阳能出力约 **45.51 GW**、负荷约 **49.21 GW**、可再生能源占负荷 **107.4%**。
5. **水务资产仍是通胀对冲型资产，但 IRR 对电费和水价更敏感。** Ras Mohaisen IWP 的公开项目口径为 **300,000 m3/day**、总投资 **SAR 2.57bn / USD 685m**、**25年**协议；在本报告模型中，水价 **0.85 USD/m3**、电价 **0.04 USD/kWh**时，未杠杆 IRR 约 **7.5%**。

## 计划校验与方法

覆盖窗口按任务意图修正为 **2026-04-27 至 2026-05-01**；原计划文件标题行写成 “2026-04-27 -> 2026-04-24”，应为笔误。2026-05-01 中国内地及香港市场休市，因此 CEA 交易窗口为 2026-04-27 至 2026-04-30；欧洲电力和 EUA 数据包含 2026-05-01。

数据源优先级：交易所或监管机构原始披露，其次为官方转载和数据供应商页面。CEA 使用上海环境能源交易所披露的全国碳市场日度/周度数据（SMM、生态中国网、新华财经转载）；EUA 使用 Investing.com EEX Dec-26 历史表；欧洲电力使用 Energy-Charts API（Bundesnetzagentur/SMARD.de，CC BY 4.0）；美国发电结构使用 EIA Open Data API `electricity/rto/fuel-type-data`、`respondent=US48`；绿证使用国家能源局；ETF 代理使用 StockAnalysis 历史价格。

## 问题一：配额收紧逻辑是否已定价？

### 方法

比较 2026-04-24、2026-04-27 至 2026-04-30 的 CEA 收盘价、成交量和 VWAP；用 EUA Dec-26 作为成熟碳市场参照，并用 2026-04-30 人民币中间价折算横向价差。政策口径用生态环境部 2026 年全国碳市场工作通知和 2026-04-30 金融机构入市表态校验。

### 发现

| 指标 | 数值 |
|---|---:|
| CEA 2026-04-30 收盘价 | 79.50元/吨 |
| CEA 周涨跌幅（相对 2026-04-24） | +2.46% |
| CEA 本周成交量 | 1,684,790吨 |
| CEA 本周成交额 | 127,601,088.30元 |
| CEA 本周 VWAP | 75.74元/吨 |
| EUA Dec-26 2026-04-30 收盘价 | 73.79 EUR/t |
| EUA Dec-26 周涨跌幅（相对 2026-04-24） | -1.48% |
| EUA 折人民币价格（2026-04-30） | 590.74元/吨 |
| EUA/CEA 折算倍数 | 7.43倍 |

**解释：** CEA 本周上涨是“扩围+金融机构入市预期”的重新定价，但价格没有突破 80元/吨区间，更没有达到计划假设的 110-115元/吨。生态环境部 2026 年工作安排已把发电、钢铁、水泥、铝冶炼纳入重点管理，并要求 2026-09-20 前完成 2025 年度核定配额注册登记，这给市场一个明确的收紧方向。上行红线在于：2024/2025 年度新行业配额仍以强度控制和整体平衡为主、监管强调市场平稳有序、且金融机构入市会被“规范和引导”。因此，配额稀缺性被部分定价，但不是充分定价。

## 问题二：氢能基础设施的盈利拐点何时出现？

### 方法

以欧盟 2026 年 CEF Energy、第二批 PCI/PMI、Hydrogen Mechanism 三组数据判断政策支持是否降低公用事业公司的资本压力。盈利拐点按“需求可见度、补贴可得性、监管资产回收机制、管网利用率”四个条件评估。

### 发现

| 指标 | 数值 |
|---|---:|
| 第二批 PCI/PMI 项目总数 | 235个 |
| 其中 electricity/offshore/smart grid | 113个 |
| 其中 hydrogen/electrolyser | 100个 |
| 其中 CO2 network | 17个 |
| 2026 CEF Energy 新一轮预算 | 600百万欧元 |
| 2026 CEF Energy 申请截止 | 2026-09-30 |
| CEF Energy 2021-2027 总预算 | 5.88十亿欧元 |
| Hydrogen Mechanism 供应项目 | 265个 |
| Hydrogen Mechanism 需求项目 | 45个 |
| 获得至少 1 个意向买方的供应机会 | 87% |
| 提供价格指引的供应机会 | 54% |

**解释：** 结论是 **2028-2030 年前后才可能出现局部盈利拐点**，且首先发生在可纳入监管资产基础或有长期容量付费的跨境管网，不会先发生在完全商用风险的电解槽项目。CEF 的 **600百万欧元**提高了研究、许可、前期工程和部分建设的可融资性，但相对 **100个** hydrogen/electrolyser 项目而言，平均支持强度有限。Hydrogen Mechanism 的 **87%**意向匹配说明需求侧开始形成，但只有 **54%**供应机会给出价格，说明商业合同价格仍不透明。对公用事业公司而言，真正减轻资产负债表压力的不是单次补助，而是长期 offtake、可计入费率的管网资产和明确的跨境成本分摊。

### 证据

- 第二批 PCI/PMI：**235个**项目，**113个**电网类、**100个** hydrogen/electrolyser、**17个** CO2 network；项目可申请 CEF。来源：[European Commission 2026-04-09](https://energy.ec.europa.eu/news/235-key-cross-border-energy-projects-boost-europes-interconnectivity-2026-04-09_en)。
- 2026 CEF Energy：2026-04-30 发布，预算 **600百万欧元**，截止 **2026-09-30**，结果预计 2027 年初。来源：[European Commission 2026-04-30](https://energy.ec.europa.eu/news/eu-grants-worth-eu600-million-available-cross-border-energy-infrastructure-projects-2026-04-30_en)。
- Hydrogen Mechanism：**265**个供应项目、**45**个需求项目，**87%**供应机会获得至少一个意向，**54%**提供价格指引。来源：[European Commission 2026-04-29](https://energy.ec.europa.eu/news/successful-first-round-eu-hydrogen-mechanism-support-market-development-providing-value-added-2026-04-29_en)。

### 局限

官方披露没有逐项目 CAPEX、补贴强度、WACC、管输费率和最终 offtake 价格。本报告的盈利拐点判断是基于政策和融资结构的推断，不是项目级财务模型。氢价、天然气价、碳价和工业需求都可能改变时间表。

## 问题三：水务资产的通胀对冲属性如何保障 IRR？

### 方法

用 Saudi Ras Mohaisen IWP 作为本周可核验的中东大型海水淡化融资样本。公开项目参数为投资 **SAR 2.57bn / USD 685m**、产能 **300,000 m3/day**、2028 年初始产水 **100,000 m3/day**、2030 年满产、协议期 **25年**。敏感性模型采用未杠杆 DCF：满产可用率 **95%**，年水量 **104.03百万 m3**，非电 O&M **0.12 USD/m3**，RO 单耗 **3.5 kWh/m3**，水价和电价分别敏感化。

### 发现

| 指标 | 数值 |
|---|---:|
| 项目总投资 | SAR 2.57bn / USD 685m |
| 设计产能 | 300,000 m3/day |
| 协议期限 | 25年 |
| 满产年化水量假设 | 104.03百万 m3 |
| 基准水价假设 | 0.85 USD/m3 |
| 基准电价假设 | 0.04 USD/kWh |
| 基准未杠杆 IRR | 7.5% |

**解释：** 水务资产的通胀对冲来自长期 WPA/PPP 结构：水价、可用性付款或成本传导条款可以把一部分通胀和电费波动传导给 offtaker。模型显示，若水价从 **0.85**降至 **0.75 USD/m3**，且电价维持 **0.04 USD/kWh**，IRR 从 **7.5%**降至 **5.5%**；若水价升至 **0.95 USD/m3**，IRR 可升至 **9.4%**。因此，水务项目不是无风险债券，核心风险仍是电力成本、可用率和水价调整公式。长期特许经营权保障 IRR 的关键是把收入从“卖水量”变成“可用容量+通胀指数化”的现金流。

### 证据

- Ras Mohaisen IWP：SWPC 宣布 financial close，总投资 **SAR 2.57bn**，产能 **300,000 m3/day**，2028 年初始 **100,000 m3/day**，2030 年满产，**25年**协议。来源：[SWPC](https://www.swpc.sa/en/swpc-announces-financial-close-for-ras-mohaisen-independent-water-desalination-plant-project-valued-at-over-sar-2-5-billion/)、[InfraPPP 2026-01-06](https://www.infrapppworld.com/news/financial-close-announced-for-us6848-million-desalination-ppp-project-in-saudi-arabia)。
- ACWA/市场披露：Ras Mohaisen IWP 为 RO 技术，产能 **300,000 m3/day**，融资约 **SAR 2.074bn / USD 553.2m**。来源：[Mubasher/ACWA disclosure](https://english.mubasher.info/news/4540298/ACWA-Power-Company-announces-the-financial-close-for-one-of-its-affiliated-water-desalination-project-companies-Ras-Mohaisen-First-Water-Desalination-Company-Ras-Mohaisen-IWP-/)。

### 局限

本报告没有项目真实水价、债务期限、DSCR、税率、EPC 支出曲线和通胀指数条款；IRR 表是未杠杆、满产稳态敏感性。原计划提到“2026-04-30 沙特 20亿美元绿色贷款水务项目”，公开检索未找到可核验事件，因此以 Ras Mohaisen IWP 替代。

## 问题四：负电价常态化如何改变储能套利模型？

### 方法

使用 Energy-Charts 的 15分钟价格、负荷和出力数据，统计德国-卢森堡与法国 2026-04-27 至 2026-05-01 的日均价、最低价、最高价、负价小时数，并抽取 2026-05-01 德国极端时段的负荷、太阳能、风电和剩余负荷。储能套利用德国 2026-05-01 最低 4 小时均价与最高 4 小时均价衡量理论价差。

### 发现

| 指标 | 德国-卢森堡 | 法国 |
|---|---:|---:|
| 2026-05-01 日均电价 | -2.08 EUR/MWh | -41.39 EUR/MWh |
| 2026-05-01 最低价 | -499.99 EUR/MWh | -498.65 EUR/MWh |
| 2026-05-01 最高价 | 232.99 EUR/MWh | 104.78 EUR/MWh |
| 2026-05-01 负价小时数 | 8.0小时 | 8.0小时 |
| 德国当日最低 4小时均价 | -379.36 EUR/MWh | 不适用 |
| 德国当日最高 4小时均价 | 152.43 EUR/MWh | 不适用 |
| 德国 4小时理论价差 | 531.79 EUR/MWh | 不适用 |
| 90%效率下每 MWh 充电理论毛利 | 516.55 EUR/MWh | 不适用 |

**解释：** 负电价把储能价值从“日内峰谷价差”提升为“吸收弃电获得负成本电量+晚高峰释放”的双边收益。2026-05-01 德国最低负价时，太阳能约 **45.51 GW**，负荷约 **49.21 GW**，可再生能源占负荷超过 **100%**，说明问题不是总装机不足，而是中午灵活性不足。储能套利模型需要从年均价差转向“事件驱动收入”：少数极端日可贡献显著收入，但容量、电网费、平衡市场限制和电池衰减会决定实际可捕捉比例。对公用事业公司，负价常态化提高了电池、抽蓄、需求响应和灵活电解槽的组合价值。

### 证据

- Energy-Charts 价格 API：`https://api.energy-charts.info/price?bzn=DE-LU&start=2026-04-27&end=2026-05-01`；`https://api.energy-charts.info/price?bzn=FR&start=2026-04-27&end=2026-05-01`，许可为 Bundesnetzagentur/SMARD.de CC BY 4.0。
- Energy-Charts 公共净发电 API：`https://api.energy-charts.info/public_power?country=de&start=2026-05-01&end=2026-05-01`。
- 新闻核验：Bloomberg 报道 2026-04-30 欧洲 Friday day-ahead 价格因太阳能和假期需求下滑接近 **-500 EUR/MWh**，法国 day-ahead 均价 **-41.4 EUR/MWh**。来源：[Bloomberg 2026-04-30](https://www.bloomberg.com/news/articles/2026-04-30/european-power-plunges-to-record-negative-prices-on-solar-surge)。

### 局限

Energy-Charts 数据是公开净发电与市场价格，不包含单个储能资产的并网点、可用容量、网费、交易费用和调频收入。15分钟价差不能直接等同于实际电池收益，尤其在同一日多次充放电、并网约束和电池衰减存在时。

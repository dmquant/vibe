## Q1：碳市场扩容后，铝冶炼企业的碳减排成本如何反映在 PPA 绿色溢价中？

### 方法

采用 2026-04-13 至 2026-04-17 全国 CEA 每日收盘价和成交量，计算周均价、成交额和 CCER 相对 CEA 溢价。PPA 绿色溢价测算以“避免外购电排放成本”为核心：`绿色溢价 = CEA 或 CCER 价格 × 替代电力排放因子`。排放因子使用 0.45、0.55、0.80 tCO2/MWh 三档，分别代表清洁省间电力替代、常规煤电边际和高煤耗自备电替代情景。铝电耗按 13.5 MWh/吨铝作为敏感性口径。

### 发现

| 指标 | 数值 |
| --- | ---: |
| CEA 周收盘均价 | 79.11 元/吨 |
| CEA 周成交量 | 1,490,500 吨 |
| CEA 周成交额 | 111,471,730.80 元 |
| 2026-04-17 CEA 收盘价 | 79.56 元/吨 |
| 2026-04-17 CCER 成交均价 | 94.34 元/吨 |
| CCER 较 CEA 溢价 | 14.78 元/吨，18.58% |

| 排放因子情景 | 按 CEA 79.11 元/吨折算的 PPA 碳成本 | 按 CCER 94.34 元/吨折算的 PPA 碳成本 | 折算至吨铝成本，按 13.5 MWh/吨 |
| --- | ---: | ---: | ---: |
| 0.45 tCO2/MWh | 35.60 元/MWh | 42.45 元/MWh | 480.60 元/吨铝 |
| 0.55 tCO2/MWh | 43.51 元/MWh | 51.89 元/MWh | 587.39 元/吨铝 |
| 0.80 tCO2/MWh | 63.29 元/MWh | 75.47 元/MWh | 854.36 元/吨铝 |

碳市场扩围本身尚未造成 CEA 价格当周上行，2026-04-13 至 2026-04-17 CEA 收盘价仅从 79.54 元/吨升至 79.56 元/吨，周内变化约 +0.03%。但 CCER 价格在 2026-04-17 达到 94.34 元/吨，相对 CEA 溢价 18.58%，说明自愿减排量已开始对高质量绿色属性进行额外定价。对铝冶炼企业而言，绿电 PPA 溢价的合理上限不应只看传统绿证溢价，而应与 35-65 元/MWh 的碳成本节约挂钩。若绿电合同能够同时降低履约缺口和 CCER 采购需求，铝厂可接受的绿色溢价会显著高于一般工商业用户。

### 局限

全国碳市场当前公开行情仍是 CEA 综合价格，没有铝、水泥、能源三个板块的独立价格序列，因此不能直接观察铝冶炼专属碳价。铝行业配额分配在 2024 年度按核查排放量、2025-2026 年度整体盈亏平衡的过渡安排下，短期实际缺口可能小于模型假设。PPA 溢价还取决于绿证归属、用电曲线匹配、偏差考核和省间输电成本，不能只由碳价决定。

## Q2：Q1 储能装机超预期后，独立储能租赁收益与现货套利是否具备长期可持续性？

### 方法

先用国家能源局 2026-03 底发电装机摘要确认新能源基数，再用 CNESA DataLink 的 2026-01 新型储能新增投运数据判断储能增速。经济性测算采用 100MW/200MWh 独立储能电站，资本开支按 0.80 元/Wh，效率按 86%，年循环 300 次；收益分为容量类、容量租赁、现货套利和辅助服务四项。比较基准为山东、宁夏等已形成容量或价差机制的省份。

### 发现

| 指标 | 数值 |
| --- | ---: |
| 2026-03 底全国累计发电装机 | 39.6 亿千瓦 |
| 2026-03 底太阳能发电装机 | 12.4 亿千瓦，同比 +31.3% |
| 2026-03 底风电装机 | 6.6 亿千瓦，同比 +22.4% |
| 2026-01 新型储能新增投运 | 3.78GW/10.90GWh |
| 2026-01 新型储能新增同比 | +62%/+106% |
| 2026-01 平均时长 | 2.88 小时 |

| 100MW/200MWh 模型项 | 假设 | 年收益估算 |
| --- | ---: | ---: |
| 容量电价或容量补偿 | 100-165 元/kW·年 | 10.00-16.50 百万元 |
| 容量租赁 | 252 元/kW·年，出租率 30% | 7.56 百万元 |
| 现货套利 | 净价差 0.30 元/kWh，300 次，效率 86% | 15.48 百万元 |
| 辅助服务 | 调频、备用等保守值 | 2.00-5.00 百万元 |
| 合计收入 | 多收益叠加 | 35.04-44.54 百万元 |
| 资本开支 | 0.80 元/Wh | 160.00 百万元 |
| 简单投资回收期 | 不含融资结构优化 | 3.6-4.6 年，扣除运维后约 4.0-5.5 年 |

独立储能的可持续性已经从“强配套建设”转向“多市场经营”，但只在高峰谷价差、容量补偿和现货准入较明确的地区成立。测算显示，100MW/200MWh 项目在容量类收入、租赁收入和套利收入同时存在时，年收入可达到 35.04-44.54 百万元，具备覆盖折旧和融资成本的基础。若去掉容量类收入，项目高度依赖现货价差，收益波动会显著放大；若去掉租赁收入，早期现金流稳定性也会下降。结论是：山东、宁夏、广东部分场景已接近长期商业可持续，全国平均口径仍不能简单外推。

## Q3：水价上调能否完全对冲折旧和财务费用上升？对 A 股主要水务标的估值提振空间多大？

### 方法

选取上海、南京作为长三角价格样本，采用居民第一阶梯供水或综合水价的调整幅度测算价格弹性。公司层面选取 601158.SH、600874.SH，使用 2025 年报公开财务数据和调价敏感性估算 ROE 提振。由于两家公司收入结构中污水处理、工程、再生水等非居民供水业务占比较高，测算采用“调价传导收入占总收入比例 20%-35%”的保守假设。

## Q4：JKM 波动下，拥有上游海外协议气源与依赖现货采购的城燃运营商毛利会分化多大？

### 方法

采用 2026-04-17 燃气行业周报中的 JKM、TTF、中国 LNG 出厂价、中国 LNG 到岸价数据，按报告给出的元/方口径直接比较。城燃毛利测算用简化公式：`配气毛利 = 终端销售价 - 综合气源成本 - 管输及损耗`。终端销售价假设为 3.7 元/方，长协气源成本为 3.0 元/方，现货采购成本按 JKM 对应元/方口径。

### 发现

| 价格项 | 2026-04-17 原始价格 | 元/方 | 周环比 | 年初至今 | 年同比 |
| --- | ---: | ---: | ---: | ---: | ---: |
| JKM 现货价 | 15.0 美元/百万英热 | 3.8 | -24.9% | +56.2% | +19.0% |

## Q5：VPP 在南方电网调度实验中的响应速度与收益分配机制有何突破？

### 方法

检索南方电网、国家能源局南方监管局、人民网和行业媒体公开资料，确认 2025-2026 年广东和南方区域 VPP 参与市场的规则与案例。重点比较三个指标：聚合容量、响应速度、收益来源。由于 2026-04-13 至 2026-04-19 当周未见公开披露的单次“调度实验”完整数据，本节以广东首批发电类 VPP 参与现货市场和南方电网全网规模化进展作为替代证据。

### 发现

| 指标 | 数值或机制 |
| --- | --- |
| 南方电网全网 VPP 聚合容量 | 近 1800 万千瓦 |
| 广东 2026 年底目标接入容量 | 400 万千瓦 |
| 广东 2026 年底目标调节能力 | 120 万千瓦 |
| 首批发电类 VPP 现货交易规模 | 深圳、佛山、中山等 5 家，合计约 33MW |
| 响应速度公开口径 | 分钟级响应、全域灵活调控 |
| 收益来源 | 需求响应收益 + 电能量现货交易收益 + 辅助服务和增值服务潜在收益 |

VPP 的突破在机制，不在单点容量。广东规则已允许发电类和负荷类 VPP 以“报量报价”方式参与中长期和现货电能量市场，使 VPP 从单纯需求响应资源变成可结算的市场主体。收益分配从过去的邀约补贴，扩展为“需求响应保底收益 + 现货价差收益 + 聚合运营商服务费”的双轮驱动模式。响应速度方面，公开资料仍以“分钟级”为主，未披露秒级调度实验的可审计数据。

# 每周深研报告（研究阶段）
## 周期：2026-04-13 至 2026-04-19

## 核心结论（先看结论）
1. **“连续降息”叙事在本周并不成立**：ECB在**2026-03-19**将存款利率维持在`2.00%`，截至本报告周（2026-04-13~04-19）并未新增降息；市场定价已从“2026年内不动”转向“先按兵不动、再讨论加息”。
2. **中国Q1增长高于计划假设**：Q1 GDP同比为`5.0%`（非4.6%），3月工业增加值同比`+5.7%`，1-3月`+6.1%`；但地产链仍深度收缩（开发投资`-11.2%`、新开工`-20.3%`、销售额`-16.7%`），尚不能判定“触底完成”。
3. **美股“软着陆交易”仍偏拥挤但有盈利支撑**：`S&P 500`报告周上涨约`+4.55%`；美国3月零售销售环比`+1.7%`、初请失业金`214k`，需求与就业韧性仍强。估值端美国12M Forward P/E约`20.3x`，需要后续财报继续兑现；FactSet Q1 blended EPS增长`13.2%`对估值形成阶段性验证。
4. **黄金与“高名义利率”并行的关键在于“实际利率+风险溢价”**：近四周黄金从`$4,492`升至`$4,830`，同期10Y实际利率（`DFII10`）由`2.01%`降至`1.90%`，替代样本相关系数约`-0.87`，并未出现“金价完全脱离实际利率”的统计证据；地缘与政策不确定性是额外溢价来源。
5. **外汇溢出并非单向**：报告周`EUR/USD +0.67%`、`USD/CAD -1.16%`（CAD走强）、`USD/AUD +1.61%`（AUD走弱），反映“政策预期+商品价格+风险偏好”共同作用，不支持简单“降息竞赛=统一弱美元路径”。

## 1) 问题一：货币政策路径与`CAD`/`AUD`溢出

### 方法
- 政策端：使用ECB官方利率决议与FOMC SEP（2026-03-18）作为“央行函数”基准。
- 市场端：用Reuters 3月末与4月下旬调查衡量“预期漂移”。
- 汇率端：用FRED日频`DEXUSEU`、`DEXCAUS`、`DEXUSAL`，计算`2026-04-10`到`2026-04-17`变化。

### 发现（数值先行）
| 指标 | 数值 | 时间点/区间 |
|---|---:|---|
| ECB存款利率 | `2.00%` | 2026-03-19（维持） |
| Fed Funds Effective Rate | `3.64%` | 2026-04-17 |
| FOMC SEP：2026年末联邦基金利率中位数 | `3.4%` | 2026-03-18 |
| `EUR/USD` 周变动 | `+0.67%` | 2026-04-10 → 2026-04-17 |
| `USD/CAD` 周变动 | `-1.16%` | 2026-04-10 → 2026-04-17 |
| `USD/AUD` 周变动 | `+1.61%` | 2026-04-10 → 2026-04-17 |

解读：
- 这周更准确的框架不是“降息竞赛”，而是“ECB短期观望 + Fed延后降息”的再定价。
- 汇率表现分化明显：CAD与AUD在同一周出现相反方向，说明商品价格弹性、风险偏好与本国增长暴露对汇率影响大于“单一政策利差”。
- 若按交易语言，当前更像“路径不确定性抬升、carry与商品beta重定价”，而非单边美元趋势。
- 对后续三个月，关键不是是否降息，而是“哪家央行先确认通胀再抬头/增长下修”。

- Fed SEP（2026-03-18）：https://www.federalreserve.gov/monetarypolicy/fomcprojtabl20260318.htm

- Reuters调查反映“经济学家预期”而非逐笔市场定价；与OIS/联邦基金期货会有偏差。

## 2) 问题二：中国刺激成效与地产是否接近尾声

### 方法
- 使用NBS英文口径：Q1 GDP、3月工业增加值、1-3月固定资产投资、社零、进出口、房地产开发投资与销售。
- 用1-2月与1-3月对比观察“斜率变化”（改善/恶化）。

### 发现（数值先行）
| 指标 | 最新值 | 对比/补充 |
|---|---:|---|
| Q1 GDP同比 | `5.0%` | 2026Q1 |
| 3月规上工业增加值同比 | `+5.7%` | 1-3月`+6.1%` |
| 1-3月固定资产投资同比 | `+1.7%` | 1-2月`+1.8%` |
| 1-3月社零同比 | `+2.4%` | 3月单月同比`+1.7%` |
| 1-3月出口同比（人民币计） | `+11.9%` | 进口`+19.6%` |
| 1-3月房地产开发投资同比 | `-11.2%` | 1-2月`-11.1%` |
| 1-3月新开工面积同比 | `-20.3%` | 1-2月`-23.1%` |
| 1-3月商品房销售面积同比 | `-10.4%` | |
| 1-3月商品房销售额同比 | `-16.7%` | |

解读：
- 工业端延续恢复（3月`+5.7%`、1-3月`+6.1%`），短期“生产修复”成立。
- 但内需链条仍偏弱：社零与FAI增速低于工业与外贸，结构上更像“生产先行、需求滞后”。
- 地产指标仍处深负区间，虽然新开工降幅较1-2月收窄，但投资与销售端尚未形成连续拐点。
- 因此“探底后期”可以讨论，但“触底确认”证据不足，政策仍需继续托底与信用修复。

### 证据

- 工业增加值（2026-04-17）：https://www.stats.gov.cn/english/PressRelease/202604/t20260417_1963356.html
- 国民经济运行（综合稿，含社零/外贸/地产摘要）：https://www.stats.gov.cn/english/PressRelease/202604/t20260416_1963326.html

### 局限
- NBS当期英文稿未直接给出“消费/投资/净出口对GDP增长拉动点数”拆解，无法严格复现“贡献度”图。
- 进出口为名义值同比，未完全剔除价格与汇率影响。

## 3) 问题三：美股估值是否被财报验证

### 方法
- 宏观需求：Census 3月零售销售 + DOL/FRED初请失业金。
- 市场与估值：GSAM周报中的`S&P 500`周涨幅与12M Forward P/E横向对比。
- 盈利验证：FactSet `Earnings Insight`（2026-04-17）Q1 blended EPS增速。

### 发现（数值先行）
| 指标 | 数值 | 时间 |
|---|---:|---|
| 美国3月零售销售环比 | `+1.7%` | 2026-04-21发布 |
| 美国初请失业金（SA） | `214,000` | week ending 2026-04-18 |
| `S&P 500`周涨幅 | `+4.55%` | week ending 2026-04-17 |
| 美国12M Forward P/E | `20.3x` | GSAM（截至2026-04-17） |
| 欧洲发达市场12M Forward P/E | `15.0x` | 同口径 |
| 新兴市场12M Forward P/E | `11.7x` | 同口径 |
| FactSet Q1 blended EPS增速 | `+13.2%` | 2026-04-17 |

解读：
- 宏观“硬数据”（零售+就业）支持增长韧性，解释了风险资产继续上冲。
- 但估值层面美国相对其他区域溢价仍高（`20.3x` vs `15.0x`/`11.7x`），意味着市场已提前计入较高盈利兑现率。
- Q1 `+13.2%`的blended EPS增长对“软着陆”定价提供了阶段性支撑，但尚不足以消化全部估值溢价。
- 结论是“未明显失真，但容错率下降”：后续财报指引若下修，估值回撤弹性会放大。

### 证据
- Census 零售销售页（2026-04-21，March 2026）：https://www.census.gov/retail/sales.html
- FRED `ICSA`：https://fred.stlouisfed.org/series/ICSA
- GSAM Market Monitor（041726）：https://am.gs.com/cms-assets/gsam-app/documents/insights/en/2026/market_monitor_041726.pdf
- FactSet Earnings Insight（041726）：https://advantage.factset.com/hubfs/Website/Resources%20Section/Research%20Desk/Earnings%20Insight/EarningsInsight_041726.pdf

### 局限
- 12M Forward P/E为一致预期口径，若盈利预期快速重置，估值会同步重估。
- FactSet blended口径在财报季早期受“已披露样本结构”影响。

## 4) 问题四：黄金为何在“高利率环境”继续上行

### 方法
- 黄金价格采用GSAM周度快照（`Gold ($/oz)`），匹配FRED `DFII10`（10Y实际利率）同周值。
- 计算替代样本相关系数，并检查方向一致性。

### 发现（数值先行）
| 周末日期 | Gold ($/oz) | 10Y Real Yield `DFII10` |
|---|---:|---:|
| 2026-03-20 | 4492.42 | 2.01% |
| 2026-03-27 | 4494.09 | 2.13% |
| 2026-04-10 | 4749.75 | 1.95% |
| 2026-04-17 | 4830.34 | 1.90% |

- 替代样本相关系数（Gold vs `DFII10`）=`-0.87`。

解读：
- 在本样本里，黄金与实际利率仍保持明显负相关，主线并未失效。
- 市场感受到的“背离”更多来自名义利率仍高、但实际利率回落以及地缘风险溢价上升。
- 金价创高并不自动意味着“储备体系发生结构性断裂”，至少从这四周数据看更像“传统框架+风险溢价抬升”。
- 若后续`DFII10`重新上行而金价不回落，才更接近“结构性脱钩”证据。

### 证据
- GSAM周报（032026/032726/041026/041726）
  - https://am.gs.com/cms-assets/gsam-app/documents/insights/en/2026/market_monitor_032026.pdf
  - https://am.gs.com/cms-assets/gsam-app/documents/insights/en/2026/market_monitor_032726.pdf
  - https://am.gs.com/cms-assets/gsam-app/documents/insights/en/2026/market_monitor_041026.pdf

- 4个点的统计稳定性有限，需后续扩展样本验证。

## 问题1：市场是否已经进入“高位钝化”？

### 方法
以 `2026-04-20` 至 `2026-04-24` 为样本窗口，读取 `bond_cb_index_jsl()` 的日度序列，统计等权指数、成交额、全市场平均转股溢价率、温度与价位分层数量，并与前一交易周 `2026-04-13` 至 `2026-04-17` 做环比比较。价位层周收益采用日收益连乘计算。

### 发现
| 指标 | 数值 |
| --- | --- |
| 集思录等权指数周涨跌 | `-0.70%` |
| 前一周集思录等权指数周涨跌 | `+3.23%` |
| JSL 对照指数 `idx_price` 周涨跌 | `+0.25%` |
| 本周日均成交额 | `793.47` 亿元 |
| `>=800` 亿元交易日天数 | `3` 天 |
| 周末全市场平均转股溢价率 | `51.17%` |
| 平均转股溢价率在近250个交易日样本中的分位 | `92.4%` |
| 周末 `>=130` 元转债占比 | `73.35%` |
| 周末 `<110` 元转债占比 | `3.29%` |
| 价位层周收益：`90-100` | `-4.97%` |
| 价位层周收益：`100-110` | `+0.20%` |
| 价位层周收益：`>=130` | `-0.27%` |

本周的核心特征不是“放量上攻”，而是“高估值下的横盘钝化”。等权指数周跌 `0.70%`，明显弱于前周的 `+3.23%`，但成交额仍维持在接近 `800` 亿元的高位，说明资金并未明显撤退，而是在高位券之间做换手。周末全市场平均转股溢价率抬升到 `51.17%`，已经站到近 `250` 个交易日样本的 `92.4%` 分位，意味着估值安全垫很薄。与此同时，`>=130` 元转债占比达到 `73.35%`，而 `<110` 元转债只剩 `11` 只，占比 `3.29%`，市场几乎没有低价缓冲层，这就是“高位钝化”的实质。

## 问题2：“退场潮”到底有多大，强赎风险如何定价？

### 方法
以 `2026-04-25` 收盘后的 `bond_cb_redeem_jsl()` 快照为基准，筛选已给出“最后交易日”的已公告强赎转债，分别统计 `2026-04-25` 至 `2026-05-15` 和 `2026-04-25` 至 `2026-05-25` 两个窗口的剩余规模；再与东方财富比价表做代码匹配，补上转股价值与转股溢价率，用来判断市场究竟在交易“赎回风险”还是“强制转股预期”。

### 发现
| 指标 | 数值 |
| --- | --- |
| `2026-04-25` 至 `2026-05-15` 已公告强赎最后交易剩余规模 | `18.95` 亿元 |
| `2026-04-25` 至 `2026-05-25` 已公告强赎最后交易剩余规模 | `37.15` 亿元 |
| 近30日已公告强赎券数 | `9` 只 |
| 最大单日退场压力 | `2026-05-22`，`18.204` 亿元 |
| 近30日样本中转股溢价率处于 `±1.5%` 内的券数 | `7/9` |
| 近30日样本中仍保留显著正溢价的券 | `128108 蓝帆转债`，`8.57%` |
| 近30日样本中深度负溢价的券 | `113618 美诺转债`，`-2.53%` |

已公告强赎的第一波退场规模并不算失控，但排布非常集中，尤其是 `2026-05-22` 当天的 `18.204` 亿元最需要警惕。更关键的是，绝大多数临近最后交易日的转债，价格已经几乎贴着转股价值交易，说明市场并不再给“强赎前溢价”额外溢酬，而是默认持有人会在截止日前完成转股。也就是说，当前强赎风险的核心已经不是定价错误，而是执行错误，包括未及时转股、账户权限不匹配、或者在最后交易日前仍把它当成普通二级市场筹码持有。例外是 `蓝帆转债` 仍有 `8.57%` 的转股溢价率，若后续正股弹性不跟，压缩风险反而更大。

### 证据
- [Jisilu 强赎列表](https://www.jisilu.cn/data/cbnew/#redeem)，字段：`最后交易日`、`剩余规模`、`强赎状态`
- [东方财富可转债比价表](https://quote.eastmoney.com/center/fullscreenlist.html#convertible_comparison)，字段：`转股价值`、`转股溢价率`
- [永创智能：关于提前赎回“永02转债”的公告（2026-04-14）](https://wap.eastmoney.com/a/202604143704709136.html)
- [锦鸡股份：关于提前赎回锦鸡转债的第七次提示性公告（公告日期 `2026-04-23`）](https://vip.stock.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?CompanyCode=80095069&gather=1&id=12155166)

### 口径与局限
- 本题使用的是“最后交易日对应剩余规模”，不是“赎回日实际未转股规模”；最终被公司按赎回价强制赎回的实际规模，仍会随着转股进度继续变化。
- `128108 蓝帆转债` 在 Jisilu 强赎表中的 `强赎价` 字段为空，推测与其临近到期/处理方式有关，因此仅引用其转股溢价率，不对“现价相对强赎价价差”做机械比较。

## 问题3：“补给潮”是否足以承接“退场潮”？

### 方法
使用 `bond_cov_issue_cninfo(start_date="20260401", end_date="20260430")` 统计 `2026-04` 已公告发行的全部可转债面值规模，再单独抽取研究周 `2026-04-20` 至 `2026-04-24` 的新增发行规模；随后与问题2中的未来15日、30日退场规模做覆盖率比较。另用东方财富比价表核对 `上市日期`，验证研究周是否有新券实际上市。

### 发现
| 指标 | 数值 |
| --- | --- |
| `2026-04` 已发行可转债数量 | `4` 只 |
| `2026-04` 已发行可转债规模 | `36.99` 亿元 |
| 研究周新增发行数量 | `1` 只 |
| 研究周新增发行规模 | `9.80` 亿元 |
| 研究周新增上市数量 | `0` 只 |
| 研究周补给对未来15日退场覆盖率 | `51.72%` |
| `2026-04` 全月补给对未来30日退场覆盖率 | `99.56%` |
| `2026-04-16` 至 `2026-05-22` 累计净供给 | `-0.163` 亿元 |

补给确实来了，但节奏明显慢于退场。就研究周本身看，只有 `123269 金杨转债` 一只新券申购，规模 `9.80` 亿元，只能覆盖未来15日退场规模的一半左右，因此短期存量收缩依然偏紧。把视角拉到整个 `2026-04`，新发 `4` 只、合计 `36.99` 亿元，恰好与未来30日已公告退场的 `37.15` 亿元几乎相当，说明“补给潮”不是即时缓冲，而是跨期对冲。另一个重要细节是，本月供给几乎全部集中在半导体、PCB、锂电结构件等硬科技/先进制造方向，而研究周内没有新券上市，这意味着补给的行业风格是偏成长的，但二级市场可交易供给并未同步落地。

### 证据
- [巨潮资讯可转债发行专题统计接口](http://webapi.cninfo.com.cn/#/thematicStatistics)，字段：`发行起始日`、`实际发行总量`
- [金杨转债发行提示性公告（`2026-04-20`）](https://static.cninfo.com.cn/finalpage/2026-04-20/1225114973.PDF)
- [东方财富可转债比价表](https://quote.eastmoney.com/center/fullscreenlist.html#convertible_comparison)，字段：`上市日期`

### 口径与局限
- 发行规模按面值口径统计，不等于上市首日可交易流通规模，也不等于中签后会立即进入二级市场的筹码量。
- 研究周内“零上市”并不代表一级供给消失，而是说明申购与上市之间存在时间错位；因此补给对二级市场估值的缓释通常滞后于发行公告。

## 问题4：下周最需要跟踪哪些券与哪类风险？

### 方法
解析 `bond_cb_redeem_jsl()` 中的 `强赎天计数` 字段，将 `x/y | 30` 拆成“已满足天数/触发所需天数”，筛选尚未给出最后交易日、但距离触发 `<=3` 天的观察池；再与东方财富比价表匹配转股价值和转股溢价率，区分“快触发但几乎无溢价”的执行型风险与“快触发且仍有双位数溢价”的压缩型风险。

### 发现
| 指标 | 数值 |
| --- | --- |
| 距强赎触发 `<=1` 天且尚未进入最后交易日 | `7` 只，`74.05` 亿元 |
| 距强赎触发 `<=3` 天且尚未进入最后交易日 | `12` 只，`132.17` 亿元 |
| 余额最大的 `<=3` 天观察券 | `127067 恒逸转2`，`29.994` 亿元 |
| 第二大 `<=3` 天观察券 | `113051 节能转债`，`26.520` 亿元 |
| 距触发 `1` 天且转股溢价率仍 `>15%` 的券 | `127067 恒逸转2`、`127103 东南转债`、`113640 苏利转债`、`113678 中贝转债` |
| 距触发 `2-3` 天且转股溢价率仍 `>19%` 的券 | `123255 鼎龙转债`、`123149 通裕转债`、`118056 路维转债` |

下周真正需要提防的，不是已经进入最后交易倒计时的老面孔，而是第二波尚未公告、但离触发只差临门一脚的大余额品种。`127067 恒逸转2`、`127103 东南转债`、`113051 节能转债` 这类券的共同点是余额大、距离触发近，而且并非全部贴着转股价值交易，一旦公告落地，溢价压缩会比已公告强赎券更急。换言之，第一波“退场潮”主要是执行风险，第二波更像估值风险。对交易层面，下周最值得跟踪的是“距离触发天数 + 剩余规模 + 当前溢价率”这三个指标的组合，而不是只看正股是否继续创新高。

### 证据
- [Jisilu 强赎列表](https://www.jisilu.cn/data/cbnew/#redeem)，字段：`强赎天计数`、`剩余规模`
- [东方财富可转债比价表](https://quote.eastmoney.com/center/fullscreenlist.html#convertible_comparison)，字段：`转股价值`、`转股溢价率`

### 口径与局限
- `强赎天计数` 采用 Jisilu 当日快照，不是基于正股历史收盘价逐笔重算；若交易所或数据商口径有修正，个别券的“距触发天数”可能发生变化。
- 触发条款并不必然立即公告强赎，发行人仍可能选择暂不实施；因此该观察池反映的是“公告风险暴露”，不是“确定退场名单”。

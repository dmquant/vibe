## 二、核心问题逐项回答

### 问题1：AAA级城投债利差处于底部，利差反弹触发点在哪里？

**方法**：以 2026/04/13-04/19 为观察周，使用财信证券/东方证券公开周报中的城投利差周变化、信用债一级发行成本、利率债基准收益率作为比较基础。触发点按“资金、供给、估值、信用事件”四类拆分。

| 指标 | 2026/04/17或周度值 | 周变化 | 判断 |
| :--- | :--- | :--- | :--- |
| 城投债AAA信用利差 | 未披露绝对bp | +3.28bp | 已出现被动走阔 |
| 城投债AA+信用利差 | 未披露绝对bp | +3.10bp | 与AAA同步走阔 |
| 城投债AA信用利差 | 未披露绝对bp | +2.30bp | 低等级未明显更差 |
| 10Y国债收益率 | 1.76% | -5.1bp | 无风险利率下行快于信用收益率 |
| AAA发行票息 | 1.87% | -7bp | 一级高等级融资成本继续低位 |
| AA+发行票息 | 2.11% | +14bp | 中等级开始出现成本分化 |

**发现**：数字上，城投债利差本周已经反弹 2-3bp，但这更像“无风险利率下行带来的被动走阔”，不是信用风险重定价。真正触发更大幅反弹的第一信号是资金价格上行或理财/基金净申购转弱，因为当前利差压缩主要由资管“钱多”驱动。第二信号是地方债、特别国债和信用债供给同时放量，导致配置盘边际承接不足。第三信号是高风险区域公开债或非标风险外溢，但本周没有公开证据显示城投公开债出现系统性折价成交。

**证据**：财信证券周报转载显示城投AAA/AA+/AA信用利差分别上行 3.28/3.10/2.30bp；东方证券周报称各省城投信用利差平均走阔约 3bp；华创证券周报指出 4月下旬资金或维持低位，1Y国股行存单或在 1.45%-1.50%低位波动。

**限制**：公开报告未披露分省份、分期限的城投G-Spread绝对值和历史分位数；Wind/Choice明细无法在本环境直接复核。因此本结论基于公开周报的变化值，不等同于全量成交逐笔测算。

### 问题2：3月销售是否支撑“白名单”项目融资实际下达？现金流是否已传导至资产负债表？

**方法**：用国家统计局 2026/01-03 累计销售、到位资金及央行住户中长期贷款数据验证需求端和融资端；用 2026/04/13-04/19 30城高频成交观察边际改善。

| 指标 | 2026/01-03或周度值 | 同比/环比 | 结论 |
| :--- | :--- | :--- | :--- |
| 新建商品房销售面积 | 19525 万平方米 | 同比 -10.4%，降幅较1-2月收窄 3.1pct | 销售边际改善 |
| 新建商品房销售额 | 17262 亿元 | 同比 -16.7%，降幅较1-2月收窄 3.5pct | 价格和结构压力仍大 |
| 房企到位资金 | 20524 亿元 | 同比 -17.3% | 融资端未同步修复 |
| 国内贷款 | 3419 亿元 | 同比 -23.7% | 银行资金传导仍弱 |
| 定金及预收款 | 5858 亿元 | 同比 -20.1% | 销售回款仍承压 |
| 个人按揭贷款 | 2204 亿元 | 同比 -34.6% | 居民购房加杠杆意愿弱 |
| 30城新房成交面积 | 155.7 万平方米 | 环比 +25.2%，同比 +4.0% | 高频成交有反弹 |
| 一季度住户中长期贷款 | 4607 亿元 | 未给同比；3月居民贷款同比少增 | 按揭端未确认趋势反转 |

**发现**：3月销售数据支持“边际修复”，但不支持“资产负债表已修复”。最关键的反证是房企到位资金同比 -17.3%，国内贷款同比 -23.7%，个人按揭贷款同比 -34.6%，均弱于销售面积降幅。高频30城成交在4月第三周改善，说明核心城市短期成交具备弹性，但累计成交仍低于去年。白名单融资更可能缓解项目交付和短期流动性，而非立刻改善房企集团层面的现金流覆盖能力。

**证据**：国家统计局 2026/04/16《2026年1-3月份全国房地产市场基本情况》；国盛证券 2026W15 房地产周报；央行 2026Q1 金融统计数据。

**限制**：白名单项目贷款投放明细通常按项目和银行内部口径统计，公开数据无法区分“批复额度、提款额度、到项目公司账面资金”。本报告只能用国内贷款、按揭、预收款等宏观科目作代理变量。

### 问题3：特殊再融资债置换后，天津、昆明等高风险区域非标违约频率是否实质性下降？

**方法**：原计划需要DMI非标违约/展期事件库。本阶段公开源无法取得DMI逐周事件，因此采用三类代理：置换债发行进度、城投公开债净融资/利差、公开新闻中的非标风险披露。

| 代理指标 | 数值 | 解释 |
| :--- | :--- | :--- |
| 置换隐债政府债券发行 | 截至 2026/04/13 约 1.3 万亿元 | 占全年 2.8 万亿元约 46%，资金落地较快 |
| 第16周特殊再融资债 | 约 930 亿元 | 国信/Reportify周度摘要口径，显示化债仍在延续 |
| 城投债周净融资 | 81 亿元或109亿元 | 财信/国信口径不同，但均为小幅正净融资 |
| 各省城投利差变化 | 平均 +3bp | 未见公开债系统性风险外溢 |
| 天津/昆明非标违约周频 | 公开源不可复核 | 无法给出实质下降结论 |

**发现**：不能确认“非标违约频率已实质性下降”。可以确认的是化债资金大规模投放，公开债市场估值仍保持稳定，且城投公开债没有出现全国性折价抛售。非标违约和展期往往披露滞后、样本不完整，并且天津、昆明这类重点区域的风险化解通常先体现为公开债刚兑预期稳定，后体现为非标展期减少。现阶段更稳妥的表述是：化债对公开债估值修复有效，对非标频率的实证检验尚缺数据。

**证据**：大河财立方 2026/04/16 披露置换隐债债券约 1.3 万亿元；东方证券周报披露城投省份利差平均走阔约 3bp；财信证券周报披露城投债信用利差按评级小幅上行。

**限制**：DMI/企业预警通/DM非标事件库均非公开全量数据，本报告无法复核天津、昆明逐周非标违约笔数、金额、展期条款和兑付状态。

### 问题4：上周评级上调与下调比例如何？是否存在行业或区域修正趋势？

**方法**：检索 2026/04/13-04/19 评级调整周报、信用债周报负面信息监测和公开评级公告摘要。将“公开文本可核验的主体级别/展望调整”和“隐含评级调整”分开。

| 口径 | 上调家数 | 下调家数 | 上调/下调比例 | 结论 |
| :--- | ---: | ---: | :--- | :--- |
| 公开文本可核验主体评级/展望 | 0 | 0 | 不适用 | 未检索到可逐项复核的本周主体评级调整清单 |
| 东方证券负面监测图表 | 未披露文本值 | 未披露文本值 | 不适用 | 网页摘要列出图表名称，但未抽取表格数据 |
| 中债隐含评级/YY信评 | 本周缺全量公开表 | 本周缺全量公开表 | 不适用 | 不能替代正式评级迁移比例 |

**发现**：本周无法用公开文本严谨计算评级上调/下调比例。可观察的信用修正更多体现在二级估值而不是正式评级：房企主体中融侨、万科、绿地、鑫苑利差走阔居前，万科债存在折价超过 10%成交；城投主体估值变化分布较散乱，没有显著区域集中。正式评级通常滞后于价格和成交异动，因此本周更应跟踪“隐含评级和折价成交”而非仅看主体评级公告。

**证据**：东方证券周报网页目录披露“主体评级或展望下调的企业”“债项评级调低的债券”等负面信息图表，但公开文本未给出数值；国金证券债券涨跌周报披露净价上涨靠前个券中以万科相关债券为主。

**限制**：评级迁移需要中诚信、联合资信、东方金诚、上海新世纪等评级机构公告全量抓取，或Wind评级调整表。公开网页摘要不足以形成完整迁移矩阵。

### 问题5：极窄信用利差下，久期策略与下沉策略哪个更具性价比？

**方法**：比较信用债利差变化、发行票息、基准利率和房企/城投风险事件。策略评价维度为票息补偿、估值波动、流动性和尾部风险。

| 策略 | 收益来源 | 当前数字依据 | 性价比判断 |
| :--- | :--- | :--- | :--- |
| 高等级适度拉久期 | 票息+利率下行资本利得 | AAA票息 1.87%；10Y国债 1.76%，周下行 5.1bp | 优于盲目下沉，但需控制久期拥挤 |
| 城投短久期下沉 | 票息挖掘 | AA城投利差仅 +2.30bp周走阔，未见明显补偿提升 | 仅限强省强市、短久期 |
| 民营地产债下沉 | 价格修复 | 万科债折价超过10%成交，部分债券净价上涨靠前 | 交易属性强，不适合作为组合底仓 |
| 中低等级拉久期 | 票息+期限利差 | AA+票息 2.11%，环比 +14bp | 成本分化已现，需谨慎 |

**发现**：本周久期策略相对更具性价比，但应以高等级、中短久期到中久期为主，不宜激进拉长到流动性弱的低等级品种。下沉策略的风险补偿不足，尤其是正式评级仍滞后、地产债价格波动大、部分房企现金流未修复。城投下沉仍可做，但应把区域财政、平台层级、债务置换受益程度和公开债到期压力作为准入条件。地产债更适合事件驱动交易，而不是信用组合的稳定票息资产。

**证据**：华创证券周报称资管“钱多”仍支撑利差压缩，1Y国股行存单或在 1.45%-1.50%低位波动；东方证券周报显示AAA发行票息下行而AA+上行；国金证券周报显示万科相关债券价格波动居前。

**限制**：未取得逐券收益率、久期、成交量和估值偏离度明细，无法计算夏普比率或持有期收益分布。

## 五、投资建议

| 组合维度 | 建议 | 执行口径 |
| :--- | :--- | :--- |
| 久期 | 高等级信用可维持中性偏长，但避免过度拥挤 | 以AAA央国企、强省城投、2-4年为核心；利率急下后不追长端低流动性券 |
| 城投 | 不做无差别下沉 | 优先江苏、浙江、福建、山东强地市；重点区域仅买短久期、强担保、公开债偿付记录清晰主体 |
| 地产 | 交易盘和配置盘分离 | 万科等高波动主体可作为事件驱动观察，不作为低波动票息底仓 |
| 一级 | 关注AA+发行成本上行 | 若AA+发行票息继续上行且取消发行增加，降低中等级新券参与比例 |
| 风险控制 | 盯住资金和供给 | DR007、1Y NCD、特别国债/地方债供给、理财申赎是利差反弹先行指标 |

## 六、后续跟踪项

| 跟踪项 | 触发阈值 | 信用含义 |
| :--- | :--- | :--- |
| 4月底政治局会议 | 地产去库存、地方债务、财政增量政策表述变化 | 决定地产债和城投债风险偏好能否继续修复 |
| DR007与1Y NCD | 1Y国股行NCD持续高于 1.50% | 资管配置行情边际降温，信用利差或反弹 |

| 房企到位资金 | 4月累计同比是否明显收窄 | 验证白名单融资是否进入资产负债表 |

## 三、逐题深度解读

### 问题 1 · 市场是否已经进入“高位钝化”？

**方法。** 以 `2026-04-20` 至 `2026-04-24` 为样本窗口，读取 `bond_cb_index_jsl()` 日度序列；统计等权指数、成交额、平均转股溢价率、温度与价位分层数量，并与前周 `2026-04-13` ～ `2026-04-17` 做环比；价位层周收益采用日收益连乘。

**核心数据。**

| 指标 | 数值 |
| --- | ---: |
| 集思录等权指数周涨跌 | `-0.70%` |
| 前一周集思录等权指数周涨跌 | `+3.23%` |
| JSL 对照指数 `idx_price` 周涨跌 | `+0.25%` |
| 本周日均成交额 | `793.47 亿元` |
| ≥800 亿元交易日天数 | `3 天` |
| 周末平均转股溢价率 | `51.17%` |
| 近 250 日样本中分位 | `92.4%` |
| 周末 ≥130 元转债占比 | `73.35%` |
| 周末 <110 元转债占比 | `3.29%` |
| 价位层周收益：90-100 | `-4.97%` |
| 价位层周收益：100-110 | `+0.20%` |
| 价位层周收益：≥130 | `-0.27%` |

**解读。** 本周的核心特征不是“放量上攻”，而是“高估值下的横盘钝化”。等权指数周跌 `0.70%`，明显弱于前周的 `+3.23%`，但成交额仍维持在接近 `800 亿元` 的高位，说明资金并未明显撤退，而是在高位券之间做换手。周末平均转股溢价率抬升到 `51.17%`，已经站到近 250 个交易日样本的 `92.4%` 分位，估值安全垫很薄。同时 `≥130 元` 转债占比 `73.35%`，`<110 元` 仅剩 11 只（占比 `3.29%`），市场几乎没有低价缓冲层——这就是“高位钝化”的实质。

**口径与局限。** 计划片段中的“2740 点附近”更接近 Jisilu 等权指数 `price` 字段，而非中证 `000832` 现货点位；本报告以 `price` 为主分析序列、`idx_price` 为对照。集思录序列当前可直接取得约 250 个交易日样本，“分位”口径对应该窗口而非全样本。

### 问题 2 · “退场潮”到底有多大，强赎风险如何定价？

**方法。** 以 `2026-04-25` 收盘后 `bond_cb_redeem_jsl()` 快照为基准，筛选已给出最后交易日的强赎转债，分别统计 15 日 / 30 日窗口剩余规模；与东方财富比价表代码匹配，补上转股价值与转股溢价率。

**核心数据。**

| 指标 | 数值 |
| --- | ---: |
| 2026-04-25 ～ 2026-05-15 已公告强赎剩余规模 | `18.95 亿元` |
| 2026-04-25 ～ 2026-05-25 已公告强赎剩余规模 | `37.15 亿元` |
| 近 30 日已公告强赎券数 | `9 只` |
| 最大单日退场压力 | `2026-05-22 / 18.204 亿元` |
| 近 30 日样本中转股溢价率 ±1.5% 内 | `7 / 9` |
| 仍保留显著正溢价 | `128108 蓝帆转债 +8.57%` |
| 深度负溢价 | `113618 美诺转债 -2.53%` |

**解读。** 第一波退场规模并不算失控，但排布非常集中——`2026-05-22` 单日 `18.204 亿元` 最需警惕。更关键的是，绝大多数临近最后交易日的转债已经几乎贴着转股价值交易，市场不再给“强赎前溢价”额外溢酬，默认持有人会在截止日前完成转股。当前强赎风险的核心已经不是定价错误，而是**执行错误**。例外是 `蓝帆转债` 仍有 `+8.57%` 溢价，若正股弹性不跟，压缩风险更大。

**口径与局限。** 本题使用的是“最后交易日对应剩余规模”，不是“赎回日实际未转股规模”；最终强制赎回的实际规模仍会随转股进度变化。`128108 蓝帆转债` 的 `强赎价` 字段为空，仅引用其转股溢价率。

**口径与局限。** 发行规模按面值口径，不等于上市首日可交易流通规模；研究周“零上市”不代表一级供给消失，而是申购与上市的时间错位；补给对二级估值的缓释通常滞后于发行公告。

### 问题 4 · 下周最需要跟踪哪些券与哪类风险？

**方法。** 解析 `bond_cb_redeem_jsl()` 的 `强赎天计数` 字段，将 `x/y | 30` 拆成“已满足天数 / 触发所需天数”，筛选尚未给出最后交易日、但距触发 ≤3 天的观察池；与东方财富比价表匹配转股价值与转股溢价率。

**核心数据。**

| 指标 | 数值 |
| --- | ---: |
| 距强赎触发 ≤1 天且未进入最后交易日 | `7 只 / 74.05 亿元` |
| 距强赎触发 ≤3 天且未进入最后交易日 | `12 只 / 132.17 亿元` |
| 余额最大的 ≤3 天观察券 | `127067 恒逸转2 / 29.994 亿元` |
| 第二大 ≤3 天观察券 | `113051 节能转债 / 26.520 亿元` |
| 距触发 1 天且转股溢价率 >15% | `127067 恒逸转2`、`127103 东南转债`、`113640 苏利转债`、`113678 中贝转债` |
| 距触发 2-3 天且转股溢价率 >19% | `123255 鼎龙转债`、`123149 通裕转债`、`118056 路维转债` |

**解读。** 下周真正需要提防的，不是已经进入最后交易倒计时的老面孔，而是**第二波尚未公告、但离触发只差临门一脚**的大余额品种。`127067 恒逸转2`、`127103 东南转债`、`113051 节能转债` 共同点是余额大、距离触发近，且并非全部贴着转股价值交易；一旦公告落地，溢价压缩会比已公告强赎券更急。第一波“退场潮”主要是**执行风险**，第二波更像**估值风险**。交易层面，下周最值得跟踪“距离触发天数 + 剩余规模 + 当前溢价率”的三因子组合。

**口径与局限。** `强赎天计数` 采用 Jisilu 当日快照，非基于正股历史收盘价逐笔重算；触发条款并不必然立即公告强赎，发行人仍可能选择暂不实施；该观察池反映“公告风险暴露”而非“确定退场名单”。

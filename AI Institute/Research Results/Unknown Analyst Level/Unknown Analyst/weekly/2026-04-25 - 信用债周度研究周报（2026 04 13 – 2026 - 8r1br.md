## 执行摘要

本周（2026/04/13-04/19）信用债一级仍热（发行 3748 亿、净融资 +717 亿），但二级城投利差出现 AAA +3.28bp / AA+ +3.10bp / AA +2.30bp 的被动走阔——幅度小、方向一致，根本原因是 10Y 国债下行 -5.1bp 至 1.76%，无风险利率跑赢信用收益率。地产端 1-3 月销售面积 -10.4%、销售额 -16.7%、到位资金 -17.3%——边际收窄但资产负债表未修复；个人按揭 -34.6% 是核心反证。化债置换债已发行约 1.3 万亿（占全年 2.8 万亿 46%），公开债估值仍稳，但非标违约频率公开数据不可复核。**策略主线：高等级、中短到中久期为底仓；城投只在强省强市做短下沉；地产债事件驱动而非配置底仓；盯 DR007、1Y NCD、AA+ 一级票息作为利差反弹先行指标。**

---

## 一、本周关键结论

1. **城投利差被动走阔，不是信用风险重定价。** AAA/AA+/AA 周变动 +3.28 / +3.10 / +2.30 bp，但 10Y 国债同期下行 -5.1 bp，AAA 一级票息 1.87%（环比 -7bp）继续探低——典型“无风险跑得快、信用跟得慢”的被动走阔，非信用面恶化。
2. **中等级一级开始分化，AA+ 票息逆势上行 +14 bp 至 2.11%。** 取消/推迟发行 3 只 / 40 亿规模可控，但 AA+ 成本上行是“资管钱多撑高等级、撑不动中低等级”的早期信号；若连续两周延续，配置盘需要降低中等级新券参与。
3. **地产销售边际收窄，但资金端未修复，白名单尚未传导至资产负债表。** 1-3 月销售面积 -10.4%（降幅较 1-2 月收窄 3.1pct），但到位资金 -17.3%、国内贷款 -23.7%、个人按揭 -34.6% 全部弱于销售面积——融资端没有同步改善。
4. **化债资金落地约 1.3 万亿（占全年 46%），公开债稳但非标不可复核。** 天津、昆明等重点区域公开债未见系统性折价抛售，但 DMI/企业预警通的非标逐周违约频率不在公开源中——“化债对公开债估值修复有效、对非标频率的实证检验尚缺数据”，这是当前最严谨的表述。
5. **配置主线：高等级久期 > 强省强市短下沉 > 民营地产事件驱动。** AAA 央国企/强省城投 2-4 年久期为核心；万科类高波动主体（折价超 10% 成交、净价上涨同时入榜）只作为事件交易，不作组合底仓。

## 三、逐题深度解读

### Q1 — AAA 城投利差已在底部，反弹触发点在哪里？

**结论：** 本周 2-3 bp 的走阔属“无风险下行更快”引发的被动走阔，不是信用风险重定价。后续真正触发更大反弹的三类信号——

| 触发点 | 当前观察 | 研判 |
| :--- | :--- | :--- |
| 资金面 / 理财申赎转弱 | 1Y 国股行 NCD 在 1.45%-1.50% 低位（华创口径） | 仍宽，但是先行变量，需每日盯 |
| 地方债 / 特别国债 / 信用债同步放量 | 本周供给温和；下周城投预计净融资 -990 亿 | 供给冲击窗口正在打开 |
| 高风险区域公开债 / 非标外溢 | 本周公开债无系统性折价；非标公开数据缺失 | 暂无外溢证据 |

**口径限制：** 未取得分省份、分期限的城投 G-Spread 绝对值与历史分位数，结论基于公开周报变化值，非全量逐笔成交。

### Q2 — 3 月销售是否支撑“白名单”项目融资落地？现金流是否已传导至资产负债表？

**结论：** 支持“边际修复”，不支持“资产负债表已修复”。最关键反证是融资端全部弱于销售面积。

| 指标 | 数值 | 同比 | 结论 |
| :--- | ---: | ---: | :--- |
| 新建商品房销售面积 | 19,525 万㎡ | -10.4% | 降幅较 1-2 月收窄 3.1pct |
| 新建商品房销售额 | 17,262 亿元 | -16.7% | 价格 / 结构压力仍大 |
| 房企到位资金 | 20,524 亿元 | -17.3% | 融资端未同步修复 |
| 国内贷款 | 3,419 亿元 | -23.7% | 银行资金传导仍弱 |
| 定金及预收款 | 5,858 亿元 | -20.1% | 销售回款承压 |
| 个人按揭贷款 | 2,204 亿元 | -34.6% | 居民加杠杆意愿弱 |
| 30 城新房成交（本周） | 155.7 万㎡ | 环比 +25.2% / 同比 +4.0% | 核心城市短期弹性 |

**口径限制：** 白名单贷款投放明细按银行内部口径统计，公开数据无法区分批复 / 提款 / 到账，本报告以宏观科目作代理。

### Q3 — 特殊再融资债置换后，天津、昆明等高风险区域非标违约频率是否实质下降？

**结论：** 不能确认实质下降。可确认的是化债资金大规模投放、公开债估值稳定、城投公开债无全国性折价抛售。

- 置换隐债政府债券累计约 1.3 万亿元（占全年 2.8 万亿元 46%），第 16 周特殊再融资债约 930 亿元；
- 城投债周净融资 81 亿（财信口径）/ 109 亿（国信口径），均小幅正；
- 各省城投利差平均走阔约 +3 bp，未见公开债系统性外溢。

**口径限制：** DMI / 企业预警通 / DM 非标事件库非公开全量；天津、昆明逐周非标违约笔数、金额、展期条款、兑付状态不可复核。当前最严谨表述是“化债对公开债估值修复有效，对非标频率检验尚缺数据”。

### Q4 — 本周评级上调与下调比例如何？是否存在行业 / 区域修正趋势？

**结论：** 本周公开文本无法严谨计算评级上调 / 下调比例。可观察的修正更多体现在二级估值——

- 房企：融侨、万科、绿地、鑫苑利差走阔居前；万科债折价超 10% 成交；
- 城投：估值变化分布散乱，无显著区域集中；
- 正式评级通常滞后于价格与成交异动，本周更应跟踪“隐含评级 + 折价成交”而非主体评级公告。

**口径限制：** 评级迁移需中诚信、联合资信、东方金诚、上海新世纪等机构公告全量抓取（或 Wind 评级调整表），公开网页摘要不足以形成完整迁移矩阵。

### Q5 — 极窄信用利差下，久期策略与下沉策略哪个更具性价比？

**结论：** 本周久期策略相对更具性价比，但应以高等级、中短到中久期为主。

| 策略 | 当前数字依据 | 性价比判断 |
| :--- | :--- | :--- |
| 高等级适度拉久期 | AAA 票息 1.87%；10Y 国债 1.76%（-5.1 bp） | 优于盲目下沉，需控制久期拥挤 |
| 城投短久期下沉 | AA 城投利差仅 +2.30 bp 走阔，无明显补偿提升 | 仅限强省强市、短久期 |
| 民营地产债下沉 | 万科债折价超 10% 成交；同时部分债券净价上涨靠前 | 事件驱动，不适合组合底仓 |
| 中低等级拉久期 | AA+ 票息 2.11%（环比 +14 bp） | 成本分化已现，谨慎参与 |

**口径限制：** 未取得逐券收益率、久期、成交量、估值偏离度明细，无法计算夏普比率或持有期收益分布。

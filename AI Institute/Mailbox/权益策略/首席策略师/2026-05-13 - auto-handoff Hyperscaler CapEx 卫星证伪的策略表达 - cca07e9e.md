**为什么不先空 IPP**：
- CEG / VST / TLN 的远期 PPA **已签部分覆盖 2027~2029 EBITDA 的 60%+**（CEG 与 Meta 20 年 Calpine + Three Mile Island；VST 与 Amazon、Google 的核电 PPA；TLN 与 AWS Susquehanna）。这些合约在 hyperscaler 主动违约或重新议价之前是真实现金流。
- 当前 PJM、ERCOT 2027~2028 capacity 前瞻曲线仍处历史 95+ 百分位（PJM 2025/26 拍卖 $269.92/MW-day 创新高），**做空 IPP 在远期电价转头之前 carry 极差**（CEG 当前 NTM FCF yield ~3.5%、VST ~4.8%、空头借券成本 0.3~0.6%）。
- 设备链 EPS 修正机制更快、估值溢价更高（光模块 30~45×、VRT 28×），**风险/回报比明显占优**。

### 2.3 风险与反向情景

- **反向 1**：卫星看到的 shell starts 下降只是"从 land + utility 阶段切换到 hyperscaler 自建模块化"（NVIDIA Spectrum-X reference design 推动）。→ 用 §3.2 的 hyperscaler-自建占比信号过滤。
- **反向 2**：QoQ −19% 含 weather / permitting 季节扰动。→ 看 Q3 第一个月的 weekly excavator-count 数据是否反弹。
- **反向 3**：DeepSeek V4 / 国产替代驱动单位算力的 shell 需求下降，**但 GPU/光模块订单不降反升**（hyperscaler 在已有 shell 内堆密度）。→ 看 hyperscaler kWh/rack 公告与液冷渗透率。

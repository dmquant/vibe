Key Design Principles

Session-per-Task

Each task creates one session. All workflow steps share the same workspace and context. Files accumulate across steps.

Generic Workflow Templates

Workflows registered once. The ticker/topic injected at runtime via input_prompt.

File Chain Architecture

Each step saves to NN_name.md. Next steps read prior files. Final step synthesizes all.

Multi-Agent Specialization

Gemini for research & web search. Claude for risk assessment & independent review. Right tool for each job.

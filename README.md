# ResearchOS · Vibe Documentation

This directory (`/vibe/`) contains the complete conceptual, technical, and operational documentation for **ResearchOS**—an ecosystem comprising the **Agent Route** (control plane) and **AI Institute** (research platform).

The documentation here is not just a technical manual; it is designed to capture the "vibe," the architectural philosophy, and the step-by-step build story behind this multi-agent orchestration system.
